## 摘要

可持续金融监管——即将环境、社会和治理（ESG）因素整合进金融服务、投资产品、资本市场和审慎监管的法律框架——已经从一种小众的自愿性实践演变为各主要司法管辖区金融监管架构的核心支柱，其推动力在于人们认识到气候变化和环境退化对金融稳定构成系统性风险，且向低碳经济的转型需要前所未有的资本动员，估计每年4-6万亿美元。韩国已建立起以金融服务委员会（FSC, 금융위원회）2020年发布的《绿色债券指引》、环境部于2021年12月公布的《韩国分类目录》（한국형 녹색분류체계, K-Taxonomy，界定了六项环境目标下的69项绿色经济活动）以及《韩国可持续金融路线图》（2023年）为基础的全面可持续金融框架，确立了强制性ESG披露、绿色债券标准和ESG评级监管的分阶段计划；作为系统重要性机构投资者的国民年金公团（NPS, 국민연금공단）已将负责任投资原则纳入其约900万亿韩元的投资组合。美国缺乏全面的联邦可持续金融框架，但通过SEC对ESG标签投资产品的监督、劳工部对ERISA管理的退休计划中ESG考量的规则、财政部关于气候相关金融风险的指引以及州级倡议来处理相关要素；政治极化产生了反向运动，许多由共和党执政的州颁布了反ESG立法，限制州养老基金在投资决策中考虑ESG因素。德国作为欧盟最大的金融市场，实施欧盟可持续金融框架，包括《可持续金融披露条例》（SFDR, 条例2019/2088）、《欧盟分类目录条例》（2020/852）、《欧盟绿色债券标准》（条例2023/2631）以及拟议的《ESG评级条例》，并辅以BaFin（联邦金融监管局, Bundesanstalt für Finanzdienstleistungsaufsicht）关于处理可持续性风险的指导通知和德意志联邦银行将气候风险纳入金融稳定分析。欧盟通过其《可持续金融行动计划》建立了世界上最全面的可持续金融监管架构，构建了一个综合框架，涵盖作为分类系统的欧盟分类目录、用于透明度义务的SFDR、用于债务资本市场的欧盟绿色债券标准、要求在投资建议和保险分销中考虑可持续性偏好的MiFID II和IDD修正案、气候基准条例以及拟议的ESG评级提供者条例，旨在将资本流动重新引导至环境可持续活动并管理气候相关金融风险。

## 韩国

### 法律框架

韩国的可持续金融框架建立在监管指引、K-Taxonomy和将可持续性因素纳入金融监管的组合之上。该方法反映了政府到2050年实现碳中和的承诺，以及对金融体系转型对实现这一目标至关重要的认识。

### 韩国绿色分类目录（K-Taxonomy）

环境部于2021年12月公布的K-Taxonomy界定了对六项环境目标作出贡献的经济活动：

| 环境目标 | 示例活动 |
|------------------------|-------------------|
| 减缓气候变化 | 可再生能源发电、绿色氢能、节能建筑 |
| 适应气候变化 | 气候适应型基础设施、预警系统 |
| 可持续水资源管理 | 水处理、节水工业过程 |
| 循环经济转型 | 回收基础设施、生态设计、再制造 |
| 污染防治 | 空气质量改善、危险废物管理 |
| 生物多样性保护 | 生态系统恢复、可持续林业和农业 |

K-Taxonomy涵盖69项绿色活动，并包括对大幅减少排放但尚未达到绿色门槛的活动设立的过渡类别，例如配备CCS技术的天然气发电——这是对韩国能源转型挑战的务实让步，鉴于其严重依赖化石燃料。

### 绿色债券市场

自FSC于2020年发布《绿色债券指引》以来，韩国绿色债券市场显著增长：

| 年份 | 绿色债券发行量（万亿韩元） | 主要进展 |
|------|-----------------------------------|-----------------|
| 2019 | 4.8 | 指引前的市场增长 |
| 2020 | 7.2 | FSC发布绿色债券指引 |
| 2021 | 15.3 | K-Taxonomy公布；首次主权绿色债券发行（7000亿韩元） |
| 2022 | 18.6 | ESG债券框架扩展至社会和可持续债券 |
| 2023 | 22.1 | K-Taxonomy对齐要求加强 |

韩国产业银行（KDB, 한국산업은행）和韩国进出口银行（KEXIM, 한국수출입은행）是主要发行人。大韩民国于2021年9月发行了其首笔主权绿色债券（7亿欧元和5亿美元两个分支），与K-Taxonomy保持一致。

### ESG基金监管

FSC制定了ESG基金披露指引，要求基金管理人披露ESG整合方法、组合ESG指标和参与活动。FSS于2023年的专题审查发现，约40%自我标榜的ESG基金缺乏系统性的ESG整合，促使加强审查并提出对ESG基金命名的强制性最低标准。

### 机构投资者的尽责管理

仿效英国《尽责管理守则》并于2016年引入的《韩国尽责管理守则》（한국 스튜어드십 코드），为机构投资者就ESG事项的参与确立了原则。截至2024年，已有200多家机构投资者采纳该守则。韩国最大的机构投资者NPS于2018年采纳该守则，并逐步将ESG因素纳入投资决策，包括对气候相关股东决议行使表决权。

## 美国

### 法律框架

美国没有全面的联邦可持续金融立法。监管活动分散在多个机构之间，且该领域已变得政治极化，拜登政府下采纳的可持续金融条款面临回退压力，州级反ESG立法在某些司法管辖区造成了独特的敌对监管环境。

### SEC ESG基金监督

SEC通过现有监管权限处理ESG投资产品：

**名称规则修正案（2023年9月）：** SEC修订了《投资公司法》名称规则（35d-1规则），要求名称中包含ESG相关术语的基金至少将80%的资产投资于与名称暗示相符的投资。这一"80%测试"适用于"可持续"、"绿色"、"ESG"和"社会负责"等术语。

**投资顾问和基金的ESG披露（2022年提议）：** SEC提议要求注册投资顾问和投资公司提供详细的ESG策略披露，包括ESG因素在投资选择和组合构建中的作用、某些气候为重点基金的温室气体排放以及ESG事项的代理投票。截至2026年，最终规则尚未通过。

### 劳工部ERISA规则

劳工部于2022年11月最终确定了一项规则（"在选择计划投资和行使股东权利时的审慎和忠诚"），明确ERISA受托人在ESG因素对风险回报分析具有重要意义时可以在投资决策中考虑ESG因素，并且行使股东权利（包括ESG事项）符合受托责任。该规则取代了特朗普时代的2020年规则，后者曾不鼓励ESG整合。劳工部ESG规则面临法律挑战，共和党州检察长辩称其超出了劳工部的权限（Utah v. Walsh，后更名为Utah v. Su，得克萨斯州北区）。

### 反ESG反弹

出现了一场重大的反向运动：

| 州行动 | 示例 |
|-------------|---------|
| **反ESG投资法** | 得克萨斯州SB 13（2021）：禁止州实体与"抵制"化石燃料的公司签订合同；佛罗里达州HB 3（2023）：禁止在州投资决策中考虑ESG因素 |
| **反抵制立法** | 西弗吉尼亚州、肯塔基州和其他州限制州资金来自被认为歧视化石燃料的金融机构 |
| **总检察长行动** | 25名州总检察长向主要资产管理公司发出信函，警告反对ESG"勾结"；对气候投资联盟的反垄断调查 |
| **国会行动** | ESG披露简化法案；禁止SEC ESG强制要求的拟议立法 |

反ESG运动导致主要气候投资倡议解散，包括几家主要资产管理公司退出"气候行动100+"和"净零资产管理人倡议"。

### 联邦审慎指引

联邦银行监管机构发布了气候相关金融风险指引：
- **OCC（2023年10月）：** 大型银行（资产>1000亿美元）气候相关金融风险管理原则
- **FDIC（2024年3月）：** 气候相关金融风险管理原则声明
- **美联储（2023）：** 与美国六家最大银行进行试点气候情景分析

该指引立即面临政治反对，共和党国会议员和州官员认为监管机构超出了其法定权限。

## 德国

### 法律框架

德国作为欧元区最大经济体和领先的绿色债券市场，全面实施欧盟可持续金融条例，并增加国家层面的监管指引和市场基础设施。

### BaFin可持续金融指引

BaFin于2019年12月发布了《关于处理可持续性风险的指导通知》（Merkblatt zum Umgang mit Nachhaltigkeitsrisiken），并于2023年更新，确立了对以下方面的监管期望：

- **银行：** 将气候和环境风险纳入风险管理（信用风险、市场风险、操作风险、流动性风险）、业务战略和治理
- **保险公司：** 将气候风险纳入承保、投资和自有风险与偿付能力评估（ORSA）
- **资产管理人：** 在组合管理、风险管理和产品治理中整合ESG

自2022年以来，BaFin已对金融机构的可持续性风险管理进行有针对性的现场检查，重点关注治理安排、数据质量和气候情景分析的稳健性。

### 德国绿色债券市场

德国是欧洲最大的绿色债券市场之一：

| 发行人类别 | 主要进展 |
|----------------|-----------------|
| **联邦政府（Bund）** | 双债券结构：与同期常规Bund同时发行绿色Bund；2024年绿色Bund存量510亿欧元；绿色债券分配报告由第三方审计 |
| **州政府（Länder）** | 北威州、巴登-符腾堡州等已发行绿色债券 |
| **KfW** | 主要绿色债券发行人（每年150亿欧元以上）；"由KfW制造的绿色债券"框架 |
| **企业** | 德国铁路、Vonovia、Evonik和公用事业公司的主要发行 |

德国绿色Bund项目于2020年9月启动，代表了一种创新的双债券方法，每只绿色Bund都有一只常规孪生债券，允许进行收益率利差比较和为"绿色溢价"（投资者为绿色标签支付的溢价）定价。

### 欧盟分类目录的实施

德国金融机构在欧盟分类目录实施方面面临重大挑战，包括中小企业贷款的数据可用性（许多德国Mittelstand公司不受CSRD报告约束）、对房地产贷款"不造成重大损害"（DNSH）标准的解释以及主权债券持有的对齐报告。BaFin发布了FAQ指引，并通过欧洲银行管理局（EBA）参与欧洲层面的解释工作。

### 联邦银行气候分析

德意志联邦银行已将气候风险纳入其金融稳定框架，在《金融稳定评论》中发表气候相关金融稳定评估，并于2023年对德国银行系统进行了自上而下的气候压力测试，发现德国银行面临中等但分布不均的转型风险敞口，区域银行（储蓄银行Sparkassen和合作银行）由于对能源密集型中小企业的集中贷款而尤其暴露。

## 欧洲人权公约与欧盟

### 欧盟可持续金融行动计划

欧盟委员会的《可持续增长融资行动计划》（2018年3月），由《更新的可持续金融战略》（2021年7月）更新，确立了全面的监管架构：

### 欧盟分类目录条例（2020/852）

作为欧盟可持续金融监管的基石，分类目录建立了基于六项环境目标的环境可持续经济活动分类系统：

1. 减缓气候变化
2. 适应气候变化
3. 可持续利用和保护水及海洋资源
4. 转向循环经济
5. 污染防治
6. 保护和恢复生物多样性和生态系统

一项活动被认定为环境可持续，必须：(a) 对一项或多项目标作出实质性贡献；(b) 不对其他目标造成重大损害；(c) 符合最低社会保障；(d) 满足授权法案中的技术筛选标准。

**争议：** 包括核能和天然气在某些条件下作为过渡活动的补充授权法案（2022/1214）引发了重大争议，奥地利和卢森堡在T-628/22案中向欧盟普通法院提出挑战（2024年驳回）。

### 可持续金融披露条例（SFDR）

SFDR为金融产品建立了三级分类：

| 分类 | 要求 | 市场术语 |
|---------------|-------------|-------------------|
| **第6条** | 披露可持续性风险整合；如认为可持续性风险不相关需说明 | "主流"产品 |
| **第8条** | 推广环境/社会特征；透明说明如何满足特征 | "浅绿"产品 |
| **第9条** | 以可持续投资为目标；证明对环境/社会目标的贡献 | "深绿"产品 |

SFDR受到批评，因为第8/9条分类被视为事实上的标签制度，尽管其设计并非如此。欧盟委员会于2023年9月启动了全面的SFDR审查，提议进行潜在重组，可能用更明确的分类系统取代第8/9条框架。

### 欧盟绿色债券标准（条例2023/2631）

《欧洲绿色债券条例》（EuGB）于2023年10月11日通过，自2024年12月21日起适用，建立了一个自愿性的"欧洲绿色债券"标识，主要特征如下：

| 特征 | 要求 |
|---------|-------------|
| **募集资金用途** | 至少85%的募集资金分配给与欧盟分类目录对齐的活动（15%灵活余地用于尚未被分类目录授权法案涵盖的活动） |
| **发行前审查** | 在营销前由注册的外部审查员进行强制性外部审查 |
| **发行后报告** | 直至完全分配为止的年度分配报告；债券存续期内至少一次影响报告 |
| **外部审查员监督** | 外部审查员在ESMA注册并受其监督 |
| **祖父条款** | 在条例适用日期前作为"绿色"营销的债券无需遵守 |

### ESG评级提供者条例

关于ESG评级活动透明度和完整性的拟议条例（COM/2023/314）于2023年6月13日提交，并于2024年初达成政治协议，将：
- 要求在欧盟运营的ESG评级提供者获得ESMA的授权和注册
- 施加治理、利益冲突管理和透明度要求
- 要求将ESG评级活动与咨询和信用评级活动分开
- 强制披露方法、数据源和模型
- 建立第三国等效和认可框架

### MiFID II / IDD可持续性偏好

授权条例（欧盟）2021/1253（MiFID II）和授权条例（欧盟）2021/1257（IDD）要求投资公司和保险分销商：
- 在适当性评估中收集客户可持续性偏好的信息
- 将产品与客户的可持续性偏好相匹配
- 如果没有满足可持续性偏好的合适产品，则予以解释

这些要求于2022年8月2日生效，由于向零售投资者解释分类目录对齐、SFDR第8/9条分类和主要不利影响指标的复杂性，实际实施一直具有挑战性。

### 欧洲央行气候活动

欧洲央行已将气候考量纳入其监管和货币政策职能：
- **2022年气候压力测试：** 涵盖104家重要银行，发现银行业气候风险敞口虽然实质但分布不均
- **监管期望（2020）：** 欧洲央行《气候相关和环境风险指南》确立了对治理、风险管理、披露和情景分析的期望
- **货币政策：** 将企业债券持有倾向于具有更好气候表现的发行人；接受可持续性挂钩债券作为抵押品
- **2024年后续行动：** 欧洲央行设定了银行完全满足气候风险预期的最后期限，对不合规情况采取升级的监管措施

## 比较分析

### 分类目录方法

欧盟分类目录是最发达和规范性最强的分类系统，为每项活动建立详细的技术筛选标准。韩国的K-Taxonomy密切仿效欧盟版本，但包括过渡类别和较少的涵盖活动，反映了不同的产业结构。美国没有联邦分类目录，这是一个有意的政策选择，反映了对可持续金融的市场驱动方法以及对政府定义的可持续性标准的政治反对。这种分类差异为跨境投资和报告造成了重大互操作性挑战。

### 监管碎片化与整合

欧盟的方法独特地整合，分类目录、SFDR、绿色债券标准、MiFID II修正案和CSRD构成一个连贯（虽然复杂）的监管生态系统。韩国正在开发类似的整合框架，但处于较早阶段。美国受到监管碎片化的困扰，不同机构（SEC、DOL、OCC、FDIC、美联储）通过各自的授权处理可持续金融，没有协调框架，反ESG立法造成相互冲突的州级要求拼凑而成的局面，使情况更加复杂。

### 政治极化

美国在经历针对可持续金融的重大政治反弹方面是独特的，18个以上州的反ESG立法为整合ESG因素的金融机构造成法律风险。这种极化在韩国、德国或欧盟没有对应物，在那里可持续金融在整个政治光谱中享有广泛的政治支持。反ESG运动产生了全球影响，导致主要气候金融联盟解散，并为跨司法管辖区运营的国际资产管理人造成不确定性。

### 绿色债券市场发展

欧盟绿色债券标准代表了规范绿色债券市场的最雄心勃勃的尝试，要求与分类目录对齐和受监督的外部审查。德国的绿色Bund项目展示了主权发行的最佳实践。韩国市场在政府支持下迅速增长，但主要依赖自愿性指引。美国拥有庞大的绿色债券市场，但没有联邦监管框架，完全依赖基于市场的标准（ICMA《绿色债券原则》）。

### 气候风险的审慎整合

所有司法管辖区都在将气候风险纳入金融监管，但速度和可执行性程度不同。欧洲央行的监管期望最为发达，并具有执法后果。BaFin的指引虽不具约束力，但有监管监督支持。韩国FSS已启动气候压力测试。美国联邦审慎指引最为试探性，且在政治上受到争议。
