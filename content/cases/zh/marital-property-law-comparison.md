## 概述

婚姻财产法——规范婚姻期间及婚姻解除时财产的所有权、管理和分割的法律规则体系——是家庭法中最具影响力的领域之一，直接影响配偶在婚姻期间和婚后的经济安全与自主权。韩国《民法》（민법）下的默认婚姻财产制度为分别财产制（별산제，第830条），即每位配偶保留以自己名义在婚姻期间取得的财产的所有权，辅以离婚时的财产分割请求权（재산분할청구권，第839-2条），允许法院在考虑每位配偶对财产取得和维护的贡献后进行公平分割；大法院（대법원）逐步扩大了对家务劳动和育儿贡献的认可，在长期婚姻中趋向推定平等分割，并认定财产分割请求权是一项不可预先放弃的独立权利。美国分为共同财产州（九个州，主要在西部和南部，遵循西班牙和法国法律传统）——婚姻期间取得的财产推定由双方平等拥有，以及公平分配州（其余四十一个州和哥伦比亚特区）——由法院根据法定因素"公平地"分割婚姻财产；最高法院在United States v. Windsor, 570 U.S. 744 (2013)和Obergefell v. Hodges, 576 U.S. 644 (2015)案中将婚姻财产权利扩展至同性伴侣。德国的默认婚姻财产制度为增益共同体（Zugewinngemeinschaft，《民法典》第1363-1390条），这是一种精密的混合制度，婚姻期间配偶各自维持独立财产，但在婚姻解除时将各自累积的增益（Zugewinn）进行均等化，增益较少的配偶有权获得差额的一半作为补偿；替代制度包括约定共同财产制（Gütergemeinschaft，《民法典》第1415-1518条）和分别财产制（Gütertrennung，《民法典》第1414条）。欧洲框架包括《欧盟婚姻财产条例》（No 2016/1103），通过加强合作程序采纳并自2019年1月29日起适用，为跨境案件中的婚姻财产事项建立了统一的法律冲突规则，以及欧洲人权法院关于婚姻解除背景下《议定书第1号》第1条财产权的判例法，和欧洲家庭法委员会的《欧洲家庭法原则》中的比较视角。

## 比较表

| 要素 | 韩国 | 美国 | 德国 | 欧洲人权公约/欧盟 |
|------|------|------|------|-------------------|
| **默认制度** | 分别财产制（별산제，第830条） | 共同财产制（9个州）或公平分配制（41个州+哥伦比亚特区） | 增益共同体（Zugewinngemeinschaft） | 欧盟条例2016/1103（法律冲突） |
| **婚姻期间所有权** | 分别（每位配偶拥有各自财产） | 共同财产：共有；公平分配：通常在解除前为分别 | 分别（尽管制度名称如此） | 由国内法管辖 |
| **离婚时分割** | 法院公平分割（第839-2条） | 共同财产：平等；公平分配：公平因素 | 增益均等化（Zugewinnausgleich） | 由国内法管辖 |
| **家务劳动贡献** | 认可（大法院判例法） | 认可（所有州） | 认可（隐含在增益均等化中） | 不适用 |
| **婚前协议** | 认可（第829条） | 认可（UPAA/UPMAA；各州法律） | 认可（Ehevertrag，《民法典》第1408条） | 可选择法律/制度（条例2016/1103） |
| **债务处理** | 一般为分别，除非共同 | 共同财产：共担；公平分配：各异 | 分别（增益共同体）；共担（共同财产制） | 由国内法管辖 |
| **专业学位/执照** | 不能作为可分割财产直接分割 | 各异：部分州估价；多数排除 | 非财产；可能影响赡养费 | 不适用 |

## 韩国

### 法律框架

韩国的婚姻财产制度受《民法》第3编第1章（민법 제3편 제1장）管辖。

### 分别财产制（별산제）

**第830条第1款：** 一方配偶在婚前拥有的财产和婚姻期间以自己名义取得的财产归该配偶个人所有（특유재산）。

**第830条第2款：** 无法确定归属于任何一方配偶的财产，推定为共同所有。

**第831条：** 每位配偶独立管理其各自的分别财产。

### 离婚时的财产分割（재산분할청구권）

**第839-2条：** 离婚时，任何一方均可向对方主张财产分割。家事法院在确定分割时考虑以下因素：

| 因素 | 描述 |
|------|------|
| 财产的数额和性质 | 婚姻期间取得的所有财产 |
| 取得贡献 | 每位配偶的经济贡献 |
| 家务劳动贡献 | 育儿、家庭管理、对另一方配偶事业的支持 |
| 未来生计 | 离婚后每位配偶的经济前景 |
| 婚姻期限 | 较长的婚姻倾向于更平等的分割 |
| 其他情况 | 所有相关情况 |

### 大法院判例法

**财产分割比例：** 大法院逐步提高了对非收入配偶贡献的认可。在长期婚姻中（通常20年以上），法院通常将40%至50%判给非收入配偶，部分判决达到50%。具体比例取决于案件的全部情况。

**大法院判决93므1002（1993年）：** 确立了家务劳动贡献构成对财产形成的真实贡献，有资格在财产分割中获得认可。

**大法院判决2012므4130（2014年）：** 认定第839-2条项下的财产分割请求权是因离婚而产生的独立权利，不同于精神损害赔偿（위자료），涵盖所有类型的婚姻财产，包括不动产、金融资产和退休福利。

**大法院判决2013므568（2013年）：** 认可了财产分割中的"扶养性要素"（부양적 요소），允许法院在仅凭贡献不足以证明合理的情况下，为经济弱势配偶提高分割比例。

### 退休福利和养老金

《国民年金法》（국민연금법）第64条规定了养老金权利的分割。在缴费期间内婚姻持续至少五年的非缴费配偶，有权在离婚时分割缴费配偶的老年养老金。该制度于1999年实施，适用于与养老金缴费期间重叠的足够期限的婚姻。

### 隐匿财产

隐匿财产的发现和估价是一项重大的实际挑战。韩国法院可发出财产调查命令（재산조사명령），2020年《家事诉讼法》修正案加强了法院调查和发现婚姻资产的权力，包括银行账户、不动产和证券持有。

## 美国

### 法律框架

美国有两种根本不同的婚姻财产制度：

### 共同财产州

九个州遵循共同财产原则：亚利桑那州、加利福尼亚州、爱达荷州、路易斯安那州、内华达州、新墨西哥州、得克萨斯州、华盛顿州和威斯康星州（通过《统一婚姻财产法》）。

**关键原则：**
- 婚姻期间通过配偶劳动或技能取得的财产为共同财产，由双方平等拥有。
- 婚前取得的财产或婚姻期间通过赠与或继承取得的财产为个人财产。
- 离婚时，共同财产平等分割（部分州允许公平偏离）。
- 婚姻期间产生的债务一般为共同债务。

**加利福尼亚州示例（Cal. Fam. Code Section 2550）：** "法院应平等分割当事人的共同财产。"

### 公平分配州

其余四十一个州和哥伦比亚特区遵循公平分配原则。

**关键原则：**
- 婚姻财产"公平地"（公正地）分割，可能但不一定意味着平等。
- 法院考虑法定因素，包括婚姻期限、每位配偶的贡献（经济和家务）、双方的年龄和健康状况、收入能力和税务后果。
- 个人财产（婚前、继承、赠与所得）通常不在分割范围内。

**纽约州示例（DRL Section 236(B)(5)(d)）：** 列出了十三项公平分配因素。

### 重要资产

| 资产类型 | 处理方式 |
|----------|----------|
| 不动产 | 如在婚姻期间取得，为可分割的婚姻财产 |
| 退休账户 | 根据ERISA通过合格家事关系令（QDRO）分割 |
| 企业权益 | 估价并分割；可能需要专家评估 |
| 专业商誉 | 因州而异：部分州纳入，部分州排除 |
| 股票期权 | 如在婚姻期间获得，一般可分割 |
| 数字资产/加密货币 | 日益受到关注；估值存在挑战 |

### 《统一婚前和婚姻协议法》（UPMAA）

UPMAA（2012年）取代了《统一婚前协议法》（UPAA，1983年），为婚前和婚后协议提供了框架。关键条款包括自愿执行的要求、资产的公平披露以及协议在执行时不得显失公平。并非所有州都已采纳UPMAA。

## 德国

### 法律框架

德国婚姻财产法提供三种制度：

### 增益共同体（默认制度）

**《民法典》第1363条：** 如果配偶未通过婚姻协议约定不同的制度，则适用增益共同体。尽管其名称为"增益共同体"，但这在婚姻期间并非共同财产制度。每位配偶保留各自财产的独立所有权和管理权。

**增益均等化（Zugewinnausgleich）：** 在制度终止时（离婚、死亡或约定终止），计算每位配偶在婚姻期间的增益（Zugewinn）：

| 步骤 | 计算 |
|------|------|
| 1. 初始资产（Anfangsvermögen） | 婚姻开始时的资产（《民法典》第1374条） |
| 2. 最终资产（Endvermögen） | 制度终止时的资产（《民法典》第1375条） |
| 3. 增益（Zugewinn） | 最终资产减去初始资产 |
| 4. 均等化请求权 | 增益较少的配偶获得差额的一半 |

**示例：** 如果配偶A的增益为200,000欧元，配偶B的增益为80,000欧元，则均等化请求权为(200,000 - 80,000) / 2 = 60,000欧元，由A支付给B。

### 替代制度

**分别财产制（Gütertrennung，《民法典》第1414条）：** 无增益均等化。婚姻解除时每位配偶保留所有财产。须在公证的婚姻协议（Ehevertrag）中约定。

**共同财产制（Gütergemeinschaft，《民法典》第1415-1518条）：** 真正的共有基金（Gesamtgut）共同所有权。因复杂性在实践中很少被选择。

**选择性增益共同体（Wahl-Zugewinngemeinschaft）：** 由2010年《德法协议》引入，这一跨国选项可供法德及其他跨境伴侣使用，基于两国制度的共同原则。

### 婚姻协议（Ehevertrag）

**《民法典》第1408条：** 配偶可通过公证的婚姻协议（Ehevertrag）修改其财产制度。协议还可涉及赡养费和养老金均等化。然而，联邦最高法院（BGH）已对婚姻协议设定了限制：

**BGH判决XII ZR 265/99（2004年）：** 法院确立了离婚后保护的核心领域（Kernbereich），不能通过婚姻协议完全放弃。累计消除所有经济保护（财产均等化、赡养费和养老金均等化）的协议可能因显失公平而受到司法审查。

### 负增益

增益共同体的一个重要特征是，负增益（一方配偶的最终资产少于初始资产）在均等化计算中被视为零。一方配偶的损失不会转嫁给另一方。2009年的改革（《增益均等化修正法》）修改了这一规则，取消了以前增益不得超过最终资产实际价值的限制，并允许计入负数初始资产。

## 欧洲人权公约与欧盟

### 欧盟婚姻财产条例（No 2016/1103）

2016年6月24日理事会条例（EU）2016/1103，在婚姻财产制度事项的管辖权、适用法律及判决的承认和执行领域实施加强合作，于2019年1月29日起适用。截至2026年，18个成员国参与。

**适用法律：** 在配偶未作出选择的情况下，适用法律按照以下级联确定：
1. 婚后配偶首个共同惯常居所地的法律
2. 婚姻缔结时配偶共同国籍国的法律
3. 婚姻缔结时配偶共同与之有最密切联系的国家的法律

**法律选择（第22条）：** 配偶可选择任何一方配偶的惯常居所地国或国籍国的法律。

**承认与执行：** 参与成员国对婚姻财产作出的判决在其他参与国无需特别程序即获承认（第36条）。

### 欧洲人权法院判例法

**Pla和Puncernau诉安道尔案（申请号69498/01，2004年7月13日）：** 法院认定在区分婚生子女和非婚生子女的继承规则的歧视性适用中，违反了第14条与第8条。虽然主要是继承案件，但对子女的婚姻财产权利有影响。

**Belli和Arquier-Martinez诉瑞士案（申请号65550/13，2018年12月11日）：** 法院处理了婚姻财产清算不公平的主张，但宣布不予受理，确认《议定书第1号》第1条可适用于财产分割请求，但各国在设计其财产制度方面享有广泛的裁量余地。

### 欧洲家庭法委员会（CEFL）

CEFL发表了《关于配偶间财产关系的欧洲家庭法原则》（2013年），提出：
- 承认共同财产制和附带均等化的分别财产制均为有效模式
- 以婚姻期间增益的平等分享作为默认规则
- 认可家务劳动贡献
- 对产生明显不公平结果的协议进行限制

这些原则不具有约束力，但作为各国改革的比较参考。

### 未婚伴侣的财产权利

欧盟框架不涉及未婚伴侣的财产权利。欧洲人权法院在Burden诉英国案（2008年）中对此有所涉及，认定将同居兄弟姐妹排除在已婚夫妇享有的税务待遇之外不构成歧视，同时承认未婚伴侣的财产权利仍属于国内法事项。

## 比较分析

### 制度设计

三个司法管辖区采用了根本不同的默认制度。韩国的分别财产制在婚姻期间最为简单，但在解除时产生重大问题。德国的增益共同体巧妙地将婚姻期间的自主权与解除时的公平分享相结合。美国的共同财产制度在婚姻期间赋予并行所有权，而公平分配州使用分别财产默认制加解除时的公平分割。每种制度反映了对婚姻作为经济伙伴关系性质的不同假设。

### 平等分割与公平分割

一个关键分歧存在于平等分割（德国增益均等化；美国共同财产州）和公平/自由裁量分割（韩国；美国公平分配州）之间。平等分割提供可预测性和一致性，但在非典型情况下可能产生不公平结果。公平分割提供灵活性，但产生不确定性并激励诉讼。

### 非经济贡献的认可

所有司法管辖区都认可家务和育儿贡献，但机制不同。德国的增益共同体固有地认可非经济贡献，因为全职家庭主妇/主夫的增益与收入配偶的增益进行均等化。美国法院明确将家务劳动列为考量因素。韩国通过大法院判例法逐步扩大了认可范围，从早期几十年的低认可度发展到长期婚姻中的近乎平等待遇。

### 养老金和退休福利

德国单独的养老保障均等化（Versorgungsausgleich）制度对养老金分割最为全面，将养老金均等化视为独立于财产分割的独特法律机制。美国通过QDRO在财产分割框架内处理养老金分割。韩国根据《国民年金法》提供养老金分割，但覆盖面不如德国全面。
