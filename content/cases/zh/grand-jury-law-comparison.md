## 概述

大陪审团——由公民组成的机构，审查检方提交的证据并决定是否有足够的合理根据（probable cause）对嫌疑人提起刑事指控（起诉书）——是一种独具英美法系特色的制度，在大多数大陆法系中没有直接对应物，反映了在谁应控制刑事起诉的入口以及公民如何参与刑事程序这一问题上的根本不同的方法。韩国不采用大陪审团制度，但通过国民参与审判制度（국민참여재판）发展了功能上的替代机制，该制度于2008年依据《国民参与刑事审判法》（국민의 형사재판 참여에 관한 법률）引入，允许公民在审判阶段（而非起诉阶段）担任陪审员，并通过检察市民委员会（검찰시민위원회，2018年设立）以及第260条的司法审查机制（재정신청）对检察官的起诉决定进行外部监督。美国是所研究的司法管辖区中唯一保留宪法规定的大陪审团制度的国家，第五修正案要求联邦案件中"死刑或其他严重犯罪"须经大陪审团起诉，但该要求并未通过第十四修正案适用于各州（Hurtado v. California, 110 U.S. 516 (1884)）；约半数州要求重罪案件须经大陪审团起诉，其余州允许检察官以检控书（information）方式提起公诉。德国从未采用大陪审团制度，而是依赖法定起诉原则（Legalitätsprinzip，强制起诉原则）、司法预审程序（StPO第203条规定的开庭裁定Eröffnungsbeschluss——法院独立判断是否有充分嫌疑开启正式审判程序）以及强制起诉程序（StPO第172条的Klageerzwingungsverfahren）作为审查检察官起诉决定和确保问责的机制。欧洲法律传统通常依赖对起诉过程的司法控制而非公民参与，欧洲人权公约第6条要求及时告知被告指控的性质和原因，欧盟2012/13/EU号指令关于刑事诉讼中的知情权确立了告知指控的最低标准。

## 韩国

### 法律框架

韩国的刑事司法制度不包含大陪审团。检察官起诉（기소）一直是提起刑事公诉的唯一途径，外部监督通过司法和公民监督机制提供。

### 检察官垄断权与起诉

根据《刑事诉讼法》第246条（공소는 검사가 제기하여 행한다——"公诉由检察官提起并进行"），检察官拥有提起刑事诉讼的专属权力。尽管2020-2022年设立高级公职人员犯罪调查处（CIO）的改革对这一垄断提出了挑战，但它仍然是韩国起诉实践的根本结构特征。

### 第260条——不起诉决定的司法审查（재정신청）

《刑事诉讼法》第260条提供了一种功能上类似于大陪审团某一方面——公民参与起诉决定——的机制：

被害人或告诉人可向高等法院提出申请（재정신청），对检察官不起诉的决定提出质疑。法院审查证据并判断是否有充分的起诉依据。如法院批准申请，案件移送至地方法院，视同已提起公诉，并由法院指定律师担任检察官角色。

该机制在2007年修正案中被显著扩大，适用范围从有限的犯罪类别扩展至所有刑事犯罪，宪法法院的后续裁决也确认了这一对检察权监督的重要性。

### 国民参与审判制度（국민참여재판）

虽非大陪审团，韩国的国民参与审判制度（2008年设立）引入了公民参与刑事诉讼程序：

| 特征 | 韩国制度 |
|---------|--------------|
| 名称 | 국민참여재판（国民参与审判） |
| 法律依据 | 《国民参与刑事审判法》（2007年） |
| 陪审员人数 | 根据犯罪严重程度为5、7或9人 |
| 适用案件 | 严重重罪（杀人、抢劫、性侵犯等） |
| 裁决 | 咨询性质（对法官无约束力，但多数情况下法官会遵循） |
| 被告选择 | 被告须申请或同意进行陪审审判 |

### 检察市民委员会（검찰시민위원회）

该委员会于2018年设立，由外部民间成员组成，审查具有重大公共利益的检察决定，提供一种类似于大陪审团民主正当性功能的公民监督形式。该委员会可以建议重新审议不起诉决定。

## 美国

### 法律框架

美国是世界上唯一在现代刑事司法制度中将大陪审团保留为宪法制度的国家。

### 第五修正案

第五修正案规定："任何人不得被迫就死刑或其他严重犯罪受审，除非经大陪审团提出起诉书或公诉书，但在战时或公共危险时期在陆海军中或民兵服役时发生的案件除外。"

### 宪法适用范围

**Hurtado v. California, 110 U.S. 516 (1884)：** 最高法院裁定第五修正案的大陪审团要求并未通过第十四修正案适用于各州。这仍是少数未被纳入适用的《权利法案》条款之一，反映了大陪审团独特的历史地位。

因此：
- **联邦：** 所有重罪均须经大陪审团起诉
- **各州：** 约23个州要求重罪案件须经大陪审团起诉；其余州允许以检控书方式起诉

### 联邦大陪审团程序

| 特征 | 说明 |
|---------|-------------|
| 人数 | 16-23名成员 |
| 法定人数 | 16名成员 |
| 起诉 | 须至少12名成员同意 |
| 任期 | 最长18个月（可延至24个月） |
| 标准 | 合理根据 |
| 证据规则 | 不适用 |
| 辩护律师 | 不得进入大陪审团审理室 |
| 保密性 | 严格保密（Fed. R. Crim. P. 6(e)） |

### 权力与功能

大陪审团拥有广泛的调查权力：
- **传唤权：** 可强制作证和提交文件；不遵守可构成藐视
- **调查功能：** 可独立于检察官调查犯罪活动（但实际上由检察官控制调查）
- **审查功能：** 审查证据以确定是否有合理根据起诉
- **保护功能：** 保护公民免受无根据的起诉（历史上的正当性依据）

### 批评与"橡皮图章"问题

大陪审团长期面临作为检察官决定的"橡皮图章"的批评。联邦大陪审团在所提交案件中约有99.9%作出起诉决定，促使法官Sol Wachtler发表了著名评论：检察官可以"起诉一个火腿三明治"。批评者认为：
- 检察官控制所有提交的证据
- 被告无权出庭或提出证据
- 大陪审团成员缺乏独立的法律指导
- 程序的保密性阻碍了公众监督
- 大陪审团程序的单方面性质使其无法发挥审查功能

### 特别大陪审团

《联邦刑事诉讼规则》第6(a)(1)条授权在人口超过500万的司法区设立特别大陪审团，其任期可长达36个月，拥有增强的调查权力，尤其针对有组织犯罪和公职腐败案件。

### 大陪审团与警察使用武力

大陪审团在涉及警察使用武力的案件中成为争议焦点，特别是在2014年密苏里州弗格森市Michael Brown枪击案和纽约市Eric Garner死亡案中大陪审团未作出起诉决定之后。批评者认为，大陪审团程序在结构上偏向警察，因为依赖警方合作的同一检察官向大陪审团提交证据。

## 德国

### 法律框架

德国从未采用大陪审团制度。对检察官起诉决定的审查通过司法机制和法定起诉原则实现。

### 开庭裁定（Eröffnungsbeschluss）——StPO第203条

当检察官提交起诉书（Anklageschrift）时，法院必须在开启正式审判程序之前独立评估是否存在"充分嫌疑"（hinreichender Tatverdacht），即被告人实施了被指控的犯罪。StPO第203条规定："如预审调查结果显示被告人有充分嫌疑实施了该犯罪行为，法院应裁定开启正式审判程序。"

这一司法审查发挥了类似于大陪审团保护功能的作用：
- 法院独立审查检察案卷
- 法院可根据StPO第202条要求补充侦查
- 如缺乏充分嫌疑，法院可拒绝开庭
- 被告在法院作出决定前有权对起诉书作出回应

### 强制起诉程序——StPO第172条

强制起诉程序允许被害人质疑检察官的不起诉决定，对检察裁量权提供了类似于大陪审团对检方监督功能的制约。

### 参审员制度（Schöffen）

虽非等同于大陪审团成员，参审员（Schöffen，非职业法官）参与初级法院（Amtsgericht）和州法院（Landgericht）层面的刑事案件裁判。参审员是从社区名册中选出的公民，与职业法官共同审理案件，在罪责和量刑问题上享有完全投票权。

| 特征 | 参审员制度 |
|---------|----------------|
| 法院层级 | 初级法院（Schöffengericht）；州法院（Große Strafkammer） |
| 参审员人数 | 2人（Schöffengericht）；2人（Große Strafkammer） |
| 职业法官人数 | 1人（Schöffengericht）；2-3人（Große Strafkammer） |
| 选任方式 | 从社区名册中随机选取；任期5年 |
| 决定权力 | 对罪责和量刑享有完全投票权 |
| 管辖范围 | 中等重罪（Schöffengericht）；严重重罪（Landgericht） |

### 历史注记

德国在19世纪曾短暂实行陪审团审判制度（Schwurgericht），受法国大革命理想的影响，但于1924年废除，以包含参审员的混合法庭制度取而代之。放弃纯粹陪审团制度反映了对将非职业和职业判断结合而非分离的偏好。

## 欧洲人权公约与欧盟

### 欧洲人权公约——关于指控知情权

欧洲人权公约第6条第(3)款第(a)项规定，每一被控犯罪的人均有权"以其能理解的语言被迅速详细地告知对其指控的性质和原因"。该条款规范了起诉程序，但并未规定特定的机制（大陪审团或其他）。

**Pelissier and Sassi v. France [GC]，申诉号25444/94，1999年3月25日判决：** 大审判庭裁定，知悉指控性质和原因的权利必须作广义解释，包括犯罪的法律定性。以与被告所被告知的指控实质不同的罪名定罪，违反第6条第(3)款第(a)项。

### 欧盟2012/13/EU号指令

关于刑事诉讼中知情权的2012/13/EU号指令（2012年5月22日通过）确立了最低标准：

**第6条——关于指控的知情权：** 须向犯罪嫌疑人和被告人提供关于其被怀疑或被指控实施的犯罪行为及犯罪的法律定性的信息，其详细程度须足以保障辩护权的有效行使。

**权利告知书：** 第4条要求在逮捕或拘留犯罪嫌疑人时向其提供书面的权利告知书，告知其程序性权利。

### 欧洲的公民参与刑事司法模式

虽然没有欧洲司法管辖区保留大陪审团，但存在多种形式的公民参与：
- **陪审团审判：** 英格兰和威尔士、爱尔兰、比利时、奥地利（严重犯罪）
- **混合法庭（非职业法官+职业法官）：** 德国、法国（Cour d'assises）、丹麦、瑞典、挪威
- **人民陪审员：** 东欧后共产主义体制

## 比较分析

### 公民对起诉的控制

大陪审团代表了公民对起诉过程最直接的控制形式，将非专业判断置于检察官与被告之间。德国的开庭裁定提供的是司法（职业）审查而非公民审查。韩国的第260条机制提供了由被害人发起的司法审查。然而，美国大陪审团近乎百分之百的起诉率令人质疑公民控制是否得到了有意义的行使。

### 透明性与保密性

美国大陪审团在严格保密下运作，理由是保护被调查但未被起诉者的名誉、防止嫌疑人潜逃以及确保证人坦诚作证。德国的预审程序不公开，但起诉书会向被告人披露。韩国的第260条程序属于司法程序，受法院程序约束。保密性与问责之间的紧张关系仍然是一个核心议题。

### 对不起诉的问责

德国通过强制起诉程序提供了质疑不起诉决定的最有力机制，赋予法院强制起诉的权力。韩国的第260条提供了类似机制。美国没有提供有效的机制，因为大陪审团程序由已经决定是否寻求起诉的检察官控制。

### 民主正当性

美国的大陪审团和韩国的国民参与审判制度均通过公民参与刑事程序获得正当性。德国的参审员制度在审判阶段发挥类似的民主功能。大陪审团在将公民判断置于起诉阶段方面独具特色，但其作为民主制约的有效性受到广泛质疑。
