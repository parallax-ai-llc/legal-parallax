## 摘要

工伤补偿——为在雇用过程中受伤或致病的雇员提供医疗、工资替代和残疾给付的强制性保险或法定福利制度——代表了最古老的社会保险形式之一，体现了"重大交易"：雇员获得无过错保障福利，同时雇主获得侵权责任豁免。韩国的《产业灾害补偿保险法》（산업재해보상보험법）于1963年首次颁布，是该国最早的社会保险项目之一，建立了由韩国劳动者补偿和福利服务机构（근로복지공단）管理的全面公共保险制度，覆盖几乎所有1人以上雇员场所的工作者；该制度提供医疗给付、按平均工资70%的临时残疾给付、通过14个残疾等级的永久残疾给付、遗属给付和职业康复，2021年进行了重大扩展，涵盖工作相关的精神健康状况，包括工作相关的抑郁症、PTSD和过劳相关的心血管疾病（과로사）。美国通过50个独立的州系统运作工伤补偿，在覆盖、福利和管理方面差异显著，由特定工人类别的联邦项目补充，包括《联邦雇员补偿法》（FECA）、《长岸和港口工人补偿法》和《黑肺福利法》；大多数州遵循无过错模式，独家救济条款禁止雇员对雇主提起侵权诉讼。德国通过Otto von Bismarck于1884年的《事故保险法》（Unfallversicherungsgesetz）开创了法定事故保险，现编入《社会法典》第VII卷（SGB VII），由Berufsgenossenschaften（雇主责任保险协会）管理，将工伤补偿职能与工作场所安全监管结合在独特的整合系统中；德国制度涵盖所有雇员、学生和志愿者，无需雇员缴费。在欧洲层面，工伤补偿仍主要是国家职权，欧盟协调仅限于关于跨境工人社会保障协调的（EC）883/2004号条例，而国际劳工组织第121号公约关于就业伤害福利（1964年）确立了国际最低标准。

## 韩国

### 法律框架

《产业灾害补偿保险法》（산업재해보상보험법）于1963年11月5日颁布，于1964年7月1日生效，使其成为朴正熙政府经济发展计划下建立的首批社会保险项目之一。最初仅涵盖500名或以上雇员的采矿和制造业工作场所，覆盖范围已逐步扩大，自2000年以来适用于所有1人以上雇员的工作场所。

### 管理

根据该法设立的韩国劳动者补偿和福利服务机构（근로복지공단，COMWEL）管理保险制度的所有方面：保费征收、索赔裁决、通过其医院和康复中心网络提供医疗、职业康复和重返工作项目。该制度完全由雇主通过经验费率保费提供资金；雇员不缴费。

### 福利结构

| 福利类型 | 描述 | 计算 |
|-------------|-------------|-------------|
| 医疗（요양급여） | 工作相关伤害/疾病的所有必要医疗 | 全额费用覆盖；无共付额 |
| 临时残疾（휴업급여） | 治疗期间的工资替代 | 平均日工资的70% |
| 永久残疾（장해급여） | 持久残疾的补偿 | 14个等级：1级（最严重）= 1,474天工资一次性付款或年金；14级 = 55天工资 |
| 遗属给付（유족급여） | 已故工人受抚养人的给付 | 年金：平均工资的52-67%；或一次性付款：1,300天工资 |
| 丧葬费（장의비） | 丧葬费用 | 120天平均工资 |
| 职业康复（직업재활급여） | 再培训和重返工作支持 | 各种项目，包括工作改造补贴 |

### 2021年精神健康和过劳改革

2021年修订显著扩大了对工作相关精神健康状况的认可：

**过劳相关疾病（과로사）：** 修订正式确立了认可由长期过劳引起的心血管和脑血管疾病（뇌심혈관계 질병）的标准，鉴于韩国密集的工作文化尤为相关。在12周内平均每周工作超过60小时并出现心脏病、中风或相关状况的工人被推定患有工作相关疾病。

**精神健康：** 工作相关的抑郁症、焦虑症、PTSD和适应障碍现明确涵盖，简化审批流程。修订还将职场欺凌（직장 내 괴롭힘）认可为可补偿精神健康状况的因果因素。

### 统计

2023年，COMWEL处理了约13万件产业事故索赔，批准率约89%。总福利支出超过8.5万亿韩元。职业死亡补偿率稳步下降，但仍高于德国或大多数欧盟成员国。

## 美国

### 法律框架

美国的工伤补偿主要由州法管辖，导致50个不同的制度，差异显著。第一部州工伤补偿法由威斯康星州于1911年颁布，到1948年所有州都建立了工伤补偿项目。

### 州制度差异

| 特征 | 多数做法 | 显著差异 |
|---------|-------------------|-------------------|
| 保险机制 | 私营保险或州基金 | 俄亥俄州、华盛顿州、怀俄明州、北达科他州：独家州基金（垄断性） |
| 工资替代率 | 平均周工资的66.67% | 范围从60%到80% |
| 最高周福利 | 通常为州平均周工资的100-150% | 差异显著 |
| 等待期 | 福利开始前3-7天 | 一些州在残疾超过指定期限时追溯支付 |
| 律师费 | 由工伤补偿委员会监管 | 风险代理费通常为15-25% |
| 独家救济 | 是，有例外 | 一些州允许对故意行为提起侵权诉讼 |

### 联邦项目

| 项目 | 覆盖 | 管理机构 |
|---------|----------|---------------------|
| 《联邦雇员补偿法》（FECA） | 联邦文职雇员 | DOL工人补偿项目办公室 |
| 《长岸和港口工人补偿法》 | 海事工人（非船员） | DOL OWCP |
| 《黑肺福利法》 | 患尘肺病的煤矿工人 | DOL OWCP |
| 《能源雇员职业病补偿项目》 | 暴露于辐射/毒素的DOE工人 | DOL OWCP |
| 《国防基地法》 | 海外军事基地承包商 | DOL OWCP |

### 当代问题

**退出运动：** 德克萨斯州是唯一不要求大多数私营雇主购买工伤补偿保险的州。俄克拉何马州在2013年颁布了允许合格雇主建立替代福利计划的退出条款，但俄克拉何马州最高法院在2016年（Vasquez诉Dillard's案）以违宪为由推翻了该法。

**零工经济：** 将零工工人归类为独立承包商已将数百万工人排除在工伤补偿覆盖之外。加州AB 5（2019年）和随后的第22号提案（2020年）创造了一个有争议的框架，试图解决这一差距。

**重大交易侵蚀：** 2015年ProPublica和NPR的调查系列《工人补偿的拆毁》记录了各州数十年来福利削减的趋势，引发了关于以保障福利换取侵权豁免的"重大交易"是否仍然公平的问题。

## 德国

### 法律框架

德国通过1884年《事故保险法》（Unfallversicherungsgesetz）建立了世界上第一个法定工伤补偿制度，该法是Otto von Bismarck总理颁布的三项社会保险法律之一。该制度现编入《社会法典》第VII卷（SGB VII），于1997年1月1日生效。

### Berufsgenossenschaften制度

德国制度独特之处在于将工伤补偿与工作场所安全功能结合在单一的机构框架内。Berufsgenossenschaften（BG）是按行业部门组织的公法法人：

| BG | 行业部门 |
|----|----------------|
| BG BAU | 建筑 |
| BG ETEM | 能源、纺织、电气、媒体 |
| BGW | 卫生服务和福利 |
| BG Verkehr | 运输 |
| BGHW | 贸易和物流 |
| BG RCI | 原材料和化学品 |
| BG Holz und Metall | 木材和金属 |
| VBG | 行政 |
| BG Nahrungsmittel | 食品工业 |

每个BG执行三项整合职能：预防（制定安全规则和进行检查）、康复（提供全面医疗和职业康复）和补偿（支付残疾和遗属福利）。

### 福利

| 福利 | 描述 |
|---------|-------------|
| 医疗（Heilbehandlung） | 完整医疗，包括专门的BG运营的创伤医院（BG-Kliniken） |
| 伤害福利（Verletztengeld） | 治疗/康复期间正常收入的80%（SGB VII第47条） |
| 残疾养老金（Verletztenrente） | 基于盈利能力降低（Minderung der Erwerbsfähigkeit, MdE）；100% MdE的全额养老金为年收入的2/3（SGB VII第56条） |
| 遗属养老金（Hinterbliebenenrente） | 鳏夫/寡妇：已故年收入的30%（如果抚养子女/47岁以上则为40%） |
| Pflegegeld | 需要持续个人护理者的护理津贴 |

### 独家救济（Haftungsprivileg）

SGB VII第104条规定，雇员不得就法定保险制度涵盖的工作场所伤害对其雇主提起侵权诉讼，除非雇主故意（Vorsatz）造成伤害。第105条将此豁免扩展至同事。这一独家救济（Haftungsprivileg）是德国对重大交易的实施。

### 职业病（Berufskrankheiten）

Berufskrankheiten-Verordnung（BKV）列出了认可的职业病（截至2021年修订当前列出82种疾病）。2021年改革（第七次SGB VII修订法）取消了工人在接受职业病福利之前必须停止致病活动的要求（Unterlassungszwang），这是索赔认可的长期障碍。改革还引入了"开放条款"，允许在新科学证据确立工作因果关系时认可未列入正式清单的疾病。

### 业绩指标

2022年，Berufsgenossenschaften处理了约82.2万起可报告的工作场所事故，并认可了约1.9万件新的职业病案件。所有BG的总支出约为167亿欧元。德国每10万名工人0.7的工作场所死亡率是世界上最低的之一，部分归功于整合的预防-补偿模式。

## 欧洲人权公约和欧盟

### 欧盟社会保障协调

关于社会保障制度协调的（EC）883/2004号条例确保在欧盟成员国之间流动的工人不会失去工伤补偿覆盖。该条例确立了确定适用立法（一般为雇用国法律）、保险期间累加和福利出口的规则。它由（EC）987/2009号实施条例补充。

### 欧盟职业病建议

委员会建议2003/670/EC建立了一个鼓励成员国采用的欧洲职业病清单。该建议涵盖由化学、物理和生物因素引起的疾病，以及职业性皮肤病、呼吸系统疾病和癌症。虽然不具有法律约束力，但它作为国家职业病清单的参考框架。

### 国际劳工组织第121号公约

国际劳工组织关于就业伤害福利的第121号公约（1964年）为工伤补偿制度确立最低标准。主要要求包括涵盖所有雇员（包括公共部门工人）、对工作场所事故和职业病的补偿、无限制的医疗、临时无能力期间至少60%的工资替代以及对永久残疾和死亡的定期支付（而不是一次性付款）。韩国于1992年批准了第121号公约；美国和德国均未批准，尽管两者都大大超过了其最低标准。

### 欧洲人权法院

欧洲人权法院通过第1议定书第1条（财产保护）和第6条（公平审判）的视角处理了与工伤补偿相关的问题。在Moskal诉波兰案（申诉号10373/05，2009年），法院判决社会保障福利（包括残疾福利）构成第1议定书第1条下的"财产"，且撤销福利必须满足比例性测试。法院还判决工伤补偿程序必须遵守第6条公平审判保障，包括合理期限和诉诸独立法庭。

## 比较分析

### 制度架构

三个司法管辖区代表三种不同的架构模式：韩国运作由单一国家机构（COMWEL）管理的中央化公共保险制度；美国运作50多个分散制度，混合私营保险、州基金和自保；德国运作由准公共自治协会（Berufsgenossenschaften）组成的基于行业的制度。德国模式独特之处在于将预防与补偿整合，为事故预防创造制度激励。

### 福利充足性

德国提供最慷慨的工资替代（80%），其次是韩国（70%）和美国（通常66.67%，受州上限限制）。然而，美国制度的最高周福利上限通常导致高薪工人的有效替代率远低于66.67%。

### 覆盖差距

美国有最显著的覆盖差距，德克萨斯州允许雇主完全退出，将零工经济工人归类为独立承包商使数百万人被排除在覆盖之外。韩国扩展至涵盖所有1人以上雇员场所，德国包括学生和志愿者，在各自制度内提供近乎普遍的覆盖。

### 职业病认可

德国2021年取消停止要求（Unterlassungszwang）代表了一项重大改革，消除了阻止许多工人在继续工作的同时获得职业病福利的障碍。韩国2021年扩展至精神健康状况和过劳相关疾病反映了对心理社会工作场所危害日益增长的认可。美国各州在职业病认可方面差异显著，许多使用限制性清单或要求证明独特的工作场所因果关系。
