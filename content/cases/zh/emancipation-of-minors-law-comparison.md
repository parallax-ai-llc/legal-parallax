## 摘要

未成年人解除亲权——使未达法定成年年龄的人摆脱父母权力并在达到标准成年年龄之前获得成年人的法律权利和责任的法律机制——在各司法管辖区通过不同的框架运作，反映了对童年、父母权力以及何种情况下适合赋予早期法律独立性的不同理念。韩国没有单独的解除亲权法，但通过《民法》（민법）中关于因婚姻获得成年地位的条款达到功能等同效果，即已婚未成年人依据第826条之2（성년의제）获得与成年人同等的法律行为能力；随着2021年成年年龄降低至18岁且结婚年龄也设定为18岁，通过婚姻解除亲权的实际适用范围已被有效消除，尽管该原则仍保留在《民法》中，且未成年人可通过法定代理人依据第5条至第7条允许的特定法律行为获得有限的功能性独立。美国提供了最发达的解除亲权框架，约40个州通过司法申请提供法定解除亲权（通常要求未成年人证明自立能力且解除亲权符合其最佳利益），婚姻或服兵役自动解除亲权，以及基于父母同意和未成年人独立生活的普通法解除亲权原则；解除亲权终止父母权力和抚养义务，同时赋予签订合同、寻求医疗和维持独立住所的权利。德国没有正式的解除亲权机制，但允许16岁及以上的未成年人在父母或法院同意下进行有限的法律行为，且2021年婚姻年龄改革前的《民法典》允许在法院批准下于16岁结婚，婚姻具有事实上的解除亲权效果；德国的方法强调渐进行为能力制度（BGB第106-113条）而非二元化的解除亲权模式。欧洲框架——特别是《联合国儿童权利公约》对发展能力的承认（第5条）和儿童表达意见权（第12条）、《欧洲人权公约》第8条对青年人自主权的保护，以及欧洲委员会关于儿童参与决策过程的建议——确立了基于权利的方法，倡导渐进自主而非正式解除亲权，同时承认未成年人在特定情况下可能需要法律独立。

## 韩国

### 法律框架

韩国没有专门的解除亲权法规。未成年人在达到法定成年年龄之前获得成年法律行为能力的主要机制是通过婚姻。

### 因婚姻获得行为能力（성년의제）

**《民法》第826条之2（민법 제826조의2）：** 已婚的未成年人在《民法》目的范围内视为已达成年。这意味着已婚的未成年人获得完全的合同行为能力、管理自己财产的能力以及独立采取法律行动的权利。

然而，随着2021年修正案将成年年龄从19岁降低至18岁，且第807条规定的最低结婚年龄也为18岁，该条款的实际适用已基本被消除。结婚年龄和成年年龄之间不再存在差距。

### 未成年人的有限自主权

《民法》为未成年人从事特定法律行为提供了机制：

| 条款 | 效果 |
|-----------|--------|
| 第5条第1款 | 未成年人进行法律行为须获得法定代理人的同意 |
| 第5条第2款 | 未成年人以法定代理人许可获得的财产进行的行为除外 |
| 第6条 | 未成年人可自由处分法定代理人专门指定供其处分的财产 |
| 第7条 | 经法定代理人授权从事特定营业的未成年人，就该营业范围内的行为具有成年人的行为能力 |

**第7条（授权营业）：** 该条款允许未成年人在法定代理人授权下经营业务，作为有限的行业性解除亲权发挥作用。未成年人对授权营业范围内的所有行为享有完全法律行为能力，但在其他方面仍为未成年人。

### 未成年演艺人员条款

韩国的娱乐产业涉及众多未成年表演者。《大众文化艺术产业发展法》（대중문화예술산업발전법）规范未成年演艺人员的工作条件，包括限制工作时间、强制教育要求以及防止剥削性合同的保护。虽然这不构成正式的解除亲权，但它为未成年人的经济活动创设了专门的框架。

### 宪法层面

宪法法院在多项裁决中审议了未成年人权利的范围，包括关于投票年龄（2017헌마1264）和教育权利的裁决。虽然不直接涉及解除亲权，但这些裁决反映了对未成年人宪法主体性和自主决策能力的不断发展的认识。

## 美国

### 法律框架

美国在所比较的司法管辖区中拥有最发达的解除亲权框架。解除亲权可通过以下方式实现：

### 法定解除亲权（司法申请）

约40个州设有司法解除亲权的法定条款。通常要求包括：

| 要求 | 典型条款 |
|-------------|-------------------|
| 最低年龄 | 14-16岁（因州而异） |
| 经济自立 | 未成年人须证明有能力养活自己 |
| 最佳利益 | 解除亲权须符合未成年人的最佳利益 |
| 成熟度 | 未成年人须展示足够的成熟度 |
| 居住安排 | 通常要求独立生活 |
| 父母通知 | 必须通知；父母同意可能相关但通常非必须 |

**加利福尼亚州示例（Cal. Fam. Code Sections 7120-7135）：**
- 最低年龄：14岁
- 未成年人须自愿独立于父母居住
- 未成年人须自行管理财务事务
- 收入须合法
- 解除亲权不得违背未成年人的最佳利益

### 自动解除亲权

| 事件 | 效果 | 普遍性 |
|-------|--------|-------------|
| 婚姻 | 完全解除亲权 | 几乎所有州 |
| 服兵役 | 完全解除亲权 | 所有州 |
| 达到法定成年年龄 | 自动生效 | 普遍适用 |

### 普通法解除亲权

即使在没有具体法规的州，普通法原则也可以基于以下情况确立解除亲权：
- 父母同意（明示或默示）
- 未成年人独立生活并自行管理事务
- 父母遗弃或未履行抚养义务
- 父母权力与未成年人的情况不相容

### 法律后果

| 权利/义务 | 解除亲权的效果 |
|-----------------|----------------------|
| 合同行为能力 | 获得签订具有约束力的合同的完全能力 |
| 医疗同意 | 有权同意自己的医疗 |
| 住所 | 有权设立自己的住所 |
| 父母抚养 | 父母抚养义务通常终止 |
| 侵权责任 | 父母侵权责任通常终止 |
| 教育 | 义务教育法律可能仍然适用 |
| 投票 | 不降低18岁以下投票年龄 |
| 酒精/烟草 | 不降低法定购买年龄 |
| 刑事管辖权 | 不影响刑事管辖权年龄 |

### 医疗解除亲权

与一般解除亲权分开，许多州为未成年人在指定情况下无需父母参与即可同意医疗提供了专门的法定授权：

| 情形 | 允许的州 |
|-------------|-------------------|
| 性传播感染 | 所有50个州+哥伦比亚特区 |
| 避孕服务 | 26个州+哥伦比亚特区 |
| 产前护理 | 32个州+哥伦比亚特区 |
| 物质滥用治疗 | 大多数州 |
| 心理健康服务 | 差异较大 |

### 重要案件

**In re Snyder, 85 Wash.2d 182 (1975)：** 华盛顿州最高法院确立了解除亲权需要证明实际独立和自立的原则，不能仅基于摆脱父母控制的愿望。

**Parker v. Stage, 43 N.Y.2d 128 (1977)：** 纽约上诉法院裁定，未成年人的婚姻不会自动终止父母的抚养义务，除非未成年人通过自立实际解除了亲权。

## 德国

### 法律框架

德国没有正式的解除亲权机制。相反，《民法典》（BGB）下的渐进行为能力制度基于年龄提供逐步的自主权：

### 渐进行为能力

| 年龄 | 法律地位 | BGB条款 |
|-----|-------------|---------------|
| 7岁以下 | Geschäftsunfähig（无行为能力） | 第104条第1款 |
| 7-17岁 | Beschränkt geschäftsfähig（限制行为能力） | 第106条 |
| 18岁以上 | Voll geschäftsfähig（完全行为能力） | 第2条 |

### 限制行为能力机制

**零花钱条款（第110条BGB）：** 未成年人未经父母同意订立的合同，若未成年人以为此目的或供其自由处分而给予的金钱完全履行了义务，则该合同有效。这一"零花钱条款"允许未成年人在其可用资金范围内进行日常购买。

**第112条BGB（授权经营）：** 经法定代理人和家庭法院同意，未成年人可被授权经营商业业务。对于该业务范围内的所有交易，未成年人具有成年人的法律行为能力。该条款与韩国的第7条相类似。

**第113条BGB（授权就业）：** 经法定代理人同意，未成年人可订立劳动关系，并独立行使该劳动关系产生的权利和义务。该条款使未成年人无需父母参与每项交易即可工作和管理其就业所得。

### 结婚年龄与解除亲权

在2017年《打击童婚法》（Gesetz zur Bekämpfung von Kinderehen）之前，未成年人在16岁时可经家庭法院批准结婚（BGB旧版第1303条）。婚姻产生事实上的解除亲权效果，尽管并未被正式表述为解除亲权。2017年的法律将最低结婚年龄无例外地设定为18岁，完全消除了这一途径。

### 青少年保护法

《青少年保护法》（Jugendschutzgesetz）和《青少年劳动保护法》（Jugendarbeitsschutzgesetz）为未成年人的公共活动和就业创设了专门框架，在受规制的范围内提供有限的自主权，而非完全解除亲权。《青少年劳动保护法》允许从13岁（轻度劳动）和15岁（有限制的正常就业）开始工作。

### 宪法框架

《基本法》第6条第2款宣布对儿童的照顾和教育是父母的自然权利和义务。这一宪法条款将父母权力确立为基本权利，使正式的解除亲权机制在宪法上变得复杂——任何终止父母权力的制度都必须与这一宪法保障相协调。

## 欧洲人权公约与欧盟

### 联合国儿童权利公约

**第5条（发展能力）：** 缔约国应尊重父母的责任、权利和义务，按照儿童各项能力的发展，为儿童行使公约所确认的权利提供适当的指导和引导。这一原则支持渐进式自主权而非二元化的解除亲权。

**第12条（表达意见权）：** 能够形成自己意见的儿童有权在所有影响其本人的事项中自由表达这些意见，其意见应根据儿童的年龄和成熟程度给予适当重视。

**第32条（经济剥削）：** 缔约国承认儿童有权受到保护，免遭经济剥削和从事任何可能有害的工作，并确定最低就业年龄。

### 《欧洲人权公约》第8条

欧洲人权法院在多种情境中处理了未成年人的自主权问题：

**Gillick v. West Norfolk and Wisbech Area Health Authority [1986] AC 112：** 虽然是英国案件而非欧洲人权法院案件，但Gillick行为能力原则（即具有充分理解力和智力的未成年人可以在未经父母同意的情况下同意医疗）已影响了欧洲人权法院的法理，并在欧洲委员会成员国中得到广泛认可。

**A.S. v. Norway (Application No. 60371/15, April 17, 2019)：** 法院在寄养背景下审理了一名14岁青少年就与亲生父母联系的决定参与权的范围，强调了发展能力原则。

### 欧洲委员会建议

**Recommendation CM/Rec(2012)2关于18岁以下儿童和青年人的参与：**
- 应将儿童视为积极的权利主体
- 参与应适合年龄并适应儿童的发展能力
- 儿童的意见应根据其年龄和成熟程度给予适当重视

**Recommendation Rec(2005)5关于寄宿机构中儿童的权利：**
- 机构中的儿童应享有与其年龄和成熟程度相符的最大可能自主权
- 应支持向独立生活的过渡

### 欧洲各国的比较方法

| 国家 | 解除亲权机制 | 年龄 |
|---------|----------------------|-----|
| 法国 | 法院裁定解除亲权（Code civil第413-2条） | 16岁 |
| 西班牙 | 父母同意、婚姻或法院解除亲权（Código Civil第314-324条） | 16岁 |
| 意大利 | 无正式解除亲权；经法院授权16岁可结婚 | 16岁 |
| 葡萄牙 | 因婚姻解除亲权 | 16岁 |
| 荷兰 | 无正式解除亲权；18岁可结婚（经法院批准16岁可结婚） | 不适用 |

法国的制度值得注意：法院可以应父母任何一方的申请，在考虑未成年人的成熟度和利益后，解除16岁或以上未成年人的亲权。被解除亲权的未成年人获得完全法律行为能力，包括从事商业活动的权利（需额外法院授权）。

## 比较分析

### 正式解除亲权与功能性解除亲权

美国提供了最发达的正式解除亲权框架，约40个州可通过司法申请实现。韩国和德国没有正式的解除亲权机制，但通过具体的法律条款（授权营业、就业授权、零花钱规则）达到功能等同效果。《联合国儿童权利公约》的发展能力原则支持德国和韩国采用的渐进方法，而非美国解除亲权的二元方法。

### 婚姻作为解除亲权途径

历史上，三个司法管辖区均将婚姻作为未成年人获得成年法律行为能力的途径。随着最低结婚年龄提高至18岁（德国2017年，韩国2021年）以及成年年龄降低至18岁，这一途径已被有效关闭。美国保留了因婚姻解除亲权的制度，但数个州也已将结婚年龄提高至18岁，缩小了其实际适用范围。

### 保护框架与自主框架

各司法管辖区反映了保护未成年人与促进其自主权之间的矛盾。美国解除亲权法律优先赋予成熟、自立的未成年人自主权，而德国和韩国法律则优先通过渐进行为能力制度进行保护。欧洲人权公约和《联合国儿童权利公约》寻求两者的平衡，要求尊重未成年人的发展能力同时确保充分的保护。

### 实际意义

随着成年年龄趋于统一为18岁，解除亲权的实际意义在所有司法管辖区均已减弱。美国解除亲权的主要剩余适用情形涉及逃离虐待或忽视家庭的未成年人、寻求经济独立的未成年演艺人员或运动员，以及怀孕或育有子女的未成年人。在韩国和德国，类似需求通过具体的法定条款而非一般性的解除亲权来满足。
