## 摘要

政党规制——规范政党的设立、登记、内部治理、经费和解散的法律框架——反映了关于有组织的政治活动、民主治理和国家权力之间关系的根本性选择。韩国依据1962年的《政党法》(정당법)及其大量修正案对政党进行规制，该法施加了重大组织要求，包括须在至少五个广域市/道设立地区分部、最低党员人数门槛以及向中央选举管理委员会(중앙선거관리위원회)登记，同时宪法法院依据宪法第8条第4款保留解散威胁民主基本秩序的政党的专属权力，该权力在2014年解散统合进步党(통합진보당)时首次也是唯一一次行使。美国在成熟民主国家中独特地没有综合性的联邦政党规制法规，政党主要通过各州选票准入法、《联邦竞选法》(FECA)的竞选资金条款以及保护政党第一修正案结社自由的宪法判例进行规制，最高法院在California Democratic Party v. Jones, 530 U.S. 567 (2000)和Tashjian v. Republican Party of Connecticut, 479 U.S. 208 (1986)等判决中以侵犯结社自由为由推翻了众多州级规定。德国政党规制以《基本法》第21条为宪法锚点，该条独特地承认政党是民主治理中宪法必要的制度，1967年的《政党法》(PartG)实施了关于党内民主(innerparteiliche Demokratie)、财务透明度的详细要求，联邦宪法法院(BVerfG)对认定违宪的政党拥有专属禁止管辖权，而欧盟框架（包括《关于欧洲政党的条例》和威尼斯委员会《政党规制指南》(CDL-AD(2010)024)）以及欧洲人权法院根据第11条（结社自由）和第一号议定书第3条（自由选举权）的判例法确立了：解散政党是最后手段措施，须有令人信服的证据证明对民主的紧迫威胁，且须与所追求的目的相称。

## 韩国

### 法律框架

韩国的政党规制在民主国家中属于最为严格的，反映了威权主义滥用政党的历史经验以及来自朝鲜的持续安全威胁。

**《政党法》(정당법)：**

| 条款 | 要求 |
|---------|-------------|
| 第3条 | 定义：公民自愿参与政治意志形成的组织 |
| 第4条 | 向中央选管会登记；须提供详细组织信息 |
| 第17条 | 须在5个以上广域市/道设立分部 |
| 第18条 | 每个分部须有1000名以上党员 |
| 第25条 | 须至少每年召开一次党员大会或代表大会 |
| 第29条 | 党内民主：通过民主程序选举领导层 |
| 第38条 | 未满足登记要求时由中央选管会解散 |
| 第44条 | 威胁民主基本秩序时由宪法法院强制解散 |

**宪法框架：**

**宪法第8条：**
1. "政党的设立是自由的，多党制受到保障。"
2. "政党在其目标、组织和活动上须具有民主性，且须具备必要的组织安排使人民能够参与政治意志的形成。"
3. "政党依法律规定享受国家保护，并可依法律规定的条件获得国家运营资金。"
4. "政党的目的或活动违反民主基本秩序时，政府可向宪法法院提起解散诉讼，该政党须依宪法法院的决定予以解散。"

### 统合进步党解散(2014)

**宪法法院裁决2013HunDa1（2014年12月19日）：**

统合进步党(통합진보당, UPP)的解散是韩国宪政史上首次也是唯一一次政党解散：

| 要素 | 认定 |
|---------|---------|
| 理由 | UPP的目标和活动违反民主基本秩序 |
| 证据 | 讨论战时起义的秘密会议；推广朝鲜意识形态（주체사상，主体思想） |
| 表决 | 8比1（法官金二洙持异议） |
| 效果 | 政党解散；5名国会议员丧失议席 |
| 比例性 | 法院认定鉴于对民主的生存性威胁，解散具有相称性 |

该裁决在国内外均引发争议：
- 威尼斯委员会对证据范围和解散的比例性表示关切
- 国际特赦组织批评该裁决不成比例
- 支持者认为这是保护民主免遭内部颠覆的必要之举

### 政党体制

韩国政党体制的特点是政党频繁创建、合并和解散：
- 主要政党经常解散后以新名称重组
- 以个人派系为基础的政治而非意识形态的持久性
- 执政党多次更名（如新国家党 -> 自由韩国党 -> 国民力量党）
- 这种流动性反映了韩国政党与西方民主国家相比制度化程度相对较弱

## 美国

### 法律框架

美国的独特之处在于主要将政党视为私人自愿结社而非需要详细规制的公共机构。

**宪法框架：**
- **第一修正案：** "国会不得制定法律……剥夺人民和平集会的权利"
- 政党作为结社组织享有强有力的第一修正案保护
- 无联邦层面规范政党设立、结构或内部治理的法定框架

**州级规制：**
各州主要通过选票准入法对政党进行规制：

| 要求 | 典型条款 |
|------------|-------------------|
| 政党承认 | 在上次选举中达到得票门槛（通常为2-5%） |
| 新政党资格 | 征集签名（注册选民的0.5-3%） |
| 初选选举 | 州规定的初选选举用于候选人提名 |
| 内部治理 | 州对此要求极少；政党自行制定规则 |

### 最高法院判例

**California Democratic Party v. Jones, 530 U.S. 567 (2000)：** 法院推翻了加利福尼亚州的全面初选制度（允许任何选民参与任何政党的初选），裁定其侵犯了政党选择自身候选人的第一修正案权利。

**Tashjian v. Republican Party of Connecticut, 479 U.S. 208 (1986)：** 法院推翻了禁止政党向无党派选民开放初选的州法律，裁定政党有权界定自身的成员资格和参与标准。

**Timmons v. Twin Cities Area New Party, 520 U.S. 351 (1997)：** 法院维持了各州禁止联合提名候选（多个政党提名同一候选人）的法律，赋予各州维护两党制稳定性的裁量空间。

**Arkansas Educational Television Commission v. Forbes, 523 U.S. 666 (1998)：** 法院维持了公共广播机构将小党候选人排除在电视辩论之外的做法，认定不构成第一修正案的违反。

### 两党制

美国拥有根深蒂固的两党制，维系因素包括：
- 单议席多数制(FPTP)选举（杜维尔杰定律）
- 各州对第三党的选票准入壁垒
- 总统辩论委员会的门槛（全国民调15%）
- 有利于主要政党候选人的竞选资金法律
- 加强两大政党安全议席的选区划分不公

**无联邦解散权：** 美国从未解散或试图解散政党。该概念被认为与第一修正案价值根本不相容。

## 德国

### 法律框架

德国的政党规制是世界上宪法层面最为发达的，反映了《基本法》为防止魏玛共和国失败重演的审慎制度设计。

**《基本法》第21条：**
1. "政党参与人民政治意志的形成。政党可自由设立。其内部组织须符合民主原则。须公开披露其资产及资金和财产的来源和用途。"
2. "以其目标或追随者行为旨在损害或废除自由民主基本秩序或危及德意志联邦共和国存在的政党为违宪。"
3. [2017年增加]"以其目标或追随者行为旨在损害或废除自由民主基本秩序或危及德意志联邦共和国存在的政党，应被排除出国家资助。"
4. "联邦宪法法院就第2款含义内的违宪问题和第3款含义内的排除国家资助问题作出裁决。"

**《政党法》(PartG)：**

| 条款 | 要求 |
|---------|-------------|
| 第1条 | 政党的宪法地位和功能 |
| 第2条 | 定义：持续影响政治意志形成的公民结社 |
| 第6条 | 须有党章和纲领 |
| 第7条 | 组织：须自下而上组织；地域结构 |
| 第9条 | 全体大会(Mitglieder-/Parteitag)为最高机关 |
| 第15条 | 党员在内部决策中享有平等投票权 |
| 第17条 | 须设党内仲裁(Parteischiedsgerichte) |
| 第23条 | 向联邦议院议长提交年度财务报告(Rechenschaftsbericht) |
| 第31c条 | 违反财务报告义务的制裁 |

### 党内民主(Innerparteiliche Demokratie)

党内民主组织的要求是德国独有的：
- **党员参与：** 党代会须为最高决策机构
- **秘密投票：** 领导层选举须以秘密投票方式进行
- **平等权利：** 所有党员须有平等投票权
- **少数派保护：** 党内少数派享有程序性权利
- **仲裁：** 内部争端须由党内仲裁法庭裁决

### 政党禁止和资金排除

**NPD禁止程序：**

| 程序 | 年份 | 结果 |
|-----------|------|---------|
| 第一次NPD禁止申请 | 2003 | 因NPD领导层中广泛使用政府线人(V-Leute)而终止(BVerfGE 107, 339) |
| 第二次NPD禁止申请 | 2017 | 申请被驳回：NPD被认定追求违宪目标但缺乏实现目标的"潜能"(BVerfGE 144, 20) |
| NPD排除出国家资助 | 2024 | 联邦宪法法院依据《基本法》第21条第3款将NPD（已更名为"Die Heimat"）排除出国家资助，期限6年 |

**BVerfGE 144, 20 (2017, NPD II)：** 法院引入了"潜能性"标准，裁定禁止政党不仅要求违宪目标，还要求有具体证据证明该政党有能力实现这些目标。由于NPD已衰退至约1%的得票率且不占据任何重要政治职位，法院认定其缺乏危及民主的潜能。然而，法院明确认定NPD的目标为违宪，从而为2024年依据新增的《基本法》第21条第3款排除其国家资助铺平了道路。

### 德国选择党(AfD)与宪法保护

联邦宪法保护局(Bundesamt fuer Verfassungsschutz, BfV)已将AfD的部分组织列为"疑似右翼极端主义"(Verdachtsfall)，并将某些州级组织列为"确认的右翼极端主义"(gesichert rechtsextremistisch)。多个州的宪法保护机构也施加了类似分类。截至2026年，关于可能提出AfD禁止申请的政治讨论仍在继续，但尚未向联邦宪法法院提交正式申请。

## 欧洲人权法院/欧盟

### 欧洲人权法院关于政党解散的判例

**Refah Partisi (The Welfare Party) and Others v. Turkey（申诉号41340/98, 41342/98, 41343/98, 41344/98，大法庭，2003年2月13日）：** 欧洲人权法院关于政党解散的里程碑案件。法院维持了土耳其解散繁荣党（当时该国最大政党）的决定，认定未违反第11条：

| 要素 | 法院认定 |
|---------|----------------|
| 干预 | 是，解散构成对结社自由的干预 |
| 法律规定 | 是，土耳其宪法法院有法律授权 |
| 合法目的 | 是，保护民主和他人权利 |
| 在民主社会中的必要性 | 是，鉴于领导人关于实施伊斯兰法律、法律多元主义和使用暴力的声明 |
| 比例性 | 是，鉴于紧迫威胁，解散具有相称性 |

法院引入了"能够自我防御的民主"（防卫性民主）原则，裁定国家可以在反民主政党有机会实施其反民主纲领之前将其解散以保护民主价值。

**United Communist Party of Turkey and Others v. Turkey（申诉号19392/92，1998年1月30日）：** 法院认定违反第11条，土耳其仅凭一个政党的名称和纲领将其解散，无证据表明该党从事或倡导任何威胁民主秩序的活动。

**Herri Batasuna and Batasuna v. Spain（申诉号25803/04，2009年6月30日）：** 法院认定解散两个与ETA有关联的巴斯克政党未违反第11条，裁定解散因该等政党对恐怖主义的支持而具有正当性。

**Republican Party of Russia v. Russia（申诉号12976/07，2011年4月12日）：** 法院认定违反第11条，俄罗斯以据称违反登记要求（最低地区分部和党员人数）为由解散一个政党，认定这些要求不成比例且解散在民主社会中并非必要。

### 威尼斯委员会指南

CDL-AD(2010)024《政党规制指南》确立了全面标准：

| 原则 | 标准 |
|-----------|----------|
| 设立 | 自由设立；最低限度的登记要求 |
| 党内民主 | 鼓励但非强制；国家规制应具比例性 |
| 资金 | 公共和私人资金平衡；透明度；独立监督 |
| 解散 | 仅可通过司法裁决；最后手段；相称；令人信服的证据 |
| 平等对待 | 国家须平等对待所有政党；不得歧视 |

### 欧盟政党规制

**第(EU, Euratom) 1141/2014号条例：**
- 欧洲政党须向欧洲政党管理机构(APPF)登记
- 要求：在一个成员国具有法人资格；在至少7个成员国有代表；尊重欧盟价值观
- 注销：APPF可对明显且严重违反欧盟价值观（《欧盟条约》第2条）的政党予以注销
- 2024年改革加强了透明度和反外国干预条款

## 比较分析

### 政党的宪法地位

各法域对政党的宪法处理差异巨大：
- **德国：** 明确承认为宪法必要的制度（《基本法》第21条），附有详细的宪法要求
- **韩国：** 宪法承认设立自由和解散权（第8条）
- **美国：** 视为享有第一修正案保护的私人结社；宪法中未提及
- **欧洲人权法院：** 受结社自由保护（第11条）；国家享有较大的裁量余地

### 防卫性民主

"防卫性民主"(streitbare Demokratie / wehrhafte Demokratie)概念——即民主国家可限制反民主力量以保护自身的原则——在各法域有不同体现：
- **德国：** 明确的政党禁止宪法框架（《基本法》第21条第2款）；最为发达的防卫性民主理论
- **韩国：** 宪法解散权（第8条第4款）；2014年行使
- **欧洲人权法院：** 在Refah Partisi案中认可；解散作为最后手段，须严格符合比例性
- **美国：** 无防卫性民主理论；政党不可被解散；依赖选举竞争和思想市场

### 党内民主要求

| 法域 | 是否要求党内民主 | 执行机制 |
|-------------|---------------------------|-------------|
| 德国 | 是（宪法要求） | 联邦宪法法院；党内仲裁法庭 |
| 韩国 | 是（法定要求） | 中央选管会监督 |
| 美国 | 否（政党自行制定规则） | 第一修正案限制州的规制 |
| 欧洲人权法院/威尼斯委员会 | 建议但非强制 | 国家自行裁量 |

### 数字时代对政党规制的挑战

社交媒体和数字竞选为政党规制带来了新挑战：

- **虚假信息：** 所有法域在规制与政党关联的虚假信息网络方面面临困难；德国的《网络执行法》(NetzDG, 2017)和欧盟的《数字服务法》对平台施加义务，但政党本身在网络言论方面面临的直接规制有限
- **微定向投送：** 通过算法定向的社交媒体政治广告引发透明度和操纵担忧；欧盟拟议中的《政治广告条例》(2024)要求披露所有在线政治广告的定向标准和资金来源
- **深度伪造和AI生成内容：** 欧盟《人工智能法》(2024)将生成政治内容的AI系统列为高风险；韩国修订《公职选举法》禁止选举前90天内的AI生成深度伪造；美国缺乏全面的联邦规制
- **加密货币捐赠：** 联邦选举委员会允许比特币及其他加密货币捐赠，适用标准捐赠限额；大多数其他法域尚未明确规定加密货币政治捐赠

### 极端政党与宽容的限度

对极端政党的处理体现了民主多元主义与民主自我防御之间的张力：

- **德国：** 联邦宪法法院2017年NPD裁决(2 BvB 1/13)在解散与无为之间创造了中间路径，确认该政党目标违宪但因缺乏实现目标的"潜能性"(Potentialitaet)而拒绝禁止；随后的《基本法》第21条第3款修正案允许排除国家资助，代表了一种精准的回应
- **韩国：** 2014年解散统合进步党(2013HunDa1)是韩国民主史上首次也是唯一一次政党解散；宪法法院要求有证据证明该政党对民主秩序构成具体危险
- **欧洲人权法院：** 法院发展了渐进的比例性框架：仅在政党构成"可信的"威胁且较轻限制性措施不足以应对时，解散方为可容许(Herri Batasuna v. Spain, 2009)
