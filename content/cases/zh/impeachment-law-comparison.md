## 概述

弹劾——通过立法机关的弹劾投票和随后的审判来罢免国家元首或其他高级公职人员因严重不当行为而去职的宪法机制——是对行政权力最重要的制约手段之一，运作于宪法、刑法和民主治理的交汇点。韩国的弹劾程序根据《宪法》第65条和第111条以及《宪法法院法》（헌법재판소법）建立，需要获得国会议员多数支持的动议（总统弹劾需三分之二），由宪法法院（헌법재판소）裁决，罢免需六名大法官的绝对多数同意，并已两次对在任总统行使：2004年对卢武铉总统的弹劾未获通过（宪法法院裁决2004HunNa1）和2017年对朴槿惠总统的弹劾成功并予以罢免（宪法法院裁决2016HunNa1），后者成为全球宪法法中的里程碑，证明了在任总统可以通过制度化机制和平地被罢免。美国宪法在第一条和第二条中建立了弹劾制度，授予众议院唯一的弹劾权（第一条第二款第五项）和参议院唯一的审判弹劾案权（第一条第三款第六至七项），参议院定罪需三分之二绝对多数，适用于"叛国罪、贿赂罪或其他重罪和轻罪"（第二条第四款），并曾对总统安德鲁·约翰逊（1868年，无罪）、比尔·克林顿（1998年，无罪）和唐纳德·特朗普两次行使（2019年，无罪；2021年，无罪）。德国不使用"弹劾"一词，但根据《基本法》第61条规定了通过总统弹劾诉讼（Präsidentenanklage）罢免总统的程序，由联邦议院或联邦参议院就故意违反《基本法》或其他联邦法律的行为向联邦宪法法院（Bundesverfassungsgericht, BVerfG）提起指控，这一机制在联邦共和国历史上从未使用过，因为德国总统的角色主要是礼仪性的，而欧盟框架通过各条约中的罢免程序处理欧盟机构领导人的问责问题，ECHR关于公正审判权（第6条）、言论自由（第10条）和选举权（第1号议定书第3条）的判例法确立了弹劾程序必须尊重基本程序保障。

## 韩国

### 法律框架

韩国的弹劾机制的独特之处在于将裁判权赋予宪法法院而非立法机关本身。

**宪法第65条（弹劾）：**

| 条款 | 内容 |
|-----------|---------|
| 第65条第1款 | 可弹劾的公职人员：总统、国务总理、国务委员、各部部长、宪法法院大法官、法官、中央选举管理委员会委员、监查院院长和监查委员及法律规定的其他公职人员 |
| 第65条第2款 | 动议：由国会议员三分之一以上提出；多数通过（总统：由多数提出；三分之二通过） |
| 第65条第3款 | 停职：被弹劾动议通过的人在判决前停止行使职权 |
| 第65条第4款 | 效力：免除职务；不免除民事或刑事责任 |

**《宪法法院法》（헌법재판소법）：**
- 第48至54条规定弹劾审判程序
- 由全体大法官（9名）裁判；罢免需6票以上
- 口头辩论；证据审查；证人证词
- 未规定时间限制，但宪法法院一贯优先审理弹劾案件

### 卢武铉弹劾案（2004年）

**宪法法院裁决2004HunNa1（2004年5月14日）：**

| 要素 | 详情 |
|---------|--------|
| 国会投票日期 | 2004年3月12日（在野党多数，193票对2票） |
| 指控事由 | 非法竞选拉票；违反《政治资金法》；不称职 |
| 法院裁决 | 弹劾驳回；总统复职 |
| 裁判理由 | 虽然总统发表党派性言论违反了选举法，但违规行为的严重程度不足以构成罢免理由；比例原则要求罢免仅适用于"严重"违规 |
| 意义 | 确立了"严重性"标准：并非所有法律违反都构成罢免理由；违规的严重程度必须与罢免这一极端后果相称 |

### 朴槿惠弹劾案（2016-2017年）

**宪法法院裁决2016HunNa1（2017年3月10日）：**

| 要素 | 详情 |
|---------|--------|
| 国会投票日期 | 2016年12月9日（234票对56票） |
| 事由 | 滥用总统权力；违反宪法；涉及崔顺实的腐败 |
| 法院裁决 | 弹劾成立；总统被罢免 |
| 投票结果 | 8票对0票（全票通过；第9个席位空缺） |
| 主要认定 | 总统违反保密义务，向崔顺实泄露机密文件；滥用总统权力胁迫企业向崔顺实控制的基金会捐款；在世越号渡轮灾难中未能保护公民的生命权；向公众和公职人员隐瞒事实 |
| 意义 | 韩国历史上首次成功的总统弹劾；展示了通过宪法机制实现和平民主权力交接 |

**后续发展：**
- 2017年5月9日举行总统紧急选举（文在寅当选）
- 朴槿惠随后被判贿赂罪、滥用权力罪和胁迫罪，判处20年（后上诉加刑至25年；2021年12月31日获赦免）
- 崔顺实被判处20年（上诉减刑至18年）

### 尹锡悦弹劾案（2024年）

尹锡悦总统于2024年12月14日被国会弹劾，此前他于2024年12月3日短暂宣布戒严。截至2026年初，宪法法院的审理程序仍在进行中，总统停止行使职权，由国务总理代行总统职责。

## 美国

### 法律框架

美国宪法建立了两阶段的弹劾程序：众议院弹劾（指控）后进入参议院审判。

**宪法条款：**

| 条款 | 内容 |
|-----------|---------|
| 第一条第二款第五项 | "众议院……应享有弹劾的唯一权力" |
| 第一条第三款第六项 | "参议院应享有审判一切弹劾案的唯一权力……非经出席议员三分之二的同意，不得定罪" |
| 第一条第三款第七项 | 判决：免职及可能取消担任公职资格；无其他法律后果 |
| 第二条第四款 | "总统、副总统和合众国的所有文职官员，因叛国罪、贿赂罪或其他重罪和轻罪被弹劾和定罪时，应予免职" |
| 第二条第二款第一项 | 总统的赦免权不适用于弹劾案 |

### 总统弹劾案

**安德鲁·约翰逊（1868年）：**
- 因违反《任期法》解除战争部长埃德温·斯坦顿而被众议院弹劾（1868年2月24日）
- 参议院审判：35票对19票宣判无罪（差一票未达三分之二多数）
- 意义：确立了国会控制行政人事的权力界限

**理查德·尼克松（1974年）：**
- 众议院司法委员会通过三项弹劾条款（1974年7月），指控妨碍司法公正、滥用权力和蔑视国会（水门事件）
- 尼克松于1974年8月9日在众议院全体投票前辞职
- 意义：证明弹劾威胁可以在无正式审判的情况下实现问责

**比尔·克林顿（1998-1999年）：**
- 因两项指控被众议院弹劾（1998年12月19日）：在大陪审团前作伪证和妨碍司法公正（与莫尼卡·莱温斯基丑闻相关）
- 参议院审判：两项指控均宣判无罪（伪证罪45票对55票；妨碍司法公正罪50票对50票；需67票定罪）
- 意义：凸显了个人不当行为是否构成"重罪和轻罪"的争论

**唐纳德·特朗普——第一次弹劾（2019-2020年）：**
- 因两项指控被众议院弹劾（2019年12月18日）：滥用权力和妨碍国会（与施压乌克兰有关）
- 参议院审判：两项指控均宣判无罪（滥用权力48票对52票；妨碍国会47票对53票）
- 参议员米特·罗姆尼就滥用权力投票定罪——历史上首次有参议员投票定罪本党总统

**唐纳德·特朗普——第二次弹劾（2021年）：**
- 因一项指控被众议院弹劾（2021年1月13日）：煽动叛乱（2021年1月6日国会大厦袭击事件）
- 参议院审判：57票对43票宣判无罪（差10票未达67票）
- 7名共和党参议员投票定罪
- 意义：首位被弹劾两次的总统；提出了前总统是否可以受审的问题（参议院以56票对44票裁定可以）

### "重罪和轻罪"

"重罪和轻罪"的含义仍有争议：
- **宽泛解释（汉密尔顿，《联邦党人文集》第65篇）：** "源于公职人员不当行为的犯罪，换言之，源于对某种公共信任的滥用或违反"
- **狭义解释：** 要求存在可起诉的犯罪行为
- **历史实践：** 犯罪行为（特朗普：煽动叛乱）和非犯罪性的权力滥用（特朗普：乌克兰事件；约翰逊：违反《任期法》）均曾被指控

### 可诉性

**Nixon v. United States, 506 U.S. 224 (1993)：** 最高法院裁定，参议院"审判"弹劾案的权力属于宪法文本明确委托给参议院的事项，因此对参议院审判程序的质疑构成不可诉的政治问题。

## 德国

### 法律框架

德国的总统弹劾机制（Präsidentenanklage）从未被使用过，反映了德国总统主要为礼仪性角色的特点。

**《基本法》第61条：**

| 条款 | 内容 |
|-----------|---------|
| 第61条第1款 | "联邦议院或联邦参议院可就联邦总统故意违反本基本法或任何其他联邦法律的行为，向联邦宪法法院提起弹劾" |
| 第61条第1款 | 动议需联邦议院或联邦参议院四分之一成员提出；决议需三分之二多数通过 |
| 第61条第2款 | 联邦宪法法院可发出暂停总统职权的临时命令 |
| 第61条第2款 | 如认定有罪，联邦宪法法院可宣告总统丧失职位 |

**《基本法》第98条第2款——法官弹劾：**
违反《基本法》原则或各州宪法秩序的联邦法官，经联邦议院申请，可由联邦宪法法院予以调任或退休。该条款曾使用过一次（1950年代，涉及一名具有纳粹时期背景的法官）。

### 总理问责

德国制度主要通过建设性不信任投票（《基本法》第67条）而非弹劾来处理行政权力问责：
- 联邦议院只能通过以多数票选出继任者并请求总统解除现任总理职务的方式，表达对联邦总理的不信任
- 该机制成功使用过一次：1982年10月1日，赫尔穆特·科尔通过建设性不信任投票取代了赫尔穆特·施密特
- 建设性机制防止了可能造成政府不稳定的纯粹消极性不信任投票

### 部长问责

联邦部长不受弹劾。他们由总理自行决定任免（《基本法》第64条），可由总统应总理请求予以解职。议会问责通过以下机制运作：
- 质询时间（Fragestunde）
- 调查委员会（Untersuchungsausschüsse，《基本法》第44条）
- 对总理的不信任投票（罢免总理即罢免整个政府）

### 与议会制国家的比较

德国的做法反映了议会制国家的普遍模式：
- **议会制国家：** 行政问责主要通过信任投票；弹劾仅保留用于国家元首
- **总统制国家（韩国、美国）：** 弹劾是行政权力的主要问责机制，因为总统不能通过不信任投票被罢免

## ECHR/欧盟

### 欧盟问责机制

**欧盟委员会：**
- **不信任动议（TFEU第234条）：** 欧洲议会可以占投票数三分之二且代表多数欧洲议会议员的投票通过对委员会的不信任动议。如获通过，所有委员必须辞职。从未成功通过（1999年针对桑特委员会的动议未获通过，但委员会先行辞职）。
- **个别委员罢免（TFEU第247条）：** "不再符合履行职责所需条件或犯有严重不当行为"的委员，可由理事会（以简单多数表决）或委员会申请，由欧盟法院予以强制退休。2006年曾对埃迪特·克勒松使用（Case C-432/04，欧盟法院认定存在不当行为，但因时间已过而未剥夺养老金权利）。

**欧盟法院法官和佐审官：**
- 可由其他法官和佐审官一致决定罢免，条件是其"不再符合任职条件或不再履行职务义务"（《欧盟法院规约》第6条）。从未使用过。

### ECHR判例法

ECHR在若干重要案件中涉及了与弹劾相关的问题：

**Paksas v. Lithuania（申请号34932/04，大审判庭，2011年1月6日）：** 法院认定，立陶宛法律在帕克萨斯总统因严重违宪而被弹劾罢免后永久禁止其参选，违反了第1号议定书第3条（参选权）。法院认为，该取消资格的永久性和不可逆性不成比例，指出即使违宪行为的严重程度也不足以证明终身禁止参加选举的合理性。

**Selahattin Demirtaş v. Turkey (No. 2)（申请号14305/17，大审判庭，2020年12月22日）：** 虽然严格来说不是弹劾案，但法院审理了一名政治领袖（人民民主党联合主席）的议会豁免权被取消和刑事起诉问题，认定违反了公约第5、10和18条以及第1号议定书第3条。该判决确立了利用刑事程序压制政治反对派有损民主多元主义。

**Merabishvili v. Georgia（申请号72508/13，大审判庭，2017年11月28日）：** 法院审理了一名前总理被拘留的案件，认定违反了第18条（限制对权利的限制的使用）结合第5条，确立了评估针对政治人物的刑事程序是否服务于隐性政治目的的框架。

### 弹劾程序标准

ECHR判例法确立了适用于弹劾的若干原则：

1. **公正审判（第6条）：** 虽然弹劾主要是政治程序，但在涉及民事权利或刑事指控的确定时，第6条的保障适用。法院认为具有准司法性质的议会程序必须提供充分的程序保障。

2. **言论自由（第10条）：** 因受保护的政治言论而对官员进行弹劾引发第10条的关切。合法的政治问责与惩罚言论之间的界限必须谨慎划定。

3. **选举权（第1号议定书第3条）：** 弹劾后的取消任职资格必须符合比例原则（Paksas v. Lithuania）。永久性、不可撤销的禁令被推定为不成比例。

4. **禁止滥用（第18条）：** 将弹劾作为消除政治对手而非处理真正不当行为的工具，可能违反第18条。

## 比较分析

### 制度设计

四个司法管辖区反映了三种不同的弹劾裁判模式：

**宪法法院模式（韩国、德国）：** 由专门的宪法法院裁判弹劾，适用宪法和法律标准。这一模式将政治性的指控行为与法律性的裁判行为分离，提供了最大程度的程序独立性。

**立法审判模式（美国）：** 参议院同时充当政治机构和事实裁判者，总统审判由首席大法官主持。这一模式使弹劾具有内在的政治性，因为参议员既是陪审员又是政治行为者。

**欧盟混合模式：** 对不同官员采用不同机制（议会不信任动议针对委员会；欧盟法院裁判针对个别委员和法官）。

### 罢免事由

| 司法管辖区 | 标准 | 宽窄程度 |
|-------------|----------|---------|
| 韩国 | "在履行职务中违反宪法或法律" | 宽泛；任何法律违反均可（但根据2004年先例需具"严重性"） |
| 美国 | "叛国罪、贿赂罪或其他重罪和轻罪" | 有争议；包括犯罪行为和政治性不当行为 |
| 德国 | "故意违反《基本法》或其他联邦法律" | 狭窄；要求故意（Vorsatz） |
| ECHR | 罢免及其后果的比例性 | 评估国家行为的框架 |

### 弹劾后的停职

一项关键的程序差异：
- **韩国：** 国会投票后自动停职（审判前）
- **美国：** 无自动停职；总统在审判期间继续在任
- **德国：** 法院可命令停职（裁量权）

这一差异具有深远的实际影响：在韩国，弹劾投票本身即将总统从日常治理中移除，而在美国，总统在审判期间继续行使全部权力。

### 频率与有效性

| 司法管辖区 | 使用次数（国家元首） | 成功罢免 | 评价 |
|-------------|---------------------|---------------------|------------|
| 韩国 | 3次（2004年、2017年、2024年） | 1次（2017年；2024年待决） | 活跃；有效的问责机制 |
| 美国 | 4次正式弹劾加1次辞职 | 0次定罪 | 党派极化阻碍定罪 |
| 德国 | 0次 | 0次 | 建设性不信任投票发挥主要功能 |
| 欧盟 | 0次（委员会不信任动议） | 0次（桑特委员会先行辞职） | 制度设计限制了使用 |

### 弹劾后的法律后果

弹劾后的法律后果在各司法管辖区之间存在显著差异：

- **韩国：** 宪法法院作出的成功弹劾裁决导致立即罢免；被罢免的总统可能因基础犯罪而面临刑事起诉。朴槿惠总统因腐败指控被判处25年监禁（后减至22年）；尹锡悦总统面临内乱罪刑事起诉
- **美国：** 参议院定罪导致罢免和可能的取消未来任职资格（通过单独投票）；根据双重危险条款分析，在普通法院的刑事起诉仍然可能；没有前总统在弹劾后被刑事起诉（尼克松获得总统赦免）
- **德国：** 联邦宪法法院可宣告丧失总统职位；罢免后无刑事豁免
- **ECHR：** 在Paksas v. Lithuania（2011年）案中，大审判庭裁定弹劾后永久且不可逆转地禁止担任议会职务违反了第1号议定书第3条（自由选举权），确立了弹劾后取消资格的比例限制

### 弹劾与民主稳定

弹劾机制与民主稳定之间存在复杂的互动关系：

- 频繁或党派性地使用弹劾可能动摇政府稳定并损害民主合法性
- 弹劾机制不健全可能纵容威权巩固
- 韩国的经验表明，功能健全的弹劾机制可以在维护宪法秩序的同时加强民主问责
- 美国的经验表明，党派极化可能使弹劾作为问责机制实际失效，无论基础行为如何
