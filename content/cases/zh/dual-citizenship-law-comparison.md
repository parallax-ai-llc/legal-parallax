## 摘要

双重国籍——同时持有两个或两个以上国家国籍——仍然是国籍法中差异最大的领域之一，各司法管辖区的态度从完全接受到近乎全面禁止不等。韩国根据《国籍法》（국적법）一般禁止双重国籍，但对某些类别存在例外，包括海外韩裔、具有特殊才能的人士以及韩国国民的外国配偶，此外男性公民的义务兵役又增加了额外的复杂性。美国在实践和司法先例上完全允许双重国籍，没有联邦法律要求入籍时放弃外国国籍，尽管效忠宣誓中包含正式的放弃条款，但该条款在实践中并未被强制执行。德国通过《国籍法现代化法》（Staatsangehoerigkeitsrechtsmodernisierungsgesetz, StARModG）经历了历史性转变，该法于2024年6月27日生效，取消了此前入籍时放弃外国国籍的要求，标志着与数十年来限制性双重国籍政策的根本性决裂。在欧洲层面，1997年《欧洲国籍公约》确立了允许各国自行确定其国籍规则的框架原则，同时禁止任意剥夺国籍，欧洲人权法院则通过第8条和第14条的视角处理了国籍问题。

## 韩国

### 法律框架

韩国的国籍法由《国籍法》（국적법，第16号法律）管辖，1948年首次颁布，1997年和2010年进行了实质性修订。该框架由《出入境管理法》（출입국관리법）和《兵役法》（병역법）作为补充。

| 条款 | 说明 | 效力 |
|------|------|------|
| 《国籍法》第10条 | 取得外国国籍后丧失韩国国籍 | 自动丧失，除非适用例外情形 |
| 《国籍法》第10条之2 | 选择性保留韩国国籍 | 对某些类别允许，需宣誓不在韩国境内行使外国国籍 |
| 《国籍法》第11条之2 | 多重国籍持有人的义务 | 须在规定期限前选择一个国籍 |
| 《国籍法》第12条 | 放弃韩国国籍 | 允许自愿放弃；不可撤销 |
| 《国籍法》第13条 | 国籍选择期限 | 女性在22岁前，男性在年满18岁当年的3月31日前，可申请延期 |
| 《兵役法》第70条 | 兵役期间禁止放弃国籍 | 男性从18岁起至兵役完成前不得放弃韩国国籍 |

### 主要原则

1. **一般禁止但有例外：** 作为默认规则，韩国不允许双重国籍。《国籍法》第10条规定，自愿取得外国国籍的韩国国民自动丧失韩国国籍。反之，取得韩国国籍的外国国民通常须在一年内放弃其原国籍（第10条第2款）。

2. **允许多重国籍的特定类别：** 2010年《国籍法》修正案为特定群体引入了有限的双重国籍，前提是他们宣誓不在韩国境内行使其外国国籍：
   - **韩国国民的外国配偶**（《国籍法》第6条第2款）
   - **在科学、经济、文化或体育领域具有特殊才能的人士**（第7条第1款）
   - **对大韩民国有重大贡献的人士**（第7条第1款）
   - **韩裔海外被收养者**（返回韩国的）
   - **年长的韩裔移民**（65岁以上）希望返回韩国的
   - **因本国法律不可能而无法放弃原国籍的人士**

3. **兵役问题：** 国籍法与义务兵役的交叉是一个关键问题。出生即持有双重国籍的男性韩国公民必须在年满18岁当年的3月31日前申报保留或放弃韩国国籍的意向。保留韩国国籍者须服兵役，且在兵役完成前不得放弃韩国国籍（《兵役法》第70条）。女性双重国籍持有人须在22岁前做出选择。

4. **不行使外国国籍宣誓：** 在例外情形下获准保持双重国籍的人士须做出正式宣誓（국적불행사서약），承诺不在韩国境内行使其外国国籍的权利。违反该宣誓可导致丧失韩国国籍。

### 重要案例

**韩国大法院，2016Du56080（2017年）：** 大法院审理了一名韩美双重国籍者未在法定期限内做出国籍选择的案件。法院维持了自动丧失韩国国籍的裁定，认为《国籍法》规定的法定期限具有强制性，未能遵守将导致依法丧失韩国国籍。

**韩国宪法法院，2006HeonMa462（2008年）：** 宪法法院审查了兵役义务期间禁止放弃韩国国籍（《兵役法》第70条）的合宪性。法院维持了该限制，认为这是对变更国籍权的相称限制，以保卫国家的宪法义务（《宪法》第39条）为正当理由。

**刘承俊案（2002-2024年）：** 韩国出生的艺人刘承俊于2002年在即将被征召服兵役前取得美国国籍，此举被广泛视为逃避兵役。他被永久禁止入境韩国。2020年，一审法院裁定应重新审查该入境禁令，但大法院于2024年维持了该禁令。该案已成为公众对逃避兵役和双重国籍态度的象征。

## 美国

### 联邦框架

美国允许双重国籍，无正式限制。法律框架包括：

- **《美国宪法》第十四修正案：**"凡在美国出生或归化美国并受其管辖的人，均为美国公民和其所居住州的公民。"这确立了出生地公民权（属地主义）。

- **《移民和国籍法》（INA），8 U.S.C. section 1101 et seq.：** 主要的联邦移民和国籍法规。第349条（8 U.S.C. section 1481）列举了可能导致丧失国籍的特定行为，但仅限于"以放弃美国国籍为意图"实施的情形。

- **效忠宣誓（8 U.S.C. section 1448）：** 入籍申请人须宣誓，其中包括："我在此郑重宣誓，绝对且完全放弃并弃绝对任何外国王子、元首、国家或主权的一切效忠和忠诚。"尽管有此措辞，该宣誓并不被解释为要求实际放弃外国公民身份。

| 可能导致丧失国籍的行为（8 U.S.C. section 1481） | 丧失条件 |
|--------------------------------------------------|----------|
| 在外国取得入籍 | 自愿行为 + 放弃意图 |
| 向外国宣誓效忠 | 自愿行为 + 放弃意图 |
| 在外国武装力量中服役 | 自愿行为 + 放弃意图 |
| 在外国政府任职 | 自愿行为 + 放弃意图 |
| 在美国外交官面前正式放弃 | 自愿行为（推定具有放弃意图） |
| 犯叛国罪 | 须经定罪 |

### 各州差异

虽然国籍事务根据《宪法》（第一条第八节第四款）属于联邦专属事项，但各州法律在以下领域可能受到双重国籍身份的影响：

- **陪审团服务：** 部分州要求进行公民身份核实以履行陪审义务；双重国籍公民完全符合资格。
- **州政府雇佣：** 某些州安全或执法职位可能对外国效忠有所限制，但完全禁止双重国籍公民的情况罕见且在宪法上存疑。
- **投票：** 具有美国公民身份的双重国籍公民享有完整投票权；持有外国公民身份不影响资格。
- **安全许可：** 双重国籍不自动取消获得联邦安全许可的资格，但在根据《国家安全裁定指南》进行审查时是一个考量因素。

### 重要案例

**Afroyim v. Rusk, 387 U.S. 253 (1967)：** 最高法院裁定，未经公民自愿放弃，国会不得剥夺美国公民的公民身份。贝斯·阿弗罗伊姆是一名出生于波兰的归化美国公民，他曾在以色列选举中投票。国务院拒绝为其续签护照，声称他已丧失公民身份。法院推翻了Perez v. Brownell（1958年）案，并裁定第十四修正案保护公民免受非自愿丧失公民身份的影响。

**Vance v. Terrazas, 444 U.S. 252 (1980)：** 法院澄清，丧失国籍需要同时具备可能导致丧失国籍的行为（如向外国宣誓效忠）和放弃美国国籍的特定意图。政府有责任以优势证据证明这两个要素。

**Kawakita v. United States, 343 U.S. 717 (1952)：** 河北友也是美日双重国籍公民，因在二战期间在日本实施的行为被判犯有叛国罪。法院裁定，其双重国籍并不减轻其作为美国公民的义务，双重国籍公民同时对两国负有效忠义务。

## 德国

### 法律框架

德国的国籍法随着《国籍法现代化法》（Staatsangehoerigkeitsrechtsmodernisierungsgesetz, StARModG）的生效经历了历史性转变，该法于2024年6月27日生效。主要法规为《国籍法》（Staatsangehoerigkeitsgesetz, StAG）。

**2024年前的框架：**

| 条款 | 此前的要求 |
|------|-----------|
| StAG 第10条第1款 | 入籍需要8年合法居住 |
| StAG 第10条第1款第4项 | 申请人须放弃或丧失原国籍 |
| StAG 第12条 | 对放弃要求的有限例外 |
| StAG 第29条 | "选择义务"（Optionspflicht）：在德国出生且通过出生取得德国国籍的外国父母子女须在18至23岁之间选择一个国籍 |

**2024年后的框架（StARModG）：**

| 条款 | 新要求 |
|------|--------|
| StAG 第10条第1款（修订） | 入籍仅需5年合法居住（从8年缩短） |
| StAG 第10条第1款第4项（删除） | **不再要求放弃原国籍** |
| StAG 第12条（删除） | 例外条款不再需要 |
| StAG 第29条（删除） | 选择义务完全废除 |
| StAG 第10条第3款（新增） | 具有特殊融合表现的人士可在3年后快速入籍 |
| StAG 第25条第1款（修订） | 取得外国国籍的德国公民不再自动丧失德国国籍 |

### 主要原则

1. **完全接受多重国籍（2024年起）：** StARModG代表了一次范式转变。数十年来，德国遵循避免多重国籍的原则（Vermeidung von Mehrstaatigkeit）。2024年改革明确放弃了这一原则，在对方国家也允许的前提下，允许双重（和多重）国籍。

2. **降低的居住要求：** 入籍的最低居住期限从八年缩短至五年，对于展示出特殊融合表现的人士（如高级语言能力、公民参与、职业成就），可在三年后快速入籍。

3. **废除选择义务：** 在此前的制度下，在德国出生且通过属地主义（2000年改革下）取得德国国籍的外国父母子女须在18至23岁之间选择德国国籍或其父母的国籍。这一"选择义务"（Optionspflicht）争议颇大，2014年部分放宽，2024年改革彻底废除。

4. **取消保留许可（Beibehaltungsgenehmigung）：** 此前，希望取得外国国籍的德国公民须获得保留许可（StAG第25条），以避免自动丧失德国公民身份。自2024年6月27日起，此要求已被取消——德国公民可自由取得额外国籍而无任何丧失德国公民身份的风险。

5. **对民主价值观的更高要求：** 2024年改革同时引入了关于承诺民主价值观的更严格要求。申请人须确认自由民主基本秩序（freiheitliche demokratische Grundordnung），并特别承认德国对纳粹时代的历史责任。

### 重要案例

**BVerfG, 2 BvR 669/04（2006年）：** 联邦宪法法院审查了选择义务（前StAG第29条）与《基本法》的兼容性。法院维持了该规定，但指出立法机关有义务监测其实际效果，并在必要时调整法律。其后选择义务的放宽（2014年）和废除（2024年）可被视为对此司法指导的立法回应。

**BVerwG, 1 C 37.02（2003年）：** 联邦行政法院（Bundesverwaltungsgericht）就前StAG第12条放弃要求的例外范围作出裁定。法院认为，"不可能或不合理"的例外情形须根据申请人的个人情况来解释，包括在原籍国放弃国籍的法律和实际障碍。

**2024年改革后的影响：** 在2024年6月改革生效后的头六个月内，德国报告入籍申请显著增加，特别是来自土耳其、叙利亚和其他此前因放弃要求而被阻止的国民。土耳其是德国最大移民社区的来源国，也允许双重国籍。

## 欧洲人权法院/欧盟框架

### 公约条款

《欧洲人权公约》不包含明确的国籍权，但多项条款具有相关性：

- **第8条（尊重私人和家庭生活的权利）：** 法院认为国籍问题可能属于第8条的范围，因为国籍影响个人的社会身份和家庭生活。任意拒绝或剥夺国籍可构成违反。

- **第14条（禁止歧视）：** 基于性别、种族、宗教或民族出身的国籍法歧视，与其他公约权利相结合，可能违反第14条。

- **第3条（禁止酷刑）：** 在极端情况下，剥夺国籍使人成为无国籍人并使其面临迫害的，可能涉及第3条。

- **第四号议定书第3条（禁止驱逐国民）：** 任何人不得被驱逐出其国籍国领土。该条款与面临可能丧失国籍后被驱逐的双重国籍人士相关。

### 主要法律文书

**《欧洲国籍公约》（ECN，1997年）：** 由欧洲委员会通过（ETS No. 166），但不属于欧洲人权法院体系。主要条款：

| 条款 | 原则 |
|------|------|
| 第3条 | 各国根据本国法律确定谁是其国民 |
| 第4条(a) | 人人有权享有国籍 |
| 第4条(b) | 应避免无国籍状态 |
| 第4条(c) | 任何人不得被任意剥夺国籍 |
| 第5条 | 国籍规则不得包含构成基于性别、宗教、种族、肤色或民族/族裔出身的歧视的区分 |
| 第7条 | 丧失国籍的许可事由的穷尽列举（依法或由国家主动） |
| 第14条 | 各国可允许多重国籍；不允许多重国籍的国家须在婚姻或出生情形下允许 |
| 第15条 | 各国不得在放弃另一国籍不可能或不合理时要求放弃 |
| 第16条 | 另一国的国籍不得作为取得或保留国籍的条件 |

### 重要判例法

**Ramadan v. Malta（2016年），申请号 76136/12：** 法院审理了撤销双重国籍者马耳他公民身份的问题。虽然在具体情况下未认定违反，但法院确认关于国籍的决定可属于第8条范围，任意剥夺国籍违反公约。

**Genovese v. Malta（2011年），申请号 53124/09：** 法院认定违反了与第8条相结合的第14条。马耳他国籍法歧视马耳他父亲的非婚生子女，拒绝给予其婚生子女可享有的公民权。法院认为国籍属于第8条的范围，因为它影响社会身份。

**Kuric and Others v. Slovenia（2012年），申请号 26828/06：** 大审判庭审理了斯洛文尼亚"被注销"居民案——前南斯拉夫国民在斯洛文尼亚独立后被非法撤销永久居留身份。法院认定违反了第8条和第13条，以及与第8条相结合的第14条，确立了对居留和国籍地位的任意变更违反公约权利。

**K2 v. United Kingdom（2017年），申请号 42387/13：** 法院审理了以国家安全为由剥夺一名英国-苏丹双重国籍者英国公民身份的案件。申请被宣布不可受理，但法院的分析确认剥夺公民身份涉及第8条，且须具有比例性。

### 成员国义务

欧盟法律为双重国籍增加了额外维度：

- **欧盟公民身份（TFEU第20条）：** 每个持有成员国国籍的人同时是欧盟公民。欧盟公民身份提供包括自由流动、欧洲议会选举投票权和外交保护在内的额外权利。持有成员国和非成员国双重国籍的人士享有完整的欧盟公民身份权利。

- **Rottmann v. Freistaat Bayern（CJEU, C-135/08, 2010年）：** 欧盟法院裁定，尽管国籍法属于成员国管辖范围，但当国籍撤销导致丧失欧盟公民身份时，须审查其是否符合欧盟法律。须评估撤销的比例性。

- **Tjebbes and Others v. Minister van Buitenlandse Zaken（CJEU, C-221/17, 2019年）：** 欧盟法院审理了荷兰国籍法关于居住在欧盟以外10年的双重国籍者自动丧失荷兰国籍（从而丧失欧盟公民身份）的规定。法院裁定，虽然此类规定原则上是允许的，但须允许进行个别比例性审查。

### 国际比较

- **日本：** 日本不承认双重国籍。《国籍法》（国籍法，第14条）要求持有双重国籍的人士在22岁前选择一个国籍。实际上执行力度很小，许多日本双重国籍者并未正式放弃其第二国籍。
- **印度：** 印度不允许双重国籍，但根据1955年《公民法》（2005年修订）提供海外印度公民（OCI）身份，提供永久居留权和工作权，但没有投票权或政府雇佣资格。
- **巴西：** 巴西根据1988年《宪法》（第12条第4款）允许双重国籍，规定巴西人不因自愿取得另一国籍而丧失其国籍，除非外国以该国籍作为居留或行使民事权利的条件。
- **澳大利亚：** 澳大利亚自2002年起允许双重国籍，当时《澳大利亚公民身份修正案（放弃公民身份）法》废除了因取得外国公民身份而自动丧失澳大利亚公民身份的规定。值得注意的是，《澳大利亚宪法》第44条(i)款取消双重国籍者担任议员的资格，这在2017年议会资格危机中得到确认。
- **墨西哥：** 墨西哥于1998年修订《宪法》（第37条），防止因取得另一国籍而丧失出生取得的墨西哥国籍。出生取得的墨西哥国籍不可撤销，但通过入籍取得的国籍可以丧失。

## 比较分析

## 政策争论与新兴趋势

### 安全与反恐维度

双重国籍日益与国家安全争论交织，尤其在反恐背景下：

- **英国：** 1981年《英国国籍法》（经2014年《移民法》修正）允许内政大臣以"有利于公共利益"为由剥夺通过入籍取得的英国公民身份，或在当事人持有另一国籍且剥夺不会导致其成为无国籍人的情况下剥夺。沙米玛·贝古姆案（2019年），在其加入ISIS后被剥夺英国公民身份，引发了关于公民身份剥夺权力范围的广泛诉讼和争论。

- **法国：** 在2015年11月巴黎恐怖袭击之后，奥朗德总统提议修改宪法，允许撤销被判犯有恐怖主义罪的双重国籍者的公民身份。该提议最终因政治反对和对创造两级公民身份的担忧而被放弃。

- **澳大利亚：** 2015年《澳大利亚公民身份修正案（对澳大利亚的效忠）法》规定，从事特定恐怖主义相关行为的双重国籍者自动丧失澳大利亚公民身份。该规定于2019年首次被援引，针对为ISIS作战的澳大利亚-斐济双重国籍者尼尔·普拉卡什。

- **韩国：** 出于安全目的的公民身份撤销在《国籍法》下是可能的，但很少被援引。解决与双重国籍者相关安全关切的主要机制是通过移民管控（入境禁令）而非公民身份剥夺。

### 双重国籍者的税收

双重国籍在某些司法管辖区产生重大纳税义务：

- **美国：** 美国是仅有的两个（另一个是厄立特里亚）不论居住地对公民征收全球收入税的国家之一。居住在海外的美国公民须每年提交联邦所得税申报表，且可能面临双重征税，可通过外国收入免税额（IRC section 911，2024年最高126,500美元）和外国税收抵免（IRC section 901）予以减轻。放弃美国公民身份的双重国籍者可能须缴纳IRC section 877A（HEART法案）下的"退出税"。

- **韩国：** 韩国对居民征收全球收入税，但不对非居民公民征税。居住在韩国的双重国籍者被视为税务居民，对全球收入征税。

- **德国：** 德国对居民征收全球收入税。2024年StARModG改革不改变税务义务；居住在德国的双重国籍者继续承担完整的德国税务责任。

- **欧盟框架：** 欧盟不统一直接税，但欧盟法院裁定，基于国籍的欧盟公民间税收歧视可能违反条约下的基本自由。

### 防止无国籍

四个司法管辖区均承认防止无国籍的重要性，与1961年《减少无国籍状态公约》一致：

- **韩国：** 《国籍法》包含有限的防止无国籍保障，但僵硬的国籍选择期限在实践中可能造成个人在未确保另一国籍的情况下丧失韩国国籍的情形。
- **美国：** 最高法院在Afroyim v. Rusk案中的裁定为美国公民免受非自愿无国籍提供了强有力的保护。
- **德国：** StAG第17条明确规定，丧失德国国籍不得导致无国籍。
- **欧盟：** 欧盟法院在Rottmann和Tjebbes案中的判决要求在国籍撤销可能导致丧失欧盟公民身份时进行比例性审查，提供了额外的保护层。

### 外交保护困境

双重国籍在外交保护方面造成复杂情况：

- 根据国际习惯法，一国一般不得对其本国国民向该人同时具有国籍的另一国行使外交保护（"主国籍规则"或Nottebohm原则，国际法院1955年）。
- 伊朗-美国索赔仲裁庭在A/18号案件（1984年）中广泛讨论了这一问题，裁定"主要和有效国籍"测试决定哪个国家可对双重国籍者行使外交保护。
- 韩国与其他国家的领事协议一般不为持对方国家护照入境的双重国籍者提供领事探视。
