## 概述

伪造货币——制造、变造或流通伪造货币——是最古老的危害公共信任和经济稳定的犯罪之一。每个现代法律体系都将其视为严重犯罪，但法定结构、刑罚范围和执法机制在各司法管辖区之间存在相当大的差异。韩国《刑法》（第207-213条）规定了全球最严厉的处罚之一，对伪造本国货币可判处终身监禁。美国主要依据18 U.S.C. 第471-489条将伪造货币定为犯罪，最高可处20年联邦监禁和25万美元罚款，由美国特勤局执法。德国通过《刑法典》第146-152b条处理伪造货币问题，对制造或流通伪币最低处一年自由刑，对商业性或有组织的运营适用加重处罚。在超国家层面，欧盟第2014/62/EU号指令在各成员国之间确立了最低统一标准，要求制造伪币的最高刑罚不低于八年、流通伪币的最高刑罚不低于五年，而欧洲人权法院提供了基本权利保护框架——包括第6条的公正审判保障和第7条的合法性原则——对各国的伪造货币追诉构成约束。所有四个司法管辖区都受到1929年4月20日在日内瓦签署的《制止伪造货币国际公约》的影响。

## 韩国

### 法律框架

韩国的伪造货币法律编入《刑法》（형법）第十一章，涵盖第207条至第213条。法定框架在伪造本国货币和外国货币之间作出了关键区分，对涉及韩元面值纸币和硬币的犯罪处以明显更重的刑罚。

**第207条——伪造或变造货币：**

- **本国货币（第207条第1款）：** 以行使为目的，伪造或变造大韩民国现行流通的硬币、纸币或银行券者，处无期徒刑或二年以上有期徒刑。
- **在国内使用的外国货币（第207条第2款）：** 伪造或变造在韩国境内流通使用的外国硬币、纸币或银行券者，处一年以上有期徒刑。
- **在国外使用的外国货币（第207条第3款）：** 伪造或变造在外国流通使用的外国货币者，处十年以下有期徒刑。

### 相关犯罪

| 犯罪行为 | 法律依据 | 处罚 |
|---------|----------------|---------|
| 明知而行使伪币 | 第208条 | 最高2年或最高500万韩元罚金 |
| 以行使为目的取得 | 第209条 | 最高5年或最高1500万韩元罚金 |
| 制造伪造工具 | 第210条 | 最高7年 |
| 伪造有价证券 | 第211-213条 | 视类型而定；本国有价证券最高可处终身监禁 |

### 执法与韩国银行的核心角色

《韩国银行法》和《外汇交易法》提供了货币保护的制度框架。韩国银行设有专门的反伪造部门，与韩国国家警察厅协调配合，检测、调查和打击伪造活动。缴获的伪币在韩国银行的鉴定中心进行分析，伪造趋势统计数据定期发布。

### 标志性案件

**朝鲜超级美元行动（1990年代-2000年代）：** 韩国曾是所谓"超级美元"——一种质量极高的伪造美国100美元钞票（被广泛认为由朝鲜国家支持生产）的重要中转站。在釜山和韩国其他港口查获了大量超级美元。这些案件引发了复杂的管辖权问题，因为伪造行为针对的是外国货币，且据称由主权国家策划。韩国当局与美国特勤局合作进行联合调查，导致根据韩国和美国法律进行了多项起诉。

**大规模韩元伪造团伙（2010年代）：** 韩国法院对一个制造和流通伪造五万韩元纸币（流通中最高面额）的国内伪造团伙成员进行了定罪。主要组织者根据第207条第1款被判处超过十年的有期徒刑，体现了两年法定最低刑期以及法院对针对本国货币的大规模行动酌定重刑的裁量权。

## 美国

### 法律框架

美国宪法在第一条第八款第六项中明确提及伪造货币，授权国会"规定对伪造合众国证券和通用货币的惩罚"。这一宪法授权通过《美国法典》第18编第471条至第489条的综合法定方案予以实施。

**核心犯罪：**

- **18 U.S.C. 第471条——伪造债务凭证或证券：** 以欺诈意图伪造、变造或仿造美国任何债务凭证或其他证券者，处罚金或最高二十年监禁，或两者并处。
- **18 U.S.C. 第472条——行使伪造债务凭证：** 以欺诈意图传递、行使、发布或出售，或试图传递、行使、发布或出售，或携带入境或以欺诈意图持有任何伪造、变造的美国债务凭证或证券者，处罚金或最高二十年监禁，或两者并处。
- **18 U.S.C. 第474条——用于伪造的印版、石版或模拟、数字或电子图像：** 将持有伪造设备定为犯罪，最高可处二十五年监禁。
- **18 U.S.C. 第478条——外国债务凭证或证券：** 将伪造禁令扩展至外国货币和证券，最高可处二十年监禁。

### 犯罪构成要件

根据第471条获得定罪，控方必须排除合理怀疑地证明：

1. **伪造、变造或仿造**美国债务凭证或证券。
2. **欺诈意图：** 被告必须具有特定的欺诈意图。与许多金融犯罪不同，控方无需证明对任何受害者造成了实际经济损失——伪造行为本身加上欺诈意图即构成犯罪。

### 处罚

| 犯罪行为 | 法规 | 最高监禁刑 | 最高罚金 |
|---------|---------|---------------------|-------------|
| 伪造美国货币 | 第471条 | 20年 | 250,000美元 |
| 行使/传递伪币 | 第472条 | 20年 | 250,000美元 |
| 经营伪币交易 | 第473条 | 20年 | 250,000美元 |
| 持有伪造印版/设备 | 第474条 | 25年 | 250,000美元 |
| 伪造外国货币 | 第478条 | 20年 | 250,000美元 |
| 伪造货币共谋 | 第371条 | 5年 | 250,000美元 |

### 美国特勤局

特勤局于1865年专门为打击伪造货币而成立，在南北战争期间伪造货币问题已达到流行程度——据估计当时流通中约三分之一至二分之一的货币为伪币。该机构保留了对伪造货币调查的主要联邦管辖权，并与联邦储备系统和雕刻与印刷局密切合作。

### 标志性案件

**Fox v. State of Ohio, 46 U.S. 410 (1847)：** 这是最高法院最早处理伪造货币的案件之一，Fox案确立了联邦和州政府均可就同一行为追诉伪造货币犯罪而不违反禁止双重追诉条款的原则。被告因传递伪造银元而被俄亥俄州法院定罪。最高法院裁定，州对个别受害者实施欺诈行为的追诉与联邦对危害国家货币体系犯罪的追诉是不同的，由此确立了伪造货币背景下的双重主权原则。

**"冒烟之龙"行动和"皇家魅力"行动（2002-2005年）：** 这两项平行的卧底行动由联邦探员在新泽西州和加利福尼亚州进行，目标是贩运超级美元——据信来自朝鲜的质量极高的伪造100美元钞票——的华人三合会犯罪集团成员。行动导致至少87人被捕，约450万美元伪币被缴获。这些案件展示了伪造货币与有组织犯罪和国家安全关切的交汇。

**Anatasios Arnaouti（2005年）：** 虽然在英格兰曼彻斯特被起诉，但此案涉及美国特勤局的重要合作。Arnaouti领导的伪造行动在18个月内制造了约3500万美元的伪造美国100美元钞票和250万英镑的伪造10英镑钞票。他被判处八年监禁。该案说明了美国伪造货币执法通过国际合作协议实现的域外效力。

## 德国

### 法律框架

德国的伪造货币条款载于《刑法典》第八章，标题为"伪造货币和有价标识"（Geldfaelschung und Wertzeichenfaelschung），涵盖第146条至第152b条。德国法律对伪造货币的处理相当严厉，反映了国内政策和该国根据欧盟第2014/62/EU号指令承担的义务。

**第146条——伪造货币（Geldfaelschung）：**

第1款规定，以下行为人：

1. 以使其作为真币流通或便利此种流通的意图伪造货币，或以此种意图变造货币使其呈现更高面值的外观，或
2. 为相同目的为自己或他人取得伪币，或
3. 出售、持有或交付如此取得的伪币，或
4. 将伪币作为真币投入流通，

处一年以上自由刑。

**第146条第2款——加重伪造：**

行为人以营利为业（gewerbsmaessig）或作为以持续伪造货币为目的的团伙（Bande）成员实施犯罪的，处二年以上自由刑。

**第146条第3款——较轻情形：**

在第1款的较轻情形中，处三个月以上五年以下自由刑。在第2款的较轻情形中，处一年以上十年以下自由刑。

### 相关犯罪

| 犯罪行为 | 法条 | 处罚 |
|---------|---------|---------|
| 流通伪币（第146条以外情形） | 第147条 | 最高5年或罚金 |
| 伪造邮票 | 第148条 | 最高5年或罚金 |
| 伪造准备行为 | 第149条 | 最高5年或罚金 |
| 伪造有价证券（等同于货币） | 第151条 | 与第146-149条相同 |
| 伪造外国货币 | 第152条 | 与第146-149条相同 |
| 扣押和没收伪币 | 第150条 | 强制没收 |

### 执法

联邦刑事警察局（BKA——Bundeskriminalamt）是德国伪造货币调查的中央机构，并与欧洲刑警组织的欧洲伪造货币中心协调合作。根据欧盟第1338/2001号条例的要求，德意志联邦银行运营着国家分析中心，负责检验和分类伪造欧元纸币。

2024年上半年，德国检测到约38,600张伪造欧元纸币，面值约240万欧元——较2023年下半年增长29%，但相对于流通中真币的数量，这一数字仍处于历史低位。

### 标志性案件

**"FAKE行动"（欧洲刑警组织协调，德国参与，2020年代）：** 德国联邦和州警察参与了由欧洲刑警组织协调的打击邮寄伪造欧元纸币的行动。在一次重大行动中，从多个欧盟成员国的邮件中缴获了超过1400万欧元的伪币。德国法院根据《刑法典》第146条第2款对多名被告以团伙商业性伪造罪进行了追诉，判处二年至八年不等的有期徒刑。

**联邦最高法院（BGH）关于"以营利为业"的判例法：** BGH通过一系列判决确立，根据第146条第2款"以营利为业"要求证明行为人意图从伪造活动中获取持续收入来源。单一的大规模伪造行为并不自动构成营利行为，除非证据表明存在重复实施的意图。这一解释直接影响二年强制最低刑期是否适用。

## 欧洲人权法院与欧盟

### 欧盟立法框架

欧盟打击伪造货币的方法依托多项法律文书，为所有成员国建立了全面的最低统一标准框架。

**第2014/62/EU号指令（通过刑法保护欧元和其他货币免受伪造）：**

该指令于2014年5月15日通过，取代了第2000/383/JHA号框架决定，是欧盟伪造货币法律的基石。其关键条款包括：

- **强制犯罪化（第3条）：** 成员国必须确保以下故意实施的行为被规定为刑事犯罪：(a) 伪造或变造货币；(b) 以欺诈方式行使伪币；(c) 以行使为目的进口、出口、运输、接收或取得伪币；(d) 以欺诈方式制造、接收、取得或持有专门适用于伪造的工具、物品、计算机程序或其他手段。
- **最低最高刑罚（第5条）：** 制造犯罪必须适用最高至少八年监禁的刑罚。流通犯罪必须适用最高至少五年的刑罚。这些是最高刑期的下限——成员国可自由规定更重的刑罚。
- **保护尚未流通的货币（第4条）：** 该指令将保护范围扩展至已指定流通但尚未发行的货币，以解决在新系列发行前预先制造伪币的问题。
- **未遂和教唆（第6条）：** 成员国必须确保伪造犯罪的未遂、帮助、教唆或煽动行为也受到处罚。
- **管辖权规则（第7条）：** 每个成员国必须对在其领土上实施的犯罪确立管辖权，对于伪造欧元的犯罪，当犯罪由其国民在国外实施时也须确立管辖权。

**其他欧盟法律文书：**

- **第(EC) 1338/2001号条例：** 确立了保护欧元免受伪造所必需的措施，包括要求信贷机构和其他支付服务提供商鉴别欧元纸币和硬币的真伪。
- **第(EC) 1339/2001号条例：** 将第1338/2001号条例的效力扩展至非欧元区成员国。
- **理事会第(EC) 2182/2004号条例：** 涉及与欧元硬币相似的奖章和代币。

### 欧洲人权法院基本权利保护

虽然欧洲人权公约不包含专门针对伪造货币的条款，但《欧洲人权公约》确立了基本权利保障，直接约束成员国调查、追诉和惩罚伪造货币犯罪的方式：

- **第6条——公正审判权：** 所有伪造货币追诉必须符合公正审判保障，包括知悉指控的权利、准备辩护的充分时间、质证权和无罪推定。欧洲人权法院已就第6条适用于金融犯罪追诉发展了广泛的判例法。
- **第7条——法无明文规定不处罚（Nulla Poena Sine Lege）：** 伪造货币刑事法规必须足够精确和可预见。不得依据在被指控犯罪行为发生时尚未生效的法律对被告定罪，也不得追溯加重刑罚。
- **第一号议定书第1条——财产权保护：** 扣押和没收伪造工具及犯罪所得必须符合比例原则并接受司法审查。
- **第8条——尊重私生活权：** 在伪造货币调查中使用的监控和调查措施（如窃听、卧底行动和数字监控）必须符合合法性、必要性和比例性要求。

### 欧洲刑警组织与制度执法

欧洲刑警组织运营着欧洲伪造货币中心，作为欧盟伪造欧元战略和行动分析的中心枢纽。2024年，欧洲中央银行报告伪造欧元纸币数量维持在历史低位，全欧盟范围内约有467,000张伪币被撤出流通——这一数字仅占流通中约300亿张真钞的极小比例。

### 标志性案件与发展

**第2000/383/JHA号框架决定（前身）：** 2000年通过的原始欧盟伪造货币框架是欧盟首次行使协调货币保护刑法的权限。它确立了伪造货币必须以"有效、相称和具有威慑力"的刑罚予以处罚的要求，但未设定具体的最低最高刑期。其被第2014/62/EU号指令取代，反映了欧盟在《里斯本条约》后向更强力协调的转变。

**欧洲央行诉委员会（机构权限）：** 欧盟机构之间关于反伪造措施权限分配的争议塑造了制度架构。欧洲中央银行主张其基于条约的保护欧元纸币完整性的角色（TFEU第128条），而委员会则依据TFEU第83条通过刑法指令推进协调。

**日内瓦公约遗产：** 所有欧盟成员国均为1929年《制止伪造货币国际公约》的缔约国，该公约要求签署国将伪造货币的制造、使用和跨境转移定为犯罪。公约确立了伪造货币为可引渡犯罪的原则，并要求各国设立中央机构以协调反伪造工作——这一结构预示了现代欧洲刑警组织的角色。

## 比较分析

### 刑罚严厉程度

韩国在四个司法管辖区中施加了最严厉的刑罚，伪造本国货币可判处终身监禁。这反映了韩国法律传统将危害国家货币完整性的犯罪视为危害国家本身的犯罪，严重程度堪比叛国罪。美国处于中间位置，最高刑期为20年，虽然大大低于韩国的上限，但通过强大的联邦执法体系予以适用。德国的结构化方法——基本伪造罪强制最低刑期一年、商业性运营两年——在低端给予法官较少裁量权，但最高刑期低于其他体系。欧盟框架对成员国最高刑期施加下限（制造八年、流通五年），但不设上限，造成了整个欧盟范围内的显著差异。

### 本国货币与外国货币

所有四个司法管辖区都保护外国货币，但处理方式存在显著差异。韩国作出了最鲜明的区分，伪造本国韩元可判处终身监禁，但伪造在国外使用的外国货币最高仅判十年。美国根据第478条对伪造外国货币的处理严厉程度几乎与本国货币相同。德国通过第152条援引第146-149条，对外国货币适用相同的刑罚框架。欧盟指令明确涵盖"任何法定货币，包括欧元纸币和硬币"以及非欧元货币。

### 主观要件

各司法管辖区的主观要件存在细微差异。韩国法律要求"行使目的"——一种特定的意图，即伪币将被投入流通。美国法律要求"欺诈意图"，法院将其广泛解释为包含任何欺骗意图，无论是否实际流通。德国法律要求伪币"被作为真币投入流通"的意图，与韩国的表述密切平行。欧盟指令要求行为"故意实施"，但未规定特定形式的特定意图，将精确的主观要件定义留给成员国。

### 执法架构

| 司法管辖区 | 主要机构 | 国际合作 |
|-------------|---------------|--------------------------|
| 韩国 | 韩国国家警察厅；韩国银行 | 国际刑警组织；与美国特勤局的双边合作 |
| 美国 | 美国特勤局 | 国际刑警组织；双边协议；域外管辖权 |
| 德国 | 联邦刑事警察局（BKA） | 欧洲刑警组织；国际刑警组织；欧盟司法互助 |
| 欧盟/欧洲人权法院 | 欧洲刑警组织（协调）；各国警察力量 | 欧洲逮捕令；欧洲司法合作组织；相互承认 |

### 宪法和人权维度

美国宪法独特地以名称提及伪造货币，授予国会明确的列举权力对其进行惩罚。韩国的宪法框架将伪造货币视为法定犯罪，没有特定的宪法提及。德国《基本法》（Grundgesetz）赋予联邦立法机关一般的刑法权限。欧洲人权法院框架增加了一个超国家人权层面，要求伪造货币法律及其执行符合公约权利——这一约束在韩国或美国体系中没有直接等价物，尽管两者都有国内宪法保护（正当程序、禁止残忍和不寻常的惩罚）发挥类似功能。

## 时间线

| 年份 | 事件 |
|------|-------|
| 1790 | 美国根据《1790年犯罪法》颁布首部伪造货币法规 |
| 1847 | 美国最高法院裁决*Fox v. State of Ohio*，确立伪造货币追诉中的双重主权原则 |
| 1865 | 美国特勤局成立，在南北战争期间打击伪造货币 |
| 1871 | 德国《帝国刑法典》编纂伪造货币犯罪（第146条及以下） |
| 1929 | 《制止伪造货币国际公约》在日内瓦签署（4月20日） |
| 1931 | 日内瓦公约生效（2月22日） |
| 1950 | 通过《欧洲人权公约》，确立适用于伪造货币追诉的公正审判和合法性保障 |
| 1953 | 韩国《刑法》颁布，包括关于伪造货币的第十一章（第207-213条） |
| 1990年代 | 朝鲜"超级美元"伪造行动成为重大国际执法关切 |
| 1999 | 《刑法典》更新以应对数字伪造方法（计算机程序、数字成像） |
| 2000 | 通过欧盟第2000/383/JHA号框架决定——欧盟首次协调伪造货币刑法 |
| 2001 | 通过欧盟第1338/2001号条例，确立欧元保护措施 |
| 2002 | 欧元纸币和硬币开始流通，成为欧洲伪造活动的新主要目标 |
| 2002-2005 | "冒烟之龙"行动和"皇家魅力"行动导致87人被捕，缴获约450万美元伪币 |
| 2009 | 韩国推出五万韩元纸币，促使更新反伪造执法重点 |
| 2014 | 通过欧盟第2014/62/EU号指令（5月15日），以更严格的最低刑罚取代第2000/383/JHA号框架决定 |
| 2016 | 成员国转化第2014/62/EU号指令的截止日期（5月23日） |
| 2020年代 | 欧洲刑警组织协调的行动打击邮寄伪造欧元，缴获超过1400万欧元 |
| 2024 | 欧洲央行报告伪造欧元检测率处于历史低位；德意志联邦银行报告2024年上半年德国检测到38,600张伪币 |
