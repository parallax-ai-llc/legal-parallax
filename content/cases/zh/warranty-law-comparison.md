## 摘要

保证法——管辖卖方和制造商对商品和服务的质量、适用性和一致性义务的规则体系——反映了不同法律传统对消费者保护和风险分配的根本不同方法，从不能通过合同排除的法定强制性保证到自由协商的商业保证加上默示条款。韩国通过《韩国民法》（민법）关于卖方保证责任的规定（담보책임，第580-584条）、《消费者基本法》（소비자기본법）和《产品责任法》（제조물 책임법）监管保证，由韩国公平贸易委员会和韩国消费者院执行与保证相关的消费者保护，并维持详细的《消费者争议解决标准》（소비자분쟁해결기준），为数百种产品类别规定修理、更换和退款时间表。美国通过《统一商法典》（UCC）第2篇关于明示保证（第2-313节）、适销性默示保证（第2-314节）和特定用途适合性默示保证（第2-315节）的规定处理保证，辅以联邦《马格努森-莫斯保证法》（15 U.S.C. §§ 2301-2312），该法规定了消费者产品的书面保证，但不强制卖方提供保证，并在给出书面保证时限制默示保证的免责声明。德国通过BGB关于卖方物质和法律缺陷责任的规定（Sachmängelhaftung和Rechtsmängelhaftung，BGB第434-442节）提供全面的保证保护，为消费者销售强制规定最低两年的法定保证期（Gewährleistungsfrist），不能通过合同缩短，救济按等级结构设计，优先修理或更换（Nacherfüllung），然后才是降价（Minderung）或解除（Rücktritt）。欧盟通过《消费者销售指令》(EU) 2019/771协调了消费者保证法，该指令取代了指令1999/44/EC，建立了最低两年的法定保证和一年的举证责任倒置，而《数字内容指令》(EU) 2019/770将一致性要求扩展到数字产品。

## 韩国

### 法律框架

韩国的保证制度将《民法》关于卖方责任的规定与消费者保护立法和详细的行政标准相结合。

**民法（민법）——卖方保证责任（담보책임）：** 第580-584条规定了卖方对出售商品缺陷的责任。第580条规定，当出售物品存在买方通过正常检查无法发现的隐藏缺陷时，买方可以在发现缺陷后六个月内解除合同或索赔损害赔偿。第581条将此扩展至出售物品缺乏卖方保证或声称的品质的情况。这些规定适用于所有销售合同，但由消费者特定立法补充。

### 消费者争议解决标准

韩国保证制度的一个独特特征是《消费者争议解决标准》（소비자분쟁해결기준），由韩国公平贸易委员会根据《消费者基本法》制定。这些标准为数百种产品类别提供详细的、针对产品的保证指南：

| 产品类别 | 保证期 | 维修保证 |
|-----------------|----------------|------------------|
| 汽车 | 3年或60,000公里（一般）；5年或100,000公里（动力总成） | 保证期内免费维修；3次主要维修后的柠檬法条款 |
| 手机 | 购买后1年 | 免费维修；相同缺陷再现3次以上则更换 |
| 家电（主要） | 1-2年，取决于产品类型 | 免费维修；相同缺陷第4次维修后更换 |
| 服装和鞋类 | 6个月至1年，取决于材料 | 30天内换货或退款；保证期内缺陷维修 |
| 软件/数字内容 | 自购买或下载起1年 | 补丁/更新；如果不能纠正则退款 |

### 产品责任法（제조물 책임법）

《产品责任法》于2000年颁布，最近一次修订是2017年，对缺陷产品造成的损害对制造商施加严格责任。第3条确立了制造缺陷、设计缺陷和未警告缺陷的责任。2017年修正案引入了可反驳的缺陷推定，在损害发生在产品正常使用期间且损害的类型通常由产品缺陷造成时，大大减轻了消费者的举证责任。

### 韩国消费者院执法

韩国消费者院（한국소비자원）通过其投诉调查和调解过程在保证执法中发挥积极作用。该院调查消费者投诉，进行产品测试，并在消费者和企业之间进行争议调解。当产品未能满足《消费者争议解决标准》时，该院可以建议采取纠正措施，必要时将事项转给KFTC进行行政执法。

## 美国

### 法律框架

美国采用多层次的保证制度，结合州采纳的UCC规定、联邦立法和普通法原则。

**UCC第2篇——销售：** 《统一商法典》在所有州都已采纳（有变化），确立了三种类型的保证：

**明示保证（第2-313节）：** 由卖方向买方作出的关于商品的任何事实确认或承诺而产生，该确认或承诺成为交易基础的一部分，商品的任何描述成为交易基础的一部分，或任何样品或模型成为交易基础的一部分。不需要使用"保证"或"担保"等特定词语，卖方也不需要有意创建保证。

**适销性默示保证（第2-314节）：** 在每一笔由商人就所售种类的商品进行的销售中自动产生。适销性要求货物在贸易中无异议通过，适合此类货物通常使用的目的，得到充分的盛装、包装和标签，并符合容器或标签上的事实确认。

**特定用途适合性默示保证（第2-315节）：** 当卖方知道或有理由知道买方需要货物的任何特定目的，且买方依赖卖方的技能或判断来选择合适的货物时产生。

### 马格努森-莫斯保证法

1975年颁布的联邦《马格努森-莫斯保证法》（15 U.S.C. §§ 2301-2312）不要求卖方提供书面保证，但对选择提供的卖方施加重大义务：

| 要求 | 条款 |
|-------------|-----------|
| 披露 | 书面保证必须用简单语言清楚和显著地披露条款（FTC规则16 CFR 701） |
| 完全与有限指定 | 必须指定为"完全"或"有限"保证（第2303节） |
| 完全保证要求 | 必须在合理时间内免费补救缺陷；无不合理条件；默示保证在期限上不能受限 |
| 默示保证保护 | 当给出书面保证时，不能免除默示保证（第2308节） |
| 联邦诉由 | 消费者可以在联邦法院（至少50,000美元争议金额）或州法院提起诉讼（第2310节） |

### 柠檬法

所有50个州都颁布了汽车"柠檬法"，为购买有严重缺陷且在合理次数的修复尝试后无法修理的新车的消费者提供增强的保证保护。尽管具体规定各不相同，但大多数柠檬法要求制造商在保证期内3-4次不成功的修理尝试或累计30天停用后提供替代车辆或全额退款。

### 保证免责声明

UCC第2-316节允许默示保证的免责声明，但须符合特定要求：适销性默示保证的免责声明必须提及"适销性"，如果是书面形式，必须显眼；适合性默示保证的免责声明必须以书面形式作出并显眼；"按现状"或"带所有缺陷"等语言可以在买方有机会检查货物的情况下免除所有默示保证。

## 德国

### 法律框架

德国的保证法在2002年的Schuldrechtsmodernisierung期间进行了全面改革，提供了世界上最强的法定保证保护之一，特别是对消费者交易。

**一致性标准（BGB第434节）：** 如果在风险转移时，货物具有约定的质量（主观要求），适合合同预期的通常使用和特定使用，已连同约定的附件和说明一起交付，并符合买方基于卖方或制造商的公开声明可以合理期待的内容（客观要求），则货物没有物质缺陷。2022年实施指令 (EU) 2019/771的改革增加了组装要求以及关于为具有数字元素的商品持续提供更新的要求。

### 救济等级（Nacherfüllung优先）

德国保证法施加严格的等级救济制度：

**第一层——Nacherfüllung（补充履行，BGB第439节）：** 买方必须首先给予卖方通过修理（Nachbesserung）或更换（Nachlieferung）来纠正缺陷的机会，由买方选择。卖方承担Nacherfüllung的所有费用，包括运输、劳动和材料（第439(2)节）。只有当买方选择的Nacherfüllung形式与另一种形式相比涉及不成比例的费用时，卖方才可拒绝。

**第二层——降价或解除（第441、323节）：** 只有在Nacherfüllung失败（通常在两次不成功尝试后）、被卖方拒绝或对买方不合理后，买方才可以进行降价（Minderung）或解除（Rücktritt）。对轻微缺陷排除解除。

**平行索赔——损害赔偿（第280-281节）：** 买方可以在等级救济之外或代替等级救济索赔损害赔偿，前提是卖方有过错（Verschulden）。

### 消费者销售规定（第474-479节）

BGB第474-479节为消费者销售（Verbrauchsgüterkauf）提供增强保护：

| 保护 | 条款 |
|-----------|-----------|
| 不可减损性 | 保证权利不能在缺陷通知之前被排除或限制（第476节） |
| 举证责任倒置 | 交付后1年内出现的缺陷推定在交付时存在（第477节） |
| 最低保证期 | 新商品2年；二手商品经协议可减至1年（第475e节） |
| 卖方追索 | 卖方可从上游供应商追回保证费用（第478节） |
| 保证声明 | 商业保证（Garantieerklärungen）具有约束力，是对法定权利的补充（第479节） |

### 数字产品保证

在转化欧盟指令2019/770后，BGB第327-327u节（自2022年1月1日起生效）确立了数字内容和数字服务的保证规则。这些规定要求符合主观（合同）和客观（合理消费者期望）标准，在消费者可以合理期待收到更新的期间内施加更新义务，并提供类似于货物保证等级的救济。

## ECHR与欧盟

### 消费者销售指令 (EU) 2019/771

指令 (EU) 2019/771于2019年5月20日通过，自2022年1月1日起适用，建立了现代欧盟消费者商品保证框架，取代了指令1999/44/EC。

**一致性要求（第5-8条）：** 该指令确立了主观一致性要求（符合合同，包括约定的质量、数量、描述、类型、功能、兼容性和互操作性）和客观一致性要求（适合普通用途、符合基于货物性质和公开声明的合理消费者期望、连同附件和说明一起交付）。当货物包含数字元素时，卖方必须确保向消费者告知并提供使货物保持一致性所需的更新。

**法定保证（第10条）：** 成员国必须确保自交付起最低两年的法定保证期。成员国可以规定更长的保证期。

**举证责任（第11条）：** 在交付后一年内变得明显的任何不一致性推定在交付时存在，除非这种推定与货物性质或不一致性不相容。成员国可以将该期间延长至两年。

**救济（第13-16条）：** 该指令建立了等级救济制度：消费者首先有权通过修理或更换（由消费者选择，除非所选救济不可能或不成比例）使货物达到一致性；当修理或更换失败、不可能或未在合理时间内完成时，消费者可以获得相应的降价或终止合同。

### 数字内容指令 (EU) 2019/770

指令 (EU) 2019/770为向消费者提供的数字内容和数字服务建立了一致性和救济规则，包括流媒体服务、云存储、可下载软件、社交媒体平台和数字文件。该指令适用于消费者支付价格或提供个人数据作为对价的合同，将类似保证的保护扩展到传统销售交易之外。

### 产品责任指令

原始产品责任指令85/374/EEC在整个欧盟建立了对缺陷产品的严格责任。新产品责任指令 (EU) 2024/2853于2024年10月23日通过，通过将AI系统和软件作为"产品"涵盖、将范围扩展到将自己作为制造商的在线平台、建立索赔人在证明缺陷时面临过度困难的可反驳缺陷推定以及规定集体救济机制，现代化了这一框架。

### CJEU保证判例

**Weber and Putz（合并案件C-65/09和C-87/09, 2011）：** 法院裁定，卖方更换缺陷货物的义务包括承担拆除缺陷货物和安装替代货物的费用，即使原始合同不包括安装，前提是此类费用不成比例。

**Faber v. Autobedrijf Hazet Ochten BV（案件C-497/13, 2015）：** 法院澄清了当时适用的指令1999/44/EC下举证责任倒置的范围，裁定消费者无需证明不一致性的原因或将其归因于卖方。

## 比较分析

### 哲学方法

各司法管辖区反映了关于卖方和买方之间质量风险分配的根本不同哲学。德国和欧盟采取最强的亲消费者立场，将法定保证视为不可减损的消费者权利，不能在消费者交易中通过合同排除或限制。韩国通过消费者争议解决标准系统采取类似方法，但在保证期方面根据产品类别具有更大的灵活性。美国占据中间位置：虽然UCC创建了默示保证，但它允许通过显著语言免除，但受到在根据马格努森-莫斯法给出书面保证时禁止免责声明的限制。

### 救济结构

救济等级代表了最重要的实际差异之一。德国和欧盟施加严格等级，要求消费者允许卖方有纠正机会（Nacherfüllung/修理或更换），然后才能进行降价或解除。这种方法优先考虑合同保存，并给予卖方"第二次机会"履行。美国允许买方更广泛的救济选择，包括拒绝不符合的货物（UCC第2-601节，"完美交货规则"）或撤销接受（第2-608节）。韩国不施加严格等级，允许消费者根据情况和消费者争议解决标准在修理、更换、退款或损害赔偿中选择。

### 期限和证明

法定保证期和举证责任规则在消费者保护水平方面造成了重大的实际差异。欧盟强制规定最低两年保证和一年举证责任倒置。德国更进一步，倒置期为一年（此前为六个月，2022年延长）。韩国的制度因产品类别而异，一些产品获得比欧盟最低要求更长的保证期。美国没有强制性保证期，而是根据UCC第2-725节规定自交付起四年的时效期间，保证索赔必须在此期间内提出。

### 数字经济适应

欧盟通过指令2019/770和指令2019/771中具有数字元素的商品的条款，在保证法对数字产品的适应方面处于领先地位，创造了德国已转化实施的综合框架。韩国通过《电子商务消费者保护法》处理数字产品保证，但没有相同水平的具体性。美国缺乏联邦数字内容保证框架，为数字产品和服务的消费者造成监管不确定性。
