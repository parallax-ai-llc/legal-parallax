## 概述

双重征税——两个或两个以上司法管辖区就同一纳税人在同一期间的同一所得或资本征收类似税款——构成了国际贸易、投资和劳动力流动的最重大障碍之一，为其避免而建立了广泛的双边税收协定网络、多边工具和国内单边减免机制。韩国已签署超过90项避免双重征税的双边税收协定（조세조약），主要以OECD税收协定范本为蓝本，并通过《国际税收协调法》（국제조세조정에 관한 법률）和《所得税法》（소득세법）的外国税额抵免提供单边减免；该国于2020年批准了OECD《关于实施税收协定相关措施以防止税基侵蚀和利润转移的多边公约》（MLI），修改其现有协定网络以纳入反滥用条款。美国拥有世界上最广泛的协定网络，签订了超过60项所得税协定，并通过外国税额抵免机制（26 U.S.C. Sections 901-909）提供单边减免，但其独特的基于公民身份的征税制度产生了基于居民身份制度所不存在的双重征税风险，通过外国劳动所得免税额（Section 911）和详细的协定条款（包括保留美国对其公民征税权的储蓄条款）加以解决。德国已签署超过95项避免双重征税协定（Doppelbesteuerungsabkommen, DBA），并通过《对外税法》（Außensteuergesetz, AStG）和《所得税法》第34c条提供单边减免，而欧盟则通过《母子公司指令》、《利息和特许权使用费指令》、《仲裁公约》（90/436/EEC）以及《欧盟争议解决指令》（2017/1852）来处理双重征税，后者为成员国之间的跨境税务争议提供了具有约束力的解决机制。

## 韩国

### 法律框架

韩国的双重征税减免框架建立在三大支柱之上：双边税收协定（조세조약）、《国际税收协调法》（국제조세조정에 관한 법률，法律第19929号）以及《所得税法》和《法人税法》中规定外国税额抵免的国内条款。

### 协定网络

韩国维持超过90项双边税收协定，覆盖所有主要贸易伙伴。与发展中国家的协定通常包含税收饶让抵免条款，反映了韩国自身从资本进口国向资本出口国的历史转型。协定关键条款通常包括：

| 条款 | 韩国标准协定立场 |
|------|-----------------|
| 股息预提税 | 5%（合格参股）；15%（组合投资） |
| 利息预提税 | 10-15% |
| 特许权使用费预提税 | 10-15% |
| 常设机构门槛 | 一般遵循OECD范本；部分协定包含服务型常设机构 |
| 股份转让所得 | 在居民国征税；不动产丰富公司例外 |

### 外国税额抵免

根据《所得税法》第57条（소득세법 제57조）和《法人税法》第57条（법인세법 제57조），居民纳税人可就已缴纳的外国所得税申请抵免，以归属于外国来源所得的韩国税额为限（纳税人可选择分国限额法或综合限额法）。超额抵免可向后结转五年。

### MLI实施

韩国于2020年9月13日交存OECD MLI批准书，公约于2021年1月1日生效。韩国采用主要目的测试（PPT）作为防止协定滥用的最低标准，并选择适用简化的利益限制（LOB）条款。MLI修改韩国的被涵盖税收协定，纳入反滥用和争议解决条款，无需进行双边重新谈判。

## 美国

### 法律框架

美国的双重征税减免源自双边税收协定（根据美国税收协定范本谈判，最近于2016年更新）、《国内收入法》第901-909条的外国税额抵免以及第911条的外国劳动所得免税额。

### 基于公民身份的征税与双重征税

美国独特的基于公民身份的征税制度产生了固有的双重征税风险。所有美国公民和永久居民无论居住地均就全球所得纳税。协定条款通过"储蓄条款"解决这一问题，该条款保留美国在协定未生效的情况下对其公民和居民征税的权利，同时为某些优惠（通常为外国税额抵免、非歧视和特定免税）设置具体例外。

### 外国税额抵免（Sections 901-909）

外国税额抵免是主要的减免机制：

| 特征 | 详情 |
|------|------|
| 合格税种 | 向外国政府缴纳或应计的所得税、战争利润税和超额利润税 |
| 限额 | 不得超过外国来源所得的美国税额（第904条限额） |
| 分篮 | 分类限额：一般类、被动类、GILTI类、外国分支机构类 |
| 结转 | 1年向前结转，10年向后结转 |
| 选择 | 抵免或扣除（第164条）；抵免通常更为有利 |

### 美国税收协定范本

美国范本在若干方面不同于OECD范本：

- **储蓄条款：** 保留美国对公民和居民的征税权
- **利益限制（LOB）：** 详细的反协定滥购条款（OECD范本中未包含）
- **零预提税：** 在许多情况下寻求消除来源国对股息、利息和特许权使用费的预提税
- **仲裁：** 在相互协商程序中包含强制性约束仲裁

### 未参与MLI

美国未签署MLI，倾向于通过双边协定重新谈判来实施BEPS措施。这反映了对MLI与美国协定网络独特特征（包括储蓄条款和详细的LOB条款）兼容性的担忧。

## 德国

### 法律框架

德国的双重征税减免体系以双边避免双重征税协定（Doppelbesteuerungsabkommen, DBA）、用于国际税收协调的《对外税法》（Außensteuergesetz, AStG）以及提供单边外国税额抵免的《所得税法》第34c条为基础。

### 协定网络与方法

德国维持超过95项双边税收协定，是全球最广泛的网络之一。德国的协定对积极经营所得和就业所得主要采用免税法（Freistellungsmethode），而对被动所得（股息、利息、特许权使用费）适用抵免法（Anrechnungsmethode）。

| 所得类型 | 德国协定典型方法 |
|----------|-----------------|
| 营业利润（常设机构） | 累进免税 |
| 就业所得 | 累进免税 |
| 股息 | 抵免法；降低预提税（通常5-15%） |
| 利息 | 抵免法；通常零预提税 |
| 特许权使用费 | 抵免法；通常零预提税 |
| 不动产所得 | 免税（在财产所在国征税） |

累进免税（Freistellung mit Progressionsvorbehalt）意味着免税的外国所得从德国税基中排除，但在确定其余德国来源所得的适用税率时予以考虑。

### 单边减免（《所得税法》第34c条）

在没有协定或协定未完全消除双重征税的情况下，《所得税法》第34c条提供单边外国税额抵免，以归属于外国所得的德国税额为限。纳税人也可选择将外国税额作为营业费用扣除。

### 转换条款与受税条款

根据《所得税法》第50d(9)条和相应的协定条款，德国适用转换条款（Umschaltklauseln），在另一缔约国不对所得征税或按降低税率征税时，将免税法转换为抵免法。这防止了"双重不征税"（Keinmalbesteuerung），即所得在两个国家均逃脱征税的情况。

### MLI实施

德国于2020年3月17日交存MLI批准书，于2020年7月1日生效。德国采用PPT作为最低标准，并选择对被涵盖税收协定适用仲裁条款。

## 欧洲人权法院与欧盟

### 欧盟消除双重征税的机制

欧盟已制定多项工具来解决单一市场内的双重征税问题：

**《母子公司指令》（2011/96/EU）：** 消除不同成员国母子公司之间股息分配的预提税（最低10%持股），并要求母公司所在国对子公司的基础税额实行免税或抵免。

**《利息和特许权使用费指令》（2003/49/EC）：** 消除不同成员国关联公司之间利息和特许权使用费支付的预提税（最低25%持股）。

**《仲裁公约》（90/436/EEC）：** 为成员国之间的转让定价争议提供具有约束力的争议解决机制。如果两年内未达成相互协议，则必须设立咨询委员会作出具有约束力的意见。

**《欧盟争议解决指令》（2017/1852）：** 该指令于2017年10月10日通过，为成员国之间的税务争议建立了全面的具有约束力的争议解决机制，范围超越转让定价，涵盖双边税收协定或仲裁公约下产生的所有双重征税情况。成员国须在2019年6月30日前转化该指令。

### 欧盟法院判例法

欧盟法院在众多案件中处理了双重征税问题，但认定在两个成员国均行使合法征税管辖权的情况下，欧盟基本自由本身并不要求消除双重征税：

**Kerckhaert-Morres（C-513/04，2006年）：** 法院认定，在双重征税系两国平行行使合法征税权所致时，资本自由流动不要求比利时消除从法国公司收到的股息的法律性双重征税。

**Block（C-67/08，2009年）：** 确认欧盟法不要求成员国消除遗产的双重征税，即使在单一市场内部。

**Damseaux（C-128/08，2009年）：** 重申在没有协定义务的情况下，居民国无须根据欧盟法对外国来源所得进行抵免或免税，仅因该所得在来源国也被征税。

### 欧洲人权公约与双重征税

欧洲人权法院很少直接处理双重征税问题，但第一议定书第1条的财产权保护在理论上限制了累积税负：

**Imbert de Trémiolles v. France（申请号25834/05和27815/05，2008年）：** 法院指出，多种税的累积效果可能构成对财产权的不成比例干预，但在案件事实上未发现违反。

## 比较分析

### 减免方法：抵免法与免税法

双重征税减免的根本政策选择是抵免法（对全球所得征税但允许抵免外国税额）与免税法（将外国所得从国内税基中排除）。美国和韩国主要使用抵免法，确保所得至少按国内税率纳税。德国对积极所得主要使用免税法，管理更为简便，当国内税率高于外国税率时对纳税人更有利，但如果外国税率较低，可能导致整体税负较低。

### 防止协定滥购

防止协定滥用的方法存在显著差异。美国依赖每项协定中详细的利益限制（LOB）条款，反映了一种具体的、基于规则的方法。OECD MLI（被韩国和德国采用但美国未采用）引入了主要目的测试（PPT），这是一种更广泛的、基于目的的反滥用规则。欧盟的《反避税指令》增加了不受协定条款约束的一般反滥用规则（GAAR）。

### 争议解决

《欧盟争议解决指令》提供了最有力的机制，以约束性仲裁作为相互协议失败时的后备手段。OECD MLI的仲裁条款为选择加入的MLI签署国提供类似机制。美国的双边协定日益包含强制仲裁，但美国未参与MLI意味着必须逐一协定谈判。

### 双重不征税

一个新兴问题是相反的问题：通过协定错配实现"双重不征税"的安排。德国的转换条款和OECD/欧盟反混合安排规则（ATAD II）通过在另一缔约国未行使其征税权时拒绝免税来解决这一问题。
