## 摘要

囚犯权利——被监禁者尽管受到监禁仍保留的基本法律保护和权利，包括人道待遇权、医疗保健权、通信权、宗教活动权、诉诸法院权以及免受暴力保护权——体现了监禁虽限制自由但不剥夺个人固有人格尊严或其宪法和人权全部内容这一基本原则。韩国的囚犯权利框架受《刑事判决执行及收容者处遇法》（형의 집행 및 수용자의 처우에 관한 법률）管辖，该法于2007年经历重大改革，取代了殖民时代的《行刑法》（행형법，最初于1950年在日本殖民统治下颁布），确立了通信权、探视权、医疗保健权、宗教活动权和请愿权，由韩国国家人权委员会（국가인권위원회）和宪法法院进行监督，宪法法院做出了包括2001HeonMa546（2004年）承认被收容者享有宪法尊严权和2013HeonMa142（2016年）认定对被收容者通信进行全面审查违宪在内的里程碑式判决。美国主要通过依据第八修正案（残忍和异常惩罚）、第一修正案（宗教、表达）、第十四修正案（正当程序、平等保护）以及《宗教土地使用和被收容者法》（RLUIPA, 42 U.S.C. Section 2000cc）的宪法诉讼发展囚犯权利，最高法院在Turner v. Safley, 482 U.S. 78 (1987)等案件中确立限制囚犯宪法权利的监狱法规必须与合法的刑罚学利益合理相关——而1995年的《监狱诉讼改革法》（PLRA）对囚犯诉讼施加了重大程序障碍。德国的囚犯权利框架属于世界上保护力度最强之列，源于联邦宪法法院在BVerfGE 33, 1 (1972)中确立的囚犯保留未被监禁目的必然限制的所有基本权利的里程碑式判决，以及BVerfGE 40, 276 (1975)中要求限制必须由法律（而非仅凭行政法规）施加的判决，促使1976年颁布《联邦监狱法》（Strafvollzugsgesetz）（2006年联邦化至各州），该法的明确法定目标是使囚犯能够过上不犯罪的、有社会责任感的生活（再社会化目标，Resozialisierungsziel）。欧洲框架由欧洲监狱规则（2020年修订）、欧洲防止酷刑委员会的拘留标准、欧洲人权法院第3条（禁止不人道待遇，在所有公约条款中产生了最多的与监狱相关的违反认定）下的广泛判例法，以及包括关于被害人权利的Directive 2012/29/EU和关于囚犯移管的Framework Decision 2008/909/JHA在内的欧盟二级立法所塑造。

## 韩国

### 法律框架

韩国的囚犯权利框架随着2007年《刑事判决执行及收容者处遇法》（형의 집행 및 수용자의 처우에 관한 법률）的颁布而经历了根本性转变，该法取代了殖民时代的《行刑法》（행형법），建立了以权利为基础的现代矫正方法。

### 宪法保护

宪法法院确立了被收容者保留宪法权利但受比例限制的原则：

**2001HeonMa546号判决（2004年）：** 法院裁定被收容者依据宪法第10条享有人格尊严的宪法权利，对被收容者权利的限制必须满足宪法第37条第（2）款的比例性检验。该判决确立了评估监狱法规合宪性的框架。

**2013HeonMa142号判决（2016年）：** 法院认定对被收容者通信进行全面审查违宪，裁定虽然基于安全目的的内容审查可能具有正当性，但在没有具体安全理由的情况下系统性阅读所有通信违反了宪法第17条和第18条规定的隐私权和通信自由。

### 法定权利

该法规定了以下权利：

| 权利 | 条款 |
|------|------|
| 医疗保健 | 第37-40条：等同于社区标准的医疗保健；对精神疾病收容者的精神科治疗 |
| 通信 | 第43-44条：收发通信权；出于安全目的的有限审查 |
| 探视 | 第41-42条：定期家属探视；2020年引入视频探视 |
| 宗教活动 | 第45条：宗教礼拜、咨询和饮食调整的权利 |
| 请愿 | 第117条：向监狱长、法务部或国家人权委员会请愿的权利 |
| 教育 | 第63-64条：获得教育计划和职业培训 |
| 劳动 | 第65-73条：劳动权；监狱劳动补偿 |
| 户外运动 | 第33条：每日户外运动（最低1小时） |

### 国家人权委员会监督

国家人权委员会（국가인권위원회）通过以下方式实施监督：
- 定期检查矫正设施
- 调查收容者个人投诉
- 就系统性问题向法务部提出建议
- 发布关于拘留条件的报告

自2007年以来，国家人权委员会已调查超过15,000件收容者投诉，并就过度拥挤、医疗保健质量、使用约束器具、单独监禁以及保护弱势群体等问题发布了大量建议。

## 美国

### 法律框架

美国主要通过宪法诉讼发展囚犯权利，形成了既承认又限制收容者权利的庞大判例法体系。

### 第八修正案——拘留条件

**Estelle v. Gamble, 429 U.S. 97 (1976)：** 最高法院裁定，对囚犯严重医疗需求的故意漠视构成残忍和异常惩罚，确立了监狱中的医疗保健权。

**Farmer v. Brennan, 511 U.S. 825 (1994)：** 法院裁定，当监狱官员对收容者面临严重伤害的实质性风险故意漠视时，包括来自其他收容者暴力的风险，即违反了第八修正案。

**Brown v. Plata, 563 U.S. 493 (2011)：** 法院维持了一个三法官审判团要求加利福尼亚州将监狱人口减少至设计容量137.5%的命令，认定极度拥挤导致了违反第八修正案的宪法不充分医疗保健。

### 第一修正案——宗教和表达

**Turner v. Safley, 482 U.S. 78 (1987)：** 法院确立了囚犯宪法申诉的审查标准：如果监狱法规与合法的刑罚学利益合理相关，则该法规有效。四项Turner要素为：（1）与合法利益的合理联系，（2）行使该权利的替代方式的可用性，（3）对看守、其他收容者和监狱资源的影响，以及（4）不存在现成的替代法规。

**Holt v. Hobbs, 574 U.S. 352 (2015)：** 法院裁定阿肯色州监狱禁止蓄须的政策对一名穆斯林收容者依据RLUIPA的宗教活动构成了实质性负担，适用严格审查（强制性利益加最小限制手段）而非尊重Turner标准。

### 《监狱诉讼改革法》（PLRA，1995年）

PLRA对囚犯诉讼施加了重大障碍：
- **穷尽行政救济要求：** 囚犯在提起诉讼前必须穷尽所有可用的行政救济途径（42 U.S.C. Section 1997e(a)）
- **身体伤害要求：** 未事先证明存在身体伤害的，不得就精神或情感伤害提起联邦民事诉讼（42 U.S.C. Section 1997e(e)）
- **三振出局规则：** 已有三项或以上诉讼被以无意义、恶意或未陈述诉由驳回的囚犯，被禁止以贫困身份继续诉讼
- **诉讼费用条款：** 囚犯必须随时间支付全部诉讼费用

### 选举权

美国囚犯选举权剥夺因州而异：
- 两个州（缅因州和佛蒙特州）从不剥夺选举权
- 14个州和华盛顿特区在出狱后恢复权利
- 16个州在完成刑期（包括假释/缓刑）后恢复权利
- 若干州施加额外等待期或要求行政特赦

截至2025年，估计有460万美国人因重罪定罪而被剥夺选举权。

## 德国

### 法律框架

德国的囚犯权利框架被广泛认为属于世界上保护力度最强之列，其基础是联邦宪法法院对囚犯保留未被监禁必然限制的所有基本权利的承认。

### 宪法基础

**BVerfGE 33, 1 (1972)——囚犯权利判决：** 法院确立了这一基本原则：收容者保留除被监禁目的必然限制之外的所有基本权利。限制必须由法律（而非仅凭行政法规）施加，并必须满足比例原则。

**BVerfGE 40, 276 (1975)——法律保留：** 法院裁定，对囚犯基本权利的限制需要正式的法律依据（Gesetzesvorbehalt），而非仅凭内部监狱规章。这一判决促使1976年颁布了《联邦监狱法》（Strafvollzugsgesetz）。

**BVerfGE 116, 69 (2006)——再社会化目标：** 法院重申国家有追求再社会化目标的积极义务，监狱条件的构建必须使收容者在释放后能够过上守法的生活。

### 再社会化目标（Resozialisierungsziel）

德国监禁的法定目标是再社会化——使囚犯在释放后能够过上不犯罪的、有社会责任感的生活。这一目标塑造了监狱管理的所有方面：
- 治疗计划和干预措施是必须提供的
- 职业培训和就业准备被优先考虑
- 积极促进家庭联系
- 监狱假（Hafturlaub）和开放式监狱（offener Vollzug）是刑期执行的标准要素

### 权利和条件

| 权利 | 标准 |
|------|------|
| 牢房面积 | 最低5.5-6平方米单人间（因州而异） |
| 医疗保健 | 等值医疗原则：监狱医疗保健必须达到社区标准 |
| 劳动 | 劳动权；按规定费率支付补偿 |
| 教育 | 教育权和职业培训权 |
| 探视 | 每月最少1小时；大多数设施大幅超过最低标准 |
| 通信 | 通信权；仅因特定安全原因进行审查 |
| 宗教活动 | 享有宗教礼拜和牧灵关怀的完整权利 |
| 选举权 | 大多数情况下保留；仅凭特定法院裁定丧失（StGB第45条） |
| 休假 | 可以定期休假；开放式监狱作为标准递进 |
| 投诉 | 对监狱决定进行司法审查的权利（StVollzG第109条） |

### 司法审查

《联邦监狱法》第109条（现已复制到各州法律中）赋予收容者在地方法院（Landgericht）的刑罚执行庭（Strafvollstreckungskammer）对任何监狱决定提出质疑的权利。这一司法审查机制比美国制度更为便利，没有与PLRA程序障碍相当的规定。

### 监禁率

德国的监禁率（2024年约为每10万人64人）属于欧洲最低之列，大约是美国比率（约每10万人650人）的十分之一，反映了在使用监禁方面根本不同的方法。

## 欧洲人权法院与欧盟

### 《欧洲人权公约》第3条——拘留条件

欧洲人权法院第3条关于监狱条件的判例法属于其最丰富的领域之一：

**Muršić v. Croatia [GC], Application No. 7334/13, 2016年10月20日判决：** 大审判庭确立了每名被拘留者最低3平方米地面面积的个人空间是第3条评估的相关标准。当个人空间低于3平方米时，产生违反的强烈推定，该推定只能通过证明空间减少是短期的、偶尔的和轻微的，且伴有牢房外充分的行动自由和适当的活动来推翻。

**Ananyev and Others v. Russia, Applications Nos. 42525/07, 60800/08, 2012年1月10日判决：** 一项试点判决，认定俄罗斯因监狱过度拥挤而系统性违反第3条，要求进行结构性改革并建立有效的国内救济途径。

**Vinter and Others v. United Kingdom [GC], Applications Nos. 66069/09 et al., 2013年7月9日判决：** 不可复审的终身监禁违反第3条，确立了每名囚犯都必须有释放的希望和展示改造的可能性的原则。

### 欧洲监狱规则（2020年修订）

部长委员会修订了欧洲监狱规则（建议CM/Rec(2006)2，2020年修订），确立了全面标准：
- 规则18：住宿必须尊重人格尊严；最低空间标准
- 规则22：营养标准，提供宗教和医疗饮食要求
- 规则24：享有等同于社区标准的医疗保健权
- 规则39：思想、良知和宗教自由
- 规则43：获得教育
- 规则49：选举权，除非依法院裁定被依法剥夺

### 欧洲防止酷刑委员会标准

欧洲防止酷刑委员会对欧洲委员会所有成员国的拘留场所进行定期检查，并发布详细的报告和标准，涉及：
- 最低牢房面积（建议单人间至少6平方米）
- 每日户外运动（最低1小时）
- 获得医疗保健
- 防止囚犯间暴力
- 工作人员培训要求
- 对妇女、少年和精神疾病囚犯的特别保护

### 欧盟框架决定2008/909/JHA

该框架决定实现了已判刑囚犯在欧盟成员国之间的移管，使其在其国籍或惯常居住地的成员国服刑，通过允许囚犯在离家庭和社区更近的地方服刑来促进再社会化。

## 比较分析

### 监禁目标

最根本的分歧涉及监禁的明确目的。德国明确的再社会化目标塑造了监狱管理和条件的所有方面。韩国2007年的改革转向康复方向，但保留了更强的矫正重点。美国没有统一的联邦目标声明，报应和隔离目标在实践中占主导地位，尽管存在一些州级别的康复举措。

### 权利保留

德国适用最具保护性的标准：囚犯保留未被监禁必然限制的所有基本权利。欧洲人权法院在第3条下适用绝对保护（不可克减）并结合其他条款的比例性分析。韩国适用宪法比例性审查。美国的Turner标准对监狱管理者最为尊重，仅要求与合法刑罚学利益存在"合理关系"。

### 医疗保健

德国和欧洲人权法院/欧洲防止酷刑委员会要求等同于社区标准的医疗保健（等值医疗原则）。美国标准（对严重医疗需求的故意漠视）明显较低，要求证明监狱官员主观上知道并忽视了严重伤害的实质性风险。韩国提供法定医疗保健权，由国家人权委员会进行监督。

### 选举权

欧洲人权法院在Hirst v. United Kingdom (No. 2) (2005)案中裁定，全面剥夺囚犯选举权违反第1号议定书第3条。德国对大多数囚犯保留选举权。韩国在监禁期间暂停选举权。美国的方法最为限制性，数百万公民被剥夺选举权。
