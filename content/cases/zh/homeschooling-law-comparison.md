## 概述

在家教育——在家庭或非机构环境中而非在公立或私立学校中教育儿童的做法，通常由父母或私人教师主导——代表了父母权利、国家教育权力和儿童福利之间最具争议的交汇点之一，各地法律框架从几乎完全自由到刑事禁止不等。韩国在在家教育问题上处于法律模糊地位，既无明确的法定授权也无禁止；《初等和中等教育法》（초·중등교육법）规定在义务教育年限内须入学，但执行机制薄弱，一个小规模但不断增长的在家教育群体（估计2,000至3,000个家庭）在法律灰色地带运作，韩国在家教育联合会（한국홈스쿨링연합）等倡导团体寻求正式承认和监管适应。美国是世界上对在家教育最为宽容的主要司法管辖区，所有50个州自1993年（密西西比州最后合法化）以来均已合法承认在家教育，截至2023-2024学年约有330万在家教育儿童（占学龄人口的5.6%），监管范围从无需通知的州（德克萨斯州、爱达荷州、阿拉斯加州）到需要课程审批、标准化考试和专业评估的州（纽约州、宾夕法尼亚州）；法律基础建立在最高法院在 Pierce v. Society of Sisters (1925) 和 Meyer v. Nebraska (1923) 中对父母教育自由的承认之上，并受 Wisconsin v. Yoder (1972) 案中宗教自由条款保护的强化。德国通过其义务到校（Schulbesuchspflicht）要求实际上禁止在家教育，通过逐步升级的行政罚款、刑事处罚和家事法院干预加以执行，联邦宪法法院（Bundesverfassungsgericht）多次以义务到校服务于国家在社会融合、民主公民素养和防止平行社会方面的合法利益为由维护该禁令；德国与瑞典一道成为欧洲限制最严格的司法管辖区。欧洲人权法院主要通过 Konrad v. Germany (2006，不予受理) 和 Wunderlich v. Germany (2019) 处理在家教育问题，裁定义务到校要求不违反《第一议定书》第2条（受教育权）或第8条（尊重家庭生活权），同时对在所有情况下完全禁止在家教育是否可能违反公约权利这一问题保持开放。

## 韩国

### 法律框架

韩国在家教育的法律框架以模糊性为特征。《初等和中等教育法》（초·중등교육법）要求父母和监护人在义务教育年限（6至15岁）内将子女送入学校，第68条授权对违反者处以最高100万韩元的罚款。然而，该法未明确涉及在家教育，对在家教育家庭的执法几乎不存在。

### 宪法背景

宪法为国家教育权力和父母权利均提供了相互竞争的基础：

**第31条第1款：**"所有公民有权根据其能力接受教育。"该权利被解释为对国家施加提供教育的义务，对父母施加确保子女接受教育的义务。

**第31条第2款：**"负有保护子女义务的所有公民有责任至少提供初等教育及法律规定的其他教育。"

**第36条第1款：**婚姻和家庭生活受到保护，包括父母对养育子女的决定权。

### 在家教育社区

尽管法律模糊，韩国已发展出一个小规模的在家教育社区：

| 特征 | 描述 |
|---------------|-------------|
| 估计家庭数 | 2,000-3,000（截至2024年） |
| 主要动机 | 宗教信仰；对应试教育体制的不满；特殊需求；国际流动性 |
| 组织支持 | 韩国在家教育联合会（한국홈스쿨링연합）；基督教在家教育网络 |
| 法律地位 | 从技术上违反入学义务；无报告的起诉案例 |
| 重新入学挑战 | 在家教育儿童可参加初中等同学力考试（검정고시）以获得认可的学历证书 |

### 검정고시（学力认定考试制度）

韩国的검정고시制度为未完成正规教育者提供了替代途径。小学、初中、高中和大学入学各级别均可参加学力认定考试。在家教育学生可通过这些考试获得认可的教育资格证书，在缺乏正式法律授权的情况下实际上为在家教育提供了一条间接通道。

### 政策辩论

韩国的在家教育辩论反映了教育体制中更广泛的紧张关系：

- **支持者**认为在家教育可以缓解韩国教育体制的极端学业压力，使符合家庭价值观的宗教或哲学教育成为可能，并支持具有国际流动性职业的家庭。
- **反对者**强调社会化的重要性、对潜在教育忽视的担忧，以及可能破坏作为韩国发展核心的普及义务教育体制的风险。

## 美国

### 法律框架

美国为在家教育提供了世界上最为宽容和发达的法律框架，所有50个州和哥伦比亚特区均予以承认。

### 历史发展

| 年份 | 事件 |
|------|-------|
| 1925 | Pierce v. Society of Sisters：选择私立教育的权利 |
| 1972 | Wisconsin v. Yoder：阿米什人免于义务出勤 |
| 1980年代 | 现代在家教育运动兴起；多州法律争议 |
| 1983 | 在家教育法律辩护协会（HSLDA）成立 |
| 1985-1993 | 所有州逐步合法化 |
| 1993 | 密西西比州成为最后一个合法化在家教育的州 |
| 2008 | 加利福尼亚上诉法院，In re Rachel L.：最初要求教师资格证，后被推翻 |
| 2020 | 新冠疫情推动在家教育大幅增长 |

### 监管范围

在家教育法律辩护协会将各州分为四个监管类别：

**无需通知（11个州）：**
爱达荷州、德克萨斯州、阿拉斯加州、康涅狄格州、印第安纳州、密歇根州、密苏里州、新泽西州、俄克拉荷马州、伊利诺伊州、爱荷华州。父母无需通知州政府其正在进行在家教育。

**仅需通知（15个州）：**
父母须通知学区或州教育机构其在家教育的意向。例如：阿拉巴马州、亚利桑那州、加利福尼亚州（以私立学校备案形式）、佛罗里达州、肯塔基州。

**通知加评估（17个州）：**
父母须提供通知并接受某种形式的教育评估（标准化考试、学习档案审查或专业评估）。例如：科罗拉多州、俄勒冈州、南卡罗来纳州、弗吉尼亚州、华盛顿州。

**高度监管（7个州加华盛顿特区）：**
父母须提供通知、接受评估，并满足额外要求，如课程审批、教师资格或家访。例如：纽约州、宾夕法尼亚州、马萨诸塞州、罗得岛州、佛蒙特州。

|------------|-------------------------------|--------------------------------------|
| 1999 | 850,000 | 1.7% |
| 2007 | 1,500,000 | 2.9% |
| 2012 | 1,770,000 | 3.4% |

| 2023-2024 | 3,300,000 | 5.6% |

### 法律挑战与问题

**父母权利与儿童福利：**父母在家教育权利与儿童福利关切之间的紧张关系引发了重大辩论，特别是在在家教育家庭中发现严重虐待案件之后（例如2018年加利福尼亚州的Turpin案，13名儿童被声称在家教育的父母囚禁）。这些案件促使多个州提出了加强监管的立法提案。

**特殊教育：**在家教育儿童可能有资格根据《残疾人教育法》（IDEA）获得部分公立学校特殊教育服务，但资格因州和学区而异。联邦层面没有要求学区向自愿在家教育的儿童提供完整的免费适当公共教育（FAPE）。

**高等教育入学：**大多数学院和大学已制定了针对在家教育申请者的招生政策，通常接受SAT/ACT成绩、学习档案和在家教育成绩单。2008年《高等教育机会法》要求机构在联邦财政援助方面对在家教育学历与公立和私立学校学历同等对待。

## 德国

### 法律框架

德国通过其义务到校（Schulbesuchspflicht）要求实际上禁止在家教育，这使其区别于大多数其他仅要求义务教育（Bildungspflicht）的民主国家。

### 宪法基础

**《基本法》第7条第1款：**"整个学校体系应处于国家监督之下。"联邦宪法法院将此解释为确立了国家全面的教育权限，不仅涵盖教育内容，还涵盖教育提供的制度形式。

**《基本法》第6条第2款：**"养育和教育子女是父母的自然权利和首要义务。国家应对其履行这一义务进行监督。"法院裁定，该父母权利不包括在家教育的权利，因为在义务教育的背景下，第7条规定的国家监督利益优先。

### 联邦宪法法院判例

**BVerfG, 1 BvR 436/03（2006年5月29日）：**法院拒绝受理一对基督教父母要求在家教育的宪法申诉，裁定：
- 义务到校服务于培养负责任公民和使少数群体融入社会的合法目的
- 国家防止"平行社会"（Parallelgesellschaften）的利益证明了对父母教育自由的限制是正当的
- 学校不仅提供学术教育，还提供不可在家复制的重要社会经验（接触不同世界观、学习应对异议）
- 鉴于所服务的重要公共利益，对父母权利的干预具有比例性

**BVerfG, 2 BvR 1693/04 (2007)：**确认了对违反义务出勤义务的刑事处罚的合宪性，包括对以宗教理由拒绝送子女上学的父母处以罚款。

### 执行实践

| 州（Land） | 执行方法 |
|--------------|---------------------|
| 巴登-符腾堡州 | 行政罚款（每个孩子每学期500-1,500欧元）；持续违反者刑事移交 |
| 巴伐利亚州 | 罚款；Schulzwang（警察护送强制执行）；移交家事法院 |
| 北莱茵-威斯特法伦州 | 递进罚款；可能的监护权措施 |
| 黑森州 | 最高2,500欧元罚款；刑事起诉（罚款或最高6个月监禁） |
| 萨克森州 | 行政罚款；家事法院干预 |

### 重要案例

**Romeike家庭（2008年）：**来自巴登-符腾堡州的福音派基督教家庭，面临超过7,000欧元的罚款以及刑事起诉和监护权诉讼的威胁。他们于2008年逃往美国，一名移民法官于2010年批准了庇护（Romeike v. Holder）。移民上诉委员会于2013年推翻该决定，认为德国的执法是具有普遍适用性的中立法律，不构成迫害。该家庭获得了无限期暂缓遣返身份。

**Wunderlich家庭（2013年）：**当局在多年不遵守义务出勤法之后，从Wunderlich家庭带走了四名子女。这些儿童被安置在临时国家照管中，之后在要求上学的条件下归还。该家庭向欧洲人权法院提起的申请导致了 Wunderlich v. Germany (2019) 判决。

### 例外情况

德国对义务到校提供了极其狭窄的例外：

- **疾病：**因病无法上学的儿童可接受由学校提供的Hausunterricht（家庭教学）
- **外交官家庭：**外交人员子女可根据国际协议获得豁免
- **暂时缺勤：**暂时在国外的家庭在缺勤期间可免于出勤义务
- **残疾：**如果没有合适的学校安排，患有严重残疾的儿童可在家接受教学，但由国家而非父母提供

## 欧洲人权公约与欧盟

### 欧洲人权法院

**Konrad v. Germany (Application No. 35504/03，2006年9月11日不予受理决定)：**法院宣布不予受理寻求在家教育其子女的德国父母的申诉，认为义务到校不违反《第一议定书》第2条或欧洲人权公约第8条。关键推理：

- 《第一议定书》第2条不保障在家接受教育的权利
- 国家有通过义务到校确保教育的合法利益
- 义务出勤所服务的社会融合目标是限制父母教育自由的有效基础
- 鉴于国家在民主和多元社会中的重要利益，该限制具有比例性

**Wunderlich v. Germany (Application No. 18925/15，2019年1月10日判决)：**法院认为临时带走一个数年来拒绝遵守义务出勤法的家庭的四名子女不违反第8条。法院裁定：

- 带走子女构成对家庭生活的严重干预
- 然而，以子女的教育需要和父母对法院命令的持续违抗来看是正当的
- 当局的回应具有比例性，特别是考虑到子女相对迅速地归还且父母获得了遵从的机会
- 在教育事务上的裁量余地较宽

**Campbell and Cosans v. United Kingdom (Application No. 7511/76，1982年2月25日判决)：**虽然涉及体罚问题，法院确立了国家必须尊重父母在教育中的哲学信念的原则，但这不延伸至免于义务教育制度本身。

### 欧洲各国差异

欧洲各国的在家教育法规差异很大：

| 国家 | 法律地位 | 监管水平 |
|---------|-------------|-----------------|
| 英国 | 合法 | 注册；地方当局检查 |
| 法国 | 合法（2021年限制） | 自2022年起需要授权；年度检查 |
| 爱尔兰 | 合法 | 宪法权利；在Tusla注册 |
| 丹麦 | 合法 | 通知；市政监督 |
| 荷兰 | 合法 | 需要豁免；检查 |
| 瑞典 | 实际上禁止 | 仅限"特殊情况"例外（自2010年起） |
| 德国 | 禁止 | 罚款；刑事处罚；监护权措施 |
| 西班牙 | 模糊 | 无明确授权；有限执行 |
| 意大利 | 合法 | 每年末考试 |

### 联合国儿童权利委员会

儿童权利委员会未就在家教育本身采取明确立场，但强调了：
- 所有儿童的受教育权（《儿童权利公约》第28条）
- 教育目的包括社会化和为民主生活做准备（《儿童权利公约》第29条）
- 国家有义务确保教育质量，无论教育在何处进行
- 在允许在家教育的地方，监测家庭教育以防止教育忽视的重要性

### 欧盟框架

教育主要仍属成员国权限。欧盟对在家教育没有统一立场。《欧盟基本权利宪章》（第14条）保障受教育权和设立教育机构的自由，承认父母确保子女按其信念接受教育的权利，但未涉及家庭教学的具体问题。

## 比较分析

### 哲学分歧

在家教育辩论揭示了关于教育目的的根本哲学分歧。德国将学校教育视为服务于社会凝聚力和民主融合的国家职能——教育不能与学校的制度环境分离。美国将教育视为主要由父母承担、国家支持但不能垄断的责任——父母保留指导子女教育的基本自由。韩国的模糊立场反映了一个通过文化共识而非严格执法实现近乎普遍入学的体制，使法律问题在很大程度上成为理论问题。

### 权利框架

各司法管辖区对相互竞争的权利赋予了不同的优先级。德国将国家教育权限（《基本法》第7条）置于父母教育自由（《基本法》第6条）之上。美国将父母自由（Pierce、Yoder）置于国家监管权力之上。欧洲人权公约赋予各国较宽的裁量余地，接受宽容和限制性方法均与公约权利相容。

### 质量控制

最重要的实际挑战是确保在家教育的教育质量。美国表现出最大的差异，一些州不要求任何评估，另一些州则要求年度标准化考试。关于在家教育成果的研究有限且方法论上存在争议，在家教育学生在标准化考试中的表现通常达到或高于公立学校平均水平，但选择偏差（有动机的父母自主选择在家教育）使解读变得复杂。德国通过禁止这一做法完全避免了质量控制问题。

### 儿童福利

在家教育与儿童福利的交叉提出了独特挑战。在家教育儿童与法定报告者（教师、学校辅导员）的隔离造成了潜在的监控缺口。在家教育家庭中的高关注虐待案件促使了加强监管的呼声，不过在家教育倡导者认为虐待发生在所有教育环境中，额外的监管不成比例地加重了守法家庭的负担。
