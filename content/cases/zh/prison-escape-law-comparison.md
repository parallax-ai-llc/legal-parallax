## 摘要

越狱在比较刑法中占据着独特而具有揭示性的地位。虽然大多数司法管辖区将逃脱合法羁押视为刑事犯罪，但以德国为首的一个值得注意的国家群体拒绝将越狱行为本身入罪，其理由是人类对自由的渴望是一种不应受到惩罚的自然本能。韩国根据《刑法》第145-151条将越狱入罪，对单纯越狱处以最高1年刑罚，对涉及暴力或破坏的加重越狱处以最高7年刑罚。美国根据18 U.S.C. SS 751在联邦层面对越狱处以最高5年刑罚，各州刑罚差异很大，最高法院严格限制了胁迫和紧急避险抗辩的适用范围。德国的独特之处在于不将囚犯自行越狱（Gefangenenselbstbefreiung）入罪，但根据StGB SS 120对协助越狱的第三方处以最高3年监禁，对疏忽的官员处以最高5年监禁。欧洲人权法院和欧盟框架并不强制要求将越狱入罪，但确立了关于重新抓捕时使用武力、重新羁押的程序权利以及通过欧洲逮捕令进行跨境合作的标准。

## 韩国

### 法律框架

越狱受韩国《刑法》（형법，1953年第293号法律，经修订）第九章管辖，该章题为"脱逃及藏匿人犯罪"（탈옥죄）。相关条款涵盖第145条至第151条。

**第145条——脱逃及违反传唤令：** 依法被逮捕或羁押的人脱逃的，处一年以下有期劳役。因天灾或依法被临时释放后无正当理由违反传唤令的，处以同等刑罚。

**第146条——特殊脱逃（특수 탈옥）：** 通过破坏拘禁设施或镣铐、对他人使用暴力或胁迫、或与二人以上共同行动而脱逃的，处七年以下有期劳役。该条款针对涉及武力、财产破坏或有组织团伙行动的加重越狱。

**第147条——帮助脱逃：** 通过打开或破坏拘禁装置、或使用暴力或胁迫手段使依法被逮捕或拘禁的人脱逃的，处七年以下有期劳役。

**第148条——看守致使脱逃：** 监狱看守或押送人员因疏忽致使囚犯脱逃的，处五年以下有期劳役。看守故意致使脱逃的，处以与脱逃者或协助者相同的刑罚。

**第149条——未遂：** 脱逃未遂和协助脱逃未遂均可处罚。

**第150条——预备和阴谋：** 为第145-148条所列犯罪进行预备或阴谋的，处二年以下有期劳役。

### 标志性案例

**辛昌原越狱案（1997年）：** 韩国历史上最臭名昭著的越狱案之一。被判谋杀罪的辛昌原于1997年1月20日从釜山监狱越狱，他经过数月节食减重后从浴室通风井挤出，并事先锯断了金属栏杆。他逃脱了大规模的全国搜捕行动，在逃两年多，成为媒体密集报道的对象。长期未能将其重新抓获成为韩国执法部门的公开耻辱，并促进了监狱安全规程的改革。

**永登浦监狱越狱案（1988年）：** 1988年10月8日，12名囚犯在从首尔永登浦监狱转押至公州监狱途中越狱。该案因一名越狱者池康宪说出"有钱无罪，无钱有罪"（돈 있으면 무죄, 돈 없으면 유죄）这句话而引起公众关注，这句话成为1980年代韩国社会对社会经济不平等的广泛引用的批评。此次越狱促使立法机构审查囚犯转押安全措施。

### 量刑实践

韩国法院将越狱作为与原定罪相独立的犯罪处理。越狱刑期与原刑期连续执行，确保越狱不会缩短囚犯的总羁押时间。恶劣的监狱条件或来自其他犯人的威胁等减轻情节在量刑中已被认可，但不构成完全抗辩。

## 美国

### 法律框架

在联邦层面，越狱根据18 U.S.C. SS 751（逃脱羁押）和SS 752（教唆或协助越狱）入罪，属于联邦刑法典第35章（"逃脱与营救"）。

**18 U.S.C. SS 751(a)——逃脱羁押：** 逃脱或企图逃脱司法部长、其授权代表或经司法部长指示关押的任何机构羁押的人，面临分级刑罚：

| 羁押类型 | 最高刑罚 |
|---------|---------|
| 重罪指控或任何定罪 | 5年监禁和/或罚金 |
| 轻罪指控（定罪前）、引渡或移民 | 1年监禁和/或罚金 |
| 少年收容 | 1年监禁和/或罚金 |

**18 U.S.C. SS 752(a)——教唆或协助越狱：** 任何营救、企图营救、教唆、帮助或协助联邦羁押人员越狱的人，面临与越狱者相同的分级刑罚。

**18 U.S.C. SS 751(b)——使用武力：** 如果在越狱或营救过程中使用武力导致死亡，该人可被判处任何年限的监禁或终身监禁。

### 各州差异

美国各州都有自己的越狱法规，刑罚差异显著：

**加利福尼亚州（Penal Code SS 4530-4532）：** 从州立监狱越狱属于重罪，可判处最高3年监禁。如果越狱涉及武力或暴力，刑罚增加至最高6年。重罪被告从县级监狱越狱面临类似刑罚。刑期与原定罪连续执行。

**得克萨斯州（Penal Code SS 38.06-38.07）：** 如果当事人因重罪被羁押，越狱属于三级重罪（2-10年）。如果越狱者使用致命武器，升级为二级重罪（2-20年）。

**纽约州（Penal Law SS 205.05-205.15）：** 越狱从A级轻罪（三级越狱——逃脱羁押）到D级重罪（一级越狱——使用危险器具或造成他人身体伤害）不等。

### 标志性案例

**United States v. Bailey, 444 U.S. 394 (1980)：** 关于越狱抗辩的最高法院主要判例。1976年8月26日，四名联邦囚犯通过拆除窗户栏杆、顺打结的床单滑下来逃脱华盛顿特区监狱，在逃一至三个半月。被告人提出胁迫和紧急避险抗辩，援引恶劣的监狱条件（火灾、暴力、威胁）。最高法院裁定：（1）"越狱"指未经许可擅自离开羁押，证明知情即足以定罪，无需证明特定的逃避拘禁意图；（2）有效的胁迫或紧急避险抗辩要求被告证明除越狱外没有合理的合法替代方案，且被告在所称的胁迫情形消除后真诚地努力自首或返回羁押。这一严格标准在后续联邦和各州案件中被一贯适用以限制越狱抗辩。

**Lovecamp v. State of California (1974)：** 加利福尼亚上诉法院在一起越狱案中承认了紧急避险抗辩，该案中两名囚犯在遭受其他囚犯反复性侵犯后逃跑，法院认定越狱者已用尽内部救济途径并在越狱后不久自愿投案。该案确立了允许胁迫作为抗辩的狭窄例外，美国最高法院后来在Bailey案中对此加以限制。

### 量刑指南

根据美国量刑指南（USSG SS 2P1.1），越狱的基础犯罪等级为13（对初犯者大约12-18个月）。如果越狱涉及暴力、使用武器或导致严重扰乱设施运行，则适用加重。联邦越狱刑期通常与原判刑期连续执行。

## 德国

### 法律框架

德国是不将囚犯自行越狱入罪的法律体系中最突出的例子。这一原则被称为Gefangenenselbstbefreiung（囚犯自我解放），深深植根于德国法律哲学，可追溯至19世纪末德意志帝国立法机构同意不应惩罚自我解放。

**哲学基础：** 德国法律承认"自然自由冲动"（naturlicher Freiheitsdrang）是人类的固有驱动力。其推理认为，虽然国家有权通过合法定罪剥夺一个人的自由，但不能因该人出于基本人类本能的行为而额外惩罚他。《基本法》（Grundgesetz）及其对人的尊严（第1条）和人身自由（第2条）的强调支撑了这一方法。

**不构成犯罪的行为：**
- 囚犯从开放式设施步行离开
- 囚犯无外援翻越围墙
- 囚犯在不伤害他人的情况下越狱
- 以上述任何方式进行的越狱未遂

**仍然构成犯罪的行为：**

**StGB SS 120——协助囚犯越狱（Gefangenenbefreiung）：**
- 第（1）款：释放囚犯或鼓励、支持其越狱的，处三年以下监禁或罚金。
- 第（2）款：如果犯罪人是有义务防止越狱的公务员或受委托从事特殊公共服务职能的人员，刑罚增加至五年以下监禁或罚金。
- 第（3）款：未遂可罚。
- 第（4）款：根据公共机关命令被关押在机构中的其他被拘留人员等同于本条所称的"囚犯"。

**StGB SS 121——囚犯暴动（Gefangenenmeuterei）：** 囚犯联合起来袭击或威胁看守、集体强行突围或煽动他人进行此类行为的，处三个月以上监禁。该条款针对的是暴力集体行为而非个人越狱。

**附带犯罪：** 虽然越狱本身不受惩罚，但越狱过程中实施的任何犯罪行为均可被完全追诉。破坏监狱财产的囚犯面临StGB SS 303（财产损害）的指控。袭击看守依据StGB SS 223-224追诉。劫持人质依据StGB SS 239b追诉。这些附带犯罪的刑罚往往远超其他司法管辖区越狱指控可能施加的刑罚。

### 实际后果

虽然越狱不构成犯罪，但并非没有后果：
- 囚犯剩余刑期继续执行，重新抓获后必须服刑
- 在监狱系统内实施纪律处分（单独监禁、取消特权、转至更高安全级别的设施）
- 逃亡期间不计入刑期
- 越狱期间造成的任何损害导致民事责任
- 假释和提前释放的前景严重降低

### 采用德国模式的其他国家

其他一些司法管辖区在不同程度上采用了这一方法：
- **奥地利：** 步行离开式越狱在奥地利法律下不构成刑事犯罪
- **比利时：** 自我解放不单独入罪
- **荷兰：** 越狱不构成刑事犯罪；荷兰法律理论认为，看到机会而不抓住的囚犯实际上是在自我惩罚
- **瑞典：** 越狱本身不产生额外刑事处罚
- **墨西哥：** 《联邦刑法》第154条规定，独自越狱的囚犯不受制裁，但与他人合谋越狱可受处罚

## 欧洲人权法院与欧盟

### 《欧洲人权公约》

欧洲人权法院不直接将越狱作为刑事犯罪加以规定，而是将此留给各成员国。然而，《公约》的几个条款创建了一个管辖囚犯权利以及国家在预防和应对越狱方面义务的框架。

**第2条——生命权：** 各国对在押人员的生命负有积极保护义务。这延伸至涉及越狱的情形：
- 使用致命武力防止越狱必须满足第2条第（2）款的严格比例性检验，该检验仅在"绝对必要"时才允许使用武力以实施合法逮捕
- 在Makaratzis v. Greece（2004年，大审判庭）案中，法院认定警方使用不成比例武力（向逃跑嫌疑人开枪多次）违反了第2条，强调枪支使用的监管框架必须提供充分的防止任意行为的保障
- 各国必须对越狱企图或重新抓捕行动中发生的任何死亡或严重伤害进行有效调查

**第3条——禁止酷刑和不人道待遇：** 拘留条件不得达到不人道或有辱人格待遇的门槛。这一原则对越狱法有以下影响：
- 在越狱由不可忍受的条件引发的情况下，法院可以考虑国家本身是否应对造成促使越狱的情形承担责任
- 重新抓获后施加的惩罚（如延长单独监禁）本身不得违反第3条
- 欧洲防止酷刑委员会（CPT）定期检查拘留设施，并就可能驱使越狱企图的条件进行报告

**第5条——自由和安全权：** 确立了合法剥夺自由的框架：
- 第5条第（1）款（a）项允许"在主管法院定罪后合法拘留某人"
- 重新抓获后的重新羁押必须符合程序保障，包括及时的司法审查
- 基于公共保护目的在越狱后进行的不确定期限羁押不得成为任意行为

**第6条——公平审判权：** 因越狱相关罪行被起诉的囚犯保留完整的公平审判权，包括法律代理和审查证据的权利。

### 欧洲监狱规则

欧洲委员会的建议Rec(2006)2关于欧洲监狱规则（2020年修订）就处理越狱情况提供了详细指导：

**使用武力（规则64-68）：**
- 监狱工作人员不得对囚犯使用武力，除非出于自卫、越狱企图或对合法命令的主动/被动抵抗，且始终作为最后手段
- 使用的武力应为必要的最低限度，持续时间为必要的最短时间
- 详细程序必须规定允许使用的武力类型、使用情形、授权人员、所需的权限级别以及强制性事后报告
- 工作人员必须接受最低限度使用武力技术的培训

**安全分类：**
- 被评估为越狱风险的囚犯应在适当的安全条件下关押
- 安全措施应为必要的最低限度，不得超出防止越狱所需
- 个别风险评估必须定期审查

### 欧盟框架：欧洲逮捕令

关于欧洲逮捕令（EAW）的理事会框架决定2002/584/JHA为欧盟内跨境追捕和移交越狱囚犯提供了机制：

- 当剩余刑期至少为4个月时，可签发欧洲逮捕令以执行监禁刑
- 成员国必须基于相互承认原则执行其他成员国签发的逮捕令
- 存在有限的拒绝理由（一事不再理、诉讼时效、属地管辖权）
- 欧洲逮捕令取代了欧盟成员国之间的传统引渡制度，显著加速了遣返越狱囚犯的过程

### 《欧盟基本权利宪章》

《欧盟宪章》第4条（禁止酷刑和不人道待遇）和第6条（自由权）与《欧洲人权公约》的保护相对应，并在成员国实施欧盟法律时适用，包括在欧洲逮捕令的背景下。

## 比较分析

### 哲学分歧

越狱法中最引人注目的分歧在于日耳曼/大陆法系方法与英美/韩国方法之间。德国、奥地利、比利时、荷兰和瑞典将越狱不入罪的做法建立在对人类自然自由本能的哲学认知之上——即认为惩罚一个人渴望自由是在合法监禁之上添加了一个不公正的刑罚。相比之下，韩国和美国将越狱视为对国家权威和司法系统完整性的侵犯，认为合法拘留必须通过刑事制裁来强制执行。

### 实际结果

尽管存在哲学分歧，实际结果的趋同程度超过法律文本所暗示的：
- 在德国，越狱的囚犯面临纪律后果、延长监禁以及对任何附带犯罪行为的追诉，这意味着越狱很少是无代价的
- 在韩国和美国，不涉及暴力的简单步行越狱通常导致适度的额外刑期（韩国1年，美国各异），这意味着增量惩罚往往有限
- 所有司法管辖区都惩罚越狱期间的暴力、财产破坏和劫持人质行为，反映出普遍共识，即这些行为超出了任何所谓"自然自由冲动"的范围

### 紧急避险抗辩

紧急避险或胁迫抗辩的可用性揭示了重要差异：
- 美国遵循Bailey案（1980年）设定了最严格的标准：被告必须证明不存在合理的合法替代方案，且必须在胁迫条件消除后及时投案
- 韩国允许基于监狱条件的减轻量刑，但不承认完全抗辩
- 德国由于不将越狱入罪，使这一问题变得毫无意义
- 欧洲人权法院框架在评估惩罚的比例性时隐含地支持考虑促使越狱的条件

### 第三方责任

所有四个体系都同意协助或促进越狱构成犯罪，即使越狱本身不构成犯罪：
- 韩国：最高7年（第147条）
- 美国：最高5年（18 U.S.C. SS 752）
- 德国：最高3年，疏忽官员最高5年（SS 120 StGB）
- 欧洲人权法院/欧盟：由成员国自行规定，但受比例性审查约束

### 使用武力

防止或重新抓捕越狱者时允许使用的武力水平是欧洲人权法院影响力最大的领域：
- 欧洲人权法院第2条下的"绝对必要"标准适用于所有欧洲委员会成员国
- 欧洲监狱规则要求武力为必要的最低限度且仅作为最后手段使用
- 韩国和美国允许使用合理武力，但在此方面受到的国际监督较少

## 时间线

| 年份 | 事件 |
|------|------|
| 1871 | 德意志帝国刑法典确立囚犯自我解放不构成刑事犯罪 |
| 1953 | 韩国颁布《刑法》（第293号法律），在第145-151条中规定了越狱犯罪 |
| 1948 | 18 U.S.C. SS 751-752作为联邦刑法典修订的一部分被编纂 |
| 1950 | 《欧洲人权公约》通过，确立第2条（生命权）和第5条（自由权） |
| 1973 | 欧洲委员会通过首批《欧洲囚犯待遇最低标准规则》 |
| 1974 | People v. Lovecamp在加利福尼亚确立了越狱案中狭窄的紧急避险抗辩 |
| 1976 | Bailey和共同被告从华盛顿特区监狱越狱，为最高法院里程碑式裁决奠定基础 |
| 1980 | United States v. Bailey, 444 U.S. 394：最高法院严格限制越狱指控的胁迫/紧急避险抗辩 |
| 1988 | 首尔永登浦监狱越狱事件；12名囚犯在转押途中越狱，引发关于不平等的公共辩论 |
| 1997 | 辛昌原从釜山监狱越狱；全国搜捕持续两年以上 |
| 2002 | 欧盟通过关于欧洲逮捕令的框架决定2002/584/JHA，实现越狱囚犯的跨境移交 |
| 2004 | Makaratzis v. Greece（欧洲人权法院大审判庭）：确立武力使用比例性标准，适用于越狱重新抓捕情形 |
| 2006 | 欧洲委员会通过修订版欧洲监狱规则（Rec(2006)2），包括越狱预防中使用武力的详细标准 |
| 2020 | 欧洲监狱规则再次修订，更新安全分类和使用武力的标准 |
