## 摘要

举报人保护法——管理对在组织内部或向公共机关报告不当行为的个人提供的法律保障——在过去二十年中经历了迅速的全球扩展，由备受瞩目的企业丑闻、政府腐败以及日益认识到内部报告机制对监管执法至关重要所推动。韩国运营世界上财务最慷慨的举报人奖励制度之一，反腐败和公民权利委员会（ACRC）根据《防止腐败和ACRC的设立和管理法》（ACRC法）有权授予高达30亿韩元（约230万美元）的奖励，并辅以《保护公共利益举报人法》（2011），该法涵盖私营部门报告。美国维持着全球最广泛和最分散的举报人框架，包括约六十部联邦法规，其中包括《虚假申报法》（31 U.S.C.第3729-3733节）及其qui tam条款、《多德-弗兰克法》的SEC举报人计划（15 U.S.C.第78u-6节），对超过100万美元的处罚提供10-30%的奖励，以及保护上市公司员工的《萨班斯-奥克斯利法》（18 U.S.C.第1514A节）。德国于2023年7月2日颁布了《举报人保护法》（Hinweisgeberschutzgesetz，HinSchG），实施欧盟《举报人指令》，要求50人或以上的组织建立内部报告渠道，对违规行为处以最高50,000欧元的行政罚款。欧盟的《举报人保护指令》（指令(EU) 2019/1937）在所有成员国之间建立了协调最低标准，要求内部和外部报告渠道以及对报复的全面保护，截至2024年，27个成员国中有25个已完成国家转换。

## 韩国

### 法律框架

韩国发展了双支柱举报人保护制度，区分反腐败报告（公共部门）和公共利益报告（私营部门）：

**《防止腐败和ACRC的设立和管理法》（ACRC法，2001，2008年修订）：**

| 条款 | 描述 |
|-----------|-------------|
| **第2(4)条** | 将"腐败"定义为公职人员在履行职责时违反法律谋取私利的行为，或造成公共财产损失的行为 |
| **第55条** | 任何知悉公共部门腐败的人可向ACRC报告 |
| **第56条** | 报告人身份必须保密；披露是刑事犯罪，最高可处3年监禁或3,000万韩元罚款 |
| **第62条** | 保护措施：禁止不利待遇；恢复原职位；纪律延期；保证继续就业 |
| **第64条** | 举报人及其家人的人身安全措施，包括身体保护和搬迁 |
| **第68条** | 奖励：如报告直接有助于追回或增加公共收入或减少支出，最高30亿韩元 |

**《保护公共利益举报人法》（2011，2017、2021年修订）：**

该法将保护扩展至超越公共部门腐败，涵盖约470部具体法律，管理：

- 公共卫生和安全
- 环境保护
- 消费者保护
- 公平竞争
- 食品安全
- 金融监管

| 条款 | 描述 |
|-----------|-------------|
| **第2(1)条** | 将"公共利益违规"定义为违反任何所列法律、侵犯公共卫生、安全、环境、消费者利益或公平竞争的行为 |
| **第6条** | 可向ACRC、调查机关、监督机构或国民大会报告 |
| **第10条** | 身份保护：在整个诉讼过程中必须保持报告人身份的保密 |
| **第15条** | 禁止不利措施：解雇、停职、降职、减薪、调动或其他报复行动 |
| **第17条** | 人身安全措施：对报告人及其亲属的身体保护 |
| **第26条** | 金钱奖励：当公共利益报告增强公共利益时最高5亿韩元，即使没有直接财务追回 |
| **第26-2条** | 救济款项：最高2亿韩元，用于医疗费用、诉讼费、工资损失和其他遭受的损害 |
| **第30条** | 报复的刑事处罚：对不利待遇最高2年监禁或2,000万韩元罚款 |

### 主要原则

**全面奖励结构：** 韩国的奖励制度是全球最慷慨的之一。ACRC管理两类财务激励：

1. **奖励（posang）：** 对直接有助于公共实体财务追回的腐败报告最高30亿韩元。按追回金额的百分比计算。
2. **金钱奖励（porogeum）：** 对公共利益违规报告最高5亿韩元，即使没有直接财务追回也可获得，承认举报的更广泛社会价值。

2002年至2023年间，ACRC支付了超过500亿韩元的总奖励，腐败报告导致5,579人被起诉和5,592人受到纪律处分。

**报告数量增加：** 举报人报告从2011年（《公共利益举报人法》第一年）的2,821起稳步增长到2023年的9,858起，反映了对保护制度的认识和信心不断增强。

### 重要案例

**KEPCO采购欺诈（2014）：** 向ACRC的举报人报告揭露了韩国电力公司系统性采购欺诈，导致多名官员被起诉并追回数十亿韩元的挪用资金。报告人获得了当时的最高奖励。

**国防部预算违规（2017）：** 内部报告揭示了国防采购的系统性预算违规。该案件导致刑事起诉并促成国防采购程序的改革。

**三星生物科学会计欺诈（2018）：** 虽然主要由监管调查驱动，但举报人报告促成了金融服务委员会对三星生物科学涉嫌会计欺诈的调查。该案件突出了公司治理、金融监管和举报人保护的交汇。

## 美国

### 联邦框架

美国拥有世界上最悠久的历史和最复杂的举报人保护制度，包括约六十部联邦法规：

**《虚假申报法》（FCA, 31 U.S.C.第3729-3733节）：**

最初在内战期间（1863）颁布，1986年大幅修订，FCA是美国最有力的举报人法规：

- **Qui Tam条款（第3730(b)节）：** 允许私人个人（关系人）代表联邦政府就对政府的欺诈提起诉讼
- **奖励结构（第3730(d)节）：** 如果政府介入并且案件成功，关系人获得追回的15-25%；如果政府拒绝介入但关系人成功进行诉讼，份额增加到25-30%
- **反报复（第3730(h)节）：** 保护员工免受解雇、降职、停职、威胁、骚扰或歧视，因在FCA诉讼中的合法行为；补救措施包括复职、双倍补发工资和特殊赔偿
- **追回记录：** 1987年至2024年间，FCA诉讼为联邦政府追回超过750亿美元，qui tam案件约占总追回的70%

**《多德-弗兰克华尔街改革和消费者保护法》（2010）：**

- **SEC举报人计划（15 U.S.C.第78u-6节）：** 根据举报人提供的原始信息，对SEC执法行动中收取的超过100万美元的货币制裁给予10-30%的奖励
- 自成立（2011）以来，截至2025年，SEC计划已向举报人授予超过22亿美元
- **CFTC举报人计划：** 类似于SEC的商品和衍生品欺诈计划
- **反报复（第922(h)节）：** 禁止对报告证券违规的员工进行报复；提供复职、双倍补发工资和律师费

**《萨班斯-奥克斯利法》（SOX, 2002）：**

- **第806节（18 U.S.C.第1514A节）：** 保护报告欺诈或证券违规的上市公司员工
- 在180天内向OSHA提出投诉（由Dodd-Frank从90天延长至180天）
- 补救措施：复职、带利息的补发工资、补偿性损害赔偿、律师费
- **第1107节（18 U.S.C.第1513(e)节）：** 报复的刑事处罚：对明知报复提供与联邦犯罪相关的真实信息的人最高10年监禁

**其他主要联邦法规：**

| 法规 | 范围 | 主要特点 |
|---------|-------|-------------|
| **《举报人保护法》（5 U.S.C.第2302节）** | 联邦员工 | 保护披露浪费、欺诈、滥用；OSC和MSPB执法 |
| **《情报界举报人保护法》（1998）** | 情报界员工 | 通过监察长的报告渠道；有限的司法审查 |
| **《国防授权法》（NDAA）条款** | 国防承包商 | 保护披露严重管理不善、欺诈或安全违规 |
| **IRS举报人计划（26 U.S.C.第7623节）** | 税务欺诈 | 对超过200万美元的税务不足信息获得15-30%的收款 |
| **AML举报人计划（2021）** | 反洗钱 | 由AMLA创建；报告BSA违规的奖励 |

### 州差异

许多州颁布了补充举报人保护：

- **加利福尼亚州《劳动法典》第1102.5节：** 禁止对报告涉嫌违反州或联邦法律的员工进行报复；提供复职、补发工资和民事处罚
- **纽约《虚假申报法》：** 针对对州政府欺诈的州级qui tam条款
- **新泽西州《良心员工保护法》（CEPA）：** 最广泛的州举报人法规之一；保护披露或拒绝参与非法或不道德活动的员工

### 重要案例

**United States ex rel. Ven-A-Care of the Florida Keys v. Various Pharmaceutical Companies（1990年代-2000年代）：** 一系列针对主要制药公司Medicaid定价欺诈的FCA qui tam诉讼导致数十亿美元的追回，并将FCA确立为打击医疗欺诈的主要工具。

**Edward Snowden（2013）：** 前NSA承包商Edward Snowden披露了机密监视计划，引发了关于政府监视和举报人保护的全球辩论。Snowden根据不提供公共利益抗辩的《间谍法》（18 U.S.C.第793节）被起诉，突出了情报界举报人保护与一般举报人法规之间的差距。Snowden于2025年1月获得总统赦免。

**Frances Haugen——Facebook/Meta（2021）：** 前Facebook数据科学家Frances Haugen向SEC和国会披露了内部文件，揭示该公司意识到其平台造成的危害。她根据Dodd-Frank举报人条款提出投诉，并在参议院商务委员会作证，引发了关于社交媒体监管的立法讨论。

## 德国

### 法律框架

德国在全面举报人保护方面相对较晚，于2023年7月2日颁布了《举报人保护法》（Hinweisgeberschutzgesetz，HinSchG），实施欧盟《举报人指令》：

**《举报人保护法》（HinSchG）：**

| 条款 | 描述 |
|-----------|-------------|
| **第1节** | 目的：保护在其职业活动中获得违规信息并报告的自然人 |
| **第2节** | 范围：涵盖报告违反欧盟法律、联邦刑事条款、涉及罚款的行政违规以及特定列举的法规 |
| **第3节** | 受保护人员：员工、公务员、学员、自营职业者、股东、志愿者、供应商和求职者 |
| **第7节** | 要求50+员工的组织建立内部报告渠道的义务 |
| **第8节** | 内部报告渠道必须允许书面和口头报告；必须保持报告人身份的保密 |
| **第12节** | 7天内确认收到；3个月内后续回应 |
| **第16节** | 向联邦司法办公室（Bundesamt fur Justiz）或主管联邦/州机关的外部报告 |
| **第28节** | 禁止任何形式的报复：解雇、降职、调动、骚扰、黑名单、扣留推荐 |
| **第33节** | 举证责任倒置：如果报告后发生不利情况，雇主必须证明与报告无关 |
| **第36节** | 赔偿：对报复造成的物质和非物质损害有权获得赔偿 |
| **第40节** | 行政罚款：阻碍报告、报复或违反保密最高50,000欧元；未建立报告渠道最高20,000欧元 |

**先前法律框架（HinSchG之前）：**

在HinSchG之前，德国举报人保护是零碎的，来源于：

- **《民法典》（BGB）第612a节：** 对雇主因行使合法权利而进行报复的一般禁令
- **《解雇保护法》（KSchG）：** 保护免受因行使法律权利（包括报告违规）而被解雇
- **联邦宪法法院判例：** Bundesverfassungsgericht在几项判决中（特别是BVerfG, 1 BvR 2049/00 (2001)）承认员工有权向当局报告犯罪行为，但须经过比例分析，平衡忠诚义务与公共利益

**《商业秘密法》（GeschGehG, 2019）：**

GeschGehG第5节规定，当获取、使用或披露商业秘密服务于合法利益的保护，特别是为了发现非法活动的目的时，是合法的。该条款为在报告违规过程中披露商业秘密的举报人提供了重要的安全港。

### 主要原则

**无匿名渠道义务：** 与某些司法管辖区不同，HinSchG不要求组织建立匿名报告渠道。但是，如果收到匿名报告，必须处理。许多组织自愿实施了匿名数字报告系统。

**内部和外部报告的平等：** HinSchG允许举报人选择内部或外部报告，而不要求首先进行内部报告。这是立法过程中的一项刻意政策选择，因为早期草案曾提议强制内部优先报告。

**范围限制：** HinSchG超越欧盟指令的最低要求，涵盖违反所有联邦刑法和行政违规条款，而不仅仅是欧盟法律的违反。这种扩展范围大大扩大了可报告行为的范围。

### 重要案例

**Wirecard AG（2020）：** Wirecard丑闻——DAX上市的支付处理器揭示了其资产负债表上19亿欧元的漏洞——暴露了BaFin监管监督的灾难性失败。记者Dan McCrum和各种内部举报人在崩溃前几年就提出了担忧。这一丑闻是加强德国举报人保护框架的主要催化剂，并为HinSchG提供了政治动力。

**Cum-Ex税务欺诈丑闻（2011年至今）：** 德国最大的金融丑闻之一，涉及股息预扣税的欺诈性追回，部分通过举报人披露暴露。斯图加特律师Eckart Seith向税务机关提供了关键信息。该案涉及估计超过100亿欧元的损害，并导致许多刑事起诉，包括2022年对Hanno Berger的定罪。该案突出了德国HinSchG之前保护的不足。

**Deutsche Bank——LIBOR操纵：** 举报人披露促成了对Deutsche Bank在LIBOR操纵丑闻中作用的调查。该银行向美国和欧洲监管机构支付了约25亿美元的罚款（2015）。该案表明了跨境举报人保护的重要性。

## ECHR / 欧盟框架

### 《公约》条款

- **第10条（表达自由）：** ECHR承认举报人披露可能构成第10条下受保护的表达，须经过比例分析。
- **第8条（尊重私人生活权）：** 与举报人身份的保密以及免受影响私人生活的工作场所报复的保护有关。

### 主要案例法

**Guja v. Moldova, App. No. 14277/04 (大法庭, 2008年2月12日)：**

大法庭建立了一个六要素测试，用于评估举报人的披露是否受第10条保护：

1. 申请人是否有其他披露渠道
2. 披露信息中的公共利益
3. 披露信息的真实性
4. 披露对雇主/当局造成的损害
5. 申请人是否出于善意行事
6. 对举报人施加的制裁的严重程度

这一框架已成为ECHR举报人保护案件的基础测试。

**Heinisch v. Germany, App. No. 28274/08 (2011年7月21日)：**

一家国营养老院的护士提出刑事投诉，指控护理实践存在缺陷并被解雇。法院认定违反第10条，认为：

- 关于老年人机构护理缺陷的信息中的公共利益具有重大意义
- 该员工在去外部之前首先使用了内部报告渠道
- 信息基本上是准确的
- 解雇是不相称的制裁

**Halet v. Luxembourg, App. No. 21884/18 (大法庭, 2023年2月14日)：**

Raphael Halet，"LuxLeaks"举报人之一，披露了机密的普华永道税务文件，揭示卢森堡对跨国公司的优惠税务裁决。大法庭（12-5）认定违反第10条，大大扩展了Guja框架：

- 法院认为披露信息中的公共利益延伸至违法行为之外，包括"虽然合法但应受谴责的做法"
- 对信息"本质"性质的评估必须从公众而非雇主的角度进行
- 法院强调举报人保护与新闻界类似，具有"看门狗"功能

**Bucur and Toma v. Romania, App. No. 40238/02 (2013)：**

一名军事情报官员披露了罗马尼亚情报部门非法窃听的信息。法院认定违反第10条，指出与非法监视披露相关的压倒性公共利益。

### 成员国义务

**欧盟《举报人保护指令》（指令(EU) 2019/1937）：**

于2019年10月23日通过，转换截止日期为2021年12月17日（对于50-249名员工的实体延长至2023年12月17日），该指令建立了协调的最低标准：

| 要求 | 描述 |
|------------|-------------|
| **内部报告渠道** | 所有50+员工的私营和公共组织强制 |
| **外部报告渠道** | 成员国必须指定接收和跟进报告的主管机关 |
| **公开披露** | 以下情况受保护：(a)报告人首先使用了内部/外部渠道但未采取适当行动；(b)对公共利益的紧迫或明显危险；(c)报复或无效补救的风险 |
| **个人范围** | 工人、自营职业者、股东、志愿者、学员、求职者、前员工 |
| **实质范围** | 违反指定领域欧盟法律：公共采购、金融服务、产品安全、运输安全、环境保护、食品安全、公共卫生、消费者保护、数据保护等 |
| **禁止报复** | 禁止报复行为的全面清单：解雇、降职、扣留培训、负面绩效评估、提前终止合同、黑名单、精神病/医疗转诊 |
| **举证责任** | 倒置：雇主必须证明任何不利措施是基于充分的正当理由 |
| **保密** | 必须保持报告人身份的保密，除非法律要求披露 |
| **后续义务** | 7天内确认；3个月内反馈 |

**实施状态（截至2024年）：**

27个欧盟成员国中有25个已转换该指令。欧盟委员会对滞后国家启动了违约程序。值得注意的国家实施：

- **法国：** Loi Sapin II（2016）早于该指令；由Loi no. 2022-401更新以符合指令要求
- **荷兰：** Wet bescherming klokkenluiders（2023）——以超出指令最低要求的扩展范围实施
- **意大利：** 第24/2023号立法令——在某些行业对少于50名员工的实体实施
- **瑞典：** 《保护报告不当行为人员法》（2021:890）

在全球范围内，日本的《举报人保护法》（2004，2020年修订）要求300+员工的公司建立内部报告渠道并扩大了受保护披露的范围。澳大利亚的《2013年公共利益披露法》（联邦）保护关于政府机构的披露，《2019年财政部法律修正案（增强举报人保护）法》将保护扩展到企业部门。巴西的《反腐败法》（Lei 12,846/2013）和最近的Lei 14,457/2022要求内部合规渠道。印度的《2014年举报人保护法》涵盖关于公务员的披露，但缺乏私营部门覆盖。

## 比较分析

| 维度 | 韩国 | 美国 | 德国 | 欧盟指令 |
|-----------|------------|---------------|---------|-------------|
| **奖励制度** | 是（最高30亿韩元） | 是（FCA 15-30%；Dodd-Frank 10-30%） | 无 | 非必需 |
| **内部报告义务** | 无——允许直接外部 | 因法规而异 | 无——选择渠道 | 鼓励，非强制 |
| **举证责任（报复）** | 雇主（初步证据后） | 因情况而异；贡献因素测试（SOX） | 雇主（HinSchG第33节） | 雇主 |
| **保密保护** | 对披露的刑事处罚 | 因情况而异；Dodd-Frank下强大 | 行政罚款；某些违反的刑事 | 强制；具体情况留给成员国 |
| **报复的刑事处罚** | 是（最高2年） | 是（SOX第1107节：最高10年） | 仅行政罚款 | 成员国确定 |
| **私营部门覆盖** | 是（《公共利益举报人法》） | 广泛（FCA、Dodd-Frank、SOX） | 是（HinSchG） | 是 |
| **匿名报告** | 接受 | 接受 | 不要求启用 | 通过外部渠道要求 |
| **有效性排名（2014年OECD）** | 高 | 高 | 中等（HinSchG之前） | N/A（指令晚于研究） |
