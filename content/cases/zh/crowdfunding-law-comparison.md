## 摘要

众筹监管——即规范通过互联网平台向大量个人募集小额资金以支持商业项目、创意计划或社会公益的法律框架——自2010年代初以来随着监管机构寻求在投资者保护与初创企业和中小企业融资之间取得平衡而迅速发展。韩国通过2015年修订的《资本市场与金融投资业务法》（자본시장과 금융투자업에 관한 법률，通称《资本市场法》）及其实施细则监管股权众筹，由金融委员会（금융위원회，FSC）管理，允许持牌众筹中介机构（크라우드펀딩 중개업자）在个人投资限额和发行人额度范围内促进证券发行。美国通过2012年《创业企业促进法》（JOBS Act）第三篇创建了全面的联邦众筹框架，编纂为1933年《证券法》第4(a)(6)条（15 U.S.C. § 77d(a)(6)），并通过SEC众筹条例（17 C.F.R. §§ 227.100-227.503）实施，该条例于2021年3月进行了重大自由化改革，将发行上限从107万美元提高到500万美元。德国最初通过2015年《小额投资者保护法》（Kleinanlegerschutzgesetz）和《资产投资法》（Vermögensanlagengesetz）监管众筹，但此后已过渡到欧盟根据条例（EU）2020/1503建立的统一框架，即《欧洲众筹服务提供商条例》（ECSPR），该条例于2023年11月10日全面适用，为借贷型和投资型众筹平台建立了单一的欧盟范围许可制度，最高发行门槛为12个月内500万欧元，同时欧洲人权法院已就证券监管与财产权和企业自由基本权利之间的紧张关系进行了审理。

## 韩国

### 法律框架

韩国的股权众筹框架通过2015年修订《资本市场法》（자본시장과 금융투자업에 관한 법률）引入，于2016年1月25日生效。该框架旨在为初创企业和中小企业创建替代融资渠道，同时维护投资者保护。

**符合条件的发行人：** 有资格进行众筹的公司须为存续不超过7年的未上市中小企业（某些技术公司可延长至10年）。特定排除行业（金融服务、房地产投机）的发行人不符合资格。

**众筹中介机构（크라우드펀딩 중개업자）：** 平台须向FSC注册为众筹中介机构，并须满足最低资本要求5亿韩元。截至2025年，约有15家注册平台在韩国运营，包括Wadiz（와디즈）、Open Trade和Crowdy。

### 投资限额

该框架根据投资者专业程度采用分级投资限额：

| 投资者类别 | 单个发行人限额 | 年度总限额 |
|-----------|--------------|-----------|
| 普通投资者（일반투자자） | 500万韩元 | 1000万韩元 |
| 收入合格投资者（소득적격투자자） | 1000万韩元 | 2000万韩元 |
| 专业投资者（전문투자자） | 无限额 | 无限额 |

**收入合格投资者**须证明年收入至少1亿韩元或金融资产至少2亿韩元。

### 披露与投资者保护

发行人须向FSC提交标准化发行文件，包含公司信息、管理层、财务状况、商业计划和风险因素。发行文件须在发行截止前至少10个工作日通过平台向投资者公开。

**冷静期：** 投资者可在投资后7个工作日内无罚款撤回投资承诺。

**锁定期：** 通过众筹购买的证券自发行截止日起受6个月的转让限制，之后可在平台公告板或其他允许的渠道上交易。

### 奖励型和捐赠型众筹

奖励型和捐赠型众筹在韩国不受证券法特别监管，因为它们不涉及金融工具的发行。但是，促进奖励型众筹的平台须遵守一般消费者保护法律，包括《电子商务消费者保护法》（전자상거래 등에서의 소비자보호에 관한 법률）和《电子金融交易法》（전자금융거래법）。

## 美国

### 法律框架

美国通过2012年JOBS Act第三篇建立了首个全面的联邦众筹框架，该法在1933年《证券法》中增加了第4(a)(6)条，为某些众筹交易创建了证券注册豁免。

**众筹条例（17 C.F.R. Part 227）：** SEC的实施规则最初于2015年10月30日通过（2016年5月16日生效），并于2021年3月15日进行了重大修订，建立了运营框架：

**发行限额（2021年3月修订后）：**
- 最高募集总额：12个月内500万美元（从107万美元提高）
- 2021年修订代表了近五倍的增长，极大地扩展了该豁免的实用性

**投资限额：**

| 投资者年收入与净资产 | 最高投资额（12个月内所有Reg CF发行的总计） |
|---------------------|------------------------------------------|
| 收入或净资产低于124,000美元 | 2,500美元或收入/净资产较低者的5%，取较大值 |
| 收入和净资产均达124,000美元及以上 | 收入/净资产较低者的10%，上限124,000美元 |
| 合格投资者 | 无限额 |

**融资门户：** 中介机构可根据《证券法》第4A(a)条（15 U.S.C. § 77d-1(a)）在SEC注册为"融资门户"，这是一种比完整经纪商注册更轻便的监管类别。融资门户还须在FINRA注册。主要平台包括Wefunder、StartEngine和Republic。

### 披露要求

| 发行规模 | 财务报表要求 |
|---------|------------|
| 不超过124,000美元 | 由首席执行官认证的财务报表 |
| 124,001-618,000美元 | 由独立公共会计师审阅的财务报表 |
| 618,001-1,235,000美元 | 经审阅的财务报表（如此前发行超过1,235,000美元则需审计） |
| 超过1,235,000美元 | 经审计的财务报表 |

发行人向SEC提交C表格，包括业务描述、高管和董事、资金用途、目标金额、截止日期、所有权和资本结构以及风险因素。须提交C-AR表格年度报告，直至发行人成为《交易法》报告公司、记录持有人少于300人或回购所有通过众筹条例出售的证券。

### Regulation A+与Regulation D比较

众筹条例与其他资本募集豁免并存：

| 特征 | Reg CF | Reg A+第一级 | Reg A+第二级 | Reg D 506(c) |
|------|--------|-------------|-------------|-------------|
| 最高发行额 | 500万美元 | 2000万美元 | 7500万美元 | 无限制 |
| 投资者限额 | 是 | 否 | 是（非合格投资者10%） | 仅限合格投资者 |
| SEC申报 | C表格 | 1-A表格 | 1-A表格 | D表格 |
| 公开招揽 | 是 | 是 | 是 | 是 |
| 转售限制 | 12个月 | 无 | 无 | 6-12个月 |

## 德国

### 法律框架

德国的众筹监管经历了从国家框架到欧盟统一制度的重大转变。

**ECSPR之前的框架（2015-2023年）：** 2015年《小额投资者保护法》（Kleinanlegerschutzgesetz）修订了《资产投资法》（VermAnlG），创建了众筹特定框架。主要规定包括：
- 发行不超过600万欧元的招股说明书豁免（VermAnlG § 2a）
- 每次发行个人投资上限10,000欧元
- 超过1,000欧元投资的自我声明要求
- Vermögensanlagen-Informationsblatt（VIB）——最多3页的简化投资信息表

**ECSPR过渡：** 条例（EU）2020/1503于2021年11月10日开始适用，现有平台的过渡期至2023年11月10日。BaFin现在根据ECSPR框架向众筹服务提供商发放许可。

**关键ECSPR条款（在德国的适用）：**

| 条款 | 要求 |
|------|------|
| 第3条 | 须获得国家主管机关（BaFin）授权 |
| 第7条 | 最低审慎保障措施（25,000欧元或未偿还贷款的0.2%） |
| 第21条 | 关键投资信息表（KIIS）——标准化6页格式 |
| 第22条 | 非专业投资者的入门知识测试和损失承受能力模拟 |
| 第23条 | 投资者知识测试的最低门槛1,000欧元 |
| 第24条 | 平台可提供二级交易公告板 |
| 第25条 | 跨境通行证：单一授权在所有欧盟/欧洲经济区国家有效 |

### 市场数据

德国拥有欧洲最大的众筹市场之一。根据剑桥替代金融中心的数据，德国的众筹和点对点借贷市场在2023年达到约18亿欧元。主要平台包括Seedmatch（股权）、Exporo（房地产）和Companisto（股权）。

### 房地产众筹

房地产众筹在德国特别受欢迎，通常采用次级参与利润贷款（partiarische Nachrangdarlehen）的结构。数起高调的平台失败案例，包括Exporo支持项目的倒闭和Zinsbaustein的破产，促使BaFin加强了对房地产众筹平台及其尽职调查实践的审查。

## 欧洲人权法院/欧盟

### 欧洲众筹服务提供商条例（ECSPR）

2020年10月7日欧洲议会和欧盟理事会的条例（EU）2020/1503为欧盟各国的众筹服务提供商建立了统一框架：

**范围：** ECSPR涵盖借贷型众筹（点对点借贷）和投资型众筹（股权和债务证券）。奖励型和捐赠型众筹不在其范围之内。

**授权和通行证：** 众筹服务提供商须根据第3条从其母国的国家主管机关获得授权。获得授权后，他们将获得欧洲通行证，可根据第18条在所有欧盟/欧洲经济区成员国提供跨境服务，无需在东道国获得额外授权。

**投资者保护框架：**

| 保护措施 | ECSPR条款 | 描述 |
|---------|-----------|------|
| 知识测试 | 第21条 | 评估投资者对众筹风险的理解 |
| 损失模拟 | 第21条 | 投资者须证明能够承受潜在损失 |
| 冷静期 | 第22条 | 4个日历日内可撤回要约或意向表示 |
| KIIS | 第23条 | 每个项目标准化6页关键投资信息表 |
| 个人投资组合管理 | 第6条 | 可选；平台代表投资者管理多元化投资组合 |

**发行门槛：** 第1(2)(c)条将ECSPR的范围限制在每个项目所有者12个月内不超过500万欧元的发行。超过此门槛的发行须根据《招股说明书条例》（EU）2017/1129提交完整招股说明书。

### ESMA监管技术标准

欧洲证券和市场管理局（ESMA）已根据ECSPR通过了广泛的监管技术标准（RTS）和实施技术标准（ITS）：
- 关于业务连续性计划的RTS（第12(16)条）
- 关于贷款个人投资组合管理的RTS（第6(7)条）
- 关于信用风险评估框架的RTS（第19(7)条）
- 关于入门知识测试的RTS（第21(8)条）
- 关于关键投资信息表格式的ITS（第23(16)条）

### 欧洲人权法院相关考量

虽然欧洲人权法院尚未直接审理众筹监管案件，但更广泛的金融监管判例中出现了相关原则：

**第1议定书第1条（财产保护）：** 限制个人投资能力或企业融资能力的证券监管构成对财产权的干预，需要符合比例原则。在金融市场监管方面赋予各国的广泛裁量余地意味着，除非众筹限制明显不成比例，否则不太可能违反公约。

**第10条（表达自由）：** 众筹发行的广告限制对商业表达自由具有潜在影响。然而，欧洲人权法院在金融服务领域一贯支持对商业言论的合比例限制。

**联合案件C-370/19和C-37/20，Commission v. United Kingdom（CJEU）：** 虽然涉及的是脱欧后的监管分歧而非众筹本身，但该案例说明了欧盟法院将金融监管视为欧盟内部市场权限合法行使的方法。

## 比较分析

### 监管理念

四个司法管辖区反映了不同的监管理念：

**韩国：** 采取保守方法，发行和投资限额相对较低，反映了在零售参与度高、替代投资领域投资者专业程度有限的市场中对投资者保护的关注。

**美国：** 最初持保守态度（最初107万美元的上限使Reg CF在很大程度上不实用），2021年改革采取了更宽松的方法，显著增加了市场活跃度。基于豁免的方法（第4(a)(6)条注册豁免）允许在不进行完整证券注册的情况下保持灵活性。

**德国/欧盟：** ECSPR代表了一种中间立场，在投资者准入与标准化保护之间取得平衡。单一通行证消除了此前欧洲众筹监管的碎片化。

### 市场影响

众筹市场发展与监管设计密切相关：

- **美国：** 2021年改革后市场出现爆发式增长，2023年Reg CF年度发行额超过7亿美元，高于2020年约2.39亿美元
- **韩国：** 市场规模仍然较小，年约500亿韩元，受低发行上限制约
- **欧盟/德国：** 预计ECSPR将增加跨境活动，尽管早期实施面临平台合规挑战

### 未解决的问题

所有司法管辖区面临共同挑战：
1. **二级市场流动性：** 尽管有公告板条款，众筹证券基本上仍然缺乏流动性
2. **信息不对称：** 简化的披露要求可能无法充分告知投资者初创企业风险
3. **平台失败风险：** 众筹平台的破产引发投资者保护问题
4. **跨境执法：** 不同的监管标准造成监管套利机会

### 代币化和基于区块链的众筹

众筹与区块链技术的交叉正在带来新的监管挑战：

**证券型代币发行（STOs）：** 通过众筹豁免发行的代币化证券将众筹的优势（更广泛的投资者准入）与区块链特性（可编程证券、自动化合规、碎片化所有权）相结合：
- **美国：** STOs可适用众筹条例、Regulation A+或Regulation D；SEC未创建单独的STO框架
- **欧盟：** MiCA不涵盖证券型代币；现有证券监管（MiFID II、《招股说明书条例》）适用；DLT试点制度允许实验
- **韩国：** FSC发布了将证券型代币视为《资本市场法》下受监管证券的指引
- **德国：** Elektronische Wertpapiergesetz（eWpG，2021年）允许发行电子和加密证券

**房地产代币化：** 通过代币化众筹实现的房地产碎片化所有权已成为重要的市场细分：
- 德国房地产众筹市场年超10亿欧元
- 韩国允许通过证券型众筹进行房地产投资
- 美国平台如RealT和Republic通过Regulation D/Regulation A+豁免提供代币化房地产

### 社会影响和社区众筹

除投资回报外，众筹日益服务于社会政策目标：

| 类型 | 特征 | 监管处理 |
|------|------|---------|
| 社区股份发行 | 当地社区对社会企业的投资 | 英国：FCA监管；德国：合作社法；美国：Regulation CF |
| 社会债券 | 与社会影响挂钩的金融回报 | 一般作为证券监管 |
| 公民众筹 | 公民资助公共项目 | 一般不受监管（捐赠型） |
| 科学众筹 | 公众支持者资助研究 | 各地不同；通常为捐赠型/奖励型 |

### 跨境众筹

尽管有监管协调努力，跨境众筹仍然面临挑战：

- **欧盟：** ECSPR创建了单一通行证，允许在一个成员国获得授权的众筹服务提供商在整个欧盟运营；然而，实际障碍仍然存在，包括语言要求、本地支付基础设施和不同的投资者期望
- **韩国：** 跨境众筹实际上受到《资本市场法》许可要求和韩语披露义务的限制
- **美国：** 众筹条例和Regulation A+发行一般仅限于国内发行人；外国非公开发行人面临重大合规负担
- IOSCO关于众筹监管的报告（2015年，2023年更新）认为监管碎片化是跨境众筹增长的主要障碍

### 投资者保护成果

众筹投资的投资者回报数据仍然有限，但现有证据表明：
- **美国：** 2016年以来约50%的Reg CF发行已失败或停止运营，尽管早期阶段投资本身就具有高失败率
- **德国：** 房地产众筹经历了显著的违约，包括数个Exporo支持项目
- **韩国：** KDIC不覆盖众筹投资；投资者损失完全由投资者承担
- 所有司法管辖区都强调投资组合多元化和风险披露在众筹投资中的重要性

### 众筹工具的二级市场

众筹投资缺乏流动性促使监管机构关注二级市场发展：

- **欧盟：** ECSPR允许众筹服务提供商运营众筹工具买卖公告板，但这远非完整的二级市场；DLT试点制度使代币化众筹工具能够在基于分布式账本的平台上交易
- **美国：** Reg CF工具受12个月转售限制；SEC已批准替代交易系统（ATS）用于Reg CF和Reg A+证券的二级交易；StartEngine Secondary和Securitize Markets等平台提供有限的二级流动性
- **韩国：** 不存在专门的众筹证券二级市场；FSC已探索将众筹工具与K-OTC非上市证券市场对接的可能性
- **德国：** BaFin已根据MiFID II投资公司许可授权数个平台进行众筹工具的二级交易
