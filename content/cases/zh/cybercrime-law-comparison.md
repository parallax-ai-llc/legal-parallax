## 概述

网络犯罪——以计算机、网络和数字系统为目标或利用其实施的犯罪活动——已成为全球增长最快的刑法领域之一，然而各国在法律范围、刑罚和执法机制方面存在显著差异。韩国主要通过《促进信息和通信网络利用及信息保护等相关法律》（정보통신망 이용촉진 및 정보보호 등에 관한 법률，通称《网络法》或정보통신망법）来应对网络犯罪，对传播恶意软件处以最高七年有期徒刑，对未经授权入侵系统处以最高五年有期徒刑。美国依赖《计算机欺诈和滥用法》（CFAA，18 U.S.C. section 1030），该法于1986年颁布并经多次修订，刑罚根据所造成损害的严重程度从一年到终身监禁不等。德国通过《刑法典》（StGB）的若干条款将网络犯罪定为刑事犯罪，特别是第202a条至第202c条（数据间谍和拦截）和第303a条至第303b条（数据篡改和计算机破坏），严重计算机破坏案件最高可处十年有期徒刑。在国际层面，2001年《布达佩斯网络犯罪公约》是首个关于网络犯罪的多边条约，为将网络犯罪行为定为犯罪和促进跨境合作提供了统一框架，截至2025年已有八十一个缔约国。四个法域均在应对勒索软件、深度伪造技术和人工智能犯罪带来的新兴挑战，促使各方全面更新立法。

## 韩国

### 法律框架

韩国的网络犯罪框架建立在多部相互关联的法律之上。主要法律工具是《促进信息和通信网络利用及信息保护等相关法律》（정보통신망법），将未经授权的访问、数据窃取、恶意软件传播和系统破坏定为犯罪。《刑法》（형법）提供补充条款，专门法律则针对特定类型的网络犯罪。

《网络法》的主要条款包括：

- **第48条第1款——未经授权入侵：** 任何人不得未经合法访问权限侵入信息和通信网络。违反者处五年以下有期徒刑或五千万韩元（约三万八千美元）以下罚金。
- **第48条第2款——传播恶意软件：** 任何人不得传输或传播可能导致信息和通信系统中断或破坏的恶意程序。违反者处七年以下有期徒刑或七千万韩元（约五万三千美元）以下罚金。
- **第48条第3款——系统破坏：** 任何人不得损坏他人信息或破坏他人信息和通信网络的运行。违反者处五年以下有期徒刑或五千万韩元以下罚金。
- **第49条——数据挪用：** 任何人不得损坏、销毁、篡改、伪造或泄露通过信息和通信网络处理、存储或传输的信息。违反者处五年以下有期徒刑或五千万韩元以下罚金。

### 深度伪造和人工智能相关犯罪

韩国对深度伪造犯罪采取了积极立场。2024年9月，国会修订了《性犯罪处罚特别法》，专门针对人工智能生成的深度伪造色情内容：

- **制作或传播**深度伪造性内容：最高七年有期徒刑。
- **持有或观看**此类内容：最高三年有期徒刑或三千万韩元（约两万两千六百美元）以下罚金。

政府启动了一项持续至2026年10月的专项打击行动，针对非法深度伪造内容的制作者、传播者和消费者。此外，《人工智能法》（定于2026年1月生效）要求对人工智能生成内容进行强制标注：任何创建或编辑人工智能生成内容并在网上发布的人必须明确标明其合成来源。

### 勒索软件

韩国虽然没有独立的勒索软件法律，但勒索软件攻击依据《网络法》第48条和第49条（恶意软件传播和系统破坏）以及《刑法》敲诈勒索条款进行起诉。韩国互联网振兴院（KISA）负责协调事件应对和支持执法调查。

### 里程碑案例

**"N号房"行动（2019-2020年）：** 赵主彬运营了一个Telegram聊天室网络，传播性剥削深度伪造和真实内容，通过数字勒索胁迫受害者。他于2022年被判处四十二年有期徒刑。该案催化了韩国对数字性犯罪和深度伪造的立法应对。

**Upbit加密货币交易所黑客攻击（2019年）：** 黑客从韩国最大加密货币交易所窃取了约三十四万两千枚以太币（当时价值约五千万美元）。韩国和国际调查人员将该攻击归因于朝鲜拉撒路组。该案凸显了网络犯罪中的跨境执法挑战。

## 美国

### 法律框架

《计算机欺诈和滥用法》（CFAA），编纂于 18 U.S.C. section 1030，是起诉网络犯罪的主要联邦法律。该法于1986年作为1984年《综合犯罪控制法》的修正案颁布，经多次修订，最近一次是2008年，以应对不断演变的威胁。CFAA将七类行为定为犯罪：

1. **Section 1030(a)(1)：** 未经授权访问计算机以获取国家安全信息（间谍活动）。
2. **Section 1030(a)(2)：** 未经授权访问以获取金融机构、政府计算机或任何受保护计算机的信息。
3. **Section 1030(a)(3)：** 未经授权访问非公开政府计算机。
4. **Section 1030(a)(4)：** 明知故犯地访问受保护计算机，以欺诈为目的并获取有价值的东西。
5. **Section 1030(a)(5)：** 明知故犯地通过传输代码、程序或命令对受保护计算机造成损害（涵盖恶意软件、勒索软件和DDoS攻击）。
6. **Section 1030(a)(6)：** 贩卖计算机密码或类似访问凭证。
7. **Section 1030(a)(7)：** 涉及威胁损坏受保护计算机或泄露被盗数据的勒索行为（直接适用于勒索软件）。

### 刑罚

| 罪行 | 初犯 | 再犯 |
|---------|--------------|----------------|
| 未经授权访问获取信息 (a)(2) | 最高一年（轻罪）或五年（重罪） | 最高十年 |
| 计算机欺诈 (a)(4) | 最高五年 | 最高十年 |
| 故意损坏受保护计算机 (a)(5)(A) | 最高十年 | 最高二十年 |
| 疏忽造成严重人身伤害的损害 (a)(5)(A) | 最高二十年 | 最高二十年 |
| 造成死亡的损害 | 最高终身监禁 | 最高终身监禁 |
| 通过计算机威胁进行勒索 (a)(7) | 最高五年 | 最高十年 |
| 密码贩卖 (a)(6) | 最高一年或十年 | 最高二十年 |

CFAA下的罚金每项罪名最高可达二十五万美元，或犯罪所得或所致损失的两倍。

### 勒索软件

勒索软件依据多项联邦法律进行起诉，包括CFAA section 1030(a)(5)（损坏受保护计算机）、section 1030(a)(7)（勒索）、《经济间谍法》（18 U.S.C. sections 1831-1839）和电信欺诈（18 U.S.C. section 1343）。司法部于2021年设立了专门的勒索软件和数字勒索工作组以协调联邦执法。2021年5月签署的第14028号行政令加强了对政府承包商和软件供应链的联邦网络安全要求。

### 深度伪造和人工智能相关犯罪

美国缺乏全面的联邦人工智能网络犯罪法律，但存在若干针对性措施：

- **《TAKE IT DOWN法》（2025年）：** 于2025年5月19日签署成为法律，将明知公布未经同意的私密视觉描绘（包括人工智能生成的深度伪造）定为犯罪。涉及成年人的犯罪最高处两年监禁，涉及未成年人的加重处罚。
- **州法律：** 超过四十个州已颁布或提议了与深度伪造相关的立法，涵盖选举虚假信息、未经同意的私密图像和欺诈。
- **现有联邦工具：** 在全面联邦立法出台前，检察官使用电信欺诈（18 U.S.C. section 1343）、身份盗窃（18 U.S.C. section 1028）和CFAA起诉人工智能网络犯罪。

### 里程碑案例

**United States v. Van Buren (2021)：** 最高法院以六比三裁定，CFAA的"超越授权访问"条款仅适用于访问其无权访问的计算机区域信息的人，而不适用于滥用其有权访问的信息的人。该判决缩小了CFAA的适用范围，解决了长期存在的巡回法院分歧。

**科洛尼尔管道公司勒索软件攻击（2021年）：** DarkSide勒索软件组织攻击了美国东部最大的燃料管道科洛尼尔管道公司，迫使其停运六天并造成大范围燃料短缺。科洛尼尔以比特币支付了四百四十万美元赎金；司法部随后追回了约二百三十万美元。该攻击加速了联邦勒索软件执法和政策制定。

**United States v. Astamirov and Vasiliev（LockBit案，2023-2024年）：** 联邦当局瓦解了全球最活跃的勒索软件即服务组织之一LockBit的运作，并对多人提起指控。该行动涉及美国、英国和欧洲刑警组织的协调行动，展示了国际合作在打击勒索软件中的重要性。

## 德国

### 法律框架

德国通过《刑法典》（StGB）的若干条款处理网络犯罪，分为两个主要类别：

**数据间谍和拦截（第202a条至第202d条）：**

- **第202a条——数据间谍（Ausspähen von Daten）：** 非法获取非针对本人且受到特别保护以防止未经授权访问的数据，通过规避该保护获取者，处三年以下有期徒刑或罚金。
- **第202b条——数据拦截（Abfangen von Daten）：** 通过技术手段从非公开数据传输或电磁辐射中拦截非针对本人的数据者，处两年以下有期徒刑或罚金。
- **第202c条——预备数据间谍和拦截（Vorbereiten des Ausspähens und Abfangens von Daten）：** 通过制造、获取、出售或分发密码、安全代码或目的在于实施第202a条或第202b条犯罪的计算机程序来预备此类犯罪者，处两年以下有期徒刑或罚金。
- **第202d条——处理被盗数据（Datenhehlerei）：** 2015年引入，将获取或转让他人通过违法行为获得的非一般可访问数据的行为定为犯罪，处三年以下有期徒刑或罚金。

**数据篡改和计算机破坏（第303a条至第303b条）：**

- **第303a条——数据篡改（Datenveränderung）：** 非法删除、压制、使之不可用或篡改数据者，处两年以下有期徒刑或罚金。
- **第303b条——计算机破坏（Computersabotage）：** 通过实施第303a条犯罪行为、以造成损害为目的输入或传输数据、或破坏、损坏、使之不可用、拆除或篡改数据处理系统来干扰对他人具有重大意义的数据处理者，处三年以下有期徒刑或罚金。在特别严重的情况下——尤其是涉及商业运营、政府机构或重要公共服务的——处六个月至十年有期徒刑。

### 深度伪造和人工智能相关犯罪

德国正在制定针对深度伪造的专门刑事条款：

- **拟议第201b条StGB——通过数字伪造侵犯人格权：** 联邦参议院于2025年8月27日向联邦议院提交了一项法案草案，提议新设一项犯罪，涵盖滥用侵犯人格权的人工智能生成或篡改的视听媒体。标准情况下拟处两年以下有期徒刑，严重情况下最高五年。为艺术、科学和讽刺等正当利益提供例外。
- **现有空白：** 现行第201条（侵犯口头言论保密性）和第201a条（通过图像侵犯私密隐私）要求真实录音录像，不涵盖人工智能生成内容，造成了拟议第201b条旨在填补的法律空白。
- **欧盟人工智能法实施：** 德国正在实施欧盟《人工智能法》（Regulation 2024/1689），该法于2026年8月2日全面实施。该法对深度伪造内容施加透明义务，并按风险等级对某些人工智能应用进行分类。

### 勒索软件

勒索软件攻击依据第303a条（数据篡改）、第303b条（计算机破坏）和第253条StGB（敲诈勒索/Erpressung）进行起诉。联邦刑事警察局（BKA，Bundeskriminalamt）主导勒索软件调查，通过既有的司法互助渠道与欧洲刑警组织和联邦调查局协调。

### 安全研究人员保护

2024年，德国政府起草了立法以保护安全研究人员在负责任地披露漏洞时免受第202a条至第202c条的刑事起诉。该草案旨在回应第202c条宽泛措辞对合法网络安全研究产生的寒蝉效应的批评。

### 里程碑案例

**联邦议院黑客攻击（2015年）：** 德国联邦议院遭受了归因于俄罗斯APT28（Fancy Bear）组织的重大网络攻击。攻击者窃取了约十六吉字节数据，包括议员的电子邮件。德国于2020年对格鲁乌军官德米特里·巴丁发出逮捕令，但尚未将其逮捕。该案突显了对国家支持网络行动进行溯源和起诉的挑战。

**杜塞尔多夫大学医院攻击（2020年）：** 针对杜塞尔多夫大学医院的勒索软件攻击迫使急诊患者转移，一名患者在被转至三十公里外的医院后死亡。德国检察官最初将这一死亡作为与网络攻击相关的过失杀人罪进行调查，但后来因难以建立直接因果关系而结案。该案是全球首批提出勒索软件致死刑事责任前景的案件之一。

## 欧洲框架

### 布达佩斯网络犯罪公约（2001年）

《网络犯罪公约》于2001年11月23日在布达佩斯开放签署，2004年7月1日生效，是首部也是最广泛采纳的网络犯罪国际条约。截至2025年，八十一个国家已批准，另有两个国家已签署但未批准。

**实体条款：** 公约要求缔约方将以下行为定为犯罪：

- 非法访问计算机系统（第2条）
- 非法拦截非公开传输（第3条）
- 数据干扰——损坏、删除或篡改数据（第4条）
- 系统干扰——阻碍计算机系统运行（第5条）
- 滥用设备——制造、销售或分发黑客工具（第6条）
- 计算机相关伪造和欺诈（第7-8条）
- 内容相关犯罪包括儿童性剥削材料（第9条）
- 侵犯版权（第10条）

**程序条款：** 公约确立了以下程序：

- 快速保全已存储的计算机数据（第16条）
- 强制披露订户数据的提交令（第18条）
- 搜查和扣押已存储的计算机数据（第19条）
- 实时收集流量数据（第20条）
- 拦截内容数据（第21条）

**司法互助：** 缔约方须建立全天候联络点以处理紧急跨境请求并在电子证据收集方面进行合作。

### 第二附加议定书（2022年）

第二附加议定书于2022年5月12日开放签署，加强了跨境合作和电子证据获取。主要条款包括：

- **第6条：** 直接向其他缔约方领土的域名注册机构请求注册信息。
- **第7条：** 跨境服务提供商披露订户信息。
- **第8条：** 跨法域快速提交订户和流量数据的命令。
- **第10条：** 涉及重大生命或安全风险的紧急司法互助。

该议定书包含健全的数据保护保障措施，包括对自动化决策的限制、数据留存要求和再转移限制。

### 欧盟《打击信息系统攻击指令》（2013/40/EU）

该指令要求欧盟成员国将非法访问信息系统、非法系统干扰、非法数据干扰和非法拦截定为犯罪。大多数犯罪规定最低最高刑不低于两年有期徒刑，涉及有组织犯罪、关键基础设施或大规模攻击的加重情形不低于五年。

| **主要法律** | 《网络法》（정보통신망법） | CFAA（18 U.S.C. section 1030） | StGB 第202a-d条、第303a-b条 | 条约（ETS 185） |
| **立法方式** | 综合性单一法律 | 多部重叠的联邦法律 | 纳入一般刑法典 | 统一化框架 |
| **深度伪造专门法** | 有（自2024年起） | 部分（《TAKE IT DOWN法》，2025年） | 拟议中（第201b条，待审） | 尚未涉及 |
| **人工智能监管** | 《人工智能法》（2026年1月生效） | 无全面联邦法律 | 实施欧盟人工智能法（2026年8月） | 尚未涉及 |
| **安全研究人员保护** | 有限 | 经Van Buren案（2021年）缩小范围 | 立法草案（2024年） | 未专门涉及 |

| **域外管辖** | 有，针对攻击韩国系统 | 有，针对"受保护计算机" | 有，针对攻击德国系统 | 通过司法互助促进 |
| **全天候联络点** | 有（KISA） | 有（DOJ/FBI） | 有（BKA） | 要求设立（第35条） |

### 主要趋势

1. **刑罚升级：** 过去十年来各法域均提高了网络犯罪刑罚，反映了对经济和国家安全影响日益增长的认识。
2. **深度伪造定罪：** 韩国以全面的深度伪造立法处于领先地位；美国已颁布针对性措施；德国已提议但尚未颁布专门条款。
3. **勒索软件优先：** 各法域均将勒索软件作为高优先执法领域，使用现有法律的组合而非独立的勒索软件法（CFAA的勒索条款部分例外）。
4. **人工智能治理空白：** 没有法域颁布了全面的人工智能网络犯罪法律，但各方均在制定将影响刑事责任的监管框架。
5. **国际合作：** 《布达佩斯公约》及其第二附加议定书提供了跨境合作的主要机制，但当攻击者在非缔约国运作时执法仍具挑战性。

## 发展时间线

| 年份 | 发展 |
|------|------------|
| 1986 | 美国颁布《计算机欺诈和滥用法》（CFAA），首部全面的联邦网络犯罪法律 |
| 1995 | 韩国颁布《信息化促进框架法》，建立早期数字治理结构 |
| 2001 | 《布达佩斯网络犯罪公约》开放签署（11月23日） |
| 2004 | 《布达佩斯公约》生效（7月1日） |
| 2007 | 德国修订StGB增加第202b条、第202c条并扩展第303b条（第41次刑法修正案） |
| 2008 | CFAA经《身份盗窃执行和赔偿法》修订，将管辖权扩展至任何用于州际商业的计算机 |
| 2013 | 欧盟指令 2013/40/EU《打击信息系统攻击指令》通过 |
| 2015 | 德国增加StGB第202d条（处理被盗数据）；联邦议院遭受重大网络攻击 |
| 2019 | 韩国"N号房"案件开始，揭露数字性剥削网络 |
| 2020 | 杜塞尔多夫大学医院勒索软件攻击引发勒索软件致死刑事责任问题 |
| 2021 | 科洛尼尔管道公司攻击促使美国对勒索软件采取行政行动；最高法院在Van Buren v. United States案中缩小CFAA范围 |
| 2022 | 《布达佩斯公约》第二附加议定书开放签署（5月12日） |
| 2023-2024 | 国际执法瓦解LockBit勒索软件组织 |
| 2024 | 韩国修订性犯罪法律将深度伪造色情内容定为犯罪；德国起草安全研究人员保护法；欧盟人工智能法（Regulation 2024/1689）通过 |
| 2025 | 美国颁布《TAKE IT DOWN法》针对深度伪造（5月）；德国联邦参议院向联邦议院提交深度伪造刑法草案（第201b条）（8月）；韩国《个人信息保护法》修订增加百分之十营业额罚款 |
| 2026 | 韩国《人工智能法》生效（1月）；欧盟人工智能法全面实施（8月）；韩国深度伪造打击行动持续至10月 |
