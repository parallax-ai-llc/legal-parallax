## 概述

未经同意的私密影像（NCII）——通常被称为"报复性色情"——包括在未经被拍摄者同意的情况下创建、传播、持有或威胁传播性暗示或私密影像。在过去十年中，由于数字通信、智能手机摄像头和人工智能生成的深度伪造技术的普及，这种基于影像的性侵犯已在越来越多的司法管辖区被确认为独立的刑事犯罪。韩国主要通过《性暴力犯罪处罚等特例法》（성폭력범죄의 처벌 등에 관한 특례법），特别是第14条来处理NCII问题，该条将未经同意的拍摄、传播和持有定为犯罪，最高可判处七年监禁；2024年扩展的深度伪造专项立法进一步将深度伪造亲密内容的制作、传播、持有甚至观看定为犯罪。美国历史上缺乏针对NCII的联邦刑事法律，于2025年5月颁布了《TAKE IT DOWN法案》——这是第一部将未经同意发布私密影像定为犯罪的联邦法律，对涉及成年人的犯罪最高可判处两年监禁，涉及未成年人的最高可判处三年——同时在2025年5月南卡罗来纳州立法后，全部50个州现已各自制定了刑事法规。德国根据《刑法典》（Strafgesetzbuch, StGB）第201a条将NCII定为犯罪，该条禁止未经授权制作和传播侵犯私密隐私的录像，最高可判处两年监禁和最高60,000欧元罚款，并辅以在影像被用作胁迫手段时适用的胁迫罪（第240条StGB）和敲诈勒索罪（第253条StGB）条款。在欧洲层面，欧洲人权法院依据《欧洲人权公约》第8条认定各国有积极义务将NCII定为犯罪并有效追诉，最显著的判例为Buturuga v. Romania（2020年）和M.S.D. v. Romania（2024年），同时欧盟第2024/1385号关于打击暴力侵害妇女和家庭暴力的指令明确要求所有成员国在2027年6月前将未经同意分享私密或篡改材料的行为定为犯罪。

## 韩国

### 法律框架

韩国在一系列备受瞩目的数字性犯罪丑闻推动下，建立了最为全面的未经同意私密影像法律框架之一。主要刑事法规是《性暴力犯罪处罚等特例法》（성폭력범죄의 처벌 등에 관한 특례법），其中第14条是针对基于摄像的性犯罪和未经同意影像传播的核心条款。该法于2010年首次颁布，经历了多次扩展——最重要的是2020年在"N号房"丑闻后以及2024年应对深度伪造性侵犯危机时的修订。

《促进信息通信网络利用及信息保护法》提供了额外的监管工具，要求平台删除非法性内容，并对在韩国管辖范围内运营的服务提供商规定了义务。

### 刑罚

| 犯罪类别 | 刑罚 |
|----------|------|
| 未经同意拍摄他人身体（第14条第1款） | 最高7年监禁或最高5,000万韩元（约42,500美元）罚款 |
| 传播、销售或公开展示未经同意的影像（第14条第2款） | 最高7年监禁或最高5,000万韩元罚款 |
| 通过信息通信网络以营利为目的的传播 | 至少3年监禁（法定最低刑） |
| 持有或观看未经同意的影像（第14条第3款） | 最高3年监禁或最高3,000万韩元（约25,500美元）罚款 |
| 惯犯（第14条第4款） | 在规定刑罚基础上加重最高二分之一 |
| 深度伪造私密影像的制作/传播（2024年法律） | 最高7年监禁（从5年提高） |
| 购买、销售或观看深度伪造色情内容 | 最高3年监禁或最高3,000万韩元罚款 |

### N号房事件（2018-2020年）

N号房事件是韩国历史上最臭名昭著的数字性犯罪案件之一。从2018年末至2020年3月，使用化名的犯罪者——最突出的是赵主彬（化名"博士"）——在Telegram加密即时通讯平台上创建了一系列加密聊天室。至少1,154名已确认的受害者（其中60.7%为20多岁或更年轻者）通过敲诈和欺骗被迫制作色情照片和视频。这些材料随后在这些"N号房"中向付费订阅者传播。

赵主彬于2020年被判处40年监禁。该案引发了广泛的公众愤怒，并直接促成了2020年夏季全面立法改革的通过，这些改革于2021年12月生效。改革加重了对在线性犯罪者的刑罚，将非法性内容的持有和购买定为犯罪，并对平台运营商施加了新的义务。

### 2024年深度伪造危机与立法回应

2024年，韩国经历了第二波数字性犯罪浪潮，涉及人工智能生成的深度伪造色情内容，主要针对女性学生和公众人物。这场危机导致了迅速的立法行动：制作和传播深度伪造亲密影像的最高刑期从五年增加到七年；首次将仅仅观看或持有深度伪造色情内容定为犯罪，最高可判处三年监禁或最高300万韩元罚款。被纳入管辖范围的平台被要求在收到受害者有效通知后48小时内删除被标记的内容。

### 执法挑战

尽管有健全的法律条款，执法仍然是一个持续的挑战。2021年人权观察的一份报告发现，在被判犯有未经同意拍摄私密影像罪的人中，79%仅被判处缓刑、罚款或两者兼有。此外，这些法律仅适用于位于韩国境内的运营商，限制了对通过Telegram等境外平台实施的犯罪的执法效力。

## 美国

### 法律框架

美国通过各州刑事法规的拼凑以及自2025年起的联邦立法来处理NCII问题。历史上，没有专门针对未经同意传播私密影像的联邦刑法，尽管检察官偶尔依据与网络跟踪（18 U.S.C. Section 2261A）、敲诈勒索或儿童剥削相关的一般性法规进行起诉。

**TAKE IT DOWN法案（2025年5月19日签署）：** 这部具有里程碑意义的联邦法律——首部在联邦层面直接将NCII定为犯罪的法律——禁止任何人在未经同意的情况下故意发布未同意的成年人或未成年人的私密视觉描绘，包括旨在造成伤害的人工智能生成的深度伪造。该法案还要求受管辖的平台（公共网站和移动应用）建立通知与下架程序，在收到有效举报后48小时内调查和删除被标记的材料。不遵守规定将被视为《联邦贸易委员会法》第18(a)(1)(B)条下的不公平或欺骗性行为，使平台面临FTC执法。

**SHIELD法案（2025年2月重新提出，待定）：** 由参议员Amy Klobuchar和参议员John Cornyn推动的《制止有害影像利用和限制传播法》将建立更广泛的联邦刑事责任，涵盖传播或威胁传播私密影像的行为，对涉及未成年人的犯罪加重刑罚。该法案于2024年在参议院通过但在众议院未获通过；已在第119届国会重新提出。

**DEFIANCE法案（2026年1月参议院通过）：** 由众议员Alexandria Ocasio-Cortez提出的《制止伪造影像和未经同意编辑法》创设了联邦民事诉因，使深度伪造亲密影像的受害者能够向制作或传播影像的人索赔。

### 刑罚

| 犯罪等级 | 联邦刑罚（TAKE IT DOWN法案） |
|----------|------------------------------|
| 发布未经同意的私密影像（成年人） | 罚款和/或最高2年监禁 |
| 发布未经同意的私密影像（未成年人） | 罚款和/或最高3年监禁 |
| 威胁发布真实私密影像（成年人） | 罚款和/或最高2年监禁 |
| 威胁发布真实私密影像（未成年人） | 罚款和/或最高3年监禁 |
| 涉及数字伪造的威胁（成年人） | 罚款和/或最高18个月监禁 |
| 涉及数字伪造的威胁（未成年人） | 罚款和/或最高30个月监禁 |
| 平台未遵守下架义务 | 作为不公平/欺骗性行为由FTC执法 |

### 各州法律

截至2025年5月，全部50个州、哥伦比亚特区和关岛均已颁布了针对未经同意传播私密影像的刑事法规。南卡罗来纳州是最后一个将NCII定为犯罪的州，于2025年5月颁布了法律。各州法律在范围、严重程度和方式上差异显著：

- **加利福尼亚州（2013年）：** 《刑法典》第647(j)(4)条将以造成情感痛苦为目的故意传播私密影像定为犯罪；属轻罪。
- **新泽西州（2004年）：** 首个颁布报复性色情立法的州，将通过未经同意披露私密影像侵犯隐私定为三级重罪，最高可判处五年监禁。
- **得克萨斯州（2015年，2019年修订）：** 最初在Ex parte Jones（2018年）案中因第一修正案理由被宣布违宪；立法机关于2019年修订了法规，增加了意图伤害的要素。
- **伊利诺伊州（2015年）：** 《未经同意传播私密性影像法》，720 ILCS 5/11-23.5；属重罪。

### 关键法律问题

**第一修正案考量：** 美国的报复性色情法律必须在言论自由的宪法保障之间找到平衡。多部法规面临宪法挑战，法院通常维护包含特定意图要求的法律。得克萨斯州刑事上诉法院在Ex parte Jones（2018年）案中以过于宽泛为由推翻了该州最初的报复性色情法规，促使立法修订。

**第230条限制：** 《通信规范法》第230条历史上为在线平台提供了对用户生成内容的广泛免责保护。TAKE IT DOWN法案代表了一项重大例外，无论第230条的保护如何，均对平台施加了积极的下架义务。

## 德国

### 法律框架

德国主要通过《刑法典》（StGB）第201a条处理未经同意的私密影像问题，该条标题为"通过拍照或使用其他技术手段侵犯私密隐私"。该条款于2004年引入，2015年和2021年进行了改革，将未经授权制作和传播侵犯个人私密生活领域的录像均定为犯罪。保护范围延伸至在私人场所——如卧室和浴室——拍摄的影像，以及任何未经被拍摄者同意而传播的私密影像，即使原始录像是在获得同意的情况下制作的。

### 刑罚

| 犯罪类别 | 刑罚 |
|----------|------|
| 未经授权制作私密录像（StGB第201a条第1款） | 最高2年监禁或罚款 |
| 未经授权传播私密录像（StGB第201a条第1款） | 最高2年监禁或罚款（最高60,000欧元） |
| 以胁迫或威胁方式传播（StGB第240条） | 最高3年监禁或罚款；情节严重者最高5年 |
| 利用私密影像敲诈勒索（StGB第253条） | 最高5年监禁或罚款；情节严重者最高15年 |
| 侵犯肖像权（KunstUrhG第33条） | 最高1年监禁或罚款 |

### 适用范围

StGB第201a条保护"最私密的生活领域"（hoechstpersoenlicher Lebensbereich）。关键是，该法律涵盖了超越典型报复性色情的多种情形：

- **经同意拍摄但未经同意传播的影像：** 即使一个人自愿向伴侣提供了私密照片或视频，未经同意的再传播也构成刑事犯罪。
- **偷拍裙底：** 2020年立法修正后，未经同意在衣物下方拍照（偷拍裙底）被StGB第184k条明确定为犯罪，最高可判处两年监禁。
- **深度伪造：** StGB第201a条适用于篡改影像，包括人工智能生成的私密描绘，因为该条款涵盖任何侵犯私密隐私的录像，无论其制作方法如何。

### 民事救济

除刑事追诉外，德国的受害者可依据《民法典》（BGB）寻求民事救济。BGB第823条提供了因侵犯人格权而产生的一般侵权责任请求权，BGB第1004条（类推适用）使受害者能够获得禁止进一步传播的禁令救济。《艺术著作权法》（KunstUrhG）第22-23条确立的肖像权（Recht am eigenen Bild）提供了额外的民事保护，要求任何发布个人肖像的行为均须获得同意。

### 批评与改革建议

德国法律学者和倡导组织批评StGB第201a条下相对较低的两年最高刑期不足以遏制基于影像的性侵犯，尤其与韩国的刑罚或新兴欧盟标准相比。"积极反对数字暴力"（Aktiv gegen digitale Gewalt）组织呼吁制定专门的报复性色情立法，加重刑罚并简化受害者支援机制。

## 欧洲人权法院与欧盟

### 欧洲人权公约——第8条

欧洲人权法院依据公约第8条（保障尊重私人和家庭生活的权利）处理未经同意的私密影像问题。法院逐步承认各国不仅有消极义务不干涉私人生活，还有积极义务采取措施保护个人免受私人行为者实施的基于影像的性侵犯。

**K.U. v. Finland（2008年）：** 虽然严格来说这不是一个报复性色情案件，但这一基础性判决（申请号2872/02）确立了第8条要求各国确保有效的刑法机制来保护个人免受通过数字手段侵犯人身完整性的原则。法院认定，当一名儿童的身份被用于具有性暗示的在线广告时，芬兰未能提供充分保护。

**Buturuga v. Romania（2020年）：** 在这一具有里程碑意义的案件（申请号56867/15）中，法院审查了罗马尼亚当局是否履行了其依据第8条保护受害者免受前伴侣传播私密照片的在线骚扰的积极义务。法院确认第8条的保护延伸至数字领域的基于影像的性侵犯，各国必须确保对此类行为的有效调查和追诉。

**M.S.D. v. Romania（2024年）：** 2024年12月3日，法院因罗马尼亚未能惩罚一名未经前伴侣同意在网上发布其私密照片的男子而对其作出谴责。法院认定违反了第8条，裁定在报复性色情——即前亲密伴侣未经同意在网上公开传播私密照片——的情况下，第8条的保护要求作出刑法回应。这一裁决加强了欧洲委员会成员国维持有效的NCII刑事框架的义务。

### 欧盟第2024/1385号指令——暴力侵害妇女和家庭暴力

2024年5月14日通过的第（EU）2024/1385号指令是首个将基于影像的性侵犯作为刑事犯罪加以处理的具有约束力的欧盟范围立法。该指令标志着欧盟刑法的分水岭时刻，为所有成员国建立了最低标准。

**关于NCII的关键条款：**

- **第5条——未经同意分享私密或篡改材料：** 将未经同意制作和传播私密影像定为犯罪，包括人工智能生成或篡改的描绘某人从事性行为的材料。传播必须"可能对受害者造成严重伤害"。
- **网络裸露闪现：** 未经他人同意向其发送私密影像被定为独立犯罪。
- **平台义务：** 成员国必须确保为受害者提供可访问且安全的基于信息通信技术的举报机制。必须建立删除令和快速证据保全机制。
- **受害者支持：** 该指令要求成员国提供专门的支持服务，包括心理咨询、法律援助和内容删除协助。

**转化期限：** 成员国必须在2027年6月14日前将该指令的要求纳入国内法。

### 欧盟补充框架

该指令与其他处理基于影像的性侵犯各方面的欧盟文书并行运作：

- **数字服务法（DSA，2022年）：** 对在线平台施加删除包括NCII在内的非法内容的义务，并建立透明度要求和可信举报人机制。
- **人工智能法（2024年）：** 处理人工智能系统带来的风险，包括深度伪造内容的生成，规定透明度和标注要求。
- **通用数据保护条例（GDPR，2016年）：** 为受害者要求删除私密影像提供数据保护依据，第17条规定的擦除权适用于由欧盟管辖范围内的数据控制者托管的未经同意的私密内容。

## 比较分析

### 刑罚严厉程度

在所调查的司法管辖区中，韩国施加了最为严厉的刑罚，对未经同意的拍摄或传播最高可判处七年监禁，对通过网络以营利为目的传播者处至少三年的法定最低监禁刑。德国StGB第201a条下两年的最高刑期是最低的，但胁迫或敲诈勒索罪的附加指控可大幅增加刑期。美国TAKE IT DOWN法案规定的联邦刑罚处于中等水平，为两至三年，但各州刑罚差异很大。欧盟指令不规定具体刑罚幅度，而是交由成员国转化实施。

### 犯罪化范围

韩国是所调查的司法管辖区中唯一明确将仅仅持有和观看未经同意的私密影像定为犯罪的国家，这一规定在2024年扩展至涵盖深度伪造内容。美国联邦层面侧重于发布和传播，未将持有定为犯罪。德国同样针对制作和传播而非持有本身。欧盟指令未要求将持有定为犯罪。

### 深度伪造覆盖范围

四个司法管辖区现在都以某种形式处理深度伪造私密影像。韩国于2024年颁布了专门的深度伪造立法，最高可判处七年。美国TAKE IT DOWN法案明确涵盖"数字伪造"。德国将现有的StGB第201a条框架适用于篡改影像。欧盟第2024/1385号指令第5条专门提及"篡改材料"，确保覆盖人工智能生成的内容。

### 平台责任

美国（TAKE IT DOWN法案）和韩国均对平台施加了48小时的具体下架义务。欧盟数字服务法为平台责任和内容审核提供了更广泛的框架，而德国作为欧盟成员国受DSA和指令附加要求的约束。欧洲人权法院框架通过国家确保有效救济的积极义务间接处理平台责任问题。

### 宪法与权利之间的紧张关系

美国在NCII犯罪化与第一修正案言论自由保障之间面临独特的紧张关系，得克萨斯州最初的报复性色情法规被司法否决即为例证。其他司法管辖区不面临类似的宪法障碍：韩国宪法法院维护了数字性犯罪条款，德国通过其既定的比例原则框架平衡隐私权与表达权，欧洲人权法院在NCII案件中一贯将第8条隐私保护置于第10条表达权之上。
