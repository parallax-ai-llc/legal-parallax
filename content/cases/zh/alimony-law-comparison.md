## 概要

扶养费——离婚后由一方前配偶向另一方支付的经济支持，又称配偶扶养、配偶赡养或婚后扶养——反映了关于婚姻解除经济后果的根本不同的法律哲学，从基于过错的补偿模式到基于需求的扶养体系，并日益向强调离婚后自力更生的限时康复性方法发展。韩国在离婚时承认两项不同的经济请求：由有过错配偶支付的慰藉金（위자료），作为根据《民法》（민법）第843条对精神损害的补偿，以及根据第839-2条的财产分割（재산분할），后者可包含配偶扶养要素；与西方扶养费制度不同，韩国法律没有独立的离婚后扶养义务，尽管法院可将持续扶养需求纳入财产分割计算中，大法院已承认财产分割可包含"扶养要素"（부양적 요소）以及反映各配偶对婚姻财产贡献的"清算要素"（청산적 요소）。美国通过各州家庭法法典管辖扶养费，全国呈现显著的改革趋势，包括2012年马萨诸塞州扶养费改革法确立了与婚姻持续时间挂钩的期限限制，多个州取消永久性扶养费而改为期限性或康复性扶养，以及2017年《减税与就业法》取消了2018年12月31日后终结的离婚中扶养费支付的联邦税前扣除和接收方的收入计入，从根本上改变了扶养费谈判的经济学。德国根据《民法典》第1569-1586b条提供离婚后配偶扶养（nachehelicher Unterhalt），自2008年《扶养法修改法》改革以来受自我责任原则（Eigenverantwortung）支配，要求每位配偶在离婚后自行维持生计，仅在照顾子女、年老、疾病或无法找到工作等列举情形下才给予扶养，且期限以合理为限。欧洲框架缺乏对配偶扶养规则的统一化，但通过欧盟扶养义务条例（No 4/2009）和2007年《海牙准据法议定书》提供冲突法和执行机制，而欧洲人权法院主要在性别歧视和程序公平的背景下处理配偶扶养问题。

## 比较表

| 要素 | 韩国 | 美国 | 德国 | 欧洲人权法院/欧盟 |

## 韩国

### 法律框架

韩国没有西方式的扶养费（持续性配偶扶养）制度。相反，离婚的经济后果通过两种机制处理：

**慰藉金（위자료，《民法》第843条）：** 有过错配偶可被命令向无辜配偶支付慰藉金，作为对因婚姻破裂而遭受的精神痛苦的补偿。金额根据过错程度、婚姻持续时间、离婚情况和双方经济状况确定。典型裁定金额为2000万至1亿韩元，但金额差异显著。

**财产分割（재산분할，《民法》第839-2条）：** 离婚时，任何一方均可请求分割婚姻财产。大法院（대법원）已确定财产分割具有两种功能：

| 功能 | 说明 |
|------|------|
| 清算要素（청산적 요소） | 分割婚姻期间通过共同努力积累的财产 |
| 扶养要素（부양적 요소） | 为经济弱势配偶的离婚后生计提供保障 |

### 大法院判例

**大法院2013므568号判决（2013年）：** 法院确认财产分割可包含扶养要素（부양적 요소），允许法院向经济弱势配偶分配超出其贡献所能证明的更大份额。该判决有效地将配偶扶养纳入了财产分割框架。

**大法院93므1002号判决（1993年）：** 法院确立了家务贡献构成财产分割请求基础的原则，承认家务劳动作为对婚姻的经济贡献。

### 实际适用

在实践中，韩国法院通常向经济弱势配偶（通常是妻子）分配约30-50%的婚姻财产，在长期婚姻或一方作出大量家务贡献的情况下向上调整比例。扶养要素允许法院在请求方配偶离婚后将面临经济困难时超出基于贡献的计算。

### 慰藉金与西方扶养费

韩国慰藉金制度与西方扶养费存在根本差异：

| 特征 | 韩国위자료 | 西方扶养费 |
|------|-----------|-----------|
| 基础 | 过错/精神损害 | 需求/经济差距 |
| 支付结构 | 通常一次性 | 通常定期支付 |
| 期限 | 一次性 | 持续（数月至永久） |
| 可变更性 | 一般不可变更 | 通常可变更 |
| 税务处理 | 无特别处理 | 可能有税务影响 |

## 美国

### 法律框架

美国的扶养费完全由州法管辖，无联邦扶养费法律。《统一婚姻和离婚法》（1970年）影响了各州改革但未被普遍采纳。法院通常考虑：

| 因素 | 说明 |
|------|------|
| 婚姻持续时间 | 较长婚姻更可能产生扶养费裁定 |
| 生活水准 | 婚姻期间生活水准作为基准 |
| 挣收能力 | 各配偶自力更生的能力 |
| 年龄和健康 | 身体和精神健康考量 |
| 贡献 | 家务、职业牺牲、资助配偶教育 |
| 婚姻不当行为 | 在部分州（非全部）予以考虑 |

### 扶养费类型

| 类型 | 目的 | 期限 |
|------|------|------|
| 永久性 | 对无法自力更生的配偶的持续扶养 | 至死亡、再婚或变更 |
| 康复性 | 过渡至自力更生期间的扶养 | 特定期间（通常与教育/培训挂钩） |
| 期限性/过渡性 | 限时扶养 | 固定期间，通常与婚姻持续时间挂钩 |
| 补偿性 | 对资助配偶教育的补偿 | 基于投资的一次性或期间 |
| 诉讼期间 | 离婚诉讼期间的临时扶养 | 至终审判决 |

### 改革运动

一场重大改革运动寻求限制或取消永久性扶养费：

**马萨诸塞州扶养费改革法（2012年）：** 确立了期限限制：
- 五年以下婚姻：扶养费不超过婚姻持续时间的50%
- 十年以下婚姻：不超过60%
- 十五年以下婚姻：不超过70%
- 二十年以下婚姻：不超过80%
- 超过二十年的婚姻：法院可裁定无限期扶养费

**佛罗里达州：** 在多次失败尝试后，佛罗里达州于2023年颁布扶养费改革法，前瞻性地取消永久性扶养费，并确立基于公式的方法和期限上限。

**得克萨斯州：** 将配偶扶养限制为每月最高5,000美元或义务人月平均总收入的20%（以较低者为准），最长期限为五至十年，取决于婚姻持续时间。

### 《减税与就业法》（2017年）

TCJA从根本上改变了2018年12月31日后终结的离婚中的扶养费经济学：

| 特征 | TCJA前 | TCJA后 |
|------|--------|--------|
| 支付方 | 可税前扣除 | 不可扣除 |
| 接收方 | 作为收入计入 | 不计入 |
| 影响 | 将税务负担转移至较低税率的接收方 | 支付方税后成本更高；达成协议的激励降低 |

这一变化减少了曾促进扶养费协议达成的"税务补贴"，据估计对和解动态产生了重大影响。

## 德国

### 法律框架

离婚后配偶扶养（nachehelicher Unterhalt）由《民法典》第1569-1586b条管辖。2008年《扶养法修改法》从根本上改革了该制度，从推定持续扶养转向推定自我责任。

### 自我责任原则（Eigenverantwortung）

**《民法典》第1569条：** 离婚后，各配偶应自行维持其生计。仅在配偶无法自行维持生计时，方可依照以下规定向对方主张扶养。

### 列举的扶养事由

| 事由 | 《民法典》条款 | 说明 |
|------|---------------|------|
| 照顾子女（Betreuungsunterhalt） | 第1570条 | 照顾三岁以下子女；可延长 |
| 年老（Altersunterhalt） | 第1571条 | 因年龄无法找到工作 |
| 疾病（Krankheitsunterhalt） | 第1572条 | 因疾病无法工作 |
| 失业（Erwerbslosigkeitsunterhalt） | 第1573条 | 无法找到适当工作 |
| 补充（Aufstockungsunterhalt） | 第1573条第2款 | 尽管就业但收入不足 |
| 教育（Ausbildungsunterhalt） | 第1575条 | 教育、培训或再培训 |
| 公平（Billigkeitsunterhalt） | 第1576条 | 其他严重理由 |

### 限制与困难

**《民法典》第1578b条：** 扶养可在期限或金额上限制在合理范围内，考虑因婚姻造成的不利条件在多大程度上已通过请求方的收入或资产得到克服。婚姻相关不利（ehebedingte Nachteile）概念是核心：扶养主要在婚姻对请求方造成经济不利（如为照顾子女而中断职业）的范围内获得正当化。

**《民法典》第1579条（困难条款）：** 在严重不公平的情况下，扶养可被拒绝、减少或限时，包括：
- 婚姻持续时间极短
- 对义务人的严重不当行为
- 故意造成需求
- 其他同等严重的理由

### 扶养计算

德国法院使用"七分之三/七分之四"或"差额法"计算扶养：扶养权利人有权获得双方调整后净收入差额的七分之三（允许义务人保留七分之四作为"就业激励奖金"——Erwerbstatigenbonus）。

### 税务分割（Realsplitting）

根据Realsplitting制度（《所得税法》第10条第1a款），扶养支付方可从应税收入中扣除最高每年13,805欧元的扶养支付，前提是接收方同意并将该支付作为收入计入。这与美国TCJA前的处理方式相似。

## 欧洲人权法院与欧盟

### 欧盟扶养义务条例（No 4/2009）

欧盟扶养义务条例以与子女抚养费相同的框架管辖跨境配偶扶养请求。对于配偶扶养，准据法由2007年《关于扶养义务准据法的海牙议定书》确定：

- **一般规则：** 权利人惯常居所地法（第3条）。
- **例外：** 如一方提出异议且最后共同惯常居所地法具有更密切联系，适用该法（第5条）。
- **法律选择：** 当事人可从指定选项中协议选择准据法（第8条）。

### 欧洲人权法院判例

**Burden v. United Kingdom（申请号13378/05，大审判庭，2008年4月29日）：** 虽然主要涉及同居者的税收优惠，大审判庭在婚姻与其他关系的经济义务方面进行了区分，指出婚姻的法律后果（包括扶养义务）证明了差别待遇的合理性。

**Airey v. Ireland（1979年）：** 对家庭诉讼程序的有效诉诸法院的要求延伸至配偶扶养请求，在诉讼程序的复杂性需要时要求提供法律援助。

**Munoz Diaz v. Spain（申请号49151/07，2009年12月8日）：** 法院认定在未承认罗姆习惯婚姻以享受遗属养老金时违反了第14条，对因未登记结合产生的扶养义务的承认具有启示意义。

### 欧盟成员国的差异

| 成员国 | 制度 | 主要特征 |
|--------|------|----------|
| 德国 | 基于需求+自我责任 | 有限期限；关注婚姻相关不利 |
| 法国 | 补偿性给付（prestation compensatoire） | 通常一次性；补偿原则 |
| 荷兰 | 基于需求 | 最长五年（长期婚姻/有子女为十二年） |
| 瑞典 | 非常有限 | 仅短期过渡性扶养 |
| 意大利 | 基于需求+生活水准 | 2017年改革（Cass. SU 18287/2018）：转向自给自足 |
| 西班牙 | 补偿性养老金（pension compensatoria） | 对经济失衡的补偿 |

## 比较分析

### 补偿性vs.需求性vs.自给自足

各法域反映了三种不同的哲学方法。韩国的慰藉金制度主要是补偿性的和基于过错的。美国历史上强调需求和婚姻生活水准，但正在向自给自足和期限限制转变。德国最充分地体现了自我责任原则，将扶养限于补偿婚姻相关不利。欧盟框架不统一这些方法，但促进其跨境适用。

### 过错的相关性

过错在各法域中扮演着递减但仍然重要的角色。在韩国，过错是慰藉金的核心。在美国，约半数州在扶养费裁定中考虑过错。德国将过错的相关性限于困难条款（第1579条BGB）。所有法域的总体趋势是减少过错的重要性，转向基于经济需求的分析。

### 时间限制

所有法域都趋向于限制配偶扶养的期限。德国第1578b条对婚姻相关不利的强调固有地限制了期限。马萨诸塞州扶养费改革法和类似的美国州改革施加了明确的期限上限。韩国的一次性慰藉金本质上是限时的。仅在特殊情况下（极长婚姻、严重残疾），这些制度中的任何一个才会考虑真正的无限期扶养。

### 税务互动

税务处理显著影响扶养费经济学。德国的Realsplitting和美国TCJA前的扣除/计入制度都通过将收入转移至较低税率的接收方为扶养费创造了税务激励。TCJA在美国取消这一激励已明显影响了和解动态，通常减少了协商的扶养费总额。韩国对慰藉金不提供特别税务处理。
