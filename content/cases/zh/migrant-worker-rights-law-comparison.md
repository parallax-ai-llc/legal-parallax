## 概述

外来劳工权利——即赋予在国籍国以外从事就业的外国公民的法律保护，涵盖劳动标准、社会保障、结社自由、工作场所安全和防止剥削——代表了移民法、劳动法和国际人权法的关键交汇点。韩国的雇用许可制度（고용허가제，EPS），根据2003年的《外国人劳动者雇用等相关法律》（외국인근로자의 고용 등에 관한 법률）建立，通过双边谅解备忘录管辖来自16个国家的约27万名低技能外来劳工的入境和就业，保障《劳动基准法》（근로기준법）的平等适用和最低工资保护，但对工作场所流动性施加限制（在4年10个月内限制更换雇主最多3次），因此持续受到工会和人权组织的批评。美国根据《公平劳动标准法》（FLSA，29 U.S.C. §§ 201-219）、《职业安全与健康法》（OSHA，29 U.S.C. §§ 651-678）和《国家劳动关系法》（NLRA，29 U.S.C. §§ 151-169），不论移民身份将大多数联邦劳动保护扩展至外来劳工，但执法因与移民执法的交叉、许多劳工的签证依赖身份（特别是H-2A和H-2B签证持有者）以及最高法院在Hoffman Plastic Compounds v. NLRB, 535 U.S. 137 (2002)案中限制对无证劳工补发工资救济的判决而复杂化。德国通过全面的《劳动派遣法》（Arbeitnehmerüberlassungsgesetz，AÜG）、《最低工资法》（Mindestlohngesetz，MiLoG）和《劳动保护监督法》（Arbeitsschutzkontrollgesetz）保护外来劳工，并在欧盟的《派遣劳工指令》（Directive 96/71/EC，由Directive (EU) 2018/957修订）、《季节性劳工指令》（Directive 2014/36/EU）和《欧盟基本权利宪章》的框架内运作，而欧洲人权法院根据第4条（禁止强迫劳动和贩运）、第8条（私人和家庭生活）和第14条（不歧视）的判例已确立了各国预防和惩罚外来劳工劳动剥削的约束性义务。

## 韩国

### 法律框架

韩国的雇用许可制度是全球最具结构性的临时劳动移民项目之一。

**《外国人劳动者雇用等相关法律》（외국인근로자의 고용 등에 관한 법률）：**

| 条款 | 规定 |
|---------|-----------|
| 第6条 | 平等待遇：劳动相关法律平等适用于外国劳工 |
| 第8条 | 外国人力政策委员会（외국인력정책위원회）确定年度配额 |
| 第9条 | 与来源国签署双边谅解备忘录 |
| 第12条 | 标准劳动合同要求 |
| 第22条 | 就业保险覆盖 |
| 第25条 | 工作场所变更限制 |

**EPS结构：**

| 组成部分 | 详情 |
|-----------|--------|
| 签证类别 | E-9（非专业就业） |
| 来源国 | 16个（菲律宾、泰国、印度尼西亚、越南、蒙古、斯里兰卡、柬埔寨、尼泊尔、缅甸、乌兹别克斯坦、孟加拉国、巴基斯坦、中国、东帝汶、老挝、吉尔吉斯斯坦） |
| 最长居留 | 4年10个月（重新入境可延长至9年8个月） |
| 工作场所变更 | 最多3次（雇主过错或不可避免的情况） |
| 韩语测试 | 出发前须通过EPS-TOPIK（雇用许可制度韩语能力测试） |

### 劳动权利

持E-9签证的外国劳工在法律上有权享有与韩国劳工相同的保护：
- **最低工资：** 9,860韩元/小时（2024年），与韩国劳工相同
- **工作时间：** 《劳动基准法》限制（每周标准40小时；加班限制）
- **工业事故保险：** 《工业事故补偿保险法》全面覆盖
- **国民健康保险：** 强制参加
- **国民年金：** 强制缴纳（与部分来源国签有退还协议）
- **出国到期保险（출국만기보험）：** 雇主资助的保险，出境时支付

### 工作场所流动性限制

EPS最受批评的方面是工作场所流动性限制。根据第25条，劳工仅可在因雇主原因（拖欠工资、工作场所关闭、虐待、不安全条件）或不可避免原因下更换雇主，最多3次。这种"绑定劳工"制度遭到以下方面的谴责：

- **国际劳工组织专家委员会：** 反复指出工作场所流动性限制造成了有利于强迫劳动的条件
- **国家人权委员会（국가인권위원회）：** 2019年建议废除3次更换限制
- **联合国消除种族歧视委员会：** 2018年结论性意见建议在雇主更换方面提供灵活性

### 农业劳工

农业部门的外来劳工面临特别严峻的条件：
- 季节性工作签证（E-8）：2015年针对农业劳动引入
- 雇主提供住房（往往是不达标的宿舍）
- 收获季节期间长时间工作普遍存在
- 农业工作场所的死伤案例促使政府进行检查和改革
- 2021年抱川市温室"柬埔寨劳工死亡"案凸显了系统性剥削

## 美国

### 法律框架

美国不论移民身份将大多数联邦劳动保护扩展至外来劳工，但执法缺口持续存在。

**关键联邦劳动保护：**

| 法规 | 保护内容 | 适用于无证劳工 |
|---------|-----------|------------------------|
| FLSA (29 U.S.C. §§ 201-219) | 最低工资、加班 | 是 |
| OSHA (29 U.S.C. §§ 651-678) | 工作场所安全 | 是 |
| NLRA (29 U.S.C. §§ 151-169) | 组织权 | 是（但救济受限） |
| Title VII (42 U.S.C. § 2000e) | 反歧视 | 是 |
| MSPA (29 U.S.C. §§ 1801-1872) | 外来季节性农业劳工 | 是 |

**Hoffman Plastic Compounds v. NLRB, 535 U.S. 137 (2002)：** 最高法院裁定NLRA未授权国家劳动关系委员会向因工会组织活动被非法解雇的无证劳工补发工资。该判决通过取消关键执法工具严重削弱了对无证劳工的劳动保护，但其他救济（最低工资追偿、歧视赔偿）仍然可用。

### H-2A农业劳工

H-2A临时农业劳工项目是季节性农业劳动的主要合法途径：

| 特征 | 要求 |
|---------|-------------|
| 雇主义务 | 证明国内劳工不足 |
| 不利影响工资率（AEWR） | 按地区设定（2024年14.53-18.65美元/小时） |
| 住房 | 符合联邦/州标准的免费住房 |
| 交通 | 入境/出境交通；每日交通 |
| 四分之三保证 | 必须提供至少合同期75%的工作时间 |
| 工伤赔偿 | 大多数州要求 |
| 合同 | 书面；经劳工部批准 |

### H-2B非农业劳工

H-2B项目用于临时非农业劳动（酒店业、园艺、海产加工），年度上限为66,000人（另有补充分配）。劳工面临与H-2A劳工类似的雇主依赖问题。

### 反贩运保护

2000年《贩运被害人保护法》（TVPA）及其后续重新授权规定：
- T签证：面向贩运被害人（每年最多5,000人）
- U签证：面向协助执法的合格犯罪（包括劳动剥削）被害人（每年10,000人）
- 持续居留：国土安全部有权允许被害人在调查/起诉期间留在美国

## 德国

### 法律框架

德国对外来劳工的保护在国内劳动法和欧盟法规的综合框架内运作。

**普遍劳动法适用：**

德国境内所有劳工，不论国籍或移民身份，均受以下保护：
- 《最低工资法》（Mindestlohngesetz，MiLoG）：法定最低工资12.41欧元/小时（2024年），12.82欧元（2025年）
- 《工作时间法》（Arbeitszeitgesetz，ArbZG）：工作时间限制（每天8小时；有补偿时最多10小时）
- 《联邦休假法》（Bundesurlaubsgesetz，BUrlG）：最少24个工作日带薪年假
- 《劳动保护法》（Arbeitsschutzgesetz，ArbSchG）：职业安全与健康
- 完全社会保险覆盖（健康、养老、失业、事故、长期护理）

**《劳动保护监督法》（Arbeitsschutzkontrollgesetz，2021年）：** 在2020年6月Tönnies肉类加工厂大规模新冠疫情暴发（感染超过1,500名劳工，主要是东欧外来劳工）之后，联邦议院颁布了《劳动保护监督法》：
- 禁止肉类加工核心行业的分包（Werkverträge）
- 禁止在肉类行业核心活动中使用临时派遣劳工
- 引入肉类行业强制电子工时记录
- 确立雇主提供住房的最低标准
- 增加劳动检查频率

**打击非法就业财务监督处（Finanzkontrolle Schwarzarbeit，FKS）：** 海关机构的打击非法就业部门进行工作场所检查，以执行最低工资合规、社会保险义务并查处非法就业。FKS约有7,300名工作人员，2023年进行了超过50,000次雇主检查。

### 欧盟自由流动劳工

欧盟/欧洲经济区国民根据TFEU第45条和Regulation (EU) No 492/2011享有完全的自由流动：
- 无需工作许可
- 与国民完全平等待遇
- 家庭团聚权利
- Regulation (EC) No 883/2004下的社会保障协调
- Directive (EU) 2018/957下的派遣劳工保护

### 季节性劳工

欧盟《季节性劳工指令》（2014/36/EU），转化为德国法律，规定：
- 任何12个月内最长居留5-9个月
- 工作条件、雇用条款和职业健康安全方面的平等待遇
- 适当的住房标准（如由雇主提供）
- 为返回的季节性劳工提供便利再入境

## 欧洲人权公约/欧盟

### 欧洲人权公约第4条——禁止强迫劳动和贩运

欧洲人权法院在外来劳工劳动剥削方面发展了重要判例：

**Siliadin v. France (Application No. 73316/01, 2005年7月26日)：** 法院认定违反第4条，一名多哥未成年人被置于家庭奴役条件下。法院确立了各国根据第4条负有将强迫劳动和奴役入罪并有效调查的积极义务。

**Rantsev v. Cyprus and Russia (Application No. 25965/04, 2010年1月7日)：** 法院裁定人口贩运属于第4条的范围。各国负有积极义务：（1）建立禁止和惩罚贩运的法律和行政框架；（2）当当局知道或应当知道存在可信贩运嫌疑的情况时保护被害人；（3）调查潜在的贩运行为。

**Chowdury and Others v. Greece (Application No. 21884/15, 2017年3月30日)：** 法院认定违反第4条，涉及希腊马诺拉达的孟加拉国草莓采摘工在强迫劳动条件下工作（暴力威胁、拖欠工资、没收证件）。该判决确立了即使外来劳工最初同意就业，后续的剥削条件也可构成强迫劳动或贩运。

**C.N. and V. v. France (Application No. 67724/09, 2012年10月11日)：** 法院澄清了奴役与强迫劳动的区别：奴役涉及在胁迫下提供服务的义务，与被害人的生活条件相关，特别是无法改变条件的不可能性。

### 欧盟派遣劳工指令

Directive (EU) 2018/957（修订Directive 96/71/EC）确立了"同一地点同工同酬"原则：

| 要素 | 规定 |
|---------|-----------|
| 薪酬 | 东道国最低工资加所有强制性薪酬组成部分 |
| 期限 | 12个月后（可延长至18个月）：适用所有东道国条款和条件 |
| 临时派遣劳工 | 与东道国同类劳工平等待遇 |
| 住宿 | 雇主提供住宿时适用东道国标准 |
| 津贴 | 薪酬之外的差旅、住宿、餐饮津贴 |

### 欧盟反贩运框架

**Directive 2011/36/EU关于预防和打击人口贩运：**
- 劳动剥削贩运的全面定义
- 最高刑罚：至少5年监禁（加重情节为10年）
- 不惩罚原则：被害人不应因作为贩运后果实施的犯罪而被起诉
- 被害人居留许可（Directive 2004/81/EC）
- 建立国家报告员或同等机制

### 国际劳工组织公约

多个司法管辖区批准的相关国际劳工组织文书包括：
- 第29号公约（强迫劳动，1930年）
- 第97号公约（移民就业，1949年）
- 第143号公约（外来劳工，1975年）
- 第189号公约（家政劳工，2011年）
- 2014年《强迫劳动公约议定书》

## 比较分析

### 工作场所流动性

与雇主绑定的签证制度造成了结构性脆弱性：
- **韩国（EPS）：** 最具限制性；最多更换雇主3次
- **美国（H-2A/H-2B）：** 与特定雇主绑定的签证；可携性有限
- **德国：** 欧盟劳工完全流动；第三国劳工在初始期后逐渐灵活
- **欧盟框架：** 《派遣劳工指令》处理临时跨境工作

所有司法管辖区都面临着与特定雇主绑定的移民管控（服务于劳动市场政策目标）和劳工自由（防止剥削）之间的根本性紧张关系。

### 执法机制

| 司法管辖区 | 主要执法机构 | 检查能力 |
|-------------|--------------------|--------------------|
| 韩国 | 雇用劳动部 | 约2,000名检查员 |
| 美国 | 劳工部工资与工时司；OSHA | 约750名WHD调查员；约1,850名OSHA检查员 |
| 德国 | FKS（海关）；Gewerbeaufsicht（贸易监督） | 约7,300名FKS工作人员；各州贸易监督机构 |
| 欧盟 | 各国机构；欧洲劳动管理局（ELA） | ELA协调；各国资源不等 |

### 社会保障获取

外来劳工获取社会保障的情况差异显著：
- **韩国：** 全面获取健康保险和养老金（与部分来源国签有双边退还协议）
- **美国：** 须缴纳社保和医疗保险税；福利获取受移民身份和最低缴纳季度要求限制
- **德国：** 合法劳工享有完全社会保险；跨境情况下的欧盟协调
- **欧盟：** Regulation (EC) No 883/2004协调在欧盟内流动的劳工的社会保障

### 招聘费用监管

剥削性招聘行为是外来劳工脆弱性的主要驱动因素：

- **韩国：** EPS以政府对政府方式运作，消除了私人招聘中介；劳工承担费用平均为900-1,500美元（在全球劳动移民项目中属于最低水平）
- **美国：** 公平招聘倡议和各州法律（如加利福尼亚州AB 450）禁止雇主收取招聘费；但在来源国的执法有限；H-2A和H-2B签证持有者尽管有正式禁令，仍经常支付数千美元的招聘费
- **德国：** 联邦就业服务局（Bundesagentur für Arbeit）通过双边协议管理政府对政府的招聘；私人招聘允许但受《劳动派遣法》（AÜG）监管
- **国际劳工组织：** 《公平招聘一般原则和业务指南》（2019年）确立了劳工不应被收取招聘费或相关费用的原则；"雇主付费"原则日益被视为最佳实践

### 平台用工与零工经济

数字劳动平台的兴起挑战了传统的外来劳工保护框架：

- **欧盟：** 《平台劳工指令》（2024年）确立了平台劳工雇佣关系的可反驳推定，对外来零工劳工有重大影响
- **韩国：** 《劳动基准法》的覆盖范围不延伸至独立承包商；外来零工劳工面临劳动保护和移民支持的双重排斥
- **美国：** 零工劳工的分类因州而异；加利福尼亚州AB 5（2019年）和第22号提案（2020年）体现了争议领域；持与就业绑定签证的外来零工劳工面临特殊脆弱性
- 数字劳动平台日益跨国运营，给劳动标准的执行带来管辖权挑战
