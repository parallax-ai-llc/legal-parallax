## 摘要

柠檬法——为在反复维修尝试后仍未达到质量和性能标准的车辆及其他产品购买者提供救济的消费者保护法规——代表了对最常见且经济意义最重大的消费者投诉之一的针对性回应：购买到有缺陷的汽车。韩国通过《汽车管理法》（자동차관리법）和《消费者基本法》（소비자기본법）处理缺陷车辆投诉，并辅以韩国消费者院（한국소비자원）的汽车争议解决机制和国土交通部的召回权力；虽然韩国没有独立的"柠檬法"法规，但2019年对《汽车管理法》（결함차 반환법）的修订引入了针对经多次维修仍无法修复的严重缺陷新车的车辆换货和退款制度（자동차 교환·환불 제도），俗称"韩国柠檬法"，于2019年1月1日生效。美国拥有世界上最发达的柠檬法框架，所有50个州和哥伦比亚特区均制定了专门的柠檬法法规（始于1982年的康涅狄格州），通常要求制造商在合理次数的维修尝试后仍无法修复的情况下更换或退还新车购买价格，并在联邦层面以《马格努森-莫斯保修法》（15 U.S.C. §§2301-2312）作为补充。德国主要通过《德国民法典》中关于保修（Gewährleistung）的销售法条款（第434-442条）、2002年《债法现代化法》的买卖法改革以及维修失败后解除合同（Rücktritt）的特定权利来处理缺陷车辆问题，提供了强有力的救济框架但没有专门的柠檬法法规。在欧洲层面，《消费者销售指令》（EU）2019/771关于货物销售为所有成员国确立了统一的两年合格保证，要求卖方修理或更换不合格货物，并在修理或更换不可能或不相称时允许价格减少或合同解除，而欧洲法院在 Weber and Putz（C-65/09和C-87/09, 2011）以及 Quelle AG v. Bundesverband（C-404/06, 2008）案中的判例阐明了缺陷商品消费者救济的范围。

## 韩国

### 法律框架

韩国的"柠檬法"制度是通过2019年1月1日生效的《汽车管理法》（자동차관리법）修正案引入的。该制度通常被称为결함차 반환법（缺陷车辆退还法）或韩国柠檬法。

### 车辆换货和退款制度（자동차 교환·환불 제도）

**适用条件：** 购买后一年内或行驶20,000公里以内（以先到者为准）的新车。

**合格缺陷：**
| 类别 | 标准 |
|------|------|
| 安全关键缺陷 | 影响制动、转向或燃油系统导致突然失灵的缺陷 |
| 反复维修失败 | 同一缺陷维修3次以上仍未解决 |
| 累计维修天数 | 车辆维修累计达30天以上 |
| 不同缺陷 | 2个以上不同缺陷，每个维修2次以上 |

**救济措施：** 消费者可要求：
- 换车：更换相同型号和配置的新车
- 退款：退还购买价格，扣除使用费用（按里程和折旧计算）

### 争议解决

当制造商和消费者意见不一致时，争议提交至根据《汽车管理法》设立的汽车争议审议委员会（자동차분쟁심의위원회）。该委员会包括技术专家、消费者权益代表和行业代表。决定为建议性质，但具有重要的实际影响力。

韩国消费者院（한국소비자원）也通过其一般消费者争议调解程序处理车辆投诉。车辆投诉持续位居投诉类别前列，每年约有8,000件汽车相关投诉。

### 召回制度

国土交通部（국토교통부）管理车辆召回制度。制造商必须报告已知缺陷并实施召回。韩国交通安全公团（한국교통안전공단）进行缺陷调查。2024年，约有720万辆车辆收到召回通知，该数字包括制造商主动发起和政府命令的召回。

### 里程碑案例：2018年宝马发动机起火危机

2018年宝马发动机起火危机是韩国柠檬法改革的催化事件。超过40辆宝马汽车（主要为520d车型）因废气再循环（EGR）阀缺陷而起火。这些事件引起了广泛的公众恐慌，导致政府对未召回的宝马车辆禁止上路——这是一项前所未有的措施。宝马韩国公司被罚款112亿韩元，该危机加速了车辆换货和退款制度的实施。

## 美国

### 法律框架

美国拥有世界上最发达和最多样化的柠檬法框架。康涅狄格州于1982年制定了第一部现代柠檬法，到1993年，所有50个州和哥伦比亚特区均已制定柠檬法法规。

### 典型的州柠檬法结构

| 要素 | 常见规定 |
|------|---------|
| 适用范围 | 在该州购买或租赁的新机动车辆 |
| 柠檬期间 | 交付后1-2年或12,000-24,000英里 |
| 维修次数 | 同一不合格项维修3-4次 |
| 停用天数 | 累计30个工作日 |
| 制造商义务 | 更换车辆或退还购买价格 |
| 使用扣减 | 基于里程的退款抵减公式 |
| 仲裁 | 许多州要求在诉讼前参加制造商仲裁 |
| 律师费 | 大多数州为胜诉消费者提供律师费 |

### 各州差异

**加利福尼亚州（Song-Beverly消费者保修法，Cal. Civ. Code §§1790-1795.8）：** 最强有力的柠檬法之一。涵盖带有制造商保修的新车和二手车。对故意违规提供最高两倍实际损失的民事处罚。诉讼前不要求仲裁。《坦纳消费者保护法》（Cal. Civ. Code §1793.22）创设了一项推定，即在首18个月或18,000英里内因同一问题维修4次或停用30天后，视为已进行了合理的维修尝试。

**纽约州（新车柠檬法，General Business Law §198-a）：** 适用于2年或18,000英里内的新车。要求4次维修尝试或30天停用。制造商必须通过纽约新车仲裁计划提供仲裁。

**得克萨斯州（柠檬法，Tex. Occ. Code §§2301.601-2301.613）：** 由得克萨斯州机动车辆管理局管理。适用于24个月或24,000英里内的新车。要求4次维修尝试、严重安全缺陷2次维修尝试或30天停用。不要求事先仲裁。

### 联邦《马格努森-莫斯保修法》

《马格努森-莫斯保修法》（15 U.S.C. §§2301-2312）为包括车辆在内的消费品提供了联邦层面的保修保护。主要特点：
- 要求书面保修标明为"完全"或"有限"保修
- 完全保修必须在合理时间内免费提供修理或更换；若修理失败，消费者可选择退款或更换
- 允许消费者对保修违规提起集体诉讼
- 为胜诉消费者提供律师费
- 不排除州柠檬法的适用；消费者可同时寻求联邦和州的救济

### 二手车柠檬法

部分州制定了涵盖二手车的柠檬法，但保护力度通常较弱：
- **纽约州：** 二手车柠檬法（GBL §198-b）适用于里程18,000-100,000英里的二手车
- **马萨诸塞州：** 适用于里程125,000英里以下的二手车
- **康涅狄格州：** 适用于持证经销商出售的二手车

### 柠檬法回购标记

当车辆根据柠檬法被回购时，大多数州要求在转售前在车辆产权证上标注"柠檬法回购"标记。这通过告知车辆缺陷历史来保护后续购买者。

## 德国

### 法律框架

德国没有专门的柠檬法法规。相反，缺陷车辆通过《德国民法典》的一般销售法条款（买卖法）处理，该条款经2002年《债法现代化法》（Schuldrechtsmodernisierung）大幅改革，以实施欧盟《消费者销售指令》。

### BGB保修（Gewährleistung）框架

**BGB第434条（物的瑕疵）：** 在风险转移时，物品符合约定的质量、适合合同规定的用途、适合通常使用且具有同类物品通常具备的且买方可期待的质量的，则无物的瑕疵。

**BGB第437条（缺陷情况下买方的权利）：** 买方可要求：
1. 补充履行（Nacherfüllung，第439条）——修理或更换
2. 解除合同（Rücktritt，第440条、第323条）——补充履行失败后
3. 减少价格（Minderung，第441条）
4. 损害赔偿（Schadensersatz，第440条、第280条、第281条）

### 救济层次

德国法律为缺陷车辆确立了严格的救济层次：

**第一步：补充履行（Nacherfüllung）：** 买方必须首先给予卖方通过修理或交付无缺陷替换品来纠正缺陷的机会（第439条）。卖方有权在修理和更换之间选择，除非所选方式不相称（第439条第4款）。

**第二步：解除合同或减价：** 若补充履行失败——判例法一般定义为同一缺陷的两次不成功维修尝试——买方可解除合同（Rücktritt）或减少价格（Minderung）。买方在行使这些次级救济前必须设定合理的额外履行期限（Nachfrist）（第323条）。

**使用补偿（Nutzungsentschädigung）：** 解除合同时，买方必须就车辆使用向卖方作出补偿，按照考虑购买价格和车辆预期使用寿命的公式计算。

### 联邦最高法院判例

联邦最高法院（BGH）就车辆保修发展了广泛的判例法：

**BGH, VIII ZR 94/13（2015年）：** 裁定柴油颗粒过滤器缺陷经两次维修仍无法修复的车辆，买方有权解除合同。

**BGH, VIII ZR 234/15（2016年）：** 确立了在卖方欺诈性隐瞒缺陷的情况下，买方可在不设定额外履行期限的情况下立即解除合同。

**柴油排放丑闻案件：** 大众柴油排放门丑闻（2015年）在德国产生了数千起诉讼案件。联邦最高法院在VIII ZR 225/17（2020年）案中裁定，大众使用非法排放作弊装置构成《德国民法典》第826条规定的故意违背善良风俗的损害行为（sittenwidrige vorsätzliche Schädigung），赋予买方在通常保修框架之外获得损害赔偿的权利。

### 举证责任倒置

BGB第477条（实施欧盟指令2019/771）规定，如果缺陷在交付后一年内显现，则推定缺陷在交付时已存在。这一期限由2022年实施更新版欧盟《消费者销售指令》的改革从六个月延长至一年。

### 诉讼时效

新车保修请求权的诉讼时效为自交付起两年（BGB第438条第1款第3项）。对于二手车，双方可协议将此缩短为一年。许多制造商提供超出法定期限的自愿制造商保修（Herstellergarantie）。

## 欧洲人权法院与欧盟

### 《消费者销售指令》（EU）2019/771

2019/771号指令关于货物销售合同的某些方面，自2022年1月1日起适用，取代了此前的1999/44/EC号指令，为包括车辆在内的缺陷商品救济提供了欧盟的基础框架。

**合格保证（第5-7条）：** 卖方对交付时存在且在交付后两年内显现的任何不合格承担责任。合格要求包括：符合合同描述、适合特定用途、具有同类商品的质量、附带所有配件和说明书交付。

**救济层次（第13-16条）：**
1. 修理或更换（消费者选择，除非不相称）
2. 价格减少或合同解除（修理/更换不可能、不相称、未在合理时间内提供或将造成重大不便时）

**举证责任倒置（第11条）：** 在交付后一年内显现的任何不合格推定为交付时已存在。成员国可将此延长至两年。

### 欧洲法院判例

**Quelle AG v. Bundesverband der Verbraucherzentralen（C-404/06, 2008）：** 法院裁定，当消费品依据合格保证被更换时，卖方不得向消费者收取更换前使用原缺陷商品期间的使用费。这一判决对车辆保修意义重大，因为它消除了德国法律此前要求的使用补偿。

**Weber and Putz（合并案件C-65/09和C-87/09, 2011）：** 法院裁定，卖方必须承担更换的全部费用，包括拆除缺陷商品和安装替换品的费用。卖方不得仅因费用不相称而拒绝更换；相反，卖方可承担适当的金额。

**Faber v. Autobedrijf Hazet Ochten（C-497/13, 2015）：** 法院裁定，举证责任倒置（如果缺陷在六个月内显现则推定为交付时已存在）具有广泛适用性，消费者只需证明不合格的存在，无需证明其原因。

### 一般产品安全法规与车辆型式认证

欧盟车辆型式认证框架（欧盟第2018/858号法规）为在欧盟销售的车辆确立了安全和环境标准。柴油排放门丑闻后，该法规得到加强，纳入了增强的市场监督、独立测试和违规处罚。该框架通过在监管层面解决系统性车辆缺陷，补充了消费者销售救济。

## 比较分析

### 专门框架与一般框架

美国是唯一拥有专门为缺陷车辆设计的柠檬法法规的司法管辖区。韩国于2019年在《汽车管理法》内引入了在功能上等同于柠檬法的车辆换货和退款制度。德国和欧盟依靠一般消费者销售法来处理车辆缺陷，没有车辆专门立法。一般框架方式的优势在于统一涵盖所有消费品，但可能无法充分解决汽车购买的独特特征。

### 维修机会门槛

触发更换或退款权利所需的维修尝试次数各不相同：
- 美国：通常3-4次（因州而异）
- 韩国：同一缺陷3次
- 德国：通常2次（联邦最高法院判例法）
- 欧盟指令："合理时间"，未指定具体次数

德国较低的门槛在这一特定维度上对消费者更为友好，但整体程序（包括额外履行期限要求）可能更为复杂。

### 使用扣减

一个关键的实际问题是，制造商能否在退还前扣除消费者使用缺陷车辆的费用：
- 美国：可以（所有州柠檬法中的里程公式）
- 韩国：可以（里程和折旧公式）
- 德国：最初可以，但欧洲法院在Quelle案中禁止了更换时的使用费收取；对于解除合同，联邦最高法院仍允许使用补偿
- 欧盟：欧洲法院禁止更换时的使用费收取；合同解除时的补偿因情况而异

### 仲裁要求

美国许多州要求消费者在提起诉讼前参加制造商主办的仲裁。此要求旨在促进高效的争议解决，但被批评为有利于制造商。韩国的争议审议委员会发挥类似功能。德国和欧盟一般不要求仲裁，但存在自愿仲裁委员会。
