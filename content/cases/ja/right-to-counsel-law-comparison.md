## 概要

弁護人依頼権——刑事犯罪で起訴された者が弁護において弁護士の援助を受けることができ、弁護人を選任する資力のない者には公費で弁護人が付与されるという法的保障——は、世界中の刑事司法制度における最も基本的な手続的保障の一つを構成し、他のすべての防御権の実効的行使を可能にするゲートウェイ的権利として機能している。韓国は弁護人依頼権を憲法第12条(4)（헌법 제12조 제4항）に明記しており、逮捕の時点から弁護人の援助を受ける権利（변호인의 조력을 받을 권리）を保障し、必要的弁護及び法律扶助に関する刑事訴訟法（형사소송법）の規定並びに大韓法律救助公団（대한법률구조공단）を設立する法律救助法（법률구조법）により補完されている。憲法裁判所は、警察取調中の弁護人立会いを要求した2004年の判決を含む画期的な決定を通じてこの権利を大幅に拡張してきた。米国は修正第6条に基づき弁護人依頼権を保障しており、連邦最高裁判所の画期的な判決であるGideon v. Wainwright, 372 U.S. 335 (1963)が、州は重罪事件の貧困な被告人に弁護人を提供しなければならないことを確立し、その後Argersinger v. Hamlin (1972)により拘禁の可能性があるすべての事件に、Douglas v. California (1963)により第一審上訴段階に拡大された。この権利はRothgery v. Gillespie County (2008)により定義されるすべての「重要な段階」において適用される。ドイツは弁護人の権利をECHR第6条(3)及び刑事訴訟法（Strafprozessordnung, StPO）第137条から第149条に基づき保障しており、重大な事件では必要的弁護（notwendige Verteidigung）が義務づけられ、国選弁護人（Pflichtverteidiger）制度が公費での弁護を提供している。EU Directive 2013/48/EUを実施した2019年の改正は、初期段階でのアクセスを大幅に強化した。ECHRは第6条(3)(c)に基づく弁護人依頼権に関する広範な判例法を発展させており、大法廷のSalduz v. Turkey判決（2008年）が最初の警察取調から弁護士へのアクセスが提供されなければならないことを確立し、EU Directive 2013/48/EUがすべての加盟国にわたり弁護士へのアクセス権を調和させている。

## 韓国

### 法的枠組み

**憲法第12条(4)：** 「逮捕又は拘禁された者は遅滞なく弁護人の援助を受ける権利を有する。刑事被告人が自力で弁護人を確保できない場合には、国家が法律の定めるところにより被告人のために弁護人を選任する。」

### 刑事訴訟法の規定

| 条文 | 内容 |
|-----------|---------|
| 第30条 | 被告人はいかなる段階においても弁護人を選任できる |
| 第33条 | 死刑又は無期懲役の事件には必要的弁護 |
| 第33条(2) | 被告人が貧困な場合、裁判所は弁護人を選任する |
| 第33条(3) | 被告人が未成年の場合、裁判所は弁護人を選任する |
| 第34条 | 被告人の家族は弁護人を選任できる |
| 第243条の2 | 検察取調中に弁護人の立会いを求める権利 |

### 憲法裁判所による権利の拡張

**憲法裁判所決定2003헌마282（2004年）：** 裁判所は、弁護人の援助を受ける権利が、取調の前後に弁護人と相談する権利のみならず、警察取調中に弁護人が物理的に立ち会う権利を包含すると判示した。この決定は韓国の刑事手続において画期的であった。警察はそれまで弁護人の立会いなしに取調を行っていたからである。

**憲法裁判所決定2008헌마550（2011年）：** 裁判所は、勾留された被疑者と弁護人との接見（접견교통권）を特定の時間帯に制限し、または不合理な条件を課すことが弁護人依頼権の憲法上の保障に違反すると判示した。裁判所は、弁護人依頼権が単なる手続的形式ではなく、刑事手続の公正さに不可欠な実体的保障であることを強調した。

### 大韓法律救助公団（대한법률구조공단）

法律救助法（법률구조법）に基づき設立されたこの公団は、弁護人を選任する資力のない者に無料の法的代理を提供している。約300名の専任弁護士を雇用し、私選弁護士と契約して溢れた事件を処理している。2024年現在、公団は年間10万件以上の事件を処理している。

### 必要的弁護

刑事訴訟法は以下の場合に必要的弁護（国選弁護人）を義務づけている：

| 状況 | 条文 |
|-------------|-----------|
| 死刑又は無期懲役に該当する罪 | 第33条(1)第1号 |
| 被告人が未成年者 | 第33条(1)第2号 |
| 被告人が70歳以上 | 第33条(1)第3号 |
| 被告人が聴覚障害者、言語障害者、又は精神障害者 | 第33条(1)第4号 |
| 被告人に心神喪失の疑い | 第33条(1)第5号 |
| 被告人が弁護人を請求し貧困な場合 | 第33条(2) |

### 課題

憲法上の保護にもかかわらず、国選弁護人の質は継続的な懸念事項である。大韓弁護士協会の調査により、国選弁護人の中には依頼人との面談に最小限の時間しか費やさず、形式的な弁護のみを提供しているケースがあることが判明している。私選弁護と比較して著しく低い国選弁護人の報酬が、寄与する要因として指摘されている。

## 米国

### 法的枠組み

**修正第6条：** 「すべての刑事訴追において、被告人は……その弁護のために弁護人の援助を受ける権利を享有する。」

### 画期的な判例

| 事件 | 年 | 判示 |
|------|------|---------|
| Powell v. Alabama | 1932年 | デュープロセスに基づく死刑事件における弁護人依頼権 |
| Gideon v. Wainwright | 1963年 | 貧困な重罪被告人に対する国選弁護人の権利 |
| Douglas v. California | 1963年 | 権利としての第一審上訴における弁護人の権利 |
| Miranda v. Arizona | 1966年 | 身柄拘束取調中の弁護人依頼権 |
| Argersinger v. Hamlin | 1972年 | 拘禁の可能性があるすべての事件に拡大 |
| Strickland v. Washington | 1984年 | 弁護の不実効に対する基準 |
| Rothgery v. Gillespie Cnty. | 2008年 | 裁判官の前への最初の出頭時に権利が発生 |
| Lafler v. Cooper | 2012年 | 権利が司法取引手続にも及ぶ |

### Gideon v. Wainwright (1963)

連邦最高裁判所のGideonにおける全員一致の判決は、修正第6条の弁護人依頼権が修正第14条を通じて州に適用される基本的権利であることを確立した。フロリダ州で重罪の建造物侵入罪で起訴されたクラレンス・アール・ギデオンは、フロリダ州法が死刑事件にのみ無料の弁護人を提供していたため、国選弁護人を拒否された。ギデオンの連邦最高裁判所への手書きの上告は、アメリカの刑事司法史上最も重大な判決の一つにつながった。

### 貧困者弁護制度

米国は貧困な被告人に弁護人を提供するために3つの主要モデルを採用している：

| モデル | 説明 | 採用州 |
|-------|-------------|-------------|
| 公設弁護人事務所 | 常勤の弁護人を擁する政府資金による事務所 | 約28州（カリフォルニア州、ニューヨーク州を含む） |
| 当番弁護人 | 裁判所が事件ごとに私選弁護士を選任 | 約15州 |
| 契約制度 | 政府が私選弁護士又は法律事務所と契約 | 約7州 |
| 混合制度 | 上記モデルの組み合わせ | 複数の州 |

### 貧困者弁護の危機

Gideonの保障にもかかわらず、米国の貧困者弁護は体系的な危機に直面している。アメリカ法曹協会は、複数の州における過剰な事件負担、不十分な資金、及び体系的な失敗を記録してきた。2024年、ルイジアナ州の公設弁護人制度は深刻な資金不足に直面し、裁判所は弁護人依頼権の体系的な侵害を宣言した。個々の公設弁護人が年間300件を超える重罪事件を抱えていた。

### Strickland v. Washington (1984)

弁護の不実効に対する二段階テストは、被告人が以下を証明することを要求する：(1) 弁護人の遂行が客観的な合理性の基準を下回ったこと（「不十分な遂行」）、及び(2) 弁護人の専門家としてふさわしくない過誤がなければ、手続の結果が異なっていたであろう合理的な蓋然性があること（「不利益」）。この基準は弁護人に対して過度に恭順であるとして批判されており、Strickland請求の認容は比較的稀である。

## ドイツ

### 法的枠組み

**StPO第137条から第149条：** これらの規定はドイツの刑事手続における弁護人の権利に関する包括的な枠組みを確立している。

### 主要な規定

| 条文 | 内容 |
|---------|---------|
| 第137条 | いかなる段階においても弁護人を選任する権利 |
| 第138条 | 弁護人の資格要件 |
| 第140条 | 必要的弁護（notwendige Verteidigung）の事件 |
| 第141条 | 国選弁護人（Pflichtverteidiger）の選任 |
| 第141a条 | 警察取調中の弁護人依頼権 |
| 第148条 | 弁護人と被告人の間の無制限の通信 |
| 第148a条 | テロ事件における通信の制限 |

### 必要的弁護（Notwendige Verteidigung、第140条）

ドイツ法は指定された状況において弁護人の選任を義務づけている：

| 状況 | 条文 |
|-------------|-----------|
| 上級地方裁判所又は地方裁判所における手続 | 第140条(1)第1号 |
| 被告人が重罪（Verbrechen）で起訴されている場合 | 第140条(1)第2号 |
| 手続が職業上の資格喪失をもたらす可能性がある場合 | 第140条(1)第3号 |
| 被告人が未決勾留中の場合 | 第140条(1)第4号 |
| 被告人が3ヶ月以上勾留されていた場合 | 第140条(1)第5号 |
| 被告人が精神科病院に収容される可能性がある場合 | 第140条(1)第7号 |
| 前任の弁護人が手続から排除された場合 | 第140条(1)第8号 |
| 事件の複雑さが弁護人を必要とする場合 | 第140条(2) |

### 2019年改正：EU Directive 2013/48/EUの実施

必要的弁護権に関する法律の改正（Gesetz zur Neuregelung des Rechts der notwendigen Verteidigung）は、2019年12月13日に施行され、EU Directive 2013/48/EUを実施して初期段階での弁護人へのアクセスを大幅に強化した：

- **最初の取調からの弁護人依頼権：** StPO第141a条は、正式な逮捕時のみならず、最初の警察取調中に弁護人の立会いを受ける権利を明示的に保障するようになった。
- **必要的弁護のより早期の発動：** 弁護人の選任は、必要的弁護が適用される事件において最初の取調の前に必要とされるようになった。
- **通知の強化：** 被疑者は取調の開始前に弁護人依頼権について告知されなければならない。

### 国選弁護人（Pflichtverteidiger）制度

国選弁護人（Pflichtverteidiger）は裁判所により地元の弁護士会から選任される。米国の公設弁護人モデルとは異なり、ドイツは政府雇用の弁護人を維持していない。代わりに、私選弁護士が選任され、弁護士報酬法（Rechtsanwaltsvergütungsgesetz）に基づき報酬が支払われる。規制されてはいるものの、報酬は一般に適切な弁護を確保するのに十分である。

## ECHR及びEU

### ECHR第6条(3)(c)

「刑事犯罪で起訴されたすべての者は、次の最低限の権利を有する：……(c) 自ら弁護し、又は自ら選任した弁護人の援助を受けること、若しくは弁護費用を支払う十分な資力がない場合において司法の利益のためにそれを必要とするときは、無償で弁護人の援助を受けること。」

### Salduz v. Turkey [GC]（申立番号36391/02、2008年）

大法廷は、弁護士へのアクセスは最初の警察取調から提供されなければならないと判示した。ただし、この権利を制限する「やむを得ない理由」がある場合はこの限りでない。裁判所は、弁護士へのアクセス権が単なる抽象的な保障ではなく、実際的かつ実効的な要件であることを確立した。弁護士へのアクセスなしに得られた供述は有罪判決の根拠として使用してはならない。

### Ibrahim and Others v. United Kingdom [GC]（申立番号50541/08、2016年）

大法廷は、警察取調中の弁護人依頼権に対する一時的制限が正当化されうるかを検討した。裁判所は、そのような制限にはやむを得ない理由が必要であり、一時的でなければならず、手続全体の公正さが維持されなければならないと判示した。裁判所は二段階テストを適用した：(1) 制限にやむを得ない理由があったか、(2) 制限が公判の全体的公正さを回復不能に害したか。

### EU Directive 2013/48/EU

刑事手続における弁護士へのアクセス権に関する指令は最低基準を確立している：

- **第3条：** 不当な遅滞なく、かつ、いかなる場合にも以下のうち最も早い時点から弁護士へのアクセス権を有する：警察による取調の前、捜査活動の実施時、又は自由の剥奪後不当な遅滞なく。
- **第3条(3)：** この権利には、被疑者が取調中に弁護人の立会いを受け、弁護人が参加することが含まれる。
- **第4条：** 自由の剥奪時に第三者に通知してもらう権利。
- **第9条：** 権利放棄は任意かつ明白でなければならず、十分な情報を得た上でなされ、記録されなければならない。

### EU Directive 2016/1919（法律扶助）

この指令は刑事手続における法律扶助の最低基準を確立し、十分な資力を欠く被疑者及び被告人が暫定的法律扶助を受ける権利を有すること、及び資力・本案の審査が公正に適用されることを保障している。

## 比較分析

### 権利の発生時期

弁護人依頼権が発生する時期は法域により異なる。韓国は逮捕の時点から権利を保障している。米国は「重要な段階」の法理を発展させ、最初の正式な司法手続の時点で権利が発生する。ドイツ及びECHRは最初の警察取調からのアクセスを保障している。

### 弁護の質

すべての法域が公費で賄われる弁護の質の確保に苦闘している。米国はStricklandを通じて最も発展した（ただし論争的な）基準を有している。ドイツは専門職業的弁護士会基準と国選弁護人制度に依拠している。韓国は国選弁護における質の格差を認識している。ECHRは手続全体の公正性の基準に基づいて質を評価している。

### 権利放棄の基準

すべての法域が弁護人依頼権の放棄を許容しているが、保障措置の程度は異なる。ECHR（Salduzに続いて）は権利放棄が任意かつ明白であり、被疑者が十分に告知された後に行われることを要求している。米国法は「知情的、賢明かつ任意的な」放棄を要求する（Johnson v. Zerbst, 1938年）。ドイツ法は放棄の前に被告人に権利について告知されることを要求している。
