## 概要

物品税――特定の商品の製造、販売、または消費に課される賦課金であり、健康や環境に有害とみなされる製品を対象とする場合には「罪の税」と呼ばれることが多い――は、歳入確保と行動変容の二重の目的に資し、各法域は従来の財政機能に加えて公衆衛生手段としても物品税政策をますます活用している。韓国は個別消費税法（개별소비세법）および酒税法（주세법）に基づき物品税を課しており、アルコール、たばこ、自動車に対して特に重要な賦課金を設け、年間約10兆ウォンの税収を生み出し交通インフラおよび環境プログラムに充当される交通・エネルギー・環境税（교통·에너지·환경세）により補完されている。米国は内国歳入法のSubtitle D（26 U.S.C. Sections 4001-5000）およびSubtitle E（アルコール、たばこ、火器）に基づき連邦物品税を課しており、アルコール・たばこ税・貿易局（TTB）およびIRSが管轄する。連邦たばこ税は1箱1.01ドルと他のOECD諸国の平均3.50〜5.00ドルと比較して概ね低率であるが、州レベルの物品税が大幅な追加負担を加え、1箱当たり0.17ドル（ミズーリ州）から5.35ドル（コネチカット州）まで変動する。ドイツはEU調和の枠組み内で物品税を課しており、エネルギー税法（Energiesteuergesetz）、たばこ税法（Tabaksteuergesetz）、蒸留酒専売法（Branntweinmonopolgesetz、2018年にAlkoholsteuergesetzに置換）、およびビール税法（Biersteuergesetz）がCouncil Directive 2011/64/EU（たばこ）、Directive 2003/96/EC（エネルギー）、およびDirective 92/83/EEC（アルコール）により確立されたEU最低税率を実施しており、ドイツは一般にEU最低税率を上回る税率を適用している。EUレベルでは、たばこ、アルコール、およびエネルギー製品に対する物品税の調和がEU税制調整の最も発展した分野のひとつを代表する一方、ECHRは第1議定書第1条に基づき物品税を国家の財政権限の正当な行使として支持してきた。

## 韓国

### 法的枠組み

韓国の物品税制度は、異なる行政機関が所管するいくつかの個別の賦課金から構成されている。個別消費税法（개별소비세법、法律第19931号）に基づく個別消費税（개별소비세）は、奢侈品、自動車、石油製品、および一部の娯楽施設に適用される。酒税法（주세법）に基づく酒税（주세）はすべての酒類に適用される。たばこ関連の賦課金は、たばこ事業法（담배사업법）、国民健康増進法（국민건강증진법）、および地方税法（지방세법）を含む複数の法令を通じて課される。

### たばこ課税

韓国のたばこ税制は多層的なアプローチで注目される。標準的なたばこ1箱の小売価格を2,500ウォンから4,500ウォンに引き上げた2015年の大幅な増税後の税構成要素は以下の通りである：

| 構成要素 | 1箱当たりの金額 |
|----------|----------------|
| 個別消費税（개별소비세） | 841ウォン |
| 国民健康増進負担金（건강증진부담금） | 841ウォン |
| 地方教育税（지방교육세） | 443ウォン |
| たばこ消費税（담배소비세、地方） | 1,007ウォン |
| 廃棄物負担金（폐기물부담금） | 24.4ウォン |
| 付加価値税（부가가치세） | 409ウォン |
| **税負担合計** | **約3,565ウォン（小売価格の約79%）** |

2015年の増税は2004年以来初のたばこ税の大幅調整であった。成人男性の喫煙率は2014年の43.7パーセントから2016年の36.7パーセントに低下したが、因果関係は議論がある。健康増進負担金は国民健康増進基金に充当されている。

### アルコール課税

酒税法は製造者または輸入者の出荷価格に対して従価税率を課す：

| 飲料種別 | 税率 |
|----------|------|
| 蒸留酒（焼酎、ウイスキー） | 72% |
| ビール（맥주） | 830.3ウォン/リットル（2020年以降容量ベース） |
| ワイン | 30% |
| 伝統酒（전통주） | 5%（優遇税率） |
| マッコリ（濁酒） | 5% |

2020年の重要な改革によりビール課税が従価税から従量税（容量ベース）に転換された。これは主に、以前はより低い輸入価格を基準に課税されていた輸入ビールに対して国内のクラフトビール醸造業者が直面していた競争上の不利を解消するためであった。

### 交通・エネルギー・環境税

専用法に基づく交通・エネルギー・環境税（교통·에너지·환경세）は石油製品に適用され、韓国の主要な燃料物品税である：

| 製品 | 税率 |
|------|------|
| ガソリン | 529ウォン/リットル |
| 軽油 | 375ウォン/リットル |
| LPG（車両用ブタン） | 160ウォン/kg |

この税は元来1994年に時限措置として制定されたが、繰り返し延長されてきた。税収は目的充当されている：80パーセントが交通インフラ（道路、鉄道）に、20パーセントが環境プログラムおよび地域開発に充てられる。追加的に燃料物品税の15パーセントの教育税（교육세）が課される。

## 米国

### 法的枠組み

連邦物品税は内国歳入法に基づき課されており、主にSubtitle D（諸物品税）およびSubtitle E（アルコール、たばこ、その他の物品税）に規定されている。財務省内のアルコール・たばこ税・貿易局（TTB）がアルコールおよびたばこの物品税を管轄する。IRSがその他の連邦物品税を管轄する。州レベルの物品税がさらに相当な追加層を加える。

### たばこ物品税

連邦たばこ税は20本入り1箱当たり1.01ドルであり、2009年の児童健康保険プログラム再認可法（CHIPRA）により最後に引き上げられた。州税率は大きく異なる：

| 州 | 1箱当たりの税率 | 連邦+州合計 |
|----|----------------|-------------|
| ミズーリ | 0.17ドル | 1.18ドル |
| バージニア | 0.60ドル | 1.61ドル |
| テキサス | 1.41ドル | 2.42ドル |
| カリフォルニア | 2.87ドル | 3.88ドル |
| ニューヨーク | 5.35ドル | 6.36ドル |
| コネチカット | 4.35ドル | 5.36ドル |
| コロンビア特別区 | 5.03ドル | 6.04ドル |

連邦たばこ表示・広告法（15 U.S.C. § 1331 et seq.）はたばこ広告の州規制を連邦専管としているが、州課税の専管は行っていない。すべてのたばこ密売防止法（PACT Act、2009年）は物品税脱税を防止するためにオンライン販売および州際たばこ密売に対処している。

### アルコール物品税

26 U.S.C. Subtitle Eに基づく連邦アルコール物品税率：

| 製品 | 連邦税率 |
|------|----------|
| 蒸留酒 | 13.50ドル/プルーフガロン |
| ビール | 18.00ドル/バレル（31ガロン）；小規模醸造業者には減税率 |
| ワイン（14%以下） | 1.07ドル/ガロン |
| ワイン（14〜21%） | 1.57ドル/ガロン |
| ワイン（21〜24%） | 3.15ドル/ガロン |

2017年のクラフト飲料近代化・税制改革法（2020年に恒久化）は小規模生産者の税率を大幅に引き下げた：2,223万プルーフガロン未満を生産する蒸留業者は最初の10万プルーフガロンに3.50ドル/プルーフガロン；200万バレル未満を生産する醸造業者は最初の6万バレルに3.50ドル/バレル。

### 連邦自動車燃料物品税

連邦ガソリン税は1ガロン当たり0.184ドル（軽油は0.244ドル）であり、1993年以来据え置きである。税収は26 U.S.C. § 9503に基づき高速道路信託基金に充当される。インフレ調整の欠如により実質価値が低下し、交通インフラの慢性的な資金不足の一因となっている。税率をインフレに連動させる、または走行距離税（VMT税）に置き換える複数の提案が立法上の推進力を得ることができていない。

州ガソリン税は約0.09ドル（アラスカ）から0.67ドル（カリフォルニア、排出権取引負担金を含む）まで変動する。いくつかの州は卸売価格またはインフレ指数に連動する変動税率メカニズムを採用している。

### 連邦砂糖税またはSSB税の不在

米国には連邦の砂糖入り飲料（SSB）税は存在しない。フィラデルフィア（1オンス当たり1.5セント、2017年）、バークレー（1オンス当たり1セント、2015年）、シアトル（1オンス当たり1.75セント、2018年）、およびボルダー（1オンス当たり2セント、2017年）を含む地方自治体がSSB税を制定している。アリゾナ、ミシガン、ワシントン州を含むいくつかの州は地方のSSB税を先制的に禁止している。

### 画期的判例

**National Federation of Independent Business v. Sebelius, 567 U.S. 519 (2012)：** 最高裁判所はACAの個人委任を課税権の正当な行使として支持し、共同責任支払いを物品税として性格づける一方、通商条項に基づく権限は否定した。

**Department of Revenue of Montana v. Kurth Ranch, 511 U.S. 767 (1994)：** 裁判所は、モンタナ州の「危険薬物税」――本質的には刑事訴追後に違法薬物に課される物品税――が二重処罰禁止条項に違反すると判示した。

## ドイツ

### 法的枠組み

ドイツの物品税制度はCouncil Directive (EU) 2020/262（物品税に関する一般的取り決め）により確立されたEU調和の枠組み内で運用されており、製品固有の指令により補完されている。国内実施はエネルギー税法（EnergieStG）、たばこ税法（TabStG）、アルコール税法（AlkStG、2018年施行）、ビール税法（BierStG）、スパークリングワイン・中間製品税法、およびコーヒー税法（Kaffeesteuergesetz――主要EU加盟国の中でドイツに固有）を通じて行われている。

### たばこ課税

ドイツのたばこ税法に基づくたばこ物品税はDirective 2011/64/EUを実施している：

| 製品 | 税率 |
|------|------|
| たばこ | 118.78ユーロ/1000本 + 小売価格の21.69% |
| 刻みたばこ | 60.82ユーロ/kg + 小売価格の13.13% |
| 葉巻 | 14ユーロ/1000本 + 小売価格の1.47% |
| 加熱式たばこ製品 | 15ユーロ/kgたばこ含有量（2022年導入） |

標準的なたばこ1箱（20本、小売約8.00ユーロ）の実効税負担は約5.40ユーロであり、小売価格の約67.5パーセントを占める。ドイツはたばこ税近代化法（Tabaksteuermodernisierungsgesetz、2021年）により2026年までの年次段階的税率引上げを実施してたばこ税率を漸進的に引き上げた。

### エネルギー課税

エネルギー税法はDirective 2003/96/ECを実施している：

| 製品 | 税率 |
|------|------|
| ガソリン（無鉛） | 0.6545ユーロ/リットル |
| 軽油 | 0.4704ユーロ/リットル |
| 天然ガス（自動車燃料） | 0.0180ユーロ/kWh |
| 軽質暖房油 | 0.0614ユーロ/リットル |
| 重質暖房油 | 0.0250ユーロ/kg |
| LPG（自動車燃料） | 0.409ユーロ/kg |

ドイツはウクライナ侵攻後の価格高騰に対応したエネルギー価格救済策の一環として、2022年夏に自動車燃料のエネルギー税率を一時的に引き下げた（「ガソリン値引き（Tankrabatt）」）。ガソリン税は約0.30ユーロ/リットル、軽油税は約0.14ユーロ/リットル、それぞれ3ヶ月間引き下げられた。

### CO2価格設定（BEHG）

燃料排出量取引法（Brennstoffemissionshandelsgesetz, BEHG、2019年）はEU ETSの対象外となっている暖房・輸送用燃料に対する国内排出量取引制度を導入した。CO2価格の推移：

| 年 | CO2価格（トン当たり） |
|----|---------------------|
| 2021年 | 25ユーロ |
| 2022年 | 30ユーロ |
| 2023年 | 30ユーロ |
| 2024年 | 45ユーロ |
| 2025年 | 55ユーロ |
| 2026年 | 市場ベース（55〜65ユーロの範囲） |

これは事実上、燃料の炭素含有量に対する追加的な物品税として機能し、2024年の価格水準でガソリンに約0.11ユーロ/リットル、軽油に約0.12ユーロ/リットルが加算される。

### アルコール課税

| 製品 | 税率 |
|------|------|
| 蒸留酒 | 13.03ユーロ/リットル純アルコール |
| アルコポップス加算 | 5,550ユーロ/ヘクトリットル純アルコール |
| ビール | 0.787ユーロ/プラト度/ヘクトリットル |
| スパークリングワイン | 136ユーロ/ヘクトリットル |
| ワイン | 0ユーロ（スティルワインに物品税なし） |
| コーヒー | 2.19ユーロ/kg（焙煎）；4.78ユーロ/kg（インスタント） |

ドイツはスティルワインに物品税を課していない点が注目されており、これはEU Directive 92/83/EEC（ワインの最低税率を0ユーロに設定）の下で許容されている。コーヒー税（Kaffeesteuer）はフリードリヒ大王の時代に遡る歴史的遺産であり、主要EU加盟国の中でドイツに固有のものであって、年間約10億ユーロの税収を生み出している。

### 砂糖入り飲料税

ドイツは公衆衛生団体の提唱にもかかわらず砂糖入り飲料税を制定していない。連邦政府は代わりに砂糖含有量の削減を目標とする自発的な産業協定（Nationale Reduktionsstrategie、2018年）を追求してきた。他のいくつかのEU加盟国はSSB税を実施している：フランス（2012年、2018年改革）、英国（Soft Drinks Industry Levy、2018年）、アイルランド（Sugar Sweetened Drinks Tax、2018年）、およびポルトガル（2017年）。

## ECHR & EU

### EU調和の枠組み

EUはいくつかの主要な指令を通じて物品税の重要な調和を達成してきた：

**Council Directive (EU) 2020/262（一般的取り決め）：** アルコール、たばこ、およびエネルギー製品に対する物品税の共通枠組みを確立し、物品移動管理制度（EMCS）を用いた加盟国間の移動中の課税停止制度を含む。

**Council Directive 2011/64/EU（製造たばこ）：** たばこ課税の最低税率および構造要件を設定。最低物品税収入は1000本当たり少なくとも90ユーロでなければならず、総課税額（物品税+付加価値税）は加重平均小売価格の少なくとも60パーセントを占めなければならない。

**Council Directive 2003/96/EC（エネルギー製品）：** 自動車燃料または暖房燃料として使用されるエネルギー製品の最低税率を確立。改正案（COM/2021/563）はエネルギー含有量および環境性能に基づく税率の再構築を提案し、化石燃料の優遇措置を廃止するものである。

**Council Directive 92/83/EEC（アルコール）：** アルコールおよびアルコール飲料に対する物品税の構造を調和し、Council Directive 92/84/EECにより最低税率が設定されている。2020年に小規模独立生産者および変性アルコールに関する規定を更新する改正が行われた。

### EU炭素国境調整メカニズム（CBAM）

Regulation (EU) 2023/956は炭素国境調整メカニズムを確立し、2023年10月から（移行的報告段階）適用され、2026年1月から完全な財務義務が発効する。CBAMは事実上、輸入品の内包された炭素含有量に対する物品税型の賦課金として機能し（当初はセメント、鉄鋼、アルミニウム、肥料、電力、水素を対象）、EU ETSの炭素価格を基準に算定される。これは炭素リーケージを防止するために設計された新しい形態の国境調整を代表している。

### 物品課税に関するECHR判例法

**Ferrazzini v. Italy（Application No. 44759/98、2001年）：** 大法廷は、物品税事項を含む税務紛争がECHR第6条の「市民的権利義務」に関わらないと判示し、第6条の公正な裁判の保障は実体的な税額査定紛争には適用されないとした（ただし「刑事上の罪」に該当する制裁には適用される）。

**Bulves AD v. Bulgaria（Application No. 3991/03、2009年）：** 物品税ではなくVATに焦点を当てた事件であるが、取引先の税務義務不履行を理由に納税者の仕入税額控除を否認することが第1議定書第1条に違反することを確立した。この原則は物品税の還付状況にも類推適用されている。

**Impar Ltd v. Lithuania（Application No. 13102/04、2010年）：** 裁判所は物品税の制裁を審査し、物品税の不備に対する不均衡な制裁が財産権を侵害し得るとし、税収確保の公益と納税者に課される負担との均衡が必要であると判示した。

## 比較分析

### 歳入と行動変容政策の緊張

3つの法域すべてが、物品税収の最大化と行動変容の達成という固有の緊張を乗り越えている。有害製品の消費を効果的に減少させる高い税率は必然的に課税ベースを縮小させる。韓国の2015年のたばこ増税は当初は税収増をもたらしたが、喫煙率の低下に伴い税収は減少した。1993年以来据え置きの米国の連邦燃料税は歳入源としても浸食され、実質価値の低下により環境目標の達成にも失敗している。

### たばこ税の有効性

3つの法域の中で、韓国の2015年のたばこ増税は物品税の有効性に関する最も明確な自然実験を提供している。たばこ価格のほぼ倍増は、特に若年成人と低所得層において喫煙率の測定可能な減少をもたらした。ドイツのたばこ税近代化法に基づく段階的増税戦略はより漸進的なアプローチをとっている。米国の分断された制度は州間の大きな税率格差により、相当な越境購入と密輸を生み出し、歳入目標と公衆衛生目標の両方を損なっている。

### 燃料課税と環境政策

ドイツはBEHGに基づくCO2価格設定を含めると3つの法域の中で最も高い実効燃料税を課しており、ガソリンの総課税額は約0.90ユーロ/リットル（約3.60ドル/ガロン）に達する。韓国の燃料物品税はOECD基準で中程度である。米国は最も低い燃料課税であり、連邦税と州平均税の合計約0.50ドル/ガロンは欧州水準のわずかな割合を占めるにすぎない。この格差は交通、都市計画、および気候政策の根本的に異なる政治経済を反映している。

### 砂糖および健康関連物品税

3つの主要法域のいずれも国レベルの砂糖入り飲料税を制定していないが、いくつかのEU加盟国（フランス、英国、アイルランド）および米国の地方自治体（フィラデルフィア、バークレー、シアトル）は制定している。新たなエビデンスはSSB税が消費を減少させ製品の改良を促すことを示唆しており、英国の段階的なSoft Drinks Industry Levyに示されているように、メーカーは税の施行前からでも砂糖含有量の削減を促されている。

### 物品税収の目的充当

韓国の交通・環境目的への燃料税収の広範な目的充当は、ドイツの一般歳入アプローチおよび米国のハイブリッドモデル（連邦燃料税は高速道路信託基金に充当されるがその他の物品税収の大部分は一般歳入）とは対照的である。目的充当はインフラ投資に対する財政的確実性を提供するが、予算の柔軟性を低下させる。
