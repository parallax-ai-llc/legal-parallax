## 概要

労災補償――業務中に負傷または疾病に罹患した従業員に対し、医療、賃金補填、および障害給付を提供する強制保険または法定給付の制度――は、最も古い社会保険の形態の一つであり、従業員が過失にかかわらず保障された給付を受け、雇用者が不法行為責任の免除を受けるという「大取引」を体現している。韓国の産業災害補償保険法（산업재해보상보험법）は1963年に同国最初期の社会保険プログラムの一つとして制定され、勤労福祉公団（근로복지공단）が管理する包括的な公的保険制度を確立し、1名以上の従業員を有する事業所のほぼすべての労働者を対象としている。同制度は医療給付、平均賃金の70%の休業給付、14等級の障害給付、遺族給付、および職業リハビリテーションを提供しており、2021年には業務関連のうつ病、PTSD、および過労関連心血管疾患（과로사）を含むメンタルヘルス疾患への大幅な拡大が行われた。米国は50の州別制度を通じて労災補償を運営しており、適用範囲、給付、および管理に大きな差異がある。これは連邦職員補償法（FECA）、港湾労働者補償法、および黒肺給付法を含む特定の労働者カテゴリーに対する連邦プログラムによって補完されている。ほとんどの州は無過失モデルに従い、従業員が雇用者を不法行為で訴えることを禁じる排他的救済規定を設けている。ドイツは1884年にオットー・フォン・ビスマルクの労災保険法（Unfallversicherungsgesetz）を通じて法定労災保険を先駆的に導入し、現在は社会法典第VII編（SGB VII）に法典化され、職業協同組合（Berufsgenossenschaften）によって管理されている。同制度は労災補償機能と職場安全規制を独自の統合的制度において組み合わせている。ドイツの制度はすべての従業員、学生、およびボランティアを対象としており、従業員の拠出は不要である。欧州レベルでは、労災補償は主に各国の権限に留まり、EUの調整は越境労働者のための社会保障調整に関する規則（EC）No 883/2004に限定されている。一方、ILO第121号条約（業務災害給付、1964年）は国際的な最低基準を確立している。

## 韓国

### 法的枠組み

産業災害補償保険法（산업재해보상보험법）は1963年11月5日に制定され、1964年7月1日に施行された。これは朴正煕政権の経済開発計画のもとで設立された最初の社会保険プログラムの一つである。当初は500名以上の従業員を有する鉱業および製造業の事業所のみを対象としていたが、適用範囲は段階的に拡大され、2000年以降はすべての1名以上の事業所に適用されている。

### 管理運営

同法に基づいて設立された勤労福祉公団（근로복지공단、COMWEL）は、保険制度のすべての側面を管理している。保険料の徴収、請求の審査、傘下の病院およびリハビリテーション・センターを通じた医療、職業リハビリテーション、および職場復帰プログラムである。制度は全額雇用者負担の経験料率保険料によって賄われており、従業員の拠出はない。

### 給付体系

| 給付の種類 | 説明 | 算定方法 |
|-----------|------|---------|
| 療養給付（요양급여） | 業務上の負傷・疾病に対するすべての必要な医療 | 全額負担；自己負担なし |
| 休業給付（휴업급여） | 療養期間中の賃金補填 | 平均日額賃金の70% |
| 障害給付（장해급여） | 永続的な障害に対する補償 | 14等級：1級（最も重度）＝1,474日分の賃金一時金または年金；14級＝55日分の賃金 |
| 遺族給付（유족급여） | 死亡した労働者の遺族への給付 | 年金：平均賃金の52〜67%；または一時金：1,300日分の賃金 |
| 葬祭費（장의비） | 葬儀費用 | 平均賃金120日分 |
| 職業リハビリテーション給付（직업재활급여） | 再訓練および職場復帰支援 | 職場改善補助金を含む各種プログラム |

### 2021年メンタルヘルスおよび過労に関する改正

2021年の改正は、業務関連メンタルヘルス疾患の認定を大幅に拡大した。

**過労関連疾病（과로사）：** 同改正は、韓国の集約的な労働文化を踏まえ特に関連性の高い、慢性的過労による心血管・脳血管疾患（뇌심혈관계 질병）の認定基準を正式に定めた。12週間の平均で週60時間を超える労働を行い、心筋梗塞、脳卒中、またはそれに関連する疾患を発症した労働者は、業務関連疾病と推定される。

**メンタルヘルス：** 業務関連のうつ病、不安障害、PTSD、および適応障害が明示的に対象となり、承認手続きが簡素化された。また、職場いじめ（직장 내 괴롭힘）が補償対象となるメンタルヘルス疾患の原因として認定された。

### 統計

2023年に、COMWELは約130,000件の産業災害請求を処理し、承認率は約89%であった。給付支出総額は8.5兆韓国ウォンを超えた。労災死亡補償率は着実に低下しているが、ドイツやほとんどのEU加盟国よりも依然として高い。

## 米国

### 法的枠組み

米国における労災補償は主に州法によって規律されており、50の異なる制度が大きな差異をもって存在している。最初の州労災補償法は1911年にウィスコンシン州で制定され、1948年までにすべての州が労災補償プログラムを確立した。

### 州制度の差異

| 特徴 | 多数派アプローチ | 注目すべき差異 |
|------|----------------|---------------|
| 保険の仕組み | 民間保険または州基金 | オハイオ州、ワシントン州、ワイオミング州、ノースダコタ州：排他的州基金（独占制） |
| 賃金補填率 | 平均週賃金の66.67% | 60%から80%の範囲 |
| 最高週給付額 | 通常、州平均週賃金の100〜150% | 大きな差異あり |
| 待機期間 | 給付開始まで3〜7日 | 障害が一定期間を超えた場合に遡及的に支払う州もある |
| 弁護士費用 | 労災委員会が規制 | 成功報酬は通常15〜25% |
| 排他的救済 | あり、例外あり | 故意の行為に対して不法行為訴訟を許可する州もある |

### 連邦プログラム

| プログラム | 適用範囲 | 管理機関 |
|-----------|---------|---------|
| 連邦職員補償法（FECA） | 連邦民間職員 | 労働省労災補償プログラム局 |
| 港湾労働者補償法 | 海事労働者（船員を除く） | 労働省OWCP |
| 黒肺給付法 | じん肺症の炭鉱労働者 | 労働省OWCP |
| エネルギー従業員職業病補償プログラム | 放射線・毒物に暴露されたDOE労働者 | 労働省OWCP |
| 国防基地法 | 海外軍事基地の請負業者 | 労働省OWCP |

### 現代の課題

**オプトアウト運動：** テキサス州はほとんどの民間雇用者に対し労災保険の加入を義務付けていない唯一の州である。オクラホマ州は2013年に適格雇用者が代替給付計画を策定できるオプトアウト規定を制定したが、オクラホマ州最高裁判所は2016年に同法を違憲と判断した（Vasquez v. Dillard's, Inc.）。

**ギグ・エコノミー：** ギグ労働者を独立請負業者として分類することにより、数百万人の労働者が労災補償の適用から除外されている。カリフォルニア州のAB 5（2019年）およびその後のProposition 22（2020年）は、この空白に対処しようとする争われた枠組みを創出した。

**大取引の侵食：** 2015年のProPublicaとNPRによる調査シリーズ「労災補償の解体」は、各州における数十年にわたる給付削減の傾向を記録し、不法行為免除と引き換えに保障された給付という「大取引」が依然として公平であるかどうかについて疑問を提起した。

## ドイツ

### 法的枠組み

ドイツは1884年の労災保険法（Unfallversicherungsgesetz）を通じて世界初の法定労災補償制度を確立した。これはビスマルク首相のもとで制定された3つの社会保険法の一つである。同制度は現在、1997年1月1日施行の社会法典第VII編（SGB VII）に法典化されている。

### 職業協同組合制度

ドイツの制度は、労災補償と職場安全機能を単一の制度的枠組みの中で統合している点で独自性を有する。職業協同組合（Berufsgenossenschaften、BG）は業種別に組織された公法上の法人である。

| BG | 産業部門 |
|----|---------|
| BG BAU | 建設 |
| BG ETEM | エネルギー、繊維、電気、メディア |
| BGW | 保健・福祉サービス |
| BG Verkehr | 運輸 |
| BGHW | 商業・物流 |
| BG RCI | 原材料・化学 |
| BG Holz und Metall | 木材・金属 |
| VBG | 事務 |
| BG Nahrungsmittel | 食品産業 |

各BGは3つの統合的機能を果たしている。予防（安全規則の策定と検査の実施）、リハビリテーション（包括的な医療および職業リハビリテーションの提供）、および補償（障害給付と遺族給付の支払い）である。

### 給付

| 給付 | 説明 |
|------|------|
| 医療（Heilbehandlung） | BG運営の専門外傷病院（BG-Kliniken）を含む完全な医療 |
| 傷病手当金（Verletztengeld） | 療養・リハビリ期間中の通常収入の80%（SGB VII第47条） |
| 障害年金（Verletztenrente） | 稼得能力の減少（Minderung der Erwerbsfähigkeit, MdE）に基づく；MdE100%の場合、年間収入の2/3の満額年金（SGB VII第56条） |
| 遺族年金（Hinterbliebenenrente） | 寡婦・鰥夫：被災者の年間収入の30%（子の養育中または47歳以上の場合は40%） |
| 介護手当（Pflegegeld） | 継続的な介護を必要とする者への介護手当 |

### 排他的救済（責任特権）

SGB VII第104条は、法定保険制度の対象となる労災について、従業員が雇用者を不法行為で訴えることを禁止しているが、雇用者が故意（Vorsatz）により負傷を引き起こした場合を除く。第105条はこの免責を同僚にも拡大している。この排他的救済（Haftungsprivileg）はドイツにおける大取引の実施形態である。

### 職業病（Berufskrankheiten）

職業病令（Berufskrankheiten-Verordnung、BKV）は認定された職業病を列挙している（2021年の改正時点で82の列挙疾病）。2021年の改正（SGB VII第7次改正法）は、職業病給付を受けるために労働者が疾病原因となる活動を中止するという要件（Unterlassungszwang）を廃止した。これは長年にわたる請求認定の障壁であった。また、同改正は正式なリストに載っていない疾病であっても、新たな科学的証拠が業務起因性を立証する場合に認定を可能とする「オープン条項」を導入した。

### 業績指標

2022年に、職業協同組合は約822,000件の報告義務のある労災事故を処理し、約19,000件の新規職業病事案を認定した。全BGの支出総額は約167億ユーロであった。ドイツの労災死亡率は労働者10万人あたり0.7と世界で最も低い水準にあり、これは統合的な予防・補償モデルに一部起因すると考えられている。

## ECHR及びEU

### EU社会保障調整

社会保障制度の調整に関する規則（EC）No 883/2004は、EU加盟国間を移動する労働者が労災補償の適用を失わないことを保証している。同規則は適用法令の決定（一般に就労国の法律）、保険期間の通算、および給付のエクスポートに関する規則を定めている。施行規則（EC）No 987/2009により補完されている。

### 職業病に関するEU勧告

欧州委員会勧告2003/670/ECは、加盟国が採用することが奨励される欧州職業病表を確立している。同勧告は化学的、物理的、および生物学的因子による疾病、ならびに職業性皮膚疾患、呼吸器疾患、およびがんを対象としている。法的拘束力はないが、各国の職業病リストの参照枠組みとして機能している。

### ILO第121号条約

ILO第121号条約（業務災害給付、1964年）は労災補償制度の最低基準を確立している。主な要件は、公的部門の労働者を含むすべての従業員の適用、労災事故および職業病に対する補償、無制限の医療、一時的労働不能時の少なくとも60%の賃金補填、ならびに永久障害および死亡に対する定期金（一時金ではなく）の支払いである。韓国は1992年に第121号条約を批准した。米国もドイツも批准していないが、両国とも最低基準を大幅に上回っている。

### 欧州人権裁判所

ECHRは第1議定書第1条（財産の保護）および第6条（公正な裁判）のレンズを通じて労災補償関連の問題に対処してきた。Moskal v. Poland (Application No. 10373/05, 2009)において、裁判所は障害給付を含む社会保障給付が第1議定書第1条に基づく「財産」に該当すること、および給付の撤回は比例性の審査を満たさなければならないと判示した。裁判所はまた、労災補償手続が合理的な期間および独立した裁判所へのアクセスを含む第6条の公正な裁判の保障に適合しなければならないとも判示した。

## 比較分析

### 制度構造

3つの法域は3つの異なる構造モデルを代表している。韓国は単一の国家機関（COMWEL）が管理する集中的な公的保険制度を運営し、米国は民間保険、州基金、および自家保険の混合による50以上の分断された制度を運営し、ドイツは準公的な自治的組合（Berufsgenossenschaften）の業種別制度を運営している。ドイツのモデルは予防と補償を統合することで、事故予防のための制度的インセンティブを生み出している点で独自性を有する。

### 給付の十分性

ドイツが最も手厚い賃金補填（80%）を提供し、韓国（70%）、米国（通常66.67%、州の上限に服する）が続く。ただし、米国の制度の最高週給付額の上限により、高賃金労働者の実効的な補填率は66.67%を大きく下回ることが多い。

### 適用範囲の空白

米国が最も大きな適用範囲の空白を有しており、テキサス州では雇用者が完全にオプトアウトすることが可能であり、ギグ・エコノミー労働者の独立請負業者としての分類は数百万人を適用から除外している。韓国の1名以上の事業所への拡大およびドイツの学生・ボランティアの包含は、それぞれの制度内でほぼ普遍的な適用を提供している。

### 職業病の認定

ドイツの2021年の活動中止要件（Unterlassungszwang）の廃止は、就労を継続しながら職業病給付を受けることを妨げていた障壁を除去する重大な改革であった。韓国の2021年のメンタルヘルス疾患および過労関連疾病への拡大は、心理社会的な職場リスクの認識の高まりを反映している。米国の各州は職業病の認定において大きな差異があり、多くの州が制限的なリストを使用するか、職場固有の因果関係の証明を要求している。
