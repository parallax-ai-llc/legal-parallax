## 概要

代理出産法――女性が依頼親のために子を妊娠・出産する合意を規律する法――は、生殖法の中でも最も倫理的・法的に争われている分野のひとつであり、法域によって完全な商業的許容から完全な刑事禁止まで幅広い対応が見られる。韓国は法的グレーゾーンにある。代理出産に関する特定の法律は存在せず、慣行は明示的には禁止されていないが、生命倫理及び安全に関する法律が関連する生殖補助技術を制限しており、商業的代理出産の取決めには法的強制力がない。米国は州レベルのアプローチのパッチワークを呈しており、カリフォルニア州（Family Code Sections 7960-7962）は報酬を伴う妊娠代理出産を出生前の親子関係決定命令とともに許容する、世界で最も代理出産に寛容な法域である一方、ミシガン州などは代理出産契約を犯罪化している。ドイツは胚保護法（Embryonenschutzgesetz、第1条第1項第7号）により代理出産を包括的に禁止しており、代理母への胚の移植を行う医師に対し刑事罰を科しているが、ドイツの裁判所は外国での代理出産取決めの承認に関する問題に次第に取り組んでいる。欧州人権裁判所は、Mennesson v. France (2014)の画期的判決およびAdvisory Opinion P16-2018-001 (2019)において、第8条に基づくプライベートライフの尊重に対する権利は、少なくとも遺伝的つながりがある場合に、外国で代理出産により出生した子と依頼親との法的親子関係を承認するためのメカニズムを加盟国が提供することを要求すると判示した。

## 韓国

### 法的枠組み

韓国は代理出産に関する特定の法律を有しておらず、慣行は法的空白状態にある。関連する法的環境は以下を含む：

**生命倫理及び安全に関する法律（생명윤리 및 안전에 관한 법률）：**

- **第23条：** 営利目的でのヒト胚、卵子および精子の売買を禁止。代理出産を直接対象としていないが、この規定は商業的代理出産の報酬に関して法的不確実性を生じさせている。
- **第24条：** 胚の研究および移植を特定の医学的目的に制限しており、代理出産目的のIVFの使用を潜在的に制約している。
- **第25条：** 妊娠以外の目的での胚の作成を禁止しており、代理母への移植のために特に胚を作成することの法的根拠を複雑にしている。

**民法（민법）：**

- **第844条：** 婚姻中に出生した子は夫の子であるとの推定を確立。韓国の家族法では出産した女性が法律上の母とされ（mater semper certa estの原則）、依頼母が出産した母ではない妊娠代理出産との根本的な法的矛盾を生じさせている。
- **第855-856条：** 認知（인지）および準正を規定するが、代理出産の取決めを想定していない。

**母子保健法：**

この法律は生殖補助医療を規制しているが、代理出産を具体的に扱っていない。妊娠治療、IVFおよび胚の取扱いの枠組みを確立しているが、代理出産を許容または禁止する枠組みは設けていない。

### 主要原則

**Mater Semper Certa Est：** 韓国の家族法は分娩した母が法律上の母であるとの原則に従う。代理出産の取決めにおいて、これは依頼母ではなく代理母が出生時に子の法律上の母として認められることを意味する。依頼親は養子縁組またはその他の法的手段を通じて親としての権利を確立しなければならないが、そのプロセスは不確実であり明確な法定の手引きがない。

**国内代理出産の衰退：** 代理出産は1990年代から2000年代初頭にかけて韓国ではより一般的で緩やかに許容されていたが、社会的偏見、法的不確実性、および国境を越えた代理出産の選択肢の利用可能性のため、慣行は大幅に衰退した。韓国の依頼親は代理出産のために米国（特にカリフォルニア州）、カナダまたは東南アジア諸国に渡航するケースが増えている。

**政府調査委員会：** 韓国保健福祉部は代理出産の法整備を検討するための研究会を複数回（2006年、2014年、2019年）招集してきたが、立法提案は国会を通過していない。血統（혈통）に対する深い文化的重視や、韓国の家族関係登録制度の複雑さが立法の不作為に寄与してきた。

### 注目すべき事例

**ソウル家庭裁判所代理出産承認事件（2018-2020年）：** いくつかの非公開の家庭裁判所判決において、ソウル家庭裁判所の裁判官は米国で代理出産を利用した依頼親に対し養子縁組命令を認めた。これらの判決は依頼遺伝親の事実上の親子関係を認めたが、代理出産の取決めに関する一般的先例を確立するものではなかった。

**韓国憲法裁判所の生命倫理関連申立て：** 生命倫理及び安全に関する法律の制限が生殖の自律を侵害するとして同法に異議を唱える申立てが提起されたが、代理出産に具体的に言及する画期的判決には至っていない。憲法裁判所は一般的に同法の生殖補助医療に関する制限を支持してきた。

## 米国

### 連邦の枠組み

米国には連邦の代理出産法がない。代理出産は完全に州法により規律されており、複雑なアプローチのパッチワークとなっている：

**統一親子法（UPA）：**

統一法委員会のUPA（最新改正2017年）は妊娠代理出産を具体的に扱う第8章を含んでいる。2017年UPAは：

- 所定の要件を満たす妊娠代理出産合意を有効とする
- 代理母への独立した法的助言を要求する
- 出生後72時間の代理母の撤回期間を義務付ける
- 妊娠代理出産と遺伝的（伝統的）代理出産の双方に代理出産条項を拡張する
- 約15州が全部または一部を採択

**州の分類：**

| カテゴリ | 州 | 主な特徴 |
|----------|-----|----------|
| **代理出産に友好的（有償代理出産許容）** | カリフォルニア、ネバダ、コネチカット、メイン、ニューハンプシャー、オレゴン、ロードアイランド、バーモント、ワシントン（2019年）、コロラド | 出生前親子関係決定命令；明確な法定枠組み；婚姻状態または性的指向を問わず利用可能 |
| **代理出産許容（利他的を優先）** | ニューヨーク（2021年改正）、バージニア、ユタ、イリノイ、フロリダ | 報酬に関する種々の制限；遺伝的つながりを求める場合あり |
| **代理出産が不明確または判例法のみ** | マサチューセッツ、オハイオ、ペンシルベニア、ジョージア、メリーランド | 法律なし；判決が限定的な指針を提供 |
| **代理出産禁止または無効** | ミシガン、インディアナ（伝統的のみ）、ルイジアナ（商業的） | ミシガン：代理出産契約は無効かつ執行不能；商業的代理出産仲介者に対する刑事罰（MCL 722.859）――最大50,000ドルの罰金および5年の禁固 |

### 州ごとの差異

**カリフォルニア州（Family Code Sections 7960-7962）：**

AB 1217（2012年）により確立されたカリフォルニア州の代理出産枠組みは、世界で最も包括的かつ寛容である：

- **第7960条：** 「妊娠キャリアのための生殖補助合意」を定義し、強制力の要件を定める
- **第7961条：** 双方が独立した法的助言を受けることを要求；代理母は胚移植前に独立した法的相談を受けなければならない
- **第7962条：** 出生前親子関係決定命令：依頼親は出生前に親子関係を確立する裁判所命令を取得でき、病院に依頼親を出生証明書に記載するよう指示する
- 婚姻状態、性的指向、性自認、または子との遺伝的関係を問わず全ての依頼親が利用可能
- 依頼親に居住要件なし
- 有償代理出産が明示的に許容；報酬条件は合意に定められなければならない

**ニューヨーク州（Child-Parent Security Act、2021年）：**

ニューヨーク州は1992年の家事関係法第122条に基づく数十年の禁止の後、2021年2月15日に有償妊娠代理出産を合法化した。CPSAは以下を含む「代理母の権利章典」を確立した：

- 代理母の身体および妊娠に関する全ての健康・福祉上の決定を行う権利
- 妊娠期間中および産後12ヶ月間の包括的な健康保険の適用を受ける権利
- 依頼親が費用を負担する独立した法的助言を受ける権利

**Baby M事件の遺産――In re Baby M, 109 N.J. 396 (1988)：**

ニュージャージー州最高裁判所の画期的判決は伝統的代理出産契約を無効とし、代理出産合意が公序良俗および子の売買を禁止する法律に反すると判示した。依頼父に親権を認める一方、裁判所は代理母に面会交流権を付与した。この事件は、代理出産契約が自発的な放棄なしには代理母の親権を終了させることができないことを確立し、全国の代理出産法制に重大な影響を及ぼした。

### 注目すべき判例

**Johnson v. Calvert, 5 Cal.4th 84 (1993)：** カリフォルニア州最高裁判所は、妊娠代理出産において依頼母――妊娠キャリアではなく――が子の法律上の母であると判示し、子を誕生させ自己の子として養育することを意図した当事者がカリフォルニア法上の子の法律上の親であると推論した。この「意図テスト」は米国の代理出産法の基礎的原則となった。

**In re Marriage of Buzzanca, 61 Cal.App.4th 1410 (1998)：** カリフォルニア州控訴裁判所は、ドナー卵子とドナー精子の双方を使用した（子との遺伝的つながりのない）依頼親が子の法律上の親であると判示し、意図に基づく親子関係分析を遺伝的つながりを超えて拡張した。

**Raftopol v. Ramey, 299 Conn. 681 (2011)：** コネチカット州最高裁判所は、同性の依頼親が妊娠代理出産の取決めにおいて出生前親子関係決定命令を取得できることを認めた。

## ドイツ

### 法的枠組み

ドイツは欧州で最も制限的な代理出産制度のひとつを維持しており、分娩した母と子の尊厳は生殖能力の商品化を排除するとの原則に根ざしている。

**胚保護法（Embryonenschutzgesetz、ESchG、1990年）：**

- **第1条第1項第7号：** 出生後に子を恒久的に放棄する用意のある女性（「代理母」）に胚を移植することを刑事犯罪とする。刑罰：3年以下の禁固または罰金。
- **第1条第1項第2号：** 卵細胞が採取された女性の妊娠を実現する以外の目的での卵細胞の人工授精を禁止し（パートナー間の卵子提供に対する狭い例外あり）、妊娠代理出産のための胚作成を実質的に防止している。
- **第1条第3項：** 刑事罰の対象は医師のみであり、代理母や依頼親は対象外。この非対称的な責任構造は、代理母として奉仕する女性を処罰することを避けるために設計された。

**養子縁組斡旋法（Adoptionsvermittlungsgesetz、AdVermiG）：**

- **第13c条：** 代理母の斡旋または仲介を禁止。代理出産を斡旋した者は最大25,000ユーロの過料に処せられる。
- **第13d条：** 代理出産サービスの広告を禁止。

**民法典（Bürgerliches Gesetzbuch、BGB）：**

- **第1591条：** 「子の母は出産した女性である。」この規定は分娩した母が法律上の母であるとの反論不可能な推定を確立しており、依頼母が分娩母ではない妊娠代理出産の前提と直接矛盾する。

### 主要原則

**人間の尊厳（Menschenwürde）：** 基本法第1条第1項（「人間の尊厳は不可侵である」）に基づくドイツの憲法理論が代理出産禁止の根拠となっている。支配的な法的見解は、代理出産の取決めが代理母と子の双方を道具化し、子を契約上の交換の対象として扱うものであるとする。

**外国代理出産の承認――発展する判例法：**

国内での禁止にもかかわらず、ドイツの裁判所は外国で代理出産により確立された親子関係の承認に関する問題に次第に取り組んでいる：

- **BGH, XII ZB 463/13 (2014年12月10日)：** 連邦通常裁判所（Bundesgerichtshof）は、代理出産により出生した子について2人のドイツ人男性（同性カップル）を親として確立するカリフォルニア州の裁判所命令をドイツで承認しなければならないと判示した。裁判所は、不承認が子の法的アイデンティティに対する権利およびプライベートライフと家族生活の尊重に対する権利を侵害すると論じた。
- **BGH, XII ZB 530/15 (2017年)：** 承認原則を拡張し、代理出産が出生国の法律に基づき合法であり依頼父が子との遺伝的つながりを有する場合、ドイツ当局は依頼親を記載した外国の出生証明書を承認しなければならないと判示した。
- **BVerfG, 1 BvR 421/05 (2018年)：** 連邦憲法裁判所は代理出産の承認について包括的な判示を避けたが、全ての家族法手続における子の最善の利益（Kindeswohl）の至上性を強調した。

### 注目すべき事例

**ケルン行政裁判所判決（2010年）：** インドで代理出産により子を得たドイツ人夫婦がドイツでの親子関係の承認を求めた。ケルンの裁判所は当初、BGB第1591条を厳格に適用して承認を拒否した。事件は最終的に依頼母による連れ子養子縁組を通じて解決され、依頼親が直面する迂回的な法的経路を例示した。

**ハンブルク家庭裁判所――ウクライナ代理出産（2019年）：** ハンブルク家庭裁判所は、ウクライナで代理母を利用したドイツ人依頼親の親子関係を、BGHの発展する承認枠組みを適用して承認した。この事件は国内の禁止と国際礼譲の間の高まる緊張を浮き彫りにした。

## ECHR / EU枠組み

### 条約上の規定

欧州人権条約は主として以下の規定を通じて代理出産を扱う：

- **第8条（プライベートライフおよび家族生活の尊重に対する権利）：** 代理出産事件で援用される中核的規定。裁判所は、子のアイデンティティに対する権利および法的親子関係を認められる権利が「プライベートライフ」の範囲に含まれると判示してきた。
- **第14条（差別の禁止）：** 代理出産により出生した子の差別的取扱いが平等の問題を提起する場合に関連する。
- **EU基本権憲章第3条：** 「医学および生物学の分野においては、特に以下が尊重されなければならない：……人体およびその部分そのものを経済的利益の源泉とすることの禁止。」これは商業的代理出産に対して援用されてきた。

### 主要判例法

**Mennesson v. France, App. No. 65192/11 (2014年6月26日)：**

フランス人夫婦がカリフォルニア州に渡航し、夫の精子とドナー卵子による妊娠代理出産で双子の女児が出生した。フランスに帰国後、当局はカリフォルニア州の出生証明書のフランス民事登録簿への転記を拒否した。ECHR裁判所は以下のように判示した：

- フランスは子のプライベートライフに対する権利に関して第8条に違反した
- 裁判所は代理出産政策一般に関して各国の広い評価の余地を認めた
- しかし、子のアイデンティティが問題となる場合、評価の余地は狭まる：「フランス法上の法的親子関係の不承認が子のプライベートライフの尊重に対する権利に及ぼす影響」は正当化され得なかった
- 依頼親と子の間に遺伝的つながりがある場合、国はその関係の法的承認のためのメカニズムを提供しなければならない

**Labassee v. France, App. No. 65941/11 (2014年)：** Mennessonと同日に判決され、同一の推論――ミネソタ州での代理出産により確立された親子関係の承認をフランスが拒否したことは第8条に違反する。

**Paradiso and Campanelli v. Italy, App. No. 25358/12 (大法廷、2017年1月24日)：**

イタリア人夫婦がロシアで代理母を利用したが、実験室の誤りにより子はいずれの依頼親とも遺伝的つながりがなかった。イタリア当局は夫婦から子を引き離し国の保護下に置いた。大法廷は11対6で以下のように判示した：

- 「家族生活」の側面に関する第8条の違反はない。関係が短すぎ法的に不確実であった
- 依頼親と子との間に遺伝的つながりがないことはMennessonとの関連性のある区別要素であった
- 各国は代理出産に関して広い評価の余地を保持しており、特に遺伝的つながりがない場合にそうである

**Advisory Opinion P16-2018-001 (2019年4月10日)：**

議定書第16号に基づきフランスの破毀院が要請したこの勧告的意見は、妊娠代理出産における依頼母の承認を扱った。大法廷は以下のように判示した：

- プライベートライフの尊重に対する権利は国内法が依頼母との法的親子関係の承認の可能性を提供することを要求する
- ただし、各国は外国の出生証明書の詳細を登録する義務を負わない；養子縁組またはその他のメカニズムで足り得る
- 子の最善の利益とアイデンティティに対する権利が最も重要な考慮事項である

**D.B. and Others v. Switzerland, App. Nos. 58817/15 and 58252/15 (2022年11月22日)：**

裁判所は、カリフォルニア州で代理出産により出生した子と依頼親（同性男性カップル）との親子関係をスイス当局が承認を拒否したことについて第8条の違反を認定した。裁判所は、子の最善の利益が親の性的指向にかかわらず法的親子関係の承認を要求すると改めて述べた。

### 加盟国の義務

ECHRの判例法に従い、加盟国は以下を行わなければならない：

1. 少なくとも遺伝的つながりがある場合に、外国で代理出産により確立された法的親子関係を承認するためのメカニズムを提供する
2. 子の最善の利益を第一の考慮事項とする
3. 代理出産に対する国内の禁止が子の法的アイデンティティまたは親子関係を剥奪しないことを確保する

加盟国のアプローチは大きく異なる：英国は1985年代理出産取決法に基づき利他的代理出産を許容するが商業的代理出産は禁止しており、2008年ヒト受精・胚法に基づく「親権命令」により親子関係が移転される。ギリシャは裁判所の許可のもとで利他的妊娠代理出産を許容している。ウクライナおよびロシアは商業的代理出産を許容しており、欧州の依頼親にとって主要な代理出産の目的地となっている。ポルトガルは2016年以降利他的代理出産を許容しているが（Law No. 25/2016）、大きな制限がある。かつて世界最大の代理出産市場であったインドは2015年に外国人向けの商業的代理出産を禁止し、2021年に代理出産（規制）法を制定して、インド人既婚夫婦のための利他的代理出産のみを許容している。オーストラリアはほとんどの州で利他的代理出産を許容するが商業的代理出産を犯罪化しており、クイーンズランド、ニューサウスウェールズおよびACTは海外での商業的代理出産の取決めに対して罰則を科している。日本は代理出産法を有しておらず、日本産科婦人科学会のガイドラインが慣行を禁止しているが犯罪化はされていない。ブラジルは連邦医学審議会のResolution No. 2,320/2022に基づき利他的代理出産を許容している。

## 比較分析

| 側面 | 韓国 | 米国 | ドイツ | ECHR枠組み |
|------|------|------|--------|------------|
| **法的明確性** | 低（特定の法律なし） | 可変（州に依存） | 高（明確な禁止） | 判例法を通じて発展中 |
| **商業的代理出産** | 規制なし；強制力なしの可能性 | 約10州で合法 | 禁止（刑事） | 要求されない；忌避 |
| **分娩母＝法律上の母** | はい（民法第844条） | 州により異なる；CAでは意図に基づく | はい（BGB第1591条、反論不可） | 各国はこの推定を適用し得る |
| **出生前命令** | 利用不可 | 代理出産に友好的な州で利用可能 | 利用不可 | 不要 |
| **外国代理出産の承認** | ケースバイケース；養子縁組ルート | 一般的に承認 | 発展中；BGHは好意的傾向 | 第8条に基づき必要（Mennesson） |
| **同性の依頼親** | 法的に不確実 | 多くの州で保護 | 外国代理出産について承認（BGH 2014年） | 保護（D.B. v. Switzerland） |
| **ハーグ条約** | 該当なし（未採択） | 該当なし | 該当なし | 策定中（親子関係/代理出産プロジェクト） |
| **傾向** | 現状維持；立法動向なし | 許容州の漸進的拡大 | 禁止維持；承認は拡大 | 親子関係の承認を強化 |

### 倫理的・政策的側面

代理出産の規制は、法域間の相違を説明する競合する倫理的枠組みに関わっている：

**生殖の自律対商品化：** 米国（特にカリフォルニア州）は生殖の自律と契約の自由のレンズを通じて代理出産を捉え、代理出産合意を個人の選択の正当な表現として扱う。ドイツの禁止は、代理出産が女性の身体と子を商品化し、基本法第1条第1項（人間の尊厳）に違反するとの対立する原則に基づいている。韓国の沈黙は文化的両面性を反映しており、生殖の自律の価値が儒教的家族構造の規範と競合している。

**フェミニストの視角：** フェミニスト学者は代理出産について深く分裂している。リベラル・フェミニストは代理出産の禁止が女性から自身の身体に対する主体性を否定すると主張する。ラディカルおよび社会主義フェミニストは、商業的代理出産がグローバル・サウスの経済的に不利な立場にある女性を不均衡に搾取し、生殖労働の搾取の一形態を構成すると反論する。ECHRはこの緊張を解消しておらず、各国の評価の余地に委ねている。

**国境を越えた代理出産（「生殖ツーリズム」）：** ドイツ、フランス、イタリアなどの国における代理出産の禁止は、主として米国、ウクライナ、ジョージア、および（かつて）インド、タイへの国境を越えた代理出産の需要を駆動してきた。この「生殖ツーリズム」は、子が無国籍となり得るか、親の本国で承認された親子関係を欠く複雑な法的状況を生み出している。ハーグ国際私法会議の親子関係/代理出産プロジェクトは国境を越えた親子関係の承認に対処する国際文書の策定を目指しているが、交渉は2026年時点で継続中である。

**子どもの権利の視角：** 国連子どもの権利条約（第3条――子の最善の利益、第7条――アイデンティティと国籍に対する権利、第35条――子の売買の防止）は、子どもの視角から代理出産を評価するための枠組みを提供している。子の売買に関する選択議定書は、商業的代理出産が子の「売買」を構成すると主張する批判者により援用されてきたが、この解釈は依然として争われている。

## 年表

| 年 | 出来事 |
|----|--------|
| 1978年 | 最初のIVF児出生（Louise Brown、英国）――妊娠代理出産を可能に |
| 1985年 | 英国が1985年代理出産取決法を制定（利他的代理出産を許容） |
| 1986年 | ニュージャージー州でBaby M事件開始 |
| 1988年 | *In re Baby M*――ニュージャージー州最高裁が代理出産契約を無効化 |
| 1990年 | ドイツが胚保護法（Embryonenschutzgesetz）を制定し代理出産を禁止 |
| 1993年 | *Johnson v. Calvert*――カリフォルニア州最高裁が意図に基づく親子関係を確立 |
| 2002年 | インドが主要な国際代理出産の目的地として台頭 |
| 2012年 | カリフォルニア州がAB 1217を制定――包括的な妊娠代理出産法 |
| 2014年 | *Mennesson v. France*――ECHRが外国代理出産の親子関係承認を要求 |
| 2014年 | BGHがドイツ人同性カップルに対するカリフォルニア州代理出産親子関係命令を承認 |
| 2015年 | インドが外国人向け商業的代理出産を禁止 |
| 2017年 | *Paradiso and Campanelli v. Italy*――ECHR大法廷が遺伝的つながりなしの承認を制限 |
| 2017年 | 統一親子法改正に代理出産条項を盛り込む |
| 2019年 | ECHR Advisory Opinion P16-2018-001――依頼母の親子関係は承認可能でなければならない |
| 2021年 | ニューヨーク州が有償代理出産を合法化（Child-Parent Security Act） |
| 2021年 | インドが代理出産（規制）法を制定――インド人既婚夫婦のための利他的のみ |
| 2022年 | *D.B. and Others v. Switzerland*――ECHRが親子関係承認を同性カップルに拡張 |
| 2023年 | ハーグ国際私法会議が親子関係/代理出産プロジェクトを継続 |
| 2025年 | 複数のEU加盟国が外国代理出産承認に関する法改正を検討 |
