## 概要

未成年者の解放――成年に達する前の者が親権から解放され、標準的な成年年齢に達する前に成人の法的権利と責任を取得する法的制度――は、各法域において異なる枠組みを通じて運用されており、子ども時代の概念、親権の在り方、および早期の法的独立が正当化される状況に関する異なる考え方を反映している。韓国には独立した解放法令は存在しないが、民法（민법）の婚姻に基づく成年擬制の規定を通じて機能的に同等の効果を達成しており、婚姻した未成年者はArticle 826-2（성년의제）に基づき成人と同等の法的能力を取得する。2021年に成年年齢が18歳に引き下げられ、婚姻年齢も18歳に設定されたことで、婚姻による解放の実際的な適用範囲は事実上消滅したが、その原則は民法に残っており、未成年者はArticles 5-7に基づき法定代理人が許可した特定の法律行為を通じて限定的な機能的独立を達成し得る。米国は最も発達した解放制度を有しており、約40州で司法上の請求（通常、未成年者が自立能力と解放が最善の利益にかなうことを証明する必要がある）による法定解放、婚姻または軍務による自動解放、および親の同意と未成年者の独立生活に基づくコモンロー上の解放が利用可能であり、解放により親権と養育費義務が終了し、契約締結権、医療同意権、および独立した住所設定権が付与される。ドイツには正式な解放制度は存在しないが、16歳以上の未成年者が親または裁判所の同意のもとで限定的な法律行為に従事することを認めており、2021年の婚姻年齢改正前の民法では、16歳での裁判所承認による婚姻が可能であり、これが事実上の解放効果を有していた。ドイツのアプローチは、二項対立的な解放モデルではなく、段階的能力制度（BGB Sections 106-113）を重視している。欧州の枠組み、特にUNCRCの発達する能力の承認（Article 5）および子どもの意見表明権（Article 12）、ECHRのArticle 8に基づく若年者の自律性の保護、ならびに子どもの意思決定過程への参加に関する欧州評議会の勧告は、正式な解放よりも段階的な自律性を支持する権利に基づくアプローチを確立しつつ、未成年者が特定の状況において法的独立を必要とし得ることを認めている。

## 韓国

### 法的枠組み

韓国には専用の解放法令は存在しない。未成年者が成年年齢前に成人の法的能力を取得するための主な制度は婚姻を通じたものである。

### 婚姻に基づく能力（성년의제）

**民法Article 826-2（민법 제826조의2）：** 婚姻した未成年者は、民法の目的上、成年に達したものとみなされる。これは、婚姻した未成年者が完全な契約能力、自己の財産を管理する能力、および独立して法的行為を行う権利を取得することを意味する。

しかし、2021年の改正により成年年齢が19歳から18歳に引き下げられ、Article 807に基づく婚姻最低年齢が18歳に設定されたことで、この規定の実際的な適用は事実上消滅した。婚姻年齢と成年年齢の間にはもはや差がない。

### 未成年者の限定的自律

民法は、未成年者が特定の法律行為に従事するための仕組みを規定している。

| 条文 | 効果 |
|------|------|
| Article 5(1) | 未成年者は法律行為に法定代理人の同意を要する |
| Article 5(2) | 法定代理人の許可を得て未成年者が取得した財産に関する行為は除外される |
| Article 6 | 未成年者は、法定代理人が処分のために特に指定した財産を自由に処分できる |
| Article 7 | 法定代理人から特定の営業を許可された未成年者は、その営業範囲内の行為について成人の能力を有する |

**Article 7（営業の許可）：** この規定は、未成年者が法定代理人の許可を得て営業を行うことを認めるもので、限定的な分野別解放として機能する。未成年者は許可された営業の範囲内の全ての行為について完全な法的能力を有するが、その他の目的については未成年者のままである。

### 子役・芸能人に関する規定

韓国の芸能産業には多数の未成年者が従事している。大衆文化芸術産業発展法（대중문화예술산업발전법）は、労働時間の制限、義務教育要件、搾取的契約に対する保護を含む、未成年芸能人の労働条件を規制している。これは正式な解放には該当しないが、未成年者の経済活動のための専門的な枠組みを創設している。

### 憲法的側面

憲法裁判所は、選挙権年齢（2017헌마1264）や教育権に関する決定を含む複数の決定において、未成年者の権利の範囲を検討している。解放を直接扱うものではないが、これらの決定は、未成年者の憲法上の人格性と自律的意思決定能力に対する認識の発展を反映している。

## 米国

### 法的枠組み

米国は、比較対象の法域の中で最も発達した解放制度を有している。解放は以下の方法で行われ得る。

### 法定解放（司法上の請求）

約40の州が司法上の解放に関する法定規定を有している。一般的な要件は以下の通りである。

| 要件 | 典型的な規定 |
|------|------------|
| 最低年齢 | 14-16歳（州により異なる） |
| 経済的自立 | 未成年者が自活能力を証明する必要がある |
| 最善の利益 | 解放が未成年者の最善の利益にかなう必要がある |
| 成熟度 | 未成年者が十分な成熟度を証明する必要がある |
| 居住形態 | 独立して生活していることが求められることが多い |
| 親への通知 | 必要。親の同意は関連する場合があるが、通常は必須ではない |

**カリフォルニア州の例（Cal. Fam. Code Sections 7120-7135）：**
- 最低年齢：14歳
- 未成年者が自発的に親と別居していること
- 未成年者が自己の財務を管理していること
- 収入が合法であること
- 解放が未成年者の最善の利益に反しないこと

### 自動的解放

| 事由 | 効果 | 普遍性 |
|------|------|--------|
| 婚姻 | 完全な解放 | 事実上全ての州 |
| 軍務 | 完全な解放 | 全州 |
| 成年年齢への到達 | 自動的 | 普遍的 |

### コモンロー上の解放

特定の法令を持たない州でも、コモンロー上の原則により以下に基づいて解放が成立し得る。
- 親の同意（明示的または黙示的）
- 未成年者が独立して生活し自己の事務を管理していること
- 親の遺棄または扶養の不履行
- 親権と未成年者の状況との両立不能

### 法的効果

| 権利・義務 | 解放の効果 |
|-----------|-----------|
| 契約能力 | 拘束力ある契約を締結する完全な能力 |
| 医療同意 | 自己の医療に同意する権利 |
| 住所 | 自己の住所を設定する権利 |
| 親の扶養 | 親の扶養義務は通常終了する |
| 不法行為責任 | 親の不法行為責任は通常終了する |
| 教育 | 義務教育法は引き続き適用される場合がある |
| 選挙権 | 選挙権年齢を18歳未満に引き下げない |
| 酒類・たばこ | 法定購入年齢を引き下げない |
| 刑事管轄権 | 刑事管轄権の年齢に影響しない |

### 医療上の解放

一般的な解放とは別に、多くの州が、指定された状況において未成年者が親の関与なしに医療に同意するための特定の法的根拠を設けている。

| 状況 | 許可する州 |
|------|-----------|
| 性感染症 | 全50州およびコロンビア特別区 |
| 避妊サービス | 26州およびコロンビア特別区 |
| 出生前ケア | 32州およびコロンビア特別区 |
| 薬物乱用治療 | ほとんどの州 |
| メンタルヘルスサービス | 大きく異なる |

### 主要判例

**In re Snyder, 85 Wash.2d 182 (1975)：** ワシントン州最高裁判所は、解放には実際の独立と自立の証明が必要であり、単に親の支配から自由になりたいという願望だけでは不十分であると判示した。

**Parker v. Stage, 43 N.Y.2d 128 (1977)：** ニューヨーク州控訴裁判所は、未成年者の婚姻は、当該未成年者が自立を通じて実際に解放されていない限り、親の養育費義務を自動的に終了させるものではないと判示した。

## ドイツ

### 法的枠組み

ドイツには正式な解放制度は存在しない。代わりに、BGBに基づく段階的能力制度が年齢に基づいた漸進的な自律性を提供している。

### 段階的能力

| 年齢 | 法的地位 | BGB条文 |
|------|---------|---------|
| 7歳未満 | Geschäftsunfähig（行為無能力） | Section 104(1) |
| 7-17歳 | Beschränkt geschäftsfähig（制限行為能力） | Section 106 |
| 18歳以上 | Voll geschäftsfähig（完全行為能力） | Section 2 |

### 制限行為能力の制度

**Taschengeldparagraph（Section 110 BGB）：** 未成年者が親の同意なく締結した契約は、未成年者がその目的のために与えられた資金またはその自由な処分のために与えられた資金で義務を全うした場合に有効となる。この「お小遣い条項」により、未成年者は利用可能な資金の範囲内で日常的な購入を行うことができる。

**Section 112 BGB（営業の許可）：** 法定代理人と家庭裁判所の同意を得て、未成年者は商業営業の遂行を許可されることができる。その営業の範囲内の全ての取引について、未成年者は成人の法的能力を有する。この規定は韓国のArticle 7に類似している。

**Section 113 BGB（雇用の許可）：** 法定代理人の同意を得て、未成年者は雇用関係に入り、その関係から生じる権利と義務を独立して行使することができる。この規定により、未成年者は各取引に対する親の関与なしに、就労し自己の給与を管理することが可能となる。

### 婚姻年齢と解放

2017年の児童婚撲滅法（Gesetz zur Bekämpfung von Kinderehen）以前は、16歳の未成年者は家庭裁判所の承認を得て婚姻することができた（BGB Section 1303旧版）。婚姻は正式にそのように特徴づけられてはいなかったものの、事実上の解放効果を有していた。2017年法は例外なく婚姻最低年齢を18歳に設定し、この経路を完全に排除した。

### 青少年保護法

青少年保護法（Jugendschutzgesetz）および青少年労働保護法（Jugendarbeitsschutzgesetz）は、未成年者の公的活動と雇用に関する特定の枠組みを創設しており、完全な解放ではなく、規制された範囲内での限定的な自律性を提供している。青少年労働保護法は、13歳からの軽作業および15歳からの制限付き通常雇用を許可している。

### 憲法上の枠組み

基本法Article 6(2)は、子どもの養育と教育は親の自然権であり義務であると宣言している。この憲法規定は親権を基本的権利として確立しており、正式な解放制度を憲法的に複雑なものにしている。親権を終了させるいかなる制度も、この憲法上の保障と調和させなければならない。

## ECHR・EU

### 国連子どもの権利条約

**Article 5（発達する能力）：** 締約国は、子どもが条約で認められた権利を行使するに際し、親が子どもの発達する能力に一致する方法で適切な指導と助言を行う責任、権利および義務を尊重する。この原則は、二項対立的な解放よりも段階的な自律性を支持するものである。

**Article 12（意見表明権）：** 自己の見解を形成する能力のある子どもは、自身に影響する全ての事項について自由に意見を表明する権利を保障され、その意見は年齢と成熟度に応じて正当に考慮される。

**Article 32（経済的搾取）：** 締約国は、経済的搾取から保護され、有害となる可能性のある労働を行うことから保護される子どもの権利を認め、雇用の最低年齢を設定する。

### ECHR Article 8

ECHRは複数の文脈で未成年者の自律性を扱っている。

**Gillick v. West Norfolk and Wisbech Area Health Authority [1986] AC 112：** ECHR判例ではなく英国の判例であるが、Gillick適格性の原則（十分な理解力と知性を示す未成年者は親の同意なしに医療に同意できるとする原則）はECHR法理に影響を与え、欧州評議会加盟国間で広く認められている。

**A.S. v. Norway (Application No. 60371/15, April 17, 2019)：** 裁判所は、里親制度の文脈における14歳の子どもの実親との面会に関する意思決定への参加権の範囲を扱い、発達する能力の原則を強調した。

### 欧州評議会勧告

**Recommendation CM/Rec(2012)2（18歳未満の子どもおよび若者の参加に関する勧告）：**
- 子どもは能動的な権利の主体として認識されるべきである
- 参加は年齢に応じたものであり、子どもの発達する能力に適合するべきである
- 子どもの意見は年齢と成熟度に応じて正当に考慮されるべきである

**Recommendation Rec(2005)5（居住型施設で生活する子どもの権利に関する勧告）：**
- 施設内の子どもは、年齢と成熟度に見合った最大限の自律性を有するべきである
- 自立生活への移行が支援されるべきである

### 欧州における比較的アプローチ

| 国 | 解放制度 | 年齢 |
|----|---------|------|
| フランス | 裁判所の決定による解放（Art. 413-2 Code civil） | 16歳 |
| スペイン | 親の同意、婚姻、または裁判所による解放（Art. 314-324 Código Civil） | 16歳 |
| イタリア | 正式な解放なし。裁判所の許可を得た16歳での婚姻 | 16歳 |
| ポルトガル | 婚姻による解放 | 16歳 |
| オランダ | 正式な解放なし。18歳で婚姻（裁判所の承認を得て16歳） | 該当なし |

フランスの制度は注目に値する。裁判所は、いずれかの親の請求に基づき、未成年者の成熟度と利益を考慮して、16歳以上の未成年者を解放することができる。解放された未成年者は、商業に従事する権利（追加の裁判所許可が必要）を含む完全な法的能力を取得する。

## 比較分析

### 正式な解放と機能的解放

米国は最も発達した正式な解放制度を提供しており、約40州で司法上の請求が利用可能である。韓国とドイツには正式な解放制度は存在しないが、特定の法的規定（営業の許可、雇用の許可、お小遣い規則）を通じて機能的に同等の効果を達成している。UNCRCの発達する能力の法理は、米国の解放に特徴的な二項対立的アプローチよりも、ドイツおよび韓国が採用する段階的アプローチを支持している。

### 解放経路としての婚姻

歴史的に、3つの法域全てが婚姻を未成年者の成人としての法的能力への経路として使用していた。婚姻最低年齢の18歳への引き上げ（ドイツは2017年、韓国は2021年）および成年年齢の18歳への引き下げにより、この経路は事実上閉鎖された。米国は婚姻に基づく解放を維持しているが、複数の州も婚姻年齢を18歳に引き上げており、その実際的な適用範囲は縮小している。

### 保護的枠組みと自律的枠組み

各法域は未成年者の保護と自律性の促進との間の緊張を反映している。米国の解放法は、成熟した自立的な未成年者の自律性を優先するのに対し、ドイツおよび韓国の法は段階的能力制度を通じた保護を優先している。ECHRおよびUNCRCは両者のバランスを追求しており、未成年者の発達する能力の尊重を求めつつ、十分な保護を確保している。

### 実際的重要性

解放の実際的重要性は、成年年齢が18歳に収斂するにつれて、全法域で低下している。米国における解放の主な残存事例は、虐待的または養育放棄的な家庭を離れる未成年者、経済的独立を求める未成年の芸能人またはアスリート、および妊娠中または育児中の未成年者に関するものである。韓国およびドイツでは、同様のニーズは一般的な解放ではなく、個別の法定規定を通じて対処されている。
