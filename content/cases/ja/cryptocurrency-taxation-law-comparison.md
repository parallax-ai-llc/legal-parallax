## 概要

暗号資産課税——ビットコイン、イーサリアムその他の暗号トークンなどのデジタル資産に関する利益、所得、取引の税務上の取扱い——は、従来の金融インフラの外で機能するよう設計された新たな資産クラスの分類、追跡、課税に世界各国の政府が苦闘する中、最も急速に進化し法域間で断片化した租税法の分野の一つとして台頭している。韓国は所得税法（소득세법）の2020年改正を通じて暗号資産利益に対する課税を立法化し、暗号資産利益を年間250万ウォンを超える部分に対して20%（地方税を含め22%）の税率で課税される「その他の所得」（기타소득）として分類したが、実施は繰り返し延期されてきた——当初2022年1月を予定していたが、2023年、2025年と延期され、直近では2024年税制改正法により2027年1月に延期されており、韓国の大規模な個人暗号資産投資家層からの強い政治的圧力を反映している。米国はIRS通知2014-21に基づき暗号資産を連邦税務上「財産」として取り扱い、利益にキャピタルゲイン税率を適用しており、2021年インフラ投資・雇用法がブローカー報告義務を拡大し（2023年12月31日以降に開始する課税年度について有効）、IRSは2024年にデジタル資産報告に関する最終規則を発出した。ドイツは暗号資産を所得税法（Einkommensteuergesetz, EStG）第23条に基づく「私的売却取引」（privates Veräußerungsgeschäft）として取り扱い、1年の保有期間を超えた利益を非課税としている（2022年のBMFガイダンスの明確化により10年から短縮された）。一方、ステーキングおよびレンディング収入はEStG第22条に基づくその他の所得として取り扱われる。EUは暗号資産市場規制（MiCA, Regulation 2023/1114）を採用しており、これは主として金融規制枠組みであるが、DAC 8指令（Directive 2023/2226）を通じて課税と接続し、暗号資産サービス提供業者に対し2026年1月1日から全加盟国の税務当局に利用者の取引データを報告することを義務付けている。

## 韓国

### 法的枠組み

韓国の暗号資産課税枠組みは、所得税法（소득세법）の2020年改正により確立され、仮想資産（가상자산）の処分による利益を第21条第1項第27号に基づく「その他の所得」（기타소득）として分類した。同法は仮想資産を特定金融取引情報の報告及び利用等に関する法律（특정 금융거래정보의 보고 및 이용 등에 관한 법률）を参照して定義しており、租税上の定義をマネーロンダリング防止規制上の定義と一致させている。

### 税制構造（実施時）

| 要素 | 詳細 |
|------|------|
| 税率 | 国税20%（＋地方所得税2%＝実効22%） |
| 年間控除額 | 250万ウォン（約2,100米ドル） |
| 計算方法 | 年間利益合計額マイナス年間損失合計額マイナス250万ウォン控除 |
| 取得価額の決定 | 実際の取得原価（実際の購入価格＋手数料） |
| 申告 | 年次確定申告；取引所が国税庁に取引データを提供 |

### 繰り返される延期

実施スケジュールは並外れた政治的敏感さを反映している：

| 年 | 実施予定 | 結果 |
|----|----------|------|
| 2020年 | 当初2022年1月を計画 | 2023年1月に延期 |
| 2022年 | 2023年1月 | 2025年1月に延期 |
| 2024年 | 2025年1月 | 2024年税制改正法により2027年1月に延期 |

延期は主として韓国の推定600万〜700万人の個人暗号資産投資家の反対に牽引されてきた。この投資家層は政治的に重要な人口統計を構成している。与党国民の力党は無期限延期を主張する一方、野党共に民主党は当初実施を求めたが、最終的に2027年への延期に同意した。

### 取引所規制

韓国で事業を行うすべての仮想資産サービス提供業者（VASP）は、特定金融取引情報の報告及び利用等に関する法律に基づき韓国金融情報分析院（KoFIU）に登録しなければならない。認定された実名韓国銀行口座と提携した取引所のみがウォン建て取引を行うことができる。この規制インフラは、課税が発効した際の税務コンプライアンスを促進するために設計されている。

## 米国

### 法的枠組み

IRSは通知2014-21において暗号資産の基礎的な取扱いを確立し、仮想通貨を連邦税務上「財産」として分類した。これは、財産取引に適用される一般的な税原則が適用されることを意味する：利益および損失は処分時に認識され、その性質（キャピタルまたは通常）は納税者における資産の性質に依存し、取得価額および保有期間のルールが適用される。

### キャピタルゲイン課税

| 取引類型 | 税務上の取扱い |
|----------|----------------|
| 法定通貨への売却 | キャピタルゲイン/ロス（1年以内の保有は短期；1年超は長期） |
| 他の暗号資産への交換 | 課税対象の処分（実現イベント） |
| 物品・サービスの支払い | 公正市場価値での課税対象の処分 |
| マイニング報酬としての受領 | 受領時のFMVで通常所得 |
| ステーキング報酬としての受領 | 受領時のFMVで通常所得（Rev. Rul. 2023-14） |
| エアドロップ | 受領時のFMVで通常所得 |
| ハードフォーク（エアドロップなし） | 処分まで課税イベントなし |
| 贈与 | 課税イベントなし；受贈者は贈与者の取得価額を引き継ぐ |

### ブローカー報告——インフラ投資・雇用法

2021年インフラ投資・雇用法（Pub. L. 117-58）は、第6045条に基づく「ブローカー」の定義を拡大し、「（対価を得て）他者に代わってデジタル資産の移転を実行するサービスを定期的に提供する責任を負う者」を含めた。ブローカーは顧客およびIRSにForm 1099-DAを発行しなければならない。IRSは2024年6月に最終規則を発出し、報告義務は段階的に導入される：

- **2025年：** 中央集権型取引所および一部のホスト型ウォレット提供業者が報告を開始。
- **2026年：** 分散型取引所のフロントエンドおよび一部の非ホスト型ウォレット提供業者が対象となり得る（進行中のルールメイキングに依存）。

### Revenue Ruling 2023-14

2023年7月31日に発出されたこのルーリングは、ステーキング報酬が納税者が報酬に対する支配権を取得した課税年度に通常所得として総所得に算入されることを明確にした。このルーリングは、*Jarrett v. United States*事件（W.D. Tenn., 2022）後の不確実性を解消した。同事件では裁判所がステーキング報酬は処分まで課税されるべきでないという主張に当初理解を示していたように見えた。

### ウォッシュセール規則

2025年時点で、第1091条に基づくウォッシュセール規則（30日以内に売却・再購入した証券の損失控除を否認する規則）は暗号資産に適用されない。暗号資産は証券ではなく財産として分類されているためである。ただし、ウォッシュセール規則をデジタル資産に拡大する立法案は繰り返し提出されている。

## ドイツ

### 法的枠組み

ドイツは暗号資産の利益をEStG第23条第1項第2号の私的売却取引（privates Veräußerungsgeschäft）の規定に基づいて取り扱っている。連邦財務省（BMF）は2022年5月10日に仮想通貨およびその他のトークンの税務上の取扱いに関する包括的ガイダンス（BMF-Schreiben IV C 1 - S 2256/19/10003 :001）を発出し、分類、所得認識、報告に関する詳細な規則を提供した。

### 保有期間による免税

ドイツの暗号資産課税で最も納税者に有利な側面は投機期間（Spekulationsfrist）である：1年超保有した暗号資産の処分による利益は、EStG第23条第1項第2号に基づき完全に非課税となる。2022年のBMFガイダンスは、暗号資産がステーキングやレンディングに使用された場合でもこの1年の保有期間が適用されることを確認し、収益を生む活動により保有期間が10年に延長されるかどうかという従前の不確実性を解消した。

### 税務上の取扱いの概要

| 活動 | 分類 | 税率 |
|------|------|------|
| 1年以内の保有期間での売却 | 私的売却益 | 通常所得税率（14〜45%） |
| 1年超の保有期間後の売却 | 免税 | 0% |
| 年間免除額（Freigrenze） | 少額免除 | 年間600ユーロまでの利益は非課税；超過した場合は全額が課税対象 |
| マイニング（事業） | 事業所得（gewerbliche Einkünfte） | 通常所得税率＋営業税（Gewerbesteuer） |
| マイニング（個人） | その他の所得（EStG第22条） | 通常所得税率 |
| ステーキング報酬 | その他の所得（EStG第22条） | 受領時に通常所得税率 |
| レンディング収入 | 資本所得またはその他の所得 | 分離課税25%（Abgeltungsteuer）または通常税率 |

### 暗号資産間の交換

BMFガイダンスは、ある暗号資産から別の暗号資産への交換が課税対象の処分に該当することを確認しており、米国および韓国のアプローチと一致する。処分価格は交換時に受領した暗号資産の公正市場価値である。

### 識別方法

どの単位が処分されたかの決定（FIFO、LIFO、または個別識別）について、BMFガイダンスはデフォルトの方法として先入先出法（FIFO）の使用を認めているが、ウォレットレベルの追跡を通じてどの特定の単位が処分されたかを納税者が証明できる場合は個別識別も認められている。

## ECHR・EU

### 暗号資産市場規制（MiCA）

2024年12月30日に完全適用となったRegulation (EU) 2023/1114（MiCA）は、EUにおける暗号資産の包括的規制枠組みを確立している。MiCAは主として金融規制（税法ではない）であるが、効果的な課税に不可欠なインフラを創設する：

- **CASPライセンス：** すべての暗号資産サービス提供業者はライセンスを取得し、運営、ガバナンス、消費者保護の要件を遵守しなければならない。
- **ステーブルコイン規制：** 資産参照トークンおよび電子マネートークンは特定の認可および準備金要件の対象となる。
- **市場の健全性：** 暗号資産市場における市場操作およびインサイダー取引を禁止する規則。

### DAC 8——暗号資産税務報告

2023年10月17日に採択されたDirective (EU) 2023/2226（DAC 8）は、行政協力指令（DAC）を改正し、暗号資産に関する自動的な情報交換を義務付けている。主要規定：

| 要素 | 要件 |
|------|------|
| 報告義務主体 | MiCAに基づくライセンスの有無にかかわらず、EUで事業を行うすべてのCASP |
| 報告対象取引 | 処分（暗号資産間を含む）、移転、暗号資産による支払い |
| 報告される情報 | 利用者の身元、暗号資産の種類別の集計取引額、取引件数 |
| 発効日 | 2026年1月1日（2026年の取引について報告） |
| 国際的整合性 | OECD暗号資産報告枠組み（CARF）と整合 |

### OECD暗号資産報告枠組み（CARF）

OECDは2022年8月にCARFを公表し、暗号資産取引の報告に関する標準化されたグローバル枠組みを提供した。CARFは仲介業者（取引所、ウォレット提供業者）に対する報告を義務付け、暗号資産と法定通貨間の交換取引、暗号資産間の交換、移転取引を対象としている。45以上の法域が2027年までにCARFを実施することを約束した。

### CJEU——VAT上の取扱い

CJEUは**Hedqvist事件（C-264/14、2015年）**において暗号資産のVAT上の取扱いを扱い、ビットコインと法定通貨の交換がVAT指令第135条第1項(e)号に基づくVAT免税の役務提供（法定通貨として使用される通貨、銀行券、硬貨に関する取引の免除）に該当すると判示した。この判決はビットコインがVAT上「支払手段」として機能することを確認した。

## 比較分析

### 分類の乖離

暗号資産の基本的な分類は法域間で異なり、重大な税務上の結果をもたらす。米国は暗号資産を財産（資本資産）として取り扱い、長期キャピタルゲインの優遇税率を可能にしている。ドイツは暗号資産を通常所得税率の対象となる私的資産として取り扱うが、1年後には完全に免税となる。韓国は利益を一律20%の税率の「その他の所得」として分類する。これらの分類のいずれも、投資財産、通貨、ユーティリティトークン、ガバナンス手段として同時に機能し得る暗号資産の多面的な性質を完全には捉えていない。

### 保有期間のインセンティブ

ドイツは1年の免税制度を通じて長期保有に対する最も強力なインセンティブを提供している——1年後の利益は金額にかかわらず完全に非課税である。米国は税率の差を提供している（長期で最大実効23.8%に対し短期で最大40.8%）。韓国が提案する一律20%の税率は保有期間に基づく区別を設けていない。

### 実施準備態勢

米国は通知2014-21以降10年以上の積極的な執行を経て、最も成熟した執行インフラを有している。ドイツは包括的なBMFガイダンスを有するが、年次確定申告を通じた自発的コンプライアンスに主として依存している。韓国はVASP登録を通じて取引所レベルの規制インフラを構築したが、課税自体はまだ発動していない。

### DeFiと新たな課題

すべての法域が分散型金融（DeFi）活動——流動性提供、イールドファーミング、ガバナンストークンのステーキング、クロスチェーンブリッジ——の課税に苦慮している。一般的な財産原則を適用する米国のアプローチは機能的だが不完全な枠組みを提供している。ドイツのBMFガイダンスは一部のDeFiシナリオに対処しているが、重大なギャップが残っている。韓国もEUも包括的なDeFi税務ガイダンスを提供していない。
