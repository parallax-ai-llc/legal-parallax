## 概要

誘拐——人の意に反する不法な逮捕、監禁、または移送——はあらゆる法体系において重大な犯罪であるが、その法律上の構造は法域によって大きく異なる。韓国は刑法の専用の章（第31章、第287条～第296条）を通じて誘拐および拉致を規定しており、被害者の年齢および拉致の目的によって犯罪を区別し、性的搾取や人身売買が関与する場合には刑罰が大幅に加重される。米国は18 U.S.C. Section 1201（リンドバーグ赤ちゃん事件を契機とする画期的な法律）によって誘拐を連邦犯罪化する一方、大多数の訴追は大きく異なる州刑法典の下で州裁判所に委ねている。ドイツは誘拐を主に自由の剥奪（Freiheitsberaubung、第239条StGB）として扱い、身代金目的の誘拐（第239a条）および人質取り（第239b条）についてはより重い別個の規定を設けている。欧州人権裁判所は、誘拐を独立した犯罪として定義するのではなく、第5条（自由および安全に対する権利）の観点からこれを扱い、恣意的な拘禁の防止および不法な拉致の捜査について署名国に積極的義務を課している——国家行為者によって、または国家行為者の共謀により実行された拉致を含む。

## 韓国

### 法的枠組み

韓国刑法第31章（형법 제31장）は誘拐および拉致の罪（약취와 유인의 죄）を規律している。韓国法は「약취」（暴行または強制による誘拐）と「유인」（誘惑または欺罔による拉致）を用語上区別している。関連条項は被害者の地位および犯人の目的に基づいて段階的な刑罰を規定している：

| 条項 | 犯罪 | 刑罰 |
|---------|---------|---------|
| 第287条 | 未成年者の誘拐または拉致 | 10年以下の懲役 |
| 第288条(1) | 営利、わいせつ目的、または労働搾取目的の誘拐・拉致 | 1年以上10年以下の懲役 |
| 第288条(2) | 韓国外への移送目的の誘拐・拉致 | 2年以上15年以下の懲役 |
| 第289条(1) | 人身売買（売買） | 7年以下の懲役 |
| 第289条(2) | 未成年者の人身売買 | 2年以上15年以下の懲役 |
| 第289条(3) | 韓国外への移送目的の人身売買 | 2年以上15年以下の懲役 |
| 第290条 | 誘拐の結果としての殺人・傷害 | 5年以上の懲役から無期懲役／死刑 |
| 第291条 | 誘拐後の殺人 | 死刑／無期懲役／10年以上の懲役 |

第288条～第292条の犯罪については、懲役に加えて最大5,000万ウォン（約38,000米ドル）の罰金が併科される場合がある。第295条の2に基づき、被害者を安全な場所に自発的に解放することは、刑を軽減しうる法定の減軽事由である。

### 主要原則

**二元的分類（약취 vs. 유인）：** 韓国法は強制による誘拐（약취）と欺罔または誘惑による拉致（유인）を独自に区別している。両者は同等の刑罰で処罰されるが、事実上の区別は起訴および量刑において重要である。贈り物の約束で子供を誘い出した者は「유인」を犯し、子供を物理的に連れ去った者は「약취」を犯す。

**目的に基づく加重：** 韓国の誘拐法は犯人の目的を中心に構成されている。第287条に基づく未成年者の単純誘拐は最大10年であるが、第288条に基づき性的搾取または人身売買の目的が加わると、最低刑は1～2年、最高刑は10～15年に加重される。

**2013年改正：** 韓国は2013年施行の改正を通じて誘拐および人身売買に関する規定を大幅に強化し、第289条に基づく人身売買の定義を拡大し、国連人身売買議定書（パレルモ議定書）の義務に合わせて全般的に刑罰を引き上げた。

### 主要判例

**趙斗淳事件（2008年）：** 主に性的暴行として起訴されたが、この事件は安山で8歳の女児が拉致されたものであり、量刑の軽さに対する全国的な怒りを引き起こした。趙は12年の懲役を宣告され、性犯罪者に対する電子監視の導入や性目的の加重誘拐の定義拡大を含む立法改革を促した。

**親による拉致事件：** 韓国は2013年に国際的な子の奪取の民事上の側面に関するハーグ条約に加入した。韓国の裁判所はその後、特に要請国における家庭内暴力の申立てがある事件において、返還命令と条約の第13条(b)の例外との均衡を図ることに取り組んでいる。

## 米国

### 法的枠組み

連邦誘拐法（18 U.S.C. Section 1201）は、州際または外国の通商に関与する場合に、何人かの不法な逮捕、監禁、誘引、おびき出し、誘拐、拉致、または連れ去りを犯罪としている。以下の場合に連邦管轄権が生じる：

- 被害者が州境または国境を越えて移送された場合
- 犯罪が特別海事管轄区域または領域管轄区域内で発生した場合
- 被害者が連邦職員、外国要人、または国際的に保護される者である場合
- 被害者が1日以内に解放されない場合、州際移送の24時間の反証可能な推定が生じる

**連邦刑罰の範囲：**

| 犯罪 | 刑罰 |
|---------|---------|
| 基本的な連邦誘拐 | 無期懲役以下 |
| 誘拐未遂 | 20年以下の懲役 |
| 誘拐の共謀 | 無期懲役以下 |
| 死亡を伴う誘拐 | 無期懲役または死刑 |
| 未成年者の誘拐（家族以外） | 最低20年の懲役 |

### 州法の差異

誘拐の訴追の大部分は州レベルで行われるため、刑罰は大きく異なる：

- **カリフォルニア州（Penal Code Section 207-209.5）：** 単純誘拐は3～8年、加重誘拐（身代金、恐喝、強盗、または性犯罪目的）は仮釈放の可能性のある無期懲役、カージャック中の誘拐は仮釈放の可能性のある無期懲役。
- **テキサス州（Penal Code Section 20.03-20.04）：** 誘拐は第3級重罪（2～10年）、加重誘拐（致死的武器の使用または脅迫、性的虐待、身代金、恐怖行為、政府機能への妨害）は第1級重罪（5～99年または無期懲役）。
- **ニューヨーク州（Penal Law Section 135）：** 第1級誘拐（身代金要求、死亡、または12時間以上の拘禁を伴う）はA-1級重罪（15～25年から無期懲役）、第2級誘拐はB級重罪（5～25年）。

### 代表的判例

**リンドバーグ誘拐事件（1932年）：** ニュージャージー州の自宅から生後20ヶ月のチャールズ・リンドバーグ・ジュニアが拉致・殺害された事件は、1932年の連邦誘拐法の制定に直接つながった。ブルーノ・リチャード・ハウプトマンは有罪判決を受け、1936年に処刑された。この事件は米国の法執行を永久に変え、州際誘拐事件に対する連邦管轄権を確立した。

**Gooch v. United States, 297 U.S. 124 (1936)：** アーサー・グーチはテキサス州で2人の警察官を誘拐し、オクラホマ州に移送し、そこで生きたまま解放した。最高裁判所は連邦誘拐法に基づく死刑判決を支持し、グーチは被害者が生存した連邦誘拐罪で処刑された米国史上唯一の人物となった。この判例は、同法における「身体的危害」が被害者の死亡を要しないことを確立した。

**Chatman v. State（ジョージア州、2019年）：** ジョージア州控訴裁判所は、誘拐と不法監禁を区別する有罪判決を支持し、被害者の移動は基礎となる犯罪に付随する程度以上でなければならないと判示した。被害者が意味のある距離を移動させられたという「移送」の要素は、全国の州誘拐訴追における重要な法的争点であり続けている。

**ジェイシー・デュガード事件（2009年）：** フィリップおよびナンシー・ガリドは1991年にカリフォルニア州サウスレイクタホで11歳のジェイシー・デュガードを誘拐し、18年間監禁した。フィリップ・ガリドは431年から無期懲役、ナンシー・ガリドは36年から無期懲役を宣告された。この事件はカリフォルニア州の仮釈放監視制度における壊滅的な欠陥を露呈し、性犯罪者監視の抜本的改革につながった。

## ドイツ

### 法的枠組み

ドイツ刑法は、刑法典（StGB）の3つの主要規定を通じて誘拐および関連犯罪を規定しており、重大性に応じて段階的に構成されている：

**第239条StGB — Freiheitsberaubung（自由の剥奪）**

基本的犯罪は、人を閉じ込めるか、その他の方法で移動の自由を奪う行為を対象としている。これはドイツ法における誘拐の基礎的規定である：

| 項 | 犯罪 | 刑罰 |
|------------|---------|---------|
| 第239条(1) | 基本的な自由の剥奪 | 5年以下の懲役または罰金 |
| 第239条(2) | 自由の剥奪の未遂 | 処罰対象 |
| 第239条(3) | 1週間を超える自由の剥奪、または重大な身体傷害を伴う場合 | 1年以上10年以下の懲役 |
| 第239条(4) | 被害者の死亡を伴う自由の剥奪 | 3年以上の懲役 |
| 第239条(5) | (3)または(4)の軽微な場合 | それぞれ6月以上5年以下または1年以上10年以下 |

**第239a条StGB — Erpresserischer Menschenraub（恐喝目的の人の略取）**

この規定は、被害者の無力さまたは被害者の安全に対する懸念を利用して身代金その他の利益を恐喝する目的で人を逮捕または拉致する行為を対象としている。基本刑は5年以上の懲役（最大15年）である。犯人がその行為によって被害者の死亡を引き起こした場合、刑罰は10年以上の懲役または無期懲役に引き上げられる。軽微な場合には、裁判所は1～10年の刑を科すことができる。

**第239b条StGB — Geiselnahme（人質取り）**

人質取り——人を逮捕し、第三者に行動を強要するために死亡、重大な身体傷害、または長期の自由剥奪で脅迫すること——は第239a条と同一の刑罰範囲を持つ：最低5年、被害者が死亡した場合は最低10年または無期懲役。

### 主要原則

**保護法益（Rechtsgut）：** 第239条は「潜在的な移動の自由」（potentielle Fortbewegungsfreiheit）を保護する。人は自身の監禁を認識している必要はない——意識不明または睡眠中の者は、意識を回復した際に退去できない状態で監禁されている場合、Freiheitsberaubungの被害者となりうる。

**同意と欺罔：** 連邦通常裁判所（Bundesgerichtshof）の最近の判例法によれば、真の目的地または旅行の目的について欺罔によって得られた監禁への同意は無効となる。2022年のBGH判決は、虚偽の口実で車両に人を誘い込み、意図しない場所に運転する行為は、被害者が当初自発的に車両に乗り込んだとしても自由の剥奪に該当すると判示した。

**独立した「誘拐」犯罪の不存在：** コモンロー法域と異なり、ドイツには「誘拐」と表示された単一の法令はない。その代わり、当該行為は状況と目的に応じて第239条、第239a条、および第239b条にわたって捕捉され、第235条StGB（Entziehung Minderjähriger——親権からの未成年者の拉致）によって補完される。第235条は最大5年の懲役を科す。

### 主要判例

**ナターシャ・カンプッシュ事件（オーストリア、2006年）：** オーストリアの事件であるが、この事件はドイツの法的議論に深い影響を与えた。ヴォルフガング・プリクロピルは1998年に10歳のカンプッシュを拉致し、隠された地下室で8年以上監禁した。この事件はオーストリアおよびドイツの立法者に、長期的な自由剥奪に対する量刑の妥当性の見直しを促した。

**ヤーコプ・フォン・メッツラー事件（2002年）：** フランクフルト警察副署長ヴォルフガング・ダシュナーは、誘拐犯マグヌス・ゲフゲンに拷問を脅して誘拐された11歳のヤーコプ・フォン・メッツラーの所在を引き出した。ゲフゲンは場所を明らかにしたが、子供はすでに死亡していた。ゲフゲンは第239a条に基づく殺人を伴う誘拐で無期懲役を宣告された。ダシュナーは職権による強要の罪で有罪となり、執行猶予付きの罰金を科された。この事件は基本法第1条（人間の尊厳）がいかなる例外をも許容するかという議論において画期的なものとなり、後に欧州人権裁判所はGaefgen v. Germany（2010年）において、拷問の脅迫は欧州人権条約第3条に違反するが、その後の裁判を不公正にするものではないと判示した。

## 欧州の枠組み

### 欧州人権条約第5条：自由および安全に対する権利

欧州人権条約第5条は、欧州において誘拐および不法拘禁が扱われる超国家的枠組みを提供している。同条は「すべての者は、身体の自由および安全に対する権利を有する」こと、および「何人も、以下の場合を除き、法律に定める手続に従ってでなければ、その自由を奪われない」ことを保障している。

### 許容される自由の剥奪

第5条(1)は、以下の6つの限定された場合にのみ自由の剥奪を許容している：

1. 管轄裁判所の有罪判決後の合法的拘禁
2. 裁判所命令の不遵守に対する逮捕または拘禁
3. 犯罪の合理的嫌疑に基づく逮捕
4. 教育的監督のための未成年者の拘禁
5. 感染症の蔓延防止のための拘禁、または精神障害者、アルコール依存症者、薬物依存症者、浮浪者の拘禁
6. 不正入国の防止または国外退去を目的とする逮捕または拘禁

これらのカテゴリー外のいかなる自由の剥奪も、国内法上合法であるか否かを問わず、第5条に違反する。

### 積極的義務

欧州人権条約は、恣意的拘禁を単に控えることを超えた積極的義務を国家に課している：

- **保護義務：** 国家は、特に脆弱な者（子供、人身売買の被害者）が関与する場合、私人による誘拐および不法拘禁を防止するための合理的措置を講じなければならない。
- **捜査義務：** 第5条は第13条（効果的な救済を受ける権利）と併読され、不法拘禁の申立てに対する迅速かつ効果的な捜査の実施を国家に要求する。
- **立法義務：** 国家は、誘拐および拉致を適切に犯罪化し抑止する国内法的枠組みを維持しなければならない。

### 代表的判例

**El-Masri v. the Former Yugoslav Republic of Macedonia, App. No. 39630/09 (2012)：** 大法廷は、マケドニアがドイツ国籍のハリド・エル＝マスリをスコプスキ・メラクホテルで23日間無起訴で拘禁し、その後CIA工作員に引き渡してアフガニスタンの秘密収容所に移送した際に、第3条、第5条、第8条、および第13条に違反したと判示した。裁判所は、拘禁がいかなる法的枠組みの完全に外にあったため第5条違反を認定した——エル＝マスリは裁判官の前に連れ出されることも、起訴されることもなく、その拘禁は第5条(1)に基づいて認められるいかなる目的にも資していなかった。この判例は、外国情報機関の誘拐作戦への国家の共謀が第5条の直接的違反を構成することを確立した。

**Rantsev v. Cyprus and Russia, App. No. 25965/04 (2010)：** キャバレーでの就労のために「アーティストビザ」でキプロスに連れてこられた若いロシア人女性オクサナ・ランツェワの死亡を受けて、裁判所は、人身売買の明確な徴候があるにもかかわらず警察が彼女を雇用主のアパートに戻した際に、キプロスが第5条に違反して彼女の自由を保護しなかったと判示した。裁判所はまた、第4条（奴隷労働および強制労働の禁止）が人身売買を包含し、被害者の特定、保護、および捜査について国家に積極的義務を課すと認定した。この判例は、私人が労働または性的搾取のために被害者を事実上誘拐し監禁する状況にまで第5条の義務を拡大した。

**Storck v. Germany, App. No. 61603/00 (2005)：** 申立人は、裁判所命令またはインフォームド・コンセントなしに、父親の要請により私立精神科施設に20ヶ月以上監禁された。裁判所は、拘禁にいかなる法的根拠も欠けていたため、ドイツが第5条(1)に違反したと判示した。重要なのは、裁判所が、私的施設による自由の剥奪について国家が第5条に基づく責任を負い、個人を監禁する権限を有する私的施設を監督・規制する義務を課したことである。

**Ilacu and Others v. Moldova and Russia, App. No. 48787/99 (2004)：** 4名のモルドバの申立人は、ロシア軍の共謀のもとトランスドニエストリア分離主義勢力によって逮捕され、未承認の裁判所により死刑または長期の懲役を宣告された。大法廷はモルドバとロシアの双方が第5条に違反していると判示し、その承認された領域外であっても、国家の機関が権限と支配を行使する場所において国家が管轄権を行使することを確立した。

| 加重刑 | 無期懲役または死刑（誘拐＋殺人） | 無期懲役または死刑（連邦、死亡の場合） | 最低5年（第239a条）、死亡の場合は無期懲役 | 損害賠償に対する国家責任 |
| 児童関連規定 | 第287条（未成年者に対する高い基本刑） | 最低20年（家族以外、連邦） | 第235条（親権からの拉致、最大5年） | 強化された積極的義務（Rantsev判決） |
| 人身売買との統合 | 第288条～第289条（2013年以降の統合的枠組み） | TVPA（別個の法律、22 U.S.C. Section 7101） | 第232条～第233a条（別個の規定） | 第4条（Rantsev解釈） |
| 同意の抗弁 | 未成年者には不可、欺罔は同意を無効にする | 一般に不可、州により若干の差異あり | 欺罔は同意を無効にする（BGH 2022年） | 国家が関与する場合、同意は無関係 |
| 公訴時効 | 犯罪の重大性に応じて5～15年 | 5年（連邦）、州により異なる | 重大性に応じて5～20年 | 第5条に基づく請求には時効なし |
| 主要な区別 | 目的に基づく等級（性的、人身売買、営利） | 州際移送が連邦管轄権を発動 | 犯罪類型（自由、身代金、人質） | 個人の刑事責任に対する国家義務 |

## 経歴年表

| 年 | 出来事 |
|------|-------|
| 1932年 | リンドバーグの赤ちゃんが誘拐される。米国連邦議会が連邦誘拐法を制定 |
| 1936年 | Gooch v. United States：最高裁判所が被害者の死亡なき誘拐に対する死刑判決を支持 |
| 1953年 | 韓国が誘拐および拉致に関する第31章を含む刑法を制定 |
| 1998年 | ナターシャ・カンプッシュがオーストリアで拉致される。2006年まで監禁 |
| 2002年 | フランクフルトでヤーコプ・フォン・メッツラー誘拐事件がドイツの取調べにおける拷問の議論を惹起 |
| 2004年 | 欧州人権裁判所がIlacu v. Moldova and Russiaにおいて恣意的拘禁への国家の共謀について判示 |
| 2005年 | Storck v. Germanyが私的施設の監禁に対する国家責任を確立 |
| 2008年 | 韓国の趙斗淳事件が誘拐および性犯罪に関する法改正を促す |
| 2010年 | Gaefgen v. Germany：欧州人権裁判所が誘拐捜査中の拷問の脅迫を扱う |
| 2010年 | Rantsev v. Cyprus and Russiaが人身売買の文脈に第5条の義務を拡大 |
| 2012年 | El-Masri v. Macedonia：欧州人権裁判所が特別引き渡しへの国家の共謀を非難 |
| 2013年 | 韓国が刑法第288条～第289条を改正し、人身売買防止規定を強化 |
| 2013年 | 韓国が国際的な子の奪取に関するハーグ条約に加入 |
| 2022年 | ドイツBGHが旅行目的地に関する欺罔は監禁への同意を無効にすると判示 |
